## 1. 직전 화 (ch001) 연결점

### 1-1. ch001 마지막 500자 요약

강신우는 연금술사 3인(흰 수염 노인 / 확대경 남자 / 둥근 안경 학자)의 검진 후 실험실에 홀로 남겨졌다. "폐기동 이송 대기" 판정. 그는 구석에 앉아 손등의 마법진을 "회로"로 인식하고, 603호의 단편 기억에서 **"엘릭서 주입 다음 일정: 8일 후"**를 인식한다. 머릿속에서 "호문클루스 구독제"라는 비유를 만들고, 최초의 상태창을 세운다.

```
[호문클루스 603호 — 현재 상태]
제조 후 경과: 약 2개월 / 남은 엘릭서: 8일분 / 다음 주입까지: 8일
월 엘릭서 비용: 금화 150닢 (2급) / 잔고: 금화 0 / 은화 0 / 동화 0
월세: 0 (연금술사 길드 실험동) / 진료비: 0 (병원비 면제) / 신용: -10 (오염체 판정)
→ 죽는 게 돈이 덜 든다
```

엔딩은 강신우가 속삭임처럼 "살 만한가 한번 계산해보자"라고 첫 자발적 발화를 내뱉으며 **살 이유(=엘릭서 1병, 금화 150닢, 다음 달 결제)**를 처음으로 수립하는 장면.

### 1-2. 열린 질문 / 서스펜스

- **폐기동 이송은 언제인가?** (ch001에서 "엘릭서 주입 주기가 남았으니 며칠 유예 가능" 발언만 나옴)
- **둥근 안경 학자가 남긴 시선의 의미는?** — AL-603 봉인 표식에 한 번 더 시선을 두고 나감. 관찰 기록을 남기겠다고 공언.
- **다음 엘릭서 주입은 실제로 들어오는가?** (폐기 대기 판정을 받은 개체에게도 주입될 것인가)
- **연금술사들이 감시 장치로 실험실을 보고 있지는 않은가?** (강신우 본인이 스스로 제기한 합리적 가능성)
- **옆 실험대에 누워 있는 "다른 호문클루스"는 누구인가?** (이리스 591호의 존재 암시 영역 — ch004에서 본격 등장 예정, ch002에서는 미등장 유지)

### 1-3. 즉각 이어야 할 상황

- ch001 엔딩 = 실험실 구석, 밤(마력등이 어두워짐), 혼자, 연기 유지 중.
- ch002 오프닝은 **"밤 → 다음 날 아침" 또는 "몇 시간 후 복귀한 누군가"**에서 시작하는 것이 자연스럽다.
- 감정 아크 연속성: ch001 엔딩 감정 = **기묘한 안도 + 첫 목표 수립** → ch002 오프닝은 **긴장 재상승(다시 누군가 들어옴)** 으로 이어가야 "코미디→고통" 톤 낙차가 살아남.
