### 서브 비트 (씬당 2~3개 마이크로 이벤트)

**씬 1 — 이리스의 관찰 (서브 시점, 800~1,000자)** [SYN 씬1]
- 공간 셋업: 폐기동 벤치 근처. 다른 홈들은 식물처럼 멈춰 있음. 햇빛·먼지·금 간 벽돌 한두 줄.
- 이리스의 관찰 기록 (내면): 603호의 눈동자가 "너무 규칙적으로 멈춘다"는 위화감. "멈추는 박자가 연기 같다."
  - 단, **FS-023(표식 감청) 언급 금지** — 박자 감지가 표식 복선으로 오독되지 않도록 "눈의 정지 박자"만 서술.
- 고독의 원인 한 줄: "이 발견을 말하지 않았다. 말할 상대가 없기 때문이다."
- 엔딩 결심 독백: "저 애가 진짜 의식이 있다면, 내 하루가 좀 덜 외로울 것이다. 한번 확인해보자."
- **이름 공개 금지** — 이 씬에서 그녀는 오직 "그녀" 또는 "591호"로 지칭. "이리스"는 씬 4에서 강신우가 지어주는 순간 최초 등장.
- 시점 이관: 씬1 마지막 문장에 "그녀는 자리에서 일어섰다"류의 행동 전환 → `---` → 씬2는 강신우 시점 풀타임.

**씬 2 — 어깨의 손 (강신우 시점, 2,500~3,000자)** [SYN 씬2]
- 도입: 강신우가 폐기동 구조·마차 주기·관리인 교대 시간을 **눈으로만** 세는 장면. 탈출 계획 **구체화 금지** — "관찰까지만"으로 제한.
  - 프로그래머 비유 **여기서 금지**. (비유는 씬3 또는 씬4에 총 1~2회.)
- 돈미새 독백 1회차: 폐기동 유지비가 아닌 **자기 자신의 남은 수명 환산**으로 톤 전환 — "하루가 금화 5닢어치 엘릭서. 오늘도 5닢 태우는 중이군." (ch003 FS-022 중복 방지용으로 **자기 몸값 환산**에 집중)
- 어깨의 손: 장면 파편화. "손가락 끝의 온도." "숨이 한 박자 멈춘다." 단문 파편.
  - 내면 패닉: `'발각인가? 연금술사인가?'`
- 돌아보기: 여성형 호문클루스의 실루엣. 백발·청록색 눈·표준 미형. **외모 묘사 3줄 이내 엄수** [TS §7.2].
- 첫 대사 (이리스): `"너, 소울 잃어버린 거 아니지?"` (속삭임)
- 내면 동결: `'어떻게 알았지?'` — 단문 파편. 이어서 외부 연기 유지 결단.
- 수다 구간: 이리스가 자기 얘기·폐기동 얘기·연금술사 얘기를 쏟아냄. 강신우는 **외부 대사 0회 유지**. 내면만 분주.
- 설정 공개: "호문클루스 기본 수명 1~2년, 특제 최대 5년" — **이리스의 대사로만** 흘러나옴. 강신우 내면이 수치를 붙잡는 장면.
- 후크 발동 키워드: "짧은 생?" 강신우 내면 — 다음 씬 폭로의 연결선.

**씬 3 — 엘릭서 폭로 (강신우 시점, 3,500~4,000자)** [SYN 씬3]
- 이리스 핵심 대사: `"엘릭서를 더 주입하면 좀 더 오래 살 수 있을 텐데. 근데 그게 얼마나 비싼 건데… 애완용이나 성노예용 홈한테나 쓰지, 우리같은 실험용 홈한테는 절대 안 써."` [SYN 핵심 대사]
- **영생 방정식 완성 단락** (돈미새 독백 2회차 / 씬3 코어):
  1. `'엘릭서 주입 = 수명 연장.'`
  2. `'주입이 영구적이면?'`
  3. `'돈만 있으면 안 죽는다.'`
  - 이 3단 도약은 **목적절 서술 없이 단문 열거**로. ch003 목적절과 리듬 차별화.
- 연기 붕괴 순간: "강신우의 눈에 초점이 살짝 들어왔다." — 3인칭 근접의 객관 묘사 한 줄.
- 이리스 즉시 감지: `"어? 뭐야 너… 지금 확실해졌다. 너 진짜 정신 있구나."`
- 강신우 첫 외부 대사 (전 화 통틀어 최초): `"…고마워."` (짧음, 1회)
- 이리스 환희: 자아 있는 존재에게 말을 걸고 싶었다는 고백. 이리스 쪽 감정 피크.
- **FS-001 씨앗 1회** (이리스 자아 근원 — 본격 공개 금지): `"나도 왜 이런지 모르겠어. 다른 591호나 592호는 다 멍해있거든. 난 처음부터 말이 많았대."` [SYN 배치 복선]
  - 주의: "마법진이 빛난다"·"고대 자장가" 같은 본격 공개 키워드 **금지**. 단순히 "왜 나만 이럴까" 한 번만.
- 공조 묵계: 외부 앞에선 무감정 연기 유지, 단둘일 때만 본래 모습. 대사는 최소, 눈짓·움직임만으로 확정.

**씬 4 — 이름·결의 (강신우 시점, 2,500~3,000자)** [SYN 씬4]
- 이리스가 "나는 591호야"라고 소개. 강신우 내면: `'번호로 부르기 싫다. 내 공범인데. 이름을 줘야겠다.'`
- 이름 부여: `"이름 지어줄게. 이리스."`
  - 이유 (내면 2~3문장): 꽃 이리스(Iris). 전생 어머니 병원 정원의 꽃 [TS §8.1 범위 — 2~3문장 엄수]. **"어머니"라는 단어는 1회까지만**, 가족 감상은 즉시 계산으로 복귀.
  - 이리스 반응: `"…예쁜 이름이네"` (감정 살짝 흔들림 — 표정 변화는 한 줄로만).
- **FS-002 씨앗 배치** (미다스 이름 이중 의미 — 이름은 아직 없음):
  - 이리스: `"너는 이제 뭐가 되고 싶은데?"`
  - 강신우: `"돈을 만지는 사람이 되겠지. 만지는 족족 금으로 바뀐다는 그 왕처럼."` [SYN 배치 복선]
  - 주의: "미다스"라는 이름 **절대 금지**. 저주받은 왕·가족이 금이 되어 죽은 왕 같은 본격 키워드도 금지 — 이 화는 **밝은 면만**("금으로 바꾸는 왕") 씨앗으로.
- 새 목표 선언 (내면, 단독 단락):
  ```
  '나는 영생한다.'
  '돈으로.'
  '호문클루스 주제에 영생한다.'
  ```
- 수치 박스 B (씬 4 말미, 필수 1회) — §9 참조.
- 엔딩 장면: 강신우·이리스 나란히 앉아 하늘을 본다. 이리스가 혼잣말처럼 **흥얼거리는 멜로디** — **FS-032 씨앗**.
  - 주의: 멜로디의 가사·정체·"고대 자장가" 연결 **금지**. 단순히 "출처 모를 음률" 정도.
- 엔딩 펀치: `'이번 생의 첫 번째 계산은 덧셈이어야 했다. 전생의 마지막 계산은 뺄셈이었으니까.'` [SYN 엔딩 훅]

## 2-A. 시점 구성 설계 (POV Design) **필수**

```markdown
## 시점 구성
- 메인 시점: 강신우 3인칭 근접 (씬2·3·4)
- 서브 시점 후보 (능동 발굴):
  1. **이리스(591호) 3인칭 근접** / 유형: 조연 시점 / 용도: "왜 이리스가 말을 걸었나" 동기 확립 + 독자에게 "이리스도 자아 있음" 선공개 / 씬 위치: 도입부 (씬1)
  2. 폐기동 관리인 3인칭 / 유형: 관찰자 / 용도: 외부 시선으로 "평범한 실패체 두 마리" 비춤 / 씬 위치: 중반 → **비채택** (관찰자 시점이 이 화 플롯 코어와 무관. 추가 장황해짐)
  3. 연금술사 B 3인칭 / 유형: 빌런·관찰자 / 용도: 폐기동 기록 관리 묘사 → **비채택** (ch004 주제는 "이리스 조우·영생 방정식"이지 "감시 리스크"가 아님. ch013 주변으로 이월)

- **최종 채택**: 이리스 3인칭 근접 서브 씬 1회 (씬1 도입부, 800~1,000자)

- 채택 사유:
  1. **연속 시점 피로 해소**: ch001·ch002·ch003 3화 연속 강신우 단독 시점. 5화 경고 임계점 근접 전 선제 환기. [continuity §9 시점 분포]
  2. **몰입감↑**: 독자에게 "이리스도 자아가 있다"를 먼저 공개하면, 씬2의 "너 소울 잃어버린 거 아니지?"가 **필연의 대사**로 읽힘. 공격적 대사가 아닌 고독한 존재의 손 내밀기로 해석됨.
  3. **동기 확립**: "왜 이리스가 603호에게 말을 걸기로 결심했는가"라는 독자의 내적 물음이 씬1에서 해소됨 → 씬2~3 전개가 개연성 확보.
  4. **정서적 회수 구조**: 씬1 이리스의 고독 → 씬4 강신우가 이름을 지어주는 장면에서 정서적으로 회수. 챕터 단위의 감정 사이클 완결. [SYN 감정·톤 배합 말미]

- 비채택 사유 (복수 시점 채택 안 한 이유): 이 화는 **강신우의 "영생 방정식 완성"이 플롯 코어**. 씬2~4는 전부 강신우의 깨달음 궤적을 **독자가 같이 체험**해야 클라이맥스(돈으로 영생)가 직격한다. 복수 서브 시점은 초점 분산.

- 분량 배분: 서브 시점 씬 총 800~1,000자 / 챕터 전체 10,000~12,000자 중 **8~10%** (가이드 20% 이내 안전 범위) [TS §1.3, SYN 시점 구성]

- 시점 이관 규칙:
  - 씬1 종료 = 이리스의 행동 전환 (예: "그녀는 자리에서 일어섰다") + `---` 씬 구분선.
  - 씬2 첫 문장 = 강신우의 감각 또는 행동으로 시작 ("강신우는 벤치 근처의 금 간 벽돌을 세고 있었다" 같은 패턴).
  - 같은 씬 내 시점 혼재 **금지** [TS §1.4].

- 이리스 시점 서술 어미 가이드:
  - 종결어미는 `~했다.`, `~였다.`, `~일 것이다.`, `~일지도 모른다.` 계열.
```

## 17. 엔딩 훅 초안 [SYN 엔딩 훅 강도 5/5]

### 엔딩 마지막 5단락 구성안

**단락 1 — 이름 부여 직후 대화**
```
"이름 지어줄게. 이리스."

그녀는 한 박자 늦게 고개를 들었다.

"…예쁜 이름이네."

감정이 살짝 흔들린 것은 목소리였다. 얼굴은 움직이지 않았다. 연기 기반이 튼튼한 쪽은 이쪽이었다.
```

**단락 2 — FS-002 씨앗 (금으로 바꾸는 왕)**
```
이리스가 무릎을 끌어안고 물었다.

"너는 이제 뭐가 되고 싶은데?"

강신우는 짧게 생각했다. 대답은 이미 씬3에서 완성되어 있었다.

"돈을 만지는 사람이 되겠지. 만지는 족족 금으로 바뀐다는 그 왕처럼."

이리스는 "뭐야 그거"라는 듯 웃었다. 웃음은 짧았다. 그리고 다시 앞을 보았다.
```

**단락 3 — 목표 갱신 내면 단독 단락**
```
'나는 영생한다.'
'돈으로.'
'호문클루스 주제에 영생한다.'
```

**단락 4 — 수치 박스 B + 멜로디 (FS-032 씨앗)**
```
(박스 B 삽입)

옆에서 이리스가 낮은 음으로 멜로디를 흥얼거리기 시작했다. 어디서 나온 음인지 그녀 자신도 몰랐다. 가사는 없었다. 박자는 느렸다. 그것은 폐기동의 저녁 공기에 섞였다.

강신우는 그 멜로디를 멈추라고 하지 않았다. 침묵보다 나쁘지 않았기 때문이었다.
```

**단락 5 — 결의 펀치 + 다음 화 예고**
```
'이번 생의 첫 번째 계산은 덧셈이어야 했다.'

강신우는 하늘을 보지 않았다. 숫자 쪽이 더 정확했다.

'전생의 마지막 계산은 뺄셈이었으니까.'

*4화 끝. 다음 화: 가축과 로봇 사이.*
```

### 엔딩 훅 강도 평가
- 유형: 목표 선언 + 철학적 대구 + FS 씨앗 3개(FS-002·FS-032·FS-001) 동시 클로저
- 강도: **5/5** [SYN 목표 달성]
- 5화 당김: "돈을 어떻게 번다는 건가?" + "저 멜로디의 정체는?" + "공범 둘은 어디까지 갈 수 있을까?" — 3중 구조적 기대.

## 19. 작업 핸드오프 노트 (chapter-creator에게)

1. **시점 약속**: 씬1은 **이리스 3인칭 근접 (서브)**, 씬2~4는 **강신우 3인칭 근접 (메인)**. `---` 3회. 시점 혼재 0.
2. **분량**: **10,000~12,000자 엄수**. 씬별 배분: 800~1,000 / 2,500~3,000 / 3,500~4,000 / 2,500~3,000 [SYN].
3. **서브 시점 규칙**: 이리스 씬은 자유 간접화법. `''` 금지 (강신우 전용). 이름 "이리스" 미사용. "591호" 또는 "그녀"로만.
4. **수치 박스**: **1회** (박스 B, 씬4 말미). "목표 갱신" 포맷. [SYN 씬4 박스 그대로 채용]
5. **FS 씨앗 3개 배치**: FS-001(씬3) + FS-002(씬4) + FS-032(씬4 엔딩). 본격 공개 키워드는 전부 금지.
6. **돈미새 독백 2~3회 / 프로그래머 비유 1~2회 / 목적절 서술 6회+** 카운트 유지.
7. **강신우 첫 외부 대사**: 씬3 "…고마워" 1회. 이후 이리스 앞에서만 3~5회 이내.
8. **이름 부여 타이밍**: 씬4에서 강신우가 "이리스"라고 부르는 순간이 그 이름의 **최초 등장**. 서술자도 씬4 이전엔 "그녀" / "591호".
9. **미다스·에녹·그레고르·세라·리노·도로시**: 이름 호명 금지.
10. **"마도 공학자 길드"·"은빛별 교단"·"Stage 1~4" 수치** 금지 [SYN 금지].
11. **연금술사 3인 이름 금지** [SYN]. 이리스의 수다에 연금술사 언급 시 "연금술사들"로만.
12. **FS-023·FS-003**: 재언급 0회.
13. **ch001~003 모티프 재사용 금지**: 스택 트레이스·믹스커피·EOF·SaaS·배터리 충전·박자 차이 표식·마력등 호흡 직유·손가락 구부리지 않기·네트워크 디펜던시·스프린트 백로그·"폐기동은 그를 잊으려 했다" [SYN 금지].
