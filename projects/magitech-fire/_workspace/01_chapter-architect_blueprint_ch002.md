### 서브 비트 (씬당 2~3개 마이크로 이벤트)

**씬 1 — 반복 검진 (D+1~5 압축)**
- 첫날: 초점 없는 눈 연기를 "유지" — 강신우의 내면 공포(심박·호흡 제어). 연금술사 B(코에 얹은 확대경, ch001 재등장)가 "소울 색 재측정". 여전히 검보라. 한숨.
- 3일차 압축: 매일 동일 대화의 변주. "603호, 손 들어." "반응 3초 지연 — 제조 스펙대로." 연금술사들의 권태가 강신우에게는 생존 공간.
- 5일차: 노장 A의 결정 — "일주일 더 본다." 내일부터 엘릭서 주입 개시 통보. 강신우 내면: 배터리 충전 예고.

**씬 2 — 603호 기억·공정 해석 (D+5~8)**
- 603호 기억 파편: 제조 가마의 따뜻한 어둠 / "소울 점화" 공정의 이질적 통증 / "주인"이라는 단어에 대한 본능적 순응 / "텔레포트"라는 단어만 남기고 지워진 실험일
- 강신우가 내면에서 퍼즐 맞추기: **호문클루스는 공장 제품 + 엘릭서는 월 구독 + 소유주 = 결제자 = 생사여탈권자**
- 산업 구조 인식: 연금술사 길드가 제조·주입·폐기 전 공정 독점. 소유자(귀족)는 엘릭서비만 부담. 무등록·미납 시 폐기.
- 돈미새 회로: '나를 살려두는 비용과 죽이는 비용을 저들이 비교 중이다. 나는 스프레드시트의 한 줄이다.'

**씬 3 — 첫 엘릭서 주입 (D+8)**
- 주입실 묘사: 유리관에 담긴 **옅은 은빛 액체**(2급 엘릭서), 주입대, 고정 벨트, 목 뒤 삽관구
- 공정: 연금술사 B가 무심히 삽관 → 액체가 몸으로 밀려들어옴
- 육체적 감각 (감각 중심, 고어 금지): 차가운 액체가 혈관을 타고 퍼지는 느낌 / 입안 쇠맛 / 손끝 시림이 풀림 / "배고픔과 비슷한 뭔가"가 사라짐
- 수치 박스 전후 노출: 주입 **전** = 잔량 1일 / 주입 **후** = 잔량 30일, "결제 성공" 메타포
- 강신우 내면 펀치라인: "Your subscription has been renewed. 그런 알림이 몸 안에서 울리는 기분이군."

**씬 4 — 판정·통보·엔딩 훅 (D+10 밤)**
- 노장 A의 최종 판정: "관찰 가치 낮음. 폐기동 이송. 내일 새벽."
- **이 화에서 "폐기동" 단어 첫 등장 허용** — 하지만 강신우는 그게 뭔지 모름 (독자도 함께 모름 유지). 603호 기억에 "폐기동" 단어가 희미하게 있음 — "거기로 간 애들은 안 돌아온다" 정도.
- 연금술사들이 나간 후 혼자 남겨진 강신우 — 손등의 마법진을 오래 응시
- FS-023 씨앗: 표식 근처 마력 흐름이 다른 마법진과 **미묘하게 다른 박자**로 뛴다는 감각. 강신우는 이유를 모르지만 이상함을 메모만 (독자에게는 씨앗).
- 내면 반전: 공포 → 계산. "실험동 = 감옥. 폐기동 = ?"라는 미지수가 **기회**일 수도 있다는 인식. 감시 밀도가 낮아질 가능성.
- 엔딩 결심: "내일 나는 실험동을 나간다. 어디로 가는지는 모르지만, 한 번 나가면 두 번째 나갈 방법도 찾을 수 있겠지."

## 5. 강신우 상태 체크 [PB §5 — 정본 업데이트 반영]

```markdown
- 육체 상태: 호문클루스 603호 (Stage 0, 제조 후 약 2개월 + 10일)
- 엘릭서 잔량: 씬1~3 = 8일→1일, 씬3 주입 후 = 30일, 씬4 말 = 27~28일 (하루 경과치만큼 차감)
- 연비 단계: Stage 0 — 1병당 30일 [MS §2.3]
- 자산: 금화 0 / 은화 0 / 동화 0 (변화 없음)
- 정신 상태: 공포 → 연기 숙련 → 관찰 → 탈출 결심의 씨앗
- 신분: 공식 "호문클루스 603호" — 실패체 판정 확정. 내일 폐기동 이송.
- 미다스 씨 신분: **미존재 유지** (Part 1 후반 구축 전)
- 신규 지식 (이 화에서 획득):
  · 엘릭서 주입 공정의 실제 감각 (원격 지식 → 체감 지식)
  · 호문클루스 산업 구조 전모 (제조·주입·폐기 독점)
  · 월 구독 = 30일/병 (Stage 0 연비)
  · "폐기동"이라는 장소의 존재 (정체 불명, 603호 기억의 희미한 단서)
  · 표식 마법진의 마력 흐름이 "다른 박자"로 뛰는 감각 (FS-023 씨앗)
- 관계도 변화: 연금술사 A/B 미세 -15 (이송 대상 판정으로 실질 관심 상실)
```

**이 화에서 반드시 서술에 반영해야 할 변화**
- 엘릭서 주입: 1일 → 30일 (수치 박스에 전/후 노출)

## 17. 엔딩 훅 초안 [프롬프트 강도 목표 4/5]

### 엔딩 마지막 4단락 구성안

**단락 1 — 통보 직후의 고요**
```
연금술사 A가 문을 닫고 나갔다. 검진실은 비었고, 강신우만 남았다. 침상 위로 복도의 마력등 빛이 기울어져 들어왔다. "폐기동"이라는 단어가 세 번 머릿속에서 돌았다. 서술자는 그 단어를 용어 분석으로 바꾸기 위해서였다.
```

**단락 2 — 손등 마법진과 FS-023 씨앗**
```
강신우는 손등을 응시했다. 표식 마법진의 빛은 다른 마법진과 달랐다. 같은 도형인데 맥박이 다르다고 해야 하나. 강신우는 손가락을 얹지 않았다. 감지했다는 사실을 들키지 않기 위해서였다.
```

**단락 3 — 내면 반전 (공포 → 계산 → 기회)**
```
'실험동은 나를 본다. 폐기동은 나를 잊으려 할 것이다.'

잊히는 편이, 보이는 쪽보다 낫다. 감시 밀도가 낮아지면, 연기의 단가도 낮아진다. 첫 번째 문을 나가면, 두 번째 문을 여는 방법도 찾을 수 있을 것이다.

공포가 계산으로 바뀌는 데 걸린 시간은 3분이었다. 강신우는 그 변화를 스스로도 기록해두었다. 이것이 이세계에서 익힌 두 번째 능력이라고. 첫 번째는 무감정 연기였고.
```

**단락 4 — 수치 박스 B + 펀치라인 + 다음 화 예고**
```
(여기에 박스 B 삽입)

실험동은 그를 보내려 하고 있었다. 강신우는 보내지는 방향이, 처음으로 반가웠다.

*2화 끝. 다음 화: 폐기동.*
```

### 엔딩 훅 강도 평가
- 유형: 내면 반전 + 공간 전환 예고 + 수치 박스
- 강도: **4/5** [프롬프트 목표 달성]
- 3화 당김: "폐기동이 대체 뭐길래?"라는 독자 궁금증 + "저 감시 밀도 ▓ 차이는 뭘 의미하지?" 구조적 기대.

## 19. 작업 핸드오프 노트 (chapter-creator에게)

1. **시점 약속**: 3인칭 근접, 강신우 전담. `---` 3회. 타 시점 0.
2. **대사 비율**: 연금술사 A/B의 업무 대사 + 강신우 외부 대사는 0~1회.
3. **수치 박스**: 2회 (주입 전/후 A + 엔딩 B). 정본 수치 (Stage 0 / 30일·병 / 150닢) 준수.
4. **FS-023 씨앗**: "박자 차이" 수준까지만. "감청"이라는 결론 금지.
5. **돈미새 독백 2회+ / 프로그래머 비유 2회(권장)~3회(상한) / 목적절 서술 7회+** 카운트 유지.
6. **이리스·미다스·세라·그레고르·리노·에녹·도로시** 이름 호명 금지. 등장 금지.
7. **"폐기동" 단어는 씬4에서 처음 등장** (씬1~3 노출 금지).
8. **"Stage 1~4" 수치 공개 금지** — 이 화는 Stage 0만 노출.
9. **"마도 공학자 길드", "은빛별 교단", "영생"** 용어 금지 [프롬프트 금지 표현].
