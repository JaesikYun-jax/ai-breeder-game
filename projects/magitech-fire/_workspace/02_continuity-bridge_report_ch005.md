## 1. 직전 2화 연결점

### ch003「폐기동」 (D+16)
- 603호 이송. 이송 전 2급 엘릭서 1병 주입 (약 45일분 덮어쓰기).
- 폐기동 도착. 감시 밀도 실험동의 1/5. 관리인 1명 + 졸음 구간 존재.
- 저녁 3급 소량 주입 수령(지원량 극미, 수명 연장 효과 거의 없음).
- 중정에서 스프린트 백로그 티켓 4개 수립 (엘릭서 조달 / 루틴 구축 / 외부 정보 / 몸 상태 유지).
- 엔딩 문장: "폐기동은 그를 잊으려 했다. …잊힌 자만이 움직일 수 있는 법이었다."

### ch004「591호」 (D+16 +α, 약 2주 관찰 + 대면)
- 서두: **591호 시점 서브 씬** — 이리스가 603호의 연기를 수일간 관찰 후 확신.
- 본문: 이리스가 603호에게 접근, "너, 소울 잃어버린 거 아니지?" 첫 접촉.
- 엘릭서 ↔ 수명 방정식 확립: 2급 1병 = 금화 150닢 = 30일. → 강신우 내면에서 "돈만 있으면 안 죽는다" 결론.
- 강신우 첫 외부 발화 "…고마워." / 두 번째 발화 "이리스."(이름 부여).
- 공범 협정: 관리인·제3자 앞에서는 연기 유지, 둘만 있을 때만 대화.
- ch004 말미 수치 박스: **남은 수명 10개월 · 엘릭서 재고 8일분 · 잔고 0 · 월 150닢**.
- 엔딩 펀치: "이번 생의 첫 번째 계산은 덧셈이어야 했다. / 전생의 마지막 계산은 뺄셈이었으니까."

### ch005 즉각 이어야 할 상황
- 시점: ch004 공범 합의 성립 **수일 뒤부터** 공조 체제 시작. 시놉시스 §메타 기준 **D+35 ~ D+60 (약 25일)** 전개.
- ch004 엔딩에서 선언된 "덧셈" 결의 → ch005는 그 덧셈의 **실행 단계 도입** (공조·관찰·기획·결의).
- 시놉시스 가정대로 ch005 시작 시 회수 요원 정기 방문에서 새 엘릭서가 주입된 상태로 출발 (엔딩 박스 = 20일분 전후).

## 3. 수명·엘릭서·자산 상태 (ECONOMY)

### 엘릭서 재고 궤적
- ch003 D+16: 2급 1병 주입 = 이론상 30일분 (덮어쓰기, 약 45일분 체내 잔량으로 강신우 추정).
- ch004 D+30 전후: 폐기동 3급 소량 저녁 주입 × 14일치 ≈ 미미(1일치 1/10 수준, 합산 체감 연장 1~2일 수준).
- **ch004 엔딩 박스 공식값: 엘릭서 재고 8일분** (정본).
- **ch005 시작 전제(시놉시스 §메타)**: 회수 요원 정기 방문 = 격주 목요일. D+35 목요일 정기 주입 1회 수령 → 새 엘릭서 2급 1병(30일분) 보충 가정.
- **ch005 내 구간 소모**: 25일 × 1일/일 = 25일분 소모. 단, 격주 회수 방문이 D+35와 D+49(목요일)에 있으므로 2회 보충 가능.
- **ch005 엔딩 박스 권고값**: **엘릭서 재고 20일분** (시놉시스 §메타 지정값). 이는 "마지막 보충 후 10일 정도 경과 + 향후 탈출 이송 준비 기간 여유분" 해석으로 일관.

### 수명
- ch004 엔딩 박스: **남은 수명 10개월**.
- ch005 시작 ~ 엔딩: 약 1개월 시간 경과 → ch005 엔딩 박스 **남은 수명 9개월**.
- 정본 참조: protagonist-bible §5, timeline.md §Part 1.

### 자산
- 금화 0 / 은화 0 / 동화 0. ch005 내 수입 이벤트 **없음** (탈출 전이므로 변동 0 유지).
- 월 운영비(엘릭서 대체원가): 금화 150닢 / 월. 지금은 길드가 결제 주체.
- 연 총비용: 1,800닢 (ch004 내면 독백에서 확정).

### 신체 개조 단계
- **Stage 0**. 변동 없음. 변두리 오두막 지하 개조는 Stage 1, ch021~025 예정 — ch005에서는 절대 시연 금지.
