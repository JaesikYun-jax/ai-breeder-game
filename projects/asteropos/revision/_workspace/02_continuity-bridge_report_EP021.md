# 연속성 보고서 — EP021

## 1. 시점
- 시간: 같은 새벽. EP020 결착 직후 (5시 반~6시). 마탑 본부 봉투 인계 직후 컷
- 장소: 학원 결투 마당
- 카엘: 마당 한 변. 봉투 손가락 안쪽
- 디트마르: 한 자락 가까이 옮긴 자리. 짧은 인정 풀어 둔 결
- 가주: 정문 너머 흰 망토 옮긴 길 (퇴장/멎음 정리 필요)
- 검우 셋, 두꺼운 교관, 라이덴 호송관 모두 마당 안

## 2. 합 진행 (EP019+EP020)
- 1단계 6합 무승부 (3+3)
- 5합에서 점 압축 한 흐름 1회 시연 (FS-209a 첫 정식)
- 가주 첫 발화 "멎으시지요" (두꺼운 교관에게, 카엘 자리까지 X)
- 두꺼운 교관 "……무승부의 자락"
- 디트마르 "평민이라기에는 한 자락 다른 길이군요"
- 동기 한 명 "한 결, 다른 길이지" (FS-214 첫 외부)
- 라이덴 호송관 자유 도시 마탑 본부 봉투 인계
- 두꺼운 교관 "2단계는 학원 명의로 풀어 받힐 길" 정리

## 3. 수치
| 항목 | 값 |
|---|---|
| 점 압축 후과 | 시야 한 변 옅게 흐려짐 며칠. 코피 X. 두통 가벼워짐 |
| 빛 두 흐름 동시 | 여전히 모자란 자리 (한 흐름만) |
| 본인용 약초 | 거의 소진 |
| 신규 약초 | 두 봉지 미개봉 |
| 한스 종군 검 | 보따리 안쪽 흠집 잔존, 결투 미사용 |
| 학원 정식 검 | 결투 후 거둠 |
| 마탑 본부 봉투 | 미개봉 |

## 4. 관계
- 카엘 ↔ 디트마르: 인정 한 자락 진입
- 카엘 ↔ 가주: 첫 발화 (간접). FS-211a 본격화 임박
- 카엘 ↔ 야콥: 빚 + 정중 동맹
- 카엘 ↔ 토비아스: 자유시 평어 (변경 결 어휘 화당 누적 회피)
- 카엘 ↔ 오토: 단답·비언어
- 카엘 ↔ 두꺼운 교관: 입회인 권위
- 카엘 ↔ 라이덴 호송관: 표면 정중·실질 감시
- 익명 동기 → 카엘: FS-214 첫 외부 시드

## 6. 클리프행어 (EP020)
라이덴 호송관 봉투 인계. 카엘 손가락 안쪽 봉투. 인장이 새벽 흙 위에 풀린 단문.

## 7. 비언어 누적 (EP019+EP020) — EP021 변주 필수
- 카엘: "검 자루 손바닥 안쪽 두꺼이 받음" 6회+ → 화당 1회 이내. 변주: 손목 한 결, 봉투 모서리, 보따리 매듭
- 디트마르: "가문 인장 옷자락 안쪽 거둠" 4회 → **EP021 금지**. 변주: 시선·호흡·발끝·직설 발화
- 야콥: "옷자락 단정하게 받음" 3회+ → 화당 1회. 변주: 한 호흡 늦게, 한 결 끄덕임
- 토비아스: "변경 결이 야물지" → **EP021 절대 금지**
- 오토: "어깨 옮김 의리" → 화당 1회. 변주: 단답 1회, 시선 머묾
- 가주: 시선 두꺼워짐 → EP021 카엘 자리 직접 발화 1회 (FS-211a 본격화)

## 8. 사이다/고구마
- EP019 사이다 / EP020 혼합 / EP021 혼합 또는 사이다 권장 (자유 도시 호출의 미지 압박감)

## 9. 클로징 자구 — EP021 사용 금지
- "카엘도 ~도 ~ 풀어 받지/받을 결이었다" (EP018·EP019)
- "~ 인장이 ~ 풀려 받힌 자리" (EP020)
- "~ 어떤 자락으로 누울지는 ~ 너머에서 풀어 받을 결이었다"
- "짐작하지 못했다" / "알게 될 것이었다" / "늦게 알게 될 자리였다" / "풀어 두지 않은 결이었다"

EP021 권고: 단문 클리프행어. 라이덴 출발/자유 도시 도착 직전 단문, 또는 셀레네 시드 인물의 첫 등장 단문 컷.

## 10. Arc 2-b 종결 / Arc 3 진입
- 자유 도시 마탑 본부 학생 연구원 자리 (Arc 3 무대, 셀레네 시드 가능)
- 디트마르 2단계 결착: **학원 명의 약식 처리** 권장 (또는 짧은 결투 + 디트마르 자청 패배). EP020에서 이미 "2단계는 학원 명의" 정리됨 → 본격 결투 X 가능성
- 검우 작별 정서적 자락 — 과잉 X
