## ALIVE-1: 메아리 대화 (0건 / PASS)

### 진단

EP001 전체 대사는 17턴(체감 4 클러스터)이며 어떤 핑퐁도 직전 화자의 키워드를 단순 반복하지 않는다.

- **L44~45 (카엘↔리나, 빵 클러스터)**
  - "왜 자꾸 챙겨." / "챙기는 게 아니야. 어차피 한 덩이는 남는 거니까."
  - → 리나의 "어차피 ~니까" 화법(설정 시드 FS-110)이 첫 노출되며 카엘의 질문을 **자기 화법으로 비껴 받는다**. 메아리가 아니라 캐릭터 정체성 노출.

- **L51~54 (우물 동행 거절)**
  - "오늘 우물물 길러 갈 거야?" / "가야지." / "같이 갈까." / "아니. 먼 길도 아닌데."
  - → 단답 핑퐁이지만 카엘의 "짐이 되지 않으려는" 미세 거절이 깔린 결. 무개성 X.

- **L70~71 (또래 아이들)**
  - "또 길어. 우물 안에 뭐 있나 보지?" / "보긴 뭘 보냐, 못 보는데."
  - → 두 아이가 각자 다른 톤(놀림→비웃음 굳히기)으로 **계단 만들기**. 같은 감정 반복 X.

- **L90~93 (시비꾼↔리나)**
  - "오, 무서워라." / "물러서." / "리나, 너 자꾸 끼니까 재미없잖아." / "한 번 더 말해야 돼? 물러서."
  - → 리나가 같은 단어("물러서")를 두 번 쓰지만 두 번째는 **같은 단어의 무게가 다른** 사용. 메아리가 아닌 **단호한 보호**(설정 보이스 키)의 시그니처.

- **L142~144 (채석장 위에서 외치는 아이들 3턴)**
  - "야, 거기 있냐?" / "안 죽었지?" / "기어 올라와. 안 데려가 줘."
  - → 3명 아이가 각자 다른 잔혹의 결(확인 → 확인 → 방치). 전원 동일 감정 X. PASS.

### 판정: PASS — 메아리·기능 대사·전원 동일 반응 모두 0건.
