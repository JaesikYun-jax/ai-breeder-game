| EP002 | 충격 → 위장 정착 → 부성·정치 두려움 → 첫사랑 폭발 | 시각 회복·금빛 손바닥 자각 | "신의 은총"으로 마을 정착 |

## 한스 (양아버지)

- **보이스 키**: 무뚝뚝 + 부드러운 부성 톤 변환 가능 + 정치 감각
- **고유 비언어**: 흙 털기, 손가락 떨림(신성모독 두려움), 침묵
- 화별 변동:
  - EP002: 부성 톤 변환 (대사 + 안대 만들기) + 손가락 떨림 (FS-116)
  - EP003: 별칭 형성 동안 한 마디도 거들지 않음 — 결심 빌드업
  - EP004: 카엘 단련 지켜봄, 부엌 결정 대화, 통보
  - EP005: 촌장 설득, 영광/인질 균형 가르침, 종군 검 선물
  - EP006: 사흘간 검 가르침, 출발 작별

## 이르마 (양어머니)

- **보이스 키**: 따뜻한 평민 말투 ("많이 먹으렴", "괜찮아", "쉬어라")
- **고유 비언어**: 손 떨림 (FS-104, 약초·민간 치유 라인 시드, 다정함과 두려움 사이 결)
- 화별 변동:
  - EP002: 시각 회복 직후 카엘 얼굴 들여다봄, "신께서 보살피신 거지"
  - EP003: 손길에 다정함과 두려움 사이 결 (별칭 회수 동안)
  - EP004: 부엌 결정 대화 (한스와)
  - EP005: 마을 회의 배경
  - EP006: 약초 봉지 + "네가 어디로 가든 너는 우리 자식이다"

## 리나 (11~12세, 첫사랑 자각 + 그늘 곡선)

- **보이스 키**: 단호한 보호 ("물러서.") + "어차피 ~니까" 화법 (FS-110 그늘 시드)
- **고유 비언어**: 좋은 향기 (카엘에게는 식별 단서, FS-105) / 한쪽 보폭이 미세하게 더 긴 발걸음 / 침묵의 분노
- **마을 어른들이 한눈에 예쁜 아이라고 부름 — 카엘만은 평생 모름 (시각 회복 전까지)**
- 화별 변동:
  - EP001: 빵 가져옴 ("어차피 한 덩이는 남는 거니까") + 우물가 단호 보호 + 채석장 후 분노의 침묵
  - EP002: 클로징 첫사랑 자각 폭발 (FS-114) — 부둥켜안고 흐느낌
  - EP003: 능력 시연 동안 한 박자 늦은 시선
  - EP004: 새벽 빵 반죽 — "어차피 너는 여기 있을 사람이 아니야" (FS-110 본격화)
  - EP005: 회의 결과 받아들임
  - EP006: 마지막 약속 ("잘 가, 카엘. 그리고 돌아와", FS-120) → 출발 아침 부엌 안쪽 한 발 떨어져 작별

## 비언어 사용 빈도 (반복 방지용)

EP001~EP006 누적 사용 — 직전 2화 대조 시 변주 필요한 표현:
- "흐느낌" (EP002·EP006 가능성) → EP002에서 사용 시 EP006은 다른 표현
- "손가락 떨림" (한스 EP002) → EP005에서 다른 표현 권장
- "어차피" (리나 EP001·EP004) → 화당 1회 이내 절제, 톤 변주
- "가만히 끌어안고" (한스 EP002) → 1회 한정
- "한 박자 늦게" (리나 EP003) → 1회 한정 — 의미 모호 시 폐기 가능
