## ast000 — 프롤로그. 별 떨어진 밤 (1,607자, 리더 카운트)

**무대**: 린덴호프 (제국 동변, 검은숲 어귀의 작은 마을)
**시점**: 마르타 할멈 밀착 자유간접 → 마지막 1단락 객관자 narrator

**비트**:
- 한밤중 동쪽 하늘이 일곱 번 갈라짐 — 일곱 줄기 빛 동시 낙하
- 그중 한 줄기가 마르타 할멈의 집 지붕을 부수고 지하실 천장까지 뚫음
- 다음 날: 이장이 운석이라 함, 영주관 신고는 다음 장날에
- 검은 고양이 녹스가 지하실로 자주 내려감 — 매끈한 녹색 알 모양 돌을 매일 밤 품음
- 며칠 후 마르타 할멈이 녹스를 부르지만 응답 없음 → 털 무더기 + 살아 있되 너무 가만한 녹스 발견 → 안도하고 떠남
- 떠난 뒤 녹스의 눈이 녹색으로 빛나며 눈꺼풀이 좌우로 감김 (도마뱀형)
- 지하실 벽돌 틈에 푸른 이끼 한 줌 돋아 있음
- 미래 예고 — 마을 이름이 신성교회 명부에 실리기까지 두 달 남음

**박힌 떡밥 (5)**: FS-001(동시 낙하)·FS-002(녹스=3번)·FS-003(도마뱀형)·FS-004(마을 실종)·FS-005(마르타 생존)

**훅**: 두 달 카운트다운 + 도마뱀 변형

## ast001 — 1화. 구덩이 (2,696자, 리더 카운트)

**무대**: 시골 마을 (린덴호프와 매우 멀리 떨어진 곳, 가칭 무명 마을)
**시점**: 카엘 12세 밀착 자유간접
**시점 인물 풀**: 카엘 / 리나 / 한스 / 이르마 / 마을 광장 어른들 / 또래 아이들

**비트**:
- 새벽 농가 일과 — 카엘이 짚을 손끝으로 골라냄, 시각 없이 자란 손의 자각
- 리나가 빵을 가져옴 — "어차피 한 덩이는 남는 거니까". 리나의 좋은 향기 = 식별 단서
- 광장 우물가 — 또래 아이들 + 술꾼의 조롱 ("못 보는 척이지?")
- 리나가 손목을 잡아 끌어당기며 "물러서." 3회 단호 경고 → 어른의 거듦으로 발걸음 흩어짐
- 해 기울 무렵 — 아이들 셋이 카엘을 외곽 낡은 채석장으로 끌고 감
- 며칠 전 채석장에서 새벽 굉음 (옛 폭약 의심 — 외부 인물의 추측에 머문 형태로 노출)
- 발로 차서 카엘 비탈로 굴림 → 흙·자갈에 굴러 바닥에 닿음 → 손에 따뜻한 무엇이 만져짐 → 무심결에 주머니 (능동적 줍기 X, 따뜻함이 좋다는 작은 감각이 먼저)
- 기어 올라와 마당 도착 — 이르마의 떨리는 손, 한스의 무뚝뚝한 흙 털기, 리나의 분노의 침묵
- 침대에서 돌이 여전히 따뜻함을 느끼며 잠듦
- 미래 예고 클로징: "그 따뜻함이 카엘의 죽은 눈을 향해 천천히 움직이기 시작했다는 것은, 카엘 자신도 모르는 일이었다."

**등장 인물**: 카엘·리나·한스·이르마 / 무명 또래 아이들·술꾼·광장 어른

**박힌 떡밥 (8)**:
- FS-101 (성석 결합 시드)
- FS-103 (한스의 옛 종군 — 이번 화는 이장과의 대화로 간접)
- FS-104 (이르마의 떨리는 손 = 약초·민간 치유 라인의 정서 시드)
- FS-105 (리나의 좋은 향기)
- FS-110 (리나의 정서적 그늘 첫 시드 — "어차피 ~니까" 자기 위치 낮추는 화법)

**감정 아크**: 무력 → 보호 → 굴욕 → 작은 따뜻함의 위로

**훅**: 미래 예고 단문 — 따뜻함이 죽은 눈으로 움직이기 시작

## ast002 — 2화. 신의 은총 (6,602자, 리더 카운트 — 2026-04-26 rewrite+polish 후)

**무대**: 같은 시골 마을, 카엘의 방·마당
**시점**: 카엘 12세 밀착 자유간접

**비트**:
- 낯선 통증으로 깨어남 — 죽은 눈에 통증 신경이 있을 리 없는데 (이상 시드)
- 시각 회복 — 천장의 결, 손등 주름, 어제 흙 자국. 두통·정보 과부하
- 손바닥 아래 옅은 금빛 → 자기 눈이라는 자각
- 한스가 들어와 입단속, "이걸 누구한테도 보여 주지 마라"
- 한스가 부드러운 부성 톤으로 어제 일 자세히 들음
- 한스의 방 수색 — 어디에서도 따뜻한 돌 못 찾음. 침대 밑에서 잃어버린 동전 한 닢 (코미디 위로) — 그러나 동전은 줍히지 않고, 한스가 카엘을 가만히 끌어안음
- **한스가 누런 소가죽 안대를 만들어 줌** — 두 개의 작은 구멍, "네가 그 빛을 네 손으로 다룰 수 있게 될 때까지만이다"
- 이르마 다시 들어와 한스의 에리히 노인(산속 명의) 변명 듣고 받아들임 → "신께서 보살피신 거지"
- 한스의 표정 = 신성모독·영주관·끌려 나가는 자리까지 알고 있는 두려움 (손가락 떨림으로 표출)
- 마을 사람들 (토마스 신부·옆집 노인) 차례로 다녀가며 "신의 은총" 정착
- 마당 단둘이 — 한스: "다만 누구한테도 다 보여 주지 마라. 너를 위해서야"
- 클로징: 리나 재회 → 카엘이 처음 시각으로 본 얼굴 = 리나 → "이렇게 생겼었구나" → "내가 상상한 것보다 훨씬 예쁘다" → "세상이 예쁜 줄 알았는데 아니었어. 네가 있는 세상이 예쁜 거였어" → 리나 흐느낌 → 부둥켜안고 둘 다 우는 첫사랑 자각 폭발

**등장 인물**: 카엘·한스·이르마·리나·토마스 신부·옆집 노인

**박힌 떡밥 (5 신규 + 4 회수/전개)**:
- 회수: FS-105 (향기 → 얼굴 회수, 정서 폭발)
- 신규: FS-114 (첫사랑 자각 폭발), FS-115 (사라진 따뜻한 돌 → 4 아크 회수), FS-116 (한스 신성모독 두려움), FS-117 (누런 소가죽 안대 = 위장+부성 시드)
- 전개: FS-101 (결합 직후, 본인은 모름), FS-104 (이르마의 손)

**감정 아크**: 충격(시각 회복) → 위장 정착 → 부성·정치 두려움 → 첫사랑 자각 폭발

**훅**: 안대 너머 두 구멍 + 첫사랑 자각이 어떻게 일상에 스며들지

## ast003 — 3화. 현자의 재림 (6,498자, 리더 카운트 — 2026-04-26 rewrite+polish 후) — 재집필 v2 (사용자 피드백 6건 반영) → v3 분량 통일 보강

**무대**: 같은 시골 마을 (며칠 ~ 몇 달 압축)
**시점**: 카엘 12세 밀착 자유간접
**핵심 방향 (사용자 피드백)**: 펠릭스 가격표·가락지 시연 폐기. **빛 컨트롤 자각이 먼저** → **두 가지 비범함**: 기억력 (법전·연대기 외우기) + 직관 (활·한계 자각·체력 단련 시작).

**비트 (재구성)**:
1. **안대 적응 며칠 압축** — 시각이 보조 도구로 자리잡음. 마을 "신의 은총" 정착
2. **거울 앞 빛 컨트롤 첫 자각** (앞쪽 이동) — 호흡으로 금빛 올리고 누르는 두 흐름이 자기 손에 있음을 확인. 안대 졸업 시작.
3. **시간 압축 — 안대 졸업 ~ 몇 달**. 마을 사람들도 안대 없는 모습이 더 익숙해짐. "세상이 보이게 됐다는 것은 시작에 지나지 않았다."
4. **첫 번째 비범함 — 기억력**: 글자 모양을 처음 보고 익힘. 마을 회관의 왕국법전·영지 연대기 두 권을 통째로 머릿속에 옮겨 앉힘. 방앗간 주인 vs 옆집 농부 마당 분쟁에서 카엘이 "왕국법전 제 14편 토지 분쟁의 항"으로 판결. **"현자의 재림" 별칭 형성** (FS-109 조기 회수).
5. **두 번째 비범함 — 직관**: 가을 추수 후 청년단 사격 연습. 카엘이 활을 처음 잡는 순간 "어디로 어떻게 쏘면 어디에 닿는지" 그림이 떠오름 → 끈을 잘라 송아지를 구함 (FS-108 회수). 잠시 후 일을 미리 보는 듯한 결의 직관.
6. **직관의 한계 + 체력 단련 시작**: 먼 표적은 바람 변화로 빗나가고, 짚 인형은 화살이 튕김. 시위 끌어당길 만큼 팔이 강하지 않음. → 직관은 손이 닿는 자리까지만 정직함을 자각 → **그날 밤부터 짚단 들기 단련 시작** (마검사형 신체 강화 시드).
7. **한스의 침묵**: 별칭이 한 바퀴 도는 동안 한스는 한 마디도 거들지 않음. 이르마의 손길은 다정함과 두려움 사이 결.
8. **클로징**: "빛의 두 흐름은 이미 카엘의 손에 있었다. 다만 빛만으로는 충분하지 않다는 것을, 카엘은 그 며칠 사이에 함께 알았다."

**등장 인물**: 카엘·한스·이르마·토마스 신부·방앗간 주인 부부·옆집 농부·청년단 우두머리 (소문 단)

**박힌 떡밥 (회수 + 신규)**:
- **FS-117 회수 진행**: 빛 컨트롤 자각 → 안대 졸업 시작
- **FS-109 조기 회수**: 사진 기억 의식적 시연 (법전·연대기) → "현자의 재림" 별칭 형성
- **FS-108 조기 회수**: 활 단발 시연 + "우연" 위장 + 직관 한계 자각
- **신규**: 카엘의 체력 단련 시작 (마검사형 신체 강화 노선 = 2-a/2-b 아크 회수와 직결)

**감정 아크**: 빛 자각 → 시각 받음의 본질 자각 (법전 외우기) → 직관 자각 → 한계 자각 → 단련 시작 → 한스 두려움

**훅**: "빛만으로는 충분하지 않다" — 다음 화 결심·진학·체력 단련 본격화

**제거된 요소** (사용자 피드백):
- 펠릭스 가격표 시연 (장황·시시함)
- 청년단 우두머리 가락지 위치 추리 (장황·몰입 약함)
- 리나의 "한 박자 늦게 시선 돌리기" 도입 (의미 불분명)

## 1~3화 누적 정착 캐논 (2026-04-25 v3 기준)

### 카엘 (12세, 결합 후 ~ 몇 달)
- **시각 상태**: 결합 직후엔 안대 사용 → 거울 앞 빛 두 흐름 자각 → 몇 달 후 안대 졸업, 평소엔 평범한 갈색 눈 (위장 기본). 금빛은 강한 발현 시만.
- **자각된 능력 (3화 끝 기준)**:
  - **사진 기억**: 카엘의 눈에 한 번 들어온 정보는 망각되기를 거부함. 잊고 싶어도 잊혀지지 않음. 왕국법전·연대기 통째로 머리에 옮김.
  - **직관**: 활을 잡으면 그림이 떠오름. 잠시 후 일을 미리 보는 듯한 결의 능력. 단, 정보·맥락이 있어야 작동.
  - **빛 두 흐름**: 호흡으로 금빛 올리고 누르기. 의식적 컨트롤 가능.
- **자각된 한계**:
  - 직관은 손이 닿는 자리까지만 정직. 시위 끌어당길 팔 힘이 부족 → 체력 훈련 시작.
  - 마을은 좁다 — 직관이 닿는 거리의 끝. "세상이 마당의 끝에서 끝나지 않는다."
- **위장 톤**: 능력 시연 시 의식적 위장. "우연이에요" / "신부님이 예전에 한 번 인용하셨던 거 같아서요" 식.

- 부드러운 부성 톤 변환 가능 ("카엘. 어제 있었던 일 하나도 빠뜨리지 말고 다 말해 봐라.")

- 행상 펠릭스 = 도시 소문 통로

### ast012 — 12화. 제국 장학생 통보 (Arc 2-a 마지막, 6,348자)
**비트**: 학장실 호출 → 제국 장학생 통보 (영광/인질 회수) → 마탑 분소 거절 + 검술 아카데미 결심 → 양부모·리나에게 편지 → 출발·호송관 첫 등장 + 미래 예고 클로징
**시드 회수**: FS-208 본격 (제국 장학생 통보) / FS-205 본격 (편지) / FS-209 시드 (검술 아카데미 결심) / FS-113 회수 (영광·인질 두 단어)
**신규 인물**: 라이덴 학원 학장 (흰 줄·잉크 자국·정치 압박 인지) / 제국 호송관 (FS-210 시드)
**감정**: 학장의 죄책감 인지 + 카엘의 결심 (자기 길)
**Arc 2-a 끝.** Arc 2-b 검술 아카데미로 진입.

### ast013 — 13화. 제국 수도 도착 (Arc 2-b 첫 화, 6,209자)
**비트**: 마차 위 며칠 압축 → 수도 진입·황궁 첨탑·제국군 행렬 → 호송관 시선·인질 자각 → 임시 숙소·결심 점검 → 도시 야경·미래 예고
**시드**: FS-210 본격 (호송관 우리에 두는 시선) / FS-113 회수 (영광/인질 거리의 결로)
**감정**: 제국 수도의 압도 + 인질 자각 + 검술 아카데미 결심 다지기

### ast014 — 14화. 검술 아카데미 지원 (6,463자)
**비트**: 마탑 분소 등록 거절 → 검술 아카데미 이동 → 시연 (마법 신체 강화 + 한스 종군 검술) → 입학 허가 → 미래 예고
**시드**: FS-209 본격 회수 (검술 아카데미 의외의 지원)
**신규 인물**: 마탑 분소장 (학자적 호기심·정치 결) / 검술 아카데미 학장 (검의 자리·직설) / 두꺼운 교관 / 마른 교관 (헛기침·정통의 단정함)
**정확 자구**: "기본기는 평민입니다 / 흡수 속도는 천재이지요" 첫 발화

### ast015 — 15화. 입학·귀족 라이벌 (6,202자)
**비트**: 기숙사 첫 아침 → 강의실·동기 풀 첫 마주 → **디트마르 첫 등장 + 노골적 멸시** → 첫 검술 시간·흡수 → 식당·기숙사·미래 예고
**시드**: FS-211 신규 (재수없는 귀족 라이벌)
**신규 인물 풀**:

**감정**: 평민·외국 라이덴 출신 이중 마이너리티 자각 + 디트마르 멸시 받아넘김 (속으로 카운터 시점 계산)

### ast016 — 16화. 첫 합동 훈련 (6,287자)
