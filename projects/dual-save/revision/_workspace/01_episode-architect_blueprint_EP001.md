## 4. 감정 곡선

```
씬1(긴장 5: 사망 압도) → 씬2(긴장 3: 어리둥절·익숙한 천장) → 씬3(긴장 4: 짐작·어머니 정서) → 씬4(긴장 4: 결심·첫 발 + 마지막 클리프 5)
```

**곡선 설계 근거**: 1회차 사망의 압도(5)에서 자각의 어리둥절(3)로 한 박자 떨어졌다가, 어머니 회상의 정서(4)와 거래자 마인드 점화의 결심(4→5)으로 점진 상승. 자조 5% 마지노선은 씬2의 *"...꿈은 아니네"*와 씬4의 *"짐꾼으로 26세까지 산 게 길었던 거다"* 두 개로 분산 — 자조→행동 1:5 비율 준수.

### 7-1. 오프닝 훅 (강도 4)

**구체적 사건/이미지/대사**:

**왜 독자가 멈추지 못하는가**:

- (3) *"...어머니?"* — 부모 미스터리 0단 떡밥 (G5 짐작만)을 *죽음의 마지막 단어*로 압축 노출

### 7-2. 중반 훅 (강도 4)
