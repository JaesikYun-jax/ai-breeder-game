### 1.1 현재 훅 구성 (마지막 500자)

```
"…적이라도 만들었나, 내가."
모니터 안에서 검은 코트가 2초 더 서 있다. 그 다음 화면 밖으로 사라졌다.
```

직전 4단락 흐름:
1. *위협의 종류가 달라졌다.* — 메타 인지 한 줄
2. `A급 신체. 펜트하우스. 경호원. 회사.` — 4목 단문 나열 (얻은 것 점검)
3. *조용한 부자가 될 줄 알았다. … 안전할 줄 알았다.* — 미래 시점 예고형 (글로벌 v2 §8 정확 적용)
4. **"…적이라도 만들었나, 내가."** — 자조형 질문 + CCTV 정지 화면 컷

### 1.2 평가

- **설계도 EP004 클리프행어 절대값** 그대로 구현 — 세부설계 §EP004 *"…적이라도 만들었나, 내가" (★ 강도 ★★★★★)*
- **위기 고조형 + 자조 질문형 복합**: R2 §4.2 5대 패턴 중 ④ 위기 고조형(검은 코트 4번째 = S급 정찰 12회의 1/3) + ① 질문형 융합. 5화 정야겸 침공 클라이맥스로 결제 동력 직결

- **G4 도윤 보이스 톤 정합**: *"…적이라도 만들었나, 내가"* — 영웅적 결의 X / 자조 질문 / 짐꾼 출신의 거리감 그대로 / 1줄 자조 → G1 *자조 1:5 비율* 통과 (직후 5화로 인계)

```
조용한 부자가 될 줄 알았다. 1차 사이클 정보로 돈을 만들고, 영약으로 몸을 바꾸고, 높은 곳에 앉으면 안전할 줄 알았다.
```

### 2.3 평가: 결함 없음, 그러나 한 가지 보강 권장

EP004 첫 1,000자에 **사이다 1방**이 약하다:
- 100억 즉결 낙찰은 *남들이 망설이는 동안 도윤만 단호히 결정* = 작은 사이다 1방 ◎
- 다만 *"100억"* 한 마디로 끝 — 정보 우위가 명시되지 않음 (검가 후계자 사망 시점에 시장 유출됐다는 건 도윤 내심에만)
