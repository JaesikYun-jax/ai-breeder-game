### S-3. 위치/거리 미세 마찰 — MINOR
- L67: "도윤이 마을 입구에 멈추었다. 산비탈에서 검은 옷 셋이 절반쯤 내려왔다. 거리 60보" — 60보.
- L77 두 줄 뒤: "도윤이 한 발 앞으로 나갔다. 거리 50보" — 한 발에 10보 좁힘. 한 발(보 단위)에 10보(거리 단위) 좁히는 산술이 부자연스러움. 한 발은 1보 수준이므로 60→59보 정도가 자연스러움.

- **교정 제안**: L77 "도윤이 한 발 앞으로 나갔다" → "도윤이 산비탈 셋이 평지로 내려오는 동안 마을 입구에서 능선 쪽으로 열 걸음 정도 나갔다. 거리 50보."

### TIMELINE 교차대조 결과

**TIMELINE 교차대조 — 1건 마찰 발견 (MAJOR)**

- **마찰점: 산행 시간 — EP019 5시진 vs EP021 1시진**
  - EP019 L13 *"우리가 갈 길은 — 다섯 시진"* + L31 *"한 시진을 올라가고 반 시진을 쉬었다"* + L51 *"오후 늦게, 두 사람은 서쪽 능선 위에 도착했다"* → 헛간 새벽 출발 → 분지 서쪽 능선 위 도달까지 **5시진 산행(서쪽 우회)**.
  - EP021 L55 *"산을 한 시진 올라간 뒤, 두 사람은 어제와 같은 분지 서쪽 능선 옆구리에 도착했다"* → 같은 새벽 헛간 출발 → 같은 분지 서쪽 능선 옆구리 도달이 **1시진 산행**.
  - EP019의 산행 5시진 + 점심에 식량 마지막 조각 + L51 *"오후 늦게"* 도달은 의식 시각(일몰 1시진 전)과 정합 — 새벽 출발 → 오후 늦게 도달이 자연스럽다.
  - EP021은 새벽 → 1시진 → *어제와 같은 분지 서쪽 능선 옆구리 도착* + *어제 의식을 봤던 자리*임을 명시 + L67 일몰 직전 마을 입구 도달이 명시 → 새벽 + 1시진 = 오전인데 마을 입구 도달은 일몰 직전. 그 사이 약 5~6시진의 공백.
  - EP021의 *"산을 한 시진 올라간 뒤"*는 EP019의 5시진 산행과 충돌. 등급 **[TIMELINE] MAJOR**.
  - **교정 방향**: L55를 *"산을 다섯 시진 올라간 뒤"* 또는 *"어제와 같은 길이라 산을 다섯 시진 올라간 뒤"* 또는 *"산을 어제와 같은 길로 다섯 시진 올라간 뒤"*로 수정. 또는 L55 직후에 능선 옆구리 도착 후 *"의식을 봤던 자리에서 의식 시작을 기다렸다"*를 추가하여 시간 공백을 메움.

### 산술 검증
- L77 *한 발 → 60→50보* 산술 마찰: SCENE S-3 참조. 한 발에 10보는 등가가 아니므로 **[TIMELINE] MINOR / SCENE 경합 — MINOR 등급으로 분류** (SCENE S-3 교정 시 동시 해결).
- L123 *거리 30보, 20보, 10보 (10보 간격으로 좁힘)* — 빈손 도윤이 가운데 자에게 다가가는 카운트다운. 산술 정상.
- L101 *"50보를 가로지르는 동안"* — 정지 안에서 검이 50보 날아가는 비행 거리 = 50보. 가운데 자가 정지 안에서 도윤 쪽으로 좁히는 평소 걸음(L95)과 검 비행이 동시 진행 → 검 끝이 가운데 자의 손목에 닿음(L105). 이 동선은 정지 안 시간(0.5초) 안에서 50보 비행 = 검 속도 100m/s 수준(50보 ≈ 30m / 0.5초). **물리 산술 실측 불가 — 메커닉 묘사로 허용 가능.** 결함 아님.

### P-1. 정지 안에서 도윤이 검을 던지는 타이밍 (MAJOR)
- L89~L99: 도윤이 1식 참 자세 → 정지 발동 → 정지 안에서 검을 던짐.
- L103 *"이번에는 — 도윤이 검을 던졌다는 것까지는 보지 못했다. 도윤이 검을 던진 시점이 — 가운데 자의 시야가 도윤에게 닿기 직전이었기 때문이었다"*
- **시점 문제**: 정지가 시작된 *직후*에 도윤이 검을 던졌다면 가운데 자는 정지 안에서 도윤의 모든 행동(검 들기 → 던지기 → 검 비행)을 시야 추적 시간 0.5초 동안 봐야 함. *시야가 도윤에게 닿기 직전*이라는 표현은 가운데 자가 정지 안에서 *50보 거리를 도윤 쪽으로 이동하면서* 시야가 천천히 도윤에게 맞춰지는 단계 묘사로 읽힘.
- 다만 EP020 L57에서 *"가운데 자가 도윤의 숨 한 번 들이쉬는 사이에 거리를 좁혔다. 가운데 자의 손은 검도 도구도 아닌 맨손이었다"* — EP020에서 가운데 자는 정지 시작과 동시에 도윤 쪽으로 *즉시* 좁힘.
- **마찰**: EP020의 가운데 자 = *정지 시작과 동시 즉시 좁힘* / EP021의 가운데 자 = *정지 시작 후 시야가 천천히 도윤에게 맞춰짐*. 두 묘사 차이가 큼.
- 또한 정지 안에서 도윤이 검을 던지는 행위 = *검의 무게가 균일하지 않아 회전이 어색했지만(L101)* + 50보 비행 + 손목 닿기 + 정지 풀림 = 정지 0.5초 안에 이뤄짐. 이 모든 사건이 0.5초 안이라는 점이 메커닉적으로는 *세계 시간 0.5초 / 정지 안 시간 ≠ 0.5초(주관적으로 더 김)*로 해석 가능 — EP020 L39~L41 *"도윤의 호흡이 정지에 얹히지 않았다 ... 도윤은 자기 손을 들 수 있었다"*가 이 해석을 지지.
- 그러나 *"가운데 자도 정지 안에서 움직였다"* + *"가운데 자가 도윤이 움직이는 것을 어제처럼 봤다(L103)"* = 가운데 자도 정지 안 시간을 동등하게 사용. 그렇다면 가운데 자가 도윤의 *검 들기 → 던지기 → 비행 50보* 전체를 시야 추적 못 했다는 설명에 추가 합리화 필요.
- **교정 제안**: L99 직후 또는 L101 직후에 한 줄 추가. 예: *"가운데 자는 정지에 들어선 직후 도윤 쪽으로 첫 걸음을 떼느라 시선이 발 끝에 머물러 있었다. 도윤이 검을 든 시점은 — 가운데 자가 고개를 들기 전이었다."* 또는 *"정지 안에서 가운데 자는 도윤을 향해 걷느라 거리를 좁히는 일에 집중했다. 검을 던지는 동작은 — 가운데 자의 시야 한 박자 바깥에 있었다."*

### P-2. 가운데 자의 정보 자백 개연성 (MAJOR — 이미 사용자가 인지)
- L141~L163: 가운데 자가 의식 잃기 직전 *카르무스 / 시간의 신 / 잃어버린 일부* 정보 흘림.
- **사용자 명시**: *"가운데 자가 미끼로 정보를 의도적으로 흘렸다는 가설을 EP022에서 본격화하므로 본 화에서는 단순 자백처럼 보임 + 사용자 깨달음은 EP022로 미룸"*.
- 본 화 L131~L133에서 도윤이 검 끝을 목에 대고 *"옷깃 안주머니에 무엇이 있는지 — 천천히 꺼내라"*고 추궁한 직후 가운데 자가 자발적으로 황보석을 꺼냄. 다음 도윤의 *"누구의 명령으로"*에 가운데 자가 *"...카르무스"*로 대답. 다음 *"카르무스가 누구인가"*에 *"시간의 신"* — 이 시퀀스에서 가운데 자가 *의식 잃기 직전*임을 강조하면 자백이 *방어 의지 상실*로 자연스럽게 읽힘.
- 그러나 EP019에서 보였던 *"입가에 미세하게 올라간 웃음"*(가운데 자가 도윤을 알아본 표정) + EP020에서 한국어 *"도윤마무. 살아 있었군"* (위계 의식 있음) → EP021에서 갑작스러운 정보 흘림은 **단순 자백으로는 어색**.
- *미끼* 가설은 EP022에서 회수되더라도 본 화에서 *작은 단서* 한 줄이 필요. 예: 가운데 자의 입꼬리가 *피 흘리는 와중 한 번 더 미세하게 올라감* 등.
- **교정 제안**: L159 *"가운데 자가 그 한 마디를 말했다. 손목의 피가 더 흘렀다. 의식이 흐려지는 직전이었다"* 직후 또는 L163 *"가운데 자의 의식이 끊겼다"* 직전에 한 줄. 예: *"가운데 자의 입가 한쪽이 — 마지막으로 한 번 더 미세하게 올라갔다. 도윤은 그것을 봤지만, 손목에서 흐르는 피의 무게에 묻혀 한 박자 뒤에야 인지했다."*

### P-3. *카르무스 / 시간의 신* 직접 노출 vs 페이싱 가드레일 (CRITICAL)
- 사용자가 [CANON]·[FORESHADOW]에서 직접 명시한 핵심 쟁점.
- **부트스트랩 §G5 (정본)**: *"시간 신이라는 단어는 100화 이후에 처음 등장 (무협·판타지 차원 빌런을 통해)"* / *"대회수자 결사는 200화 이후"* / *"부모 살해범 #7은 50~70화 이후 첫 등장"*.
- **플롯훅가이드 §G-PLOT-5 (정본)**: *"25화: 시간 신이라는 단어 처음 노출 (무협 사부 입)"* / *"이 페이싱은 변형 금지. 25화에 시간 신을 노출하면 100화의 시간 신 조각 시그니처가 약해진다"*. §4.1 EP025 비트 = *"무협 차원 사부의 입에서 시간 신은 — 우리도 못 본 신이었지 한 줄"*.
- **두 정본 충돌**: 부트스트랩 = *100화 이후*. 플롯훅가이드 = *25화 사부 입*. 플롯훅가이드가 더 세부 설계 문서이고 §G-PLOT-5에서 *25화 첫 노출*을 절대값으로 명시하므로 **플롯훅가이드가 우선 정본**.
- 본 화 EP021은 25화가 아니라 21화. 발화 주체도 정파 사부가 아니라 외경자 빌런. 다음과 같은 위반 항목들:
  1. **시점 위반**: 25화 첫 노출 → 21화로 4화 앞당김.
  2. **발화 주체 위반**: 정파 사부 → 외경자 빌런.
  3. **고유명 *카르무스* 등장**: 부트스트랩 §13.1·§4.6·플롯훅가이드·캐릭터시트 어디에도 *카르무스* 고유명이 정본 등록되어 있지 않음. 본 화에서 처음 등장. **신규 고유명사 — 정본 미등록**. *카르무스 = 시간의 신*이라는 동등화도 본 화에서 처음 단언.
  4. **잃어버린 일부 시그니처**: L163 *"도윤마무는 — 카르무스의 — 잃어버린 일부"* — 이는 캐릭터시트 §3.4 *"부모 살해범 #7 무기 = 시간 신 아티팩트 부서진 조각 1편"* + 캐릭터시트 §3.2 *"부모가 시간 신 아티팩트 발견·파괴 + 도윤마무 옮기기"*와 정합하나, *카르무스의 잃어버린 일부*는 부트스트랩 §4.6 *"신급 아티팩트 = 시간 신의 일부"*를 직접 진술하는 결정타. **종착 떡밥(200화+ 대회수자)의 핵심을 21화에 노출**.
- **결정적 마찰**: 본 화의 결말 클리프(L197 *"카르무스 = 시간의 신 = 도윤마무의 본주"* + L201 *"도윤마무가 누군가의 잃어버린 일부 ... 부모님이 도윤에게 카르무스의 일부를 옮겨 담은 셈"*)는 부트스트랩·플롯훅가이드의 *단계적 페이싱 절대값*과 정면 충돌.
