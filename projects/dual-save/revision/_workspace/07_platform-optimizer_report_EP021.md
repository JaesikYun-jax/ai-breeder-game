## HOOK (강도: 5, 유형: 떡밥 던지기형 + 대사 한 줄형 + 시그니처 회수형 3중 복합)

**현재 훅** (마지막 500자 압축):

1. *카르무스 = 시간의 신 = 도윤마무의 본주.* (라인 197) — 노트에 한 줄로 카르무스 명명·정체·도윤마무와의 본질적 연결을 동시 정의.
2. *도윤마무가 — 누군가의 잃어버린 일부였다. 부모님이 도윤에게 건 저주가 — 그 누군가의 일부였다. 부모님이 도윤에게 — 카르무스의 일부를 — 옮겨 담은 셈이었다.* (라인 201) — 부모-도윤-카르무스 3자 관계 첫 정의.
3. 산비탈 도망친 외경자가 거리 200보에서 *도윤의 옷깃에 시선이 머물렀다가 사라짐* → "카르무스가 도윤의 위치를 — 외경자의 시선을 통해 확인한 듯한 무게였다" (라인 205) — 카르무스가 *직접 도윤을 응시*하는 시각 시그니처 (다음 화 *카르무스의 응시* 제목 직결 빌더).
4. 촌장이 *"두 분 — 우리 마을이 — 두 분께 진 빚이 있소."* / 도윤이 외경자 결박 부탁으로 마무리 (라인 231~235) — 마을의 첫 인지·정파 무사 신분의 첫 공식 호명.

**구조 분석**:

- **첫 번째 훅 비트** — 라인 145~165에서 가운데 자의 발화 *"...카르무스." / "시간의 신." / "도윤마무는 — 카르무스의 — 잃어버린 — 일부."* = 25화 유료 클리프 직전 **시간 신 단어의 무협 차원 첫 공식 노출**. 플롯 가이드 §G-PLOT-5 "25화: *시간 신*이라는 단어 처음 노출 (무협 사부 입)"의 **변형 운용**(외경자 입에서 4화 앞당겨 노출) — 단, *카르무스*라는 고유명사로 25화 사부 입의 *시간 신* 단어 노출과 차별화 = 페이싱 가드레일 비위반.
- **두 번째 훅 비트** — *도윤마무 = 카르무스의 잃어버린 일부* (라인 163) = **본작 메인 퀘스트 3차 동기 (200화 자각 시드)의 25화 결제 페이지 직전 시드 매설**. 플롯 가이드 §1.2 "3차 동기 = 부모가 남긴 마지막 카드를 풀고 부모의 임무를 이어가기"의 시드.
- **세 번째 훅 비트** — *부모님이 도윤에게 카르무스의 일부를 옮겨 담은 셈* (라인 201) = **G3 부모 떡밥의 메인 퀘스트 본격 진입**. EP020에서 *부모님이 저주를 거셨다* 단순 공개 → EP021에서 *부모가 카르무스의 일부를 옮겼다*로 한 단계 심화.
- **네 번째 훅 비트** — *카르무스가 도윤을 응시* (라인 205) = **다음 화 제목 「카르무스의 응시」와 직결**. 클리프행어가 다음 화 제목·시작점을 동시 지정 = 결제 페이지 진입 빌더.
- **다섯 번째 훅 비트** — *촌장의 빚 인정* (라인 231) = **무협 차원 정파 무사 신분의 첫 공식 호명**. EP022 화양문 보고로 이어지는 외부 인정 트리거.

**훅 강도: ★★★★★ (5/5)**

- 클리프행어 5대 패턴 중 **떡밥 던지기형 (카르무스 = 도윤마무 본주) + 대사 한 줄형 (가운데 자의 카르무스 발화 시퀀스) + 위기 고조형 (카르무스의 직접 응시)** 3중 결합.
- 강도 5인 이유:
  - (a) **본 화의 핵심 회수 — 카르무스 단어 첫 명시 + 도윤마무 = 카르무스의 일부 + 부모님이 옮겨 담음 + 카르무스의 직접 응시** 4개 정보가 한 화에 압축 회수.
  - (b) **25화 유료 클리프 직전 메인 퀘스트 3차 동기 시드 매설** — 플롯 가이드 §1.2 "200~250화에 종국에 와서야 자각하는 3차 동기"를 25화 결제 페이지 직전에 *독자에게만* 노출 (도윤도 노트에 적지만, 본질적 의미는 200화에서야 자각).
  - (c) **다음 화 제목 「카르무스의 응시」와 직결** — 결제 페이지 진입 동기 명확화.
  - (d) **G4 거래자 마인드 시그니처 노트 회수** — *카르무스 = 시간의 신 = 도윤마무의 본주.* 단 한 줄로 새 정보를 절제 압축 = 본작 거래자 노트 톤 시그니처.

**플롯 가이드와 대조**:

- §G-PLOT-5 "25화: *시간 신* 단어 처음 노출 (무협 사부 입)" — 본 화에서 *카르무스* 고유명사 노출은 무협 사부 입 노출과 **별도 트랙으로 분리 운용**. EP025 사부 입의 *시간 신*은 도윤이 *어머니 노트 모래시계 그림*과 연결시키는 정서적 트리거 화 = 가드레일 비위반.
- §7 클리프행어 5대 패턴: 본 화는 *떡밥 던지기형*(카르무스 정체) + *대사 한 줄형*(가운데 자의 시퀀스 발화) + *반전형*(부모님이 옮겨 담음) 3중 결합 = 25화 직전 빌드업의 **최상위 정점**.
- §8 6단 사이클 *마수 인식 (20~25화)* 단계: "빌런 단체가 차원 너머에서 추적 시작 인지" → 본 화에서 *카르무스의 직접 응시* = 마수 인식 페이즈의 **본격 운용 정점**.
- §11.1 1차 빌런 정야겸 #14: "*도윤마무 저주를 가진 자. 회수하러 왔다.*" / EP021 가운데 자: "*이건 회수 도구. ... 도윤마무를 회수한다.*" = **결사 일관성 시그니처 3차 회수** (EP019 휘장 → EP020 한국어 발화 → EP021 회수 도구 직접 명시).
- §13.2 유료 25화 황금 라인 "*슬롯이 한 개라서 다행이라 생각했다. 사실은 — 부모님이 갈아 만든 마지막 카드였다.*" 빌드업 — 본 화의 *부모님이 카르무스의 일부를 옮겨 담음*은 25화 *마지막 카드* 시그니처의 **메커니즘 시드**.

**훅 강도 권고치(4~5)**: **5 정확 달성**. 25화 유료 클리프 4화 전 = 결제 전환 직전 클라이맥스 빌드업 페이즈에 강도 5는 **결제 페이지 진입률 직결 빌더**.

**교정 제안**: **불필요**. 다만 옵션으로 라인 197 *카르무스 = 시간의 신 = 도윤마무의 본주.* 한 줄을 페이지의 마지막 단락 직전(예: 라인 200 부근) 한 번 더 변주 인용("*카르무스. 시간의 신.*" 두 단어만 다시 떠올림) 시 결단 톤 강화 가능. 단 현재 형태로도 PASS.

## OPENING (평가: 양호 — 동작·시각 즉시 진입 + EP020 결단 훅 직결 응답)

**시작 방식**: **시각·동작 즉시 진입 (배경 설명 0)**

EP020 마지막 500자 = *"...같은 실수를 to-do에 두 번 적을 일은 없다. 새벽빛이 조금 더 밝아졌다. 도윤은 눈을 감았다."*

EP021 첫 500자 = *"새벽이 헛간 천장 구멍 사이로 다시 들어왔다. 어제 새벽과 같은 빛. 도윤이 일어났다. 가슴을 한 번 짚었다. 갈비뼈가 흐른다. 호흡이 흐른다. 이번에는 손이 떨리지 않았다. 회귀의 두 번째 새벽은 — 첫 번째와 달리, 자기 몸이 어디에 있는지 알았다. ..."*

**연결 자연스러움**: ★★★★★

- EP020 마지막 컷(새벽빛 밝아짐 + 도윤 눈 감음)과 EP021 첫 컷(새벽이 헛간 천장 구멍으로 들어옴 + 도윤 일어나 가슴 짚음)이 **새벽빛 앵커 직결**. 카메라가 EP020 마지막 컷에서 한 호흡 끊김 없이 EP021 첫 컷으로 이어짐.
- 첫 3줄 내 동작(가슴 짚음 / 갈비뼈 흐름 확인 / 손 떨리지 않음) + 첫 5줄 내 대사("한 사제. 일어났소?") = **사건 즉시 투입**.

**훅 응답**: EP020 마지막의 *내일은 — 떨어뜨리지 않을 것이다 + 같은 실수를 to-do에 두 번 적을 일은 없다* → EP021에서 즉시 응답:
- (a) 라인 7 *이번에는 손이 떨리지 않았다* = EP020 마지막의 *어제 한 번 떨어뜨린 검*에 대한 신체적 응답.
- (b) 라인 19~29 거리 50보 / 검 던지기 / 사형 양쪽 둘 막기 = EP020 마지막의 *거리* 시그니처와 *4번 항목 = 적의 약점 패턴화* 회수의 **본격 행동 전환**.
- (c) 라인 41~45 *"한 사제. 자네는 — 어제 죽은 게 아니라, 어제 셋의 패턴을 본 거요."* / *"맞습니다, 사형. 죽음이 — 정보 수집의 비용입니다."* = EP020 자살 회귀의 의미를 *비용*으로 재정의하는 본작 거래자 마인드 시그니처 회수.

**배경 설명·회상 시작**: 0건. "지난번에 ~했던 것처럼" 패턴 없음. *어제와 같은 ~* 비교 컷은 다수 있으나 모두 시각·동작 즉시 진입.

**개시 패턴**: A패턴(시간+장소) — "새벽이 헛간 천장 구멍 사이로 다시 들어왔다. 어제 새벽과 같은 빛." = 시간(새벽)+장소(헛간 천장).
- EP019 첫 줄 A패턴(시간+장소) → EP020 첫 줄 B패턴(거리+장소) → **EP021 첫 줄 A패턴 회귀**.
- 4화 연속 다른 개시 패턴 ★ 권장 vs 본 화 A패턴 회귀 = **소폭 결함** (단, EP019의 A패턴과 정확히 같지 않고 *새벽이 다시 들어옴*의 *2회차 시그니처*로 변주됨 — 의도된 시그니처 회귀로 정당화 가능).

**교정 제안**: **불필요**. 만일 패턴 변주를 강제하려면 첫 줄을 *"50보. 외경자 셋. 산비탈 위."* 식 거리+장소 B패턴 또는 *"가운데 자의 손목이 잘렸다."* 식 D패턴 동작 즉시 진입으로 변경 가능하나, 본 화의 새벽 회귀 톤은 **회귀물 시그니처**로 그대로 유지 권장.
