## ALIVE-1: 메아리 대화 (0건)

EP003은 **단독 1인칭 화**로 직접 대화 핑퐁이 발생하지 않는다. 정보 운반은 모두 *서류·계약·고용*으로 처리되어 메아리 대화 발생 구조 자체가 차단됨.

- **도윤 발화 3회** (L31·L69·L101): 모두 시그니처 #6 자조 변주 *"이게 미래 정보 가진 자의 N억이구나"* + 부모 떡밥 클리프. 자기 자신과의 핑퐁이라 메아리 X.
- **사설 조사관**: 직접 대사 0건. 산출물(스물두 장 서류 + 빨간 펜 밑줄 + 정보원 단편 세 줄)만 등장 → 메아리 대화 진단 비대상.
- **부모 회상**: 직접 대사 0건. 한경수·윤다인 이름 노출은 *서류 묶음 안의 명패 형식* (L87) → 메아리 진단 비대상.
- **신예 헌터·회사 직원·정보원**: 무명 NPC, 모두 *고용·계약·매수* 행위로만 처리. 발화 0건.
- **1회차 회상 B급 탱커** (L17): *"MK헌터 산 놈들 다 울었다더라"* — 1줄짜리 정보 통로. 도윤이 받는 줄·반박하는 줄 없음. 가십 → 행동(공매도)로 즉시 압축. **기능 대사가 아니라 단발 회상 디테일**로 작동 → 보존.

**진단**: ALIVE-1 위반 0건. 단독 1인칭 화 특성상 보이스 분리는 *도윤 ↔ 부모 회상 ↔ 사설 조사관 산출물* 세 톤으로 자동 분리됨 (각각 자조·과거 시제 풍경·검은 줄 검열문서).
