# Story Analysis Report — EP020

## SCENE (3건)

### 시공간 앵커 검증

EP020은 4개의 `---` 구분(상기 grep 결과 4건 매칭, 헤더 1 + 씬 전환 3)으로 4씬 구조:
1. **씬1 (L5~L26)**: "100보 거리. 산비탈 위 셋. 마을 한가운데 도윤과 유연." — 시공간 앵커 명시 (잿더미 직후 두 번째 마을 입구, 일몰 직전). 양호.
2. **씬2 (L29~L96)**: 시간 정지 발동 → 가운데 자 접근 → 자살 회귀. **앵커 누락**. "가운데 자의 손이 올라왔다"로 시작. 씬1과 같은 마을·같은 시각에서 연속되므로 큰 문제는 아니나, 시간 정지 시작점을 알리는 한 줄(예: "일몰 직전, 산비탈의 셋이 50보 거리까지 내려왔다.") 추가 권장. → MINOR.
3. **씬3 (L99~L141)**: "도윤이 눈을 떴다. 천장의 구멍 사이로 별 한 개" + L113 "전날 밤. 헛간. 같은 시각." — 시공간 앵커 명시. 양호.
4. **씬4 (L145~L243)**: 대화 시작 — 앵커 누락(헛간 안 같은 시각 연속이긴 하나 한 줄 없음). 다만 씬3 직후 연속이므로 묵인 가능. → MINOR.

### [ADJACENT] 인접 화 검증

- **EP019 결말 → EP020 도입 연속성**: EP019 마지막은 "가운데 자의 입가가 미세하게 올라갔다. … 웃었다. 그게 외경자의 첫 표정이었다. 그리고 도윤을 알아본 자의 표정이었다." (100보 거리, 도윤이 마을 한가운데서 한국어로 외친 직후, 가운데 자가 도윤을 알아봄). EP020은 "100보 거리. 산비탈 위 셋. 마을 한가운데 도윤과 유연." — **거리·위치 일치, 자연스러운 연속**. ★ 양호.

- **EP018→EP019→EP020 동선**: 헛간(EP018 야간) → 분지 서쪽 능선(EP019 낮) → 두 번째 마을 입구(EP019 일몰) → 마을 한가운데(EP020 일몰) → **회귀 후 EP018 야간 헛간으로 복귀**. 동선 일관.
- **L31 "도윤의 호흡이 한 박자 거꾸로 들이쉬어졌다"** — EP018 L95 / EP019 L87에 이미 같은 패턴 사용. 시그니처로 의도된 반복이지만 **3화 연속 동일 문장 반복은 UNIFORM 측면에서 신호**. → MAJOR (UNIFORM 절에서 다룸).

### TIMELINE 교차대조 결과

#### [TIMELINE-1] CRITICAL — "어제 헛간 새벽" vs 실제 깨어남 시점의 불일치

EP020 L101 묘사:
> "헛간이었다. **어제 잿더미 마을 입구의 헛간.** 가죽 외투가 도윤의 어깨에 덮여 있었다. 부싯돌이 짚 위에서 작은 불을 흔들고 있다."

EP020 L113:
> "**전날 밤. 헛간. 같은 시각.**"

EP020 L115:
> "회귀 지점은 — *마지막 안전 지점*이었다. … 무협 차원에서는 ***어제 헛간 새벽***이었다."

**모순 분석**:
- L101 묘사는 "부싯돌이 짚 위에서 작은 불을 흔들고 있다" + 유연이 짚을 옆에 놓는 자세 → **저녁/밤 분위기** (불 피우는 시점, 잠들기 전).
- L113은 명시적으로 **"전날 밤"**.
- L115는 메커니즘 정의 문장에서 **"어제 헛간 새벽"**.

→ 같은 회귀 지점을 EP020 안에서 **"전날 밤"(L113) vs "어제 헛간 새벽"(L115)** 두 표현으로 기술. 회귀 지점이 EP018 마지막(헛간에서 잠들기 전 야간)인지 EP019 새벽(잠 깨어남)인지 불일치.

**정본 후보 확인**:
- EP018 L237~L239: "두 사람은 가죽 외투를 깔고 누웠다. 헛간 천장의 구멍 사이로 별이 한 개 보였다. 산맥 깊은 곳의 별은 밝았다. … *EP018 끝. 다음: EP019 두 번째 관찰과 시간 왜곡.*"
- EP019 L5~L7: "새벽이 헛간 천장의 구멍 사이로 옅게 들어왔다. 유연이 먼저 일어났다. 부싯돌은 밤사이 꺼졌고 짚 위에 한 줌의 재가 남았다. 도윤이 눈을 떴을 때 유연은 이미 가죽 외투를 정리하고 짐 끈을 묶었다."

**EP019 L5의 깨어남 = 5일째 새벽**. 만약 회귀 지점이 "어제 헛간 새벽"(L115 정의)이라면 → EP019 L5의 새벽 = 회귀 지점이어야 함. 그러나 EP020 L101은 "부싯돌이 짚 위에서 작은 불을 흔들고 있다" + 유연이 "짚을 한 줌 부싯돌 옆에 놓고 있었다"(L103) — 이는 EP018 L215 "마른 짚을 한 줌 더 얹었다"와 같은 EP018 야간 묘사. EP019 L5 새벽 묘사는 "부싯돌은 밤사이 꺼졌고 짚 위에 한 줌의 재가 남았다" — 불이 꺼진 상태.

→ **L101 묘사 = EP018 야간 (불 켜진 시점) ≠ L115 정의(어제 헛간 새벽). 결론: 회귀 지점이 EP018 야간(잠들기 전)인데, 도윤이 그것을 "어제 헛간 새벽"으로 기술 = CRITICAL 불일치.**

→ **[TIMELINE] CRITICAL**. 교정 방향:
- 옵션 A: L101을 "어제 헛간 새벽" 분위기로 수정. 예: "부싯돌은 밤사이 꺼졌고 짚 위에 한 줌의 재가 남아 있었다. 새벽빛이 천장 구멍 사이로 옅게 들어왔다." (EP019 L5와 정합)
- 옵션 B: L115의 회귀 지점 정의를 "어제 헛간 — 잠들기 전 야간"으로 수정.
- **옵션 A 권장**(EP019 L5 정본 보존 + 도윤의 의식적 자각 = 새벽 깨어남으로 정합).

#### [TIMELINE-2] MAJOR — "모레 일몰"의 시간 기준 모호

EP020 L173:
> "내일 — 아니, 정확히는 **모레 일몰**. 마을 한가운데서. 외경자 가운데 자에게 가슴을 짚혔습니다."

회귀 후 도윤이 깨어난 시점이 **5일째 새벽**(EP019 L5와 동일 시점)이라면:
- 죽은 시점(첫 사이클) = 5일째 일몰 직전 = "오늘" 일몰(L171 유연 발화 "오늘 — 죽었던 거요?")
- 회귀 후 가는 길 = 5일째 정찰(EP019의 분지 서쪽 능선) → 5일째 일몰 두 번째 마을 → 죽음.
- **즉, 두 번째 시도의 마을 도달 = 5일째 일몰. 도윤이 깨어난 새벽 = 5일째 새벽. → "내일" 또는 "오늘"이 맞음.**

도윤이 "모레 일몰"이라고 한 것은:
- 5일째 새벽 깨어남 → 5일째 정찰(EP019) → 5일째 일몰 = "오늘 일몰" (24시간 안)
- 그런데 도윤은 "모레 일몰"로 표현 → 이틀 뒤 일몰.

→ **불일치 가능성**: 만약 회귀 지점이 EP018 야간(4일째 밤)이라면, 4일째 밤 깨어남 → 5일째 정찰 → 5일째 일몰 죽음 시점 = "내일 일몰"(~30시간 후)도 가능. 그러나 "모레 일몰"은 36~48시간 후 → 회귀 지점이 더 앞 시점(예: 3일째 새벽)이어야 성립.

→ **[TIMELINE] MAJOR**. 교정 방향:
- 옵션 A: "내일 일몰" 또는 "오늘 일몰"로 수정. 회귀 지점이 5일째 새벽이면 같은 날 일몰이 죽음 시점.
- 옵션 B: 회귀 지점을 4일째 밤 또는 3일째 새벽으로 수정 → 전체 타임라인 재정렬.
- **옵션 A 권장** (회귀 지점 = 5일째 새벽, 죽음 시점 = "오늘 일몰" 또는 "내일 일몰" 정합).

#### [TIMELINE-3] MINOR — EP019 "다섯 시진" vs EP020 "0.5시진" 단위 일관성

EP019 L13: "우리가 갈 길은 — 다섯 시진" (1시진 = 2시간, 5시진 = 10시간)
EP019 L167: "셋의 발걸음이 — 우리보다 0.3시진 빠릅니다"
EP020 L23: "외침이 마을을 깨운 0.5시진"

**0.3시진 = 36분 / 0.5시진 = 1시간**. 무협 차원에서 "0.3시진", "0.5시진" 같은 소수점 시진 단위 사용은 무협 차원 토착민(유연)의 어휘로는 어색. 도윤(외지 출신)이 노트에 적는 단위로는 정합. 다만 서술자 보이스에서 "0.5시진"(L23)으로 등장하는 것은 도윤 POV이므로 허용. → MINOR.

### TIMELINE 교차대조 종합

- [TIMELINE-1] CRITICAL: 회귀 지점 표기 불일치 (L101 야간 분위기 vs L113 "전날 밤" vs L115 "어제 헛간 새벽").
- [TIMELINE-2] MAJOR: "모레 일몰"의 시간 기준 모호 (회귀 지점과 죽음 시점의 일수 차).
- [TIMELINE-3] MINOR: 시진 단위 일관성.
