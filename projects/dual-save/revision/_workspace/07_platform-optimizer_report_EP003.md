## 1. HOOK 진단 (마지막 500자)

### 강도 평가: **★★★★ (4 — 떡밥형)** — 설계 정합

**현재 훅 본문 (마지막 500자 발췌)**:

```
"...부모님이 뭔가 큰 걸 쫓고 계셨네."

자조 한 줄이었다. 무게가 더 실리지 않은 톤이었다.
…
서류 스물두 장을 다시 정리해서 책상 서랍에 넣었다. 서랍을 닫았다.
잠금장치가 딸깍 소리를 냈다.

부서 분류 위 단계 정보를 사는 데 들 비용을 노트 한 구석에 적어두었다.
0이 한 자리 더 붙는 가격대였고, 그건 영약 단계 이후에 풀어야 할 카드였다.
…
강남 사무실 창밖에서 봤던 야경이 떠올랐다. 반지하 유리창 너머 보도블록 위
구두 소리가 지나갔다. 두 풍경 사이에 800억이 끼어 있다.

일단 영약부터였다.
```

### 훅 유형 분류 — **떡밥 던지기형** (R2 §4.2 / 플롯가이드 §7 정합)

세부설계 EP003 명시 클리프행어:
> *"...부모님이 뭔가 큰 걸 쫓고 계셨네"* + 자료 서랍 닫는 그림 + 강남 야경 (★ 강도 ★★★★) — 부모 미스터리 1단 떡밥

**플롯가이드 §7 떡밥형 예시문 합치도**:
> *"어머니 노트 한 장. 모래시계 같은 그림. 도윤은 노트를 닫았다."* (3화)

→ **"서랍을 닫았다 / 잠금장치가 딸깍 / 다음은 영약"** 의 3단 닫음 그림이 설계의 *서랍에 닫음*을 명확히 구현. 정합.

### 훅 강도 4 근거

- (a) **떡밥의 무게**: *베를린 외곽 / 미회수 / 부서 분류 위 단계는 0이 한 자리 더 / 비밀 부서 가족 실종 패턴* — 부모 미스터리의 4단 단서 노출. 25화·50화·100화·200화 단계적 노출의 1단계로 적정 (G5 페이싱 정합).
- (b) **닫음의 미학**: 서랍 닫음 → 잠금장치 딸깍 → "더 파고들지 않았다" → "일단 영약부터" 의 4단 차단 그림. 호기심 유보를 *유료 결제 동력*으로 전환.
- (c) **사이다 예고 결합**: 마지막 줄 *"일단 영약부터였다"* — EP004 영약 + A급 비트로 직결. R2 §4.2 사이다 예고형 결합.
- (d) **두 풍경 한 줄**: *"두 풍경 사이에 800억이 끼어 있다"* — 800억 누적 사이다의 정서적 정점을 한 줄로 압축. 글로벌 서술체 v2 § 미래 시점 예고와 동격의 *공간 예고* 효과.

### 결제 전환 포인트 검사

R2 §3.1 표준 유료화 타임라인 → **EP003은 무료 1~5화 도입부 후반부**. 유료 25화 전환점은 아니지만, **무료 5화 안에 첫 큰 사이다 + 다음 화 강한 추진력** (R2 §5 5화 비트표) 단계의 *직전 누적 화*. 800억 사이다 폭발 + 부모 단서로 떡밥 → EP004 영약 → EP005 자살 회귀 시그니처로 1차 사이클 클로즈하는 흐름의 핵심 전환점.

→ **강도 4 합당.** 강도 5는 EP005 자살 회귀 시그니처에 배정해야 하므로, EP003에 5를 배정하면 5화의 임팩트가 떨어진다. **현재 4가 최적.**

### 훅 직전 긴장감 해소 검사

- ✓ "더 파고들지 않았다. 이번 회차 우선순위는 강해짐이지 부모 추적이 아니다." — 일견 *해소처럼 보이는* 차단 멘트지만, 실제로는 *부서 분류 위 단계 + 0이 한 자리 더 + 비밀 부서 가족 실종 패턴*이라는 더 큰 떡밥을 던지고 닫는 구조. 차단 자체가 *지연된 호기심*을 강화. 결함 아님.

### "내일 ~한다" 행동 예고 검사

- ✓ 마지막 줄 *"일단 영약부터였다"* — 구체 시점·구체 행동 없는 **선언**으로 처리. 서스펜스 해소 없음. 적정.

- *"...부모님이 뭔가 큰 걸 쫓고 계셨네"* + *"자조 한 줄이었다. 무게가 더 실리지 않은 톤이었다"* — 자조 노출 + **자조 비중 절제 자체에 대한 메타 코멘트**가 한 줄. 톤 다이얼 정합. 비중도 5% 마지노선 내 (실측 자조 분량 ~3%, 세부설계 명시값 합치).

## 2. OPENING 진단 (첫 1,000자)

### 시작 방식: **시간 점프 + 사이다 누적 + 휴직 선언** (EP002 훅에 직답)

**오프닝 첫 4단락 (인용)**:

```
6월에 GATE 토큰이 800% 올랐다. 노트에 적어둔 대로였다. 8월에 분할 매도하고 나니
100억이 손에 들어왔다. 12억으로 시작해서 다섯 달 만이었다.

100억이 통장에 찍힌 그날, 도윤은 짐꾼 등록을 정지시켰다. 휴직 처리.
협회 시스템상 짐꾼 휴직은 별도 사유 없이 1년까지 가능했고, 1년이면 충분했다. …

자기단련은 멈추지 않았다. 새벽 러닝, 헬스장, 검도장. 매주 같은 루틴으로 돌렸다. …

2034년 9월 15일. 자취방.
```

### 평가 — 양호

**(a) 즉시 사건 투입**: ◎
- 첫 줄 *"6월에 GATE 토큰이 800% 올랐다"* — 배경 설명 0줄. 결과부터 박음. R2 §4.3 "1. 액션으로 시작" 정합.
- *"노트에 적어둔 대로였다"* — 1회차 정보 우위 시연이 한 줄 안에. R2 §4.3 "3. 고구마-사이다 밸런스 — 1화 안 작은 사이다 1번"의 3화판 응용.

**(b) 첫 3줄 안 행동**: ◎
- 1줄: 800% 폭등 결과 (사이다)
- 2줄: 8월 분할 매도 → 100억 (행동 + 결과)
- 3줄: 12억 → 100억 누적 사실 (1회차 정보 활용 함의)
- 단, 큰따옴표 대화는 없음 — *내적 행동·결과만으로* 사건 투입 성공. 결함 아님.

**(c) 직전 화(EP002) 훅에 대한 응답**: ◎ — **완벽 직답**
- EP002 훅: *"이게 미래 정보 가진 자의 12억이구나." / 다음은 800%였다.*
- EP003 첫 줄: *"6월에 GATE 토큰이 800% 올랐다."* → *"12억으로 시작해서 다섯 달 만이었다."*
- 12억 → 800% → 100억 의 누적 사이다가 첫 단락에 압축 체결. **R2 §4.3 5번 가독성 + §5 3화 비트(6개월 800억)** 절대값 정합.

**(d) 회상 시작 ("지난번에 ~") 결함**: ✓ 없음
- *"노트에 적어둔 대로였다"* 한 줄로 1회차 정보 우위만 신호 송출. 회상 늘어짐 0.

**(e) EP002 마지막 500자 → EP003 첫 1,000자 연결**: ◎
- EP002: 12억 / 다음은 800% (예고)
- EP003: 800% 사이다 → 100억 → 짐꾼 휴직 선언 (행동 전환)
- 휴직은 EP002 마지막의 *"라면을 끓이는 건 변하지 않았다. 아직 아니니까. 이건 시작이니까"* 의 *시작*을 행동으로 입증. 정서·시간선·행동 일관.

### 평가: **양호 — 사이다 직답 우수**

EP002 훅 강도 4(사이다 예고형) → EP003 오프닝 사이다 결제. 결함 없음. 교정 불요.
