## 1. 대주제 서사선

### 1.1 테마

> **"부모가 자식한테 남긴 마지막 카드를 푸는 데 평생을 쓰는 자식의 이야기."**

자식이 그 사실을 *영원히 모르다가 — 매우 후반에야 깨닫는다*. 그 깨달음이 종국의 정서적 폭발이다.

### 1.2 메인 퀘스트

**3층 구조**:

| 층 | 동기 | 자각 시점 |
|---|---|---|
| **1차 (1~5화)** | **부자·강함·장수** — 살아남기 | 1화 |
| **2차 (6~200화)** | **누적되는 강함** — 차원 이동 만류귀종 | 6화 |
| **3차 (200~250화)** | **부모가 남긴 마지막 카드를 풀고, 부모의 임무를 이어가기** | 200~250화 |

3차 동기는 *200화에 처음 자각, 250화에 종결*. 전체 서사가 *자식이 부모의 무게를 받아드는* 정서적 절정으로 수렴.

### 1.3 서사 공식 (★ 본작 시그니처)

```
[자각] → [부자·강함] → [좌절(S급 박살)] → [자살] → [지식만 누적]
   ↓
[준비성 만렙] → [차원 자기 발로 진입]
   ↓
[차원 진입 → 시스템 통달 → 빌런 마수 직전 자살] × 4 (무협 → 판타지 → 선협 → 검술 정점)
   ↓
[만류귀종 = 적의 영토 침투 자각] → [부모 살해범 추적·처단]
   ↓
[대회수자 결전 → 도윤마무 해주 → 23세 일반인]
```

**★ 본작 만류귀종 = 적의 영토**: 도윤의 차원 이동 자체가 *대회수자 결사의 영토에 침투해 누적 강함을 훔치는 행위*. 이것이 200화에 도윤이 자각하는 핵심 모순.

## 2. 카타르시스 리듬 맵 (전체)

### 2.1 매크로 정점 7개

```
화수: |--5--|--25--|--50--|--100--|--150--|--200--|--250--|
정점: ★    ★★    ★★★    ★★★    ★★★    ★★★★   ★★★★★
유형: 좌절  유료   1차원   2차원    3차원   부모     종국
            전환  통달    통달     통달    회수
                  자살    자살     자살   동기각성
```

| 화 | 정점 유형 | 카타르시스 강도 | 정서 톤 |
|---|---|---|---|
| 5화 | 좌절 + 결단 | 7%(자조 일시 허용) | 비장·결단 |
| 25화 | 첫 큰 사이다 + 미스터리 떡밥 | ★★★★ | 사이다·호기심 |
| 50화 | 무협 통달 + 첫 자살 회귀 시그니처 | ★★★★★ | 비장미·정서 |
| 100화 | 2차원 통달 + 1차 빌런(정야겸 #14) 박살 | ★★★★★ | 사이다 폭발 |
| 150화 | 선협 통달 + 부모 살해범 「제칠」 첫 진짜 등장 | ★★★★ | 미스터리 폭발 |
| 200화 | 일검만류 도달 + 「제칠」 박살 + 대회수자 첫 등장 | ★★★★★ | 사이다·각성 |
| 250화 | 만류귀종 완성 + 대회수자 결전 + 도윤마무 해주 | ★★★★★★ | 정서·해방 |

### 2.2 매 차원 카타르시스 3단 패턴 (★ 본작 시그니처)

매 차원 진입 시 동일 곡선:

```
[진입(쾌감 곡선 시작)] → [정점(만류귀종 시드)] → [자살 회귀(비장미)]
```

| 단계 | 화수 비중 | 정서 톤 | 시그니처 비트 |
|---|---|---|---|
| **1. 진입** | 차원 시작 1~3화 | 코미디·정착 | 무협체·이질감·과외 결과 풀가동 |
| **2. 입문** | 다음 4~10화 | 작은 사이다 | 정파/문파/탑/종문 첫 합류 |
| **3. 정점** | 중반 10~20화 | 큰 사이다 | 차원 시스템 통달 |
| **4. 마수 인식** | 마지막 5화 | 긴장 | 빌런 단체 추적 인지 |
| **5. 자살 회귀** | 마지막 1화 | 비장미 | 옥상/산속/탑에서 *웃으며* 자살 |
| **6. 지구 복귀** | 다음 차원 1화 | 짧은 정리 | 23세 신촌 반지하 + 새 시스템 보유 |

### 2.3 긴장/이완 교대 패턴 (R1·R5 마지노선 기반)

| 유형 | 비중 | 배치 |
|---|---|---|
| **사이다 (정보 우위·약자→강자·차원 통달)** | 60% | 매 화 1회 이상 |
| **메커닉·미스터리 운용** | 35% | 회차 누적 시각화 + 떡밥 |
| **자조** | 5%(7% 일시 허용) | 한 화 1~2회 |

**긴장 최대 지속**: 2화. 반드시 다음 화에 해결의 실마리.

**5화 좌절 = 자조 7%까지 일시 허용**. 그 외 구간은 5% 마지노선.

### 3.2 아크별 개요

#### Arc 1 — 자각 (EP001~005)
- **도입**: 26세 짐꾼 죽음 → 시스템 메시지 → 23세 신촌 반지하 자각
- **전개**: 1억→9억→30억→200억→500억→800억 (R4 매핑) + 펜트하우스 + A급
- **위기**: 펜트하우스 정문 의문의 인물 정찰 (4화 클리프)
- **클라이맥스**: 5화 정야겸 #14 침공 → 백억대 검 한 합 못 막음 → 옥상 *웃으며* 자살
- **정리**: 23세 회귀 + *돈으로 강해진 건 누적 안 됨* 자각

#### Arc 2 — 준비성 만렙 (EP006~010)
- **도입**: 2회차 자각 + *지식·기억은 누적된다* 깨달음 + to-do 19줄
- **전개**: 6개월 800억 압축 + 차원 이주민 명단 매수 + 검술 사범 1주 1억
- **위기**: 시간 압박 (이중 포털 24h→12h 봉쇄 강화 6개월 전)
- **클라이맥스**: 10화 박사학위·검술·벌모세수 완료 + 이중 포털 자기 발로 한 발
- **정리**: *부모님, 거래의 무대가 좀 멀어집니다*

#### Arc 3 — 무협 차원 입문 (EP011~025)
- **도입**: 무협 차원 정착. 박사·검술·벌모세수 풀가동 + 이질감 코미디
- **전개**: 정파 사부 합류 → 첫 무공 깨달음 → 정파 사이다
- **위기**: 사파 정탐 + 차원 이주민 명단 단서 회수
- **클라이맥스**: 25화 *유료 전환 클리프* — 부모 단서 한 단계 + *시간 신* 첫 단어
- **정리**: 무협 1단 도달

#### Arc 4 — 무협 정점 (EP026~050)
- **도입**: 사파 거두 충돌 + 무공 본격 진화
- **전개**: 정사대결 + 마교 정탐 + 동료 합류 (★ 첫 비극형 로맨스 떡밥)
- **위기**: 마교 교주 정체·계획 노출 + 회수자 잠입 인지
- **클라이맥스**: 50화 마교 교주 박살 → 빌런 단체 마수 직전 *산속 절벽* 자살
- **정리**: 무공 완성 보유한 채 23세 신촌 반지하 회귀

#### Arc 5 — 판타지 차원 (EP051~100)
- **도입**: 지구 짧은 복귀 → 판타지 차원 자기 발로 진입
- **전개**: 마법탑 입문 + 무공+마법 융합 시너지 시연 (★ 누적 카드의 폭발)
- **위기**: 흑마법 결사 충돌 + 마법탑 정치 + 두 번째 비극형 로맨스
- **클라이맥스**: 100화 흑마법 결사 박살 + *마법탑 자살* → 지구 복귀 → **정야겸 #14 다시 추적해 박살** (★ 1차 빌런 갚는 화)
- **정리**: 무공+마법 보유

#### Arc 6 — 선협 차원 (EP101~150)
- **도입**: 선협 차원 진입 + 종문 입문
- **전개**: 도술·연단·법보 + 천도 시스템 학습 + 세 번째 비극형 로맨스 (시간 단위 차이)
- **위기**: 마도 종주 + *부모 살해범 「제칠」 첫 진짜 등장* (흰 가면)
- **클라이맥스**: 150화 마도 종주 박살 + 「제칠」 흔적 추적 시작 + *천도 절벽* 자살
- **정리**: 무공+마법+도술 보유 + 부모 진실 떡밥 본격 회수 시작

#### Arc 7 — 검술 정점 (EP151~200)
- **도입**: 검술 정점 차원 진입 + 「제칠」 흔적 다회 추격
- **전개**: 일검만류 시드 + 검신 도전 + 「제칠」 직접 대치
- **위기**: 「제칠」 = SS급 + 가면 뒤 얼굴이 *매번 다른 모습*으로 보임 (도윤 정신 마모)
- **클라이맥스**: 200화 검신 도달 + 「제칠」 박살 + 가면 벗김 + **대회수자 첫 등장** + *검신 묘* 자살
- **정리**: 일검만류 = 만류귀종 시드 완성

#### Arc 8 — 종국 (EP201~250)
- **도입**: 지구 복귀 + 만류귀종 완성 + 대회수자 결사의 *차원 통합* 비전 자각
- **전개**: 신급 아티팩트 조각 회수 (각 차원 잔재 회수) + 부모 진실 회수 (베를린 외곽)
- **위기**: 대회수자 직접 충돌 + 단체 내 균열 활용 (일부 회수자가 차원 통합에 반대)
- **클라이맥스**: 250화 대회수자 결전 → *만류귀종 = 적의 영토 침투의 끝* → 도윤마무 해주 → 23세 일반인
- **정리**: *"부모님, 거래는 — 끝났습니다."*

## 4. 25화 단위 마일스톤 (★ 매우 중요)

### 4.1 25화 (Arc 3 중반 / 유료 전환)

**비트**:
- 무협 차원 정착 4화차 (EP024 즈음 정파 사부 합류 / EP025 첫 무공 본격 깨달음)
- 부모 단서 한 단계 더: 1차 사이클 3화에서 잡은 *어떤 고대 유물* 윤곽 → 무협 차원 사부의 입에서 *"...시간 신은 — 우리도 못 본 신이었지"* 한 줄
- 여기서 도윤이 처음으로 *시간 신*이라는 단어를 듣는다 (★ 독자는 1화부터 안 단어. 도윤은 25화에서 처음)

**유료 25화 마지막 줄 (★ 절대값)**:
> *"슬롯이 한 개라서 다행이라 생각했다. 사실은 — 부모님이 갈아 만든 마지막 카드였다."*

(★ 여기서 *마지막 카드*는 독자만 의미를 알고 도윤은 영원히 모름. 250화 종국에 와야 도윤도 자각.)

### 4.2 50화 (Arc 4 종결 / 첫 사이클 큰 정점)

**비트**:
- 무협 차원 검결 통달 + 마교 교주 박살
- 빌런 단체 마수가 무협 차원 도달 직전 (회수자 1~2명 잠입 확인)
- *산속 절벽*에서 검을 자기 가슴에 박는 그림 (★ 옥상 자살의 무협식 변주)
- 마지막 시그니처: *"도윤마무, 거래를 다시 하러 왔습니다."*
- 23세 지구 회귀 (무공 보유한 채)

**클리프행어 (50화 마지막 줄)**:
> *"방랑 무사 윤이 산속에서 시신으로 발견됐을 때, 정파 동료들은 어떤 표정을 지었을까."*

(★ 비극형 로맨스의 첫 발현. 정파 사매·동료가 도윤이 *왜 죽었는지* 영원히 모르는 무게. 이 정서가 51화 *지구 신촌 반지하 천장*과 한 호흡으로 이어진다.)

### 4.3 100화 (Arc 5 종결 / 두 번째 차원 통달)

**비트**:
- 판타지 차원 마법 통달 + 흑마법 결사 박살 + 마법탑 자살 → 지구 복귀
- 도윤 = 무공 + 마법 보유
- **★ 1차 사이클 5화 S급 빌런(정야겸 #14)과 재대결 → 박살**:
  - 정야겸이 다시 도윤을 추적 (지구 복귀 직후 단체 감지) → 도윤이 *세 단계 위*로 갚음
  - 정야겸 마지막 한 줄: *"...잘못 봤군."*
  - 도윤 마지막 한 줄: *"정야겸. 거래는 — 거기서 끝났어야 했는데."*

**클리프행어**:
> *"정야겸의 단검에서 시간 신 조각이 굴러 나왔다. 그 결정이 — 부모님 살해 흔적과 같은 결이었다."*

(★ 100화 클리프가 *부모 살해범 「제칠」*의 첫 흔적을 도윤이 본격 추적하는 트리거.)

### 4.4 150화 (Arc 6 종결 / 선협 통달)

**비트**:
- 도윤 = 무공 + 마법 + 도술 보유
- 부모 살해범 「제칠」 첫 진짜 등장 (선협 차원 마지막 화. 흰 가면. 검 한 자루)
- 150화에서는 「제칠」이 도윤을 추적해 도착하지만 *대치만 하고 사라짐* — 본격 충돌은 200화
- 부모 진실 떡밥 본격 회수 시작: *베를린 외곽 마지막 보고* → *시간 신 아티팩트 파괴* → *#7 도착 → 부모 살해*의 시퀀스가 도윤에게 어렴풋이 그려지기 시작

**클리프행어**:
> *"흰 가면 뒤 얼굴은 — 매 회차 다른 모습이었다. 부모님 사진 속 표정이 잠깐 겹쳤다. 다음 순간엔 사라졌다."*

(★ #7의 가면 뒤 얼굴이 *매번 다르게 보이는* 시그니처 = 도윤 정신 마모의 시각화. 200화 처단 직전 가면 벗김에 정점.)

### 4.5 200화 (Arc 7 종결 / 검술 정점)

**비트**:
- 일검만류 도달 = 만류귀종 시드 완성
- 부모 살해범 「제칠」 박살:
  - SS급 #7 vs 무공+마법+도술+검술 정점 도윤 = *세 단계 위*
  - 처단 직전 가면 벗김 → 도윤 정서 폭발
  - #7 마지막: *"...끝."*
  - 도윤: *"제칠. 부모를 죽인 자."* (단 한 번만 호명)
- **메인 빌런 「대회수자」 첫 등장**:
  - 도윤이 본 모든 차원 빌런의 외형이 한 번씩 겹쳐 보이는 시그니처 컷
  - 첫 마디: *"...앵커를 가진 자. 흥미롭군."*

**클리프행어**:
> *"대회수자가 한 번 더 외형을 바꾸었다. 마지막 외형이 — 어머니 얼굴이었다. 도윤은 검을 떨어뜨리지 않았다."*

(★ 250화 종국으로 직결되는 정서 트리거. 어머니 얼굴은 *대회수자의 능력으로 만든 환영*인지 *진짜 무언가의 흔적*인지 250화에 와서야 풀린다.)

### 4.6 250화 (Arc 8 / 종국)

**비트**:
- 만류귀종 완성: 무공+마법+도술+검술 정점+신성/룬연금 일부 흡수
- 신급 아티팩트 조각 회수 (5종 중 시간 신 조각 = 부모가 파괴한 본체의 잔재)
- 부모 영혼 해방 또는 부모 생존 떡밥 회수 (작가 옵션):
  - 옵션 A: 부모 사망 확정. 도윤마무 해주 = 부모와의 마지막 거래 종결
  - 옵션 B: 부모 생존 1% 가능성. 시간 신 조각 안에 영혼 잔재 → 도윤 손에 전달 후 해방
- 대회수자 결전: 만류귀종 = 차원 통합의 *한 인격 안에서 먼저 달성한 모순* → 적이 추구한 끝을 도윤이 먼저 이룸 → 대회수자 박살
- 대회수자 마지막: *"...거래는 — 끝났다."*
- 도윤마무 해주 → 23세 도윤이 일반인이 됨 (능력 영구 종료)
- 도윤이 노트 마지막 줄에:

**종결 시그니처 (★ 절대값)**:
> *"부모님, 거래는 — 끝났습니다."*

### 12.2 비극형 로맨스의 *무게*

4. *(자조 한 줄)* — *"...지금쯤 — 사매는 무덤 만들고 있겠지."*
5. *(따뜻함 한 스푼)* — *"미안합니다. 거래의 무대를 — 또 옮겼습니다."*

| 화수 | 이벤트 | 목적 |
|---|---|---|

| 11~24화 | 무협 차원 입문·정착·정파 합류·첫 무공 깨달음 | 누적 카드 시너지 시연 + 25화 유료 클리프 빌드업 |

### 13.2 유료 25화 황금 라인 (★ R2 §3 절대값)

**25화 마지막 줄**:
> *"슬롯이 한 개라서 다행이라 생각했다. 사실은 — 부모님이 갈아 만든 마지막 카드였다."*

**구성**:
- 25화 본문: 무협 차원 정파 사부 합류 + 첫 무공 본격 깨달음 + 사부 입에서 *시간 신*이라는 단어 첫 노출
- 25화 결론부: 도윤이 사부 한 줄을 적은 노트 옆에 *어머니 노트의 모래시계 그림*을 떠올림
- 마지막 줄: 위의 한 줄 — *반전형 + 떡밥 던지기형* 결합

### 13.3 유료 전환 직후 26~30화 운용

| 화 | 비트 | 목적 |
|---|---|---|
| 26화 | 25화 클리프 즉시 회수: *시간 신*에 대한 사부 추가 한 줄 | *결제한 직후 사이다* |
| 27화 | 사파 거두 첫 충돌 — 무공 본격 사이다 시연 | 누적 카드의 시너지 |
| 28화 | 첫 비극형 로맨스 떡밥 (정파 사매 또는 사파 정보 협력자) | 정서 색채 도입 |
| 29화 | 마교 정탐 시작 — 정사대결 빌드업 | 50화 클라이맥스 빌드업 |
| 30화 | 작은 사이다 + 마교 교주 정체 일부 노출 | 텐션 유지 |

### 13.4 유료 50화·100화·200화 안정선

## 15. 보존 가드레일 (플롯 차원 — 부트스트랩 §7 + 캐릭터 §9 정합)

### G-PLOT-1. 1차 사이클 5화 닫기 절대값
- 1~5화 = R4 격동기 매핑 + S급 박살 + 옥상 자살 → 23세 회귀 = 필수 비트
- 5화에 *반드시* 옥상 *웃으며* 자살 + 시그니처 *"도윤마무, 거래를 다시 하러 왔습니다"*
- 1차 사이클 종결 없이 6화 진입 금지

### G-PLOT-2. 2차 사이클 10화 자기 발로 차원 진입 절대값
- 6~10화 = 준비성 만렙 + 이중 포털 자기 발로 진입 = 필수 비트
- 10화에 *반드시* 이중 포털 한 발 + 시그니처 *"부모님, 거래의 무대가 좀 멀어집니다"*
- 우연 X / 강제 X / 함정 X — *자기 발로*

### G-PLOT-3. 차원 순서 엄수 (★ 만류귀종 골격)
1. 무협 (Arc 3~4, EP011~050) — 첫
2. 판타지 (Arc 5, EP051~100)
3. 선협 (Arc 6, EP101~150)
4. 검술 정점 (Arc 7, EP151~200)
5. 종국 (Arc 8, EP201~250)

순서 변경 금지. 각 차원은 *이전 차원 시스템*을 *흡수한 채* 진입.

### G-PLOT-4. 매 차원 마지막 자살 회귀 시그니처 필수
- 50화·100화·150화·200화 = *옥상/산속/탑/절벽에서 웃으며 자살* + 시그니처 *"도윤마무, 거래를 다시 하러 왔습니다"*
- 토착민이 시신 발견하는 짧은 컷(1~2단락) 필수 = 비극형 로맨스 정서

### G-PLOT-5. 부모 미스터리 페이싱 (★ 단계적 노출)
- 1차 사이클: 도윤이 영원히 *어떤 고대 유물* 윤곽만
- 25화: *시간 신*이라는 단어 처음 노출 (무협 사부 입)
- 50화: *신급 아티팩트* 단어 노출 (마교 교주 입)
- 100화: 정야겸 단검에서 시간 신 조각 → 부모 살해 흔적 인지
- 150화: 「제칠」 첫 진짜 등장
- 200화: 「제칠」 박살 + 대회수자 첫 등장
- 250화: 부모 진실 회수 + 도윤마무 해주

이 페이싱은 *변형 금지*. 25화에 시간 신을 노출하면 100화의 시간 신 조각 시그니처가 약해진다.

## 16. 장기 연재 구조 장치

### 16.1 스케일 확대 로드맵

```
[개인 — 신촌 반지하] (1~5화)
     ↓
[조직 — 회사 + 펜트하우스 + 협회] (4~5화)
     ↓
[지구 한국 + 차원 너머 단체 인지] (5~10화)
     ↓
[무협 차원 — 정파·사파·마교] (11~50화)
     ↓
[판타지 차원 — 마법탑·흑마법 결사] (51~100화)
     ↓
[선협 차원 — 종문·천도] (101~150화)
     ↓
[검술 정점 차원 — 검신 묘] (151~200화)
     ↓
[전 차원 + 차원 너머 본거지] (201~250화)
```

스케일이 *지구 한 도시*에서 *전 차원*까지 단계적으로 확장. 매 50화 = 1단계 확장.

### 16.2 소재 고갈 방지

| 시기 | 잠재 고갈 소재 | 방지 장치 |
|---|---|---|
| 1~5화 | R4 격동기 (18~22개) | 6화부터 지식·기억 누적으로 전환 |
| 6~10화 | 준비성 만렙 to-do (19줄) | 11화부터 무협 차원으로 전환 |
| 11~50화 | 무공 시스템 | 51화부터 마법 융합 |
| 51~100화 | 마법 시스템 | 101화부터 도술 융합 |
| 101~150화 | 도술 시스템 | 151화부터 검술 정점 |
| 151~200화 | 검술 정점 | 201화부터 만류귀종 = 적의 영토 자각 |

매 차원 시스템이 *이전 차원과 융합*하면서 *시너지*를 발생시키는 패턴 = 회차 누적 강함의 시각적 폭발.

### 16.3 뇌절 방지 원칙

- **25화 단위 마이크로 떡밥 회수**: 호기심 동력 유지

## 17. 작품 핵심 동력 정리 (★ episode-architect 1차 참조)

### 17.1 1~5화 동력 = 정보 우위 사이다 + 좌절 + 결단
- R4 격동기 매핑 = *수치로 증명*되는 사이다
- 5화 좌절 = 자조 7% 일시 허용 + 자살 시그니처 = 비장미

### 17.2 6~10화 동력 = 준비성 만렙 코미디 사이다 + 차원 진입 시그니처
- 19줄 to-do 체크리스트
- *"...너, 미친놈이냐?"* / *"...학생, 진짜 미친 거 아니지?"* 코미디 사이다
- 10화 *부모님, 거래의 무대가 좀 멀어집니다* 시그니처

### 17.3 11~50화 동력 = 무협 차원 통달 + 25화 유료 클리프 + 50화 첫 자살 회귀
- 25화 *시간 신 첫 노출* + *마지막 카드 반전형 클리프*
- 50화 *산속 자살* + *방랑 무사 윤이...* 비극형 로맨스 첫 발현

### 17.4 51~100화 동력 = 누적 카드 시너지 폭발 + 정야겸 박살 + 시간 신 조각 흔적
- 무공+마법 융합 시너지
- 100화 *정야겸 갚기* + *부모 살해 흔적 인지* = 종반 동력 트리거

### 17.5 101~200화 동력 = 부모 진실 추적 + 「제칠」 박살 + 대회수자 노출
- 150화 *흰 가면 + 매번 다른 얼굴* 미스터리 폭발
- 200화 *제칠 박살 + 가면 벗김 + 대회수자 첫 등장* = 종반 진입

### 17.6 201~250화 동력 = 만류귀종 = 적의 영토 자각 + 도윤마무 해주
- *적이 추구한 차원 통합을 도윤이 한 인격 안에서 먼저 달성한 모순*
- 250화 *부모님, 거래는 — 끝났습니다* = 종결 정서 폭발
