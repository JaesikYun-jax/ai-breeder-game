## 2. 한 줄 컨셉 + 로그라인 + 셀링포인트

### 한 줄 컨셉
부모가 죽기 전 시간 신 아티팩트를 부수면서 자식에게 옮긴 *마지막 카드* 「도윤마무」 저주. 짐꾼으로 26세에 죽은 한도윤은 23세 신촌 반지하로 회귀해, 부자·강함·장수를 추구하다 S급한테 박살난 후 **누적되는 강함은 지식뿐**임을 깨닫고 차원을 옮겨 다니며 무협·판타지·선협을 통달한다.

### 60자 로그라인
> *짐꾼으로 죽은 23세 회귀자가 차원을 옮겨 다니며 누적된 지식만으로 인류 구원자급에 도달, 부모가 남긴 저주를 푸는 만류귀종 회귀물.*

### ★ 한 줄 셀링포인트
> ★ **"부모님이 죽기 전에 신급 아티팩트의 시간 회귀 저주를 자식한테 걸어버렸다. 이름을 도윤마무로 — 자식이 알아채도록."**

### 부수 셀링포인트 카피 3개

1. **"돈으로 강해진 줄 알았다. SS급도 아니고 S급한테 박살나면서 깨달았다. 누적되는 강함이 필요하다고."**
2. **"이중 포털은 못 돌아오는 길이라고? 나는 죽으면 23세 신촌 반지하로 돌아간다. 그러니까 — 일방통행이 아니다."**
3. **"박사학위 받으면서 무공 익히는 사람은, 전 우주에 나밖에 없을 듯."**

## 3. 장르 정체성 (genre-dna-framework 기반 / R1·R5 적용)

### 3.1 회귀물 5대 코어 (R1) 적용 방식

R1이 정의한 회귀물 독자 5대 만족 코어를 본작은 다음과 같이 충족한다.

| 코어 | 본작 적용 |
|---|---|
| **정보 우위** | R4 사건 연표 격동기(2033~2036) 18~22개 이벤트를 도윤이 분 단위로 보유 → 코인·신약·합병 차익 거래로 1차 사이클 800억 → 수천억대 |
| **약자→강자 역전** | F급 미각성 짐꾼(평균 수명 1년 미만) → 1차 사이클 A급 → 차원 통달 후 인류 구원자급 |
| **복수** | 부모 살해범(차원 너머 회수자 #7) — 매우 후반(50~70화 이후) 떡밥 회수 |
| **재건 서사** | 표준 회귀물의 "문파/가문 재건" 변주 → **"부모가 남긴 저주를 풀고 부모의 임무를 이어가는 재건"** |
| **개연성 있는 먼치킨** | 누적되는 건 *지식·기억*뿐. 회차마다 영약·자산·신체는 사라짐 → 먼치킨이지만 *매번 처음부터 다시 만들어야 하는* 먼치킨 |

### 3.2 헌터물 정통 사이다 코드 (R3 적용)

R3의 한국 헌터 협회 3층 구조와 짐꾼 클리셰를 그대로 흡수해 *시장 친숙성 7*을 확보한다.

| 흡수 코드 | 본작 적용 |
|---|---|
| F급/짐꾼 멸칭 | 도윤 1회차 정체성. 자조 시그니처: *"짐꾼으로 26세까지 산 게 길었던 거다"* |
| 게이트·등급(F~SS) | 1차 사이클 신체 등급 진화의 무대 (미각성→A급) |
| 영약·아티팩트 시장 | 1~4화 주요 매수 채널 (5억/주 영약·백억대 검) |
| 길드·신예 후원 | 1차 사이클 신예 후원 시스템 (5화 호위 헌터의 정체) |
| 협회 비공개 부서 | 차원 이주민 관리·신급 아티팩트 추적·이중 포털 봉쇄 3종 (★ 본작 시그니처) |

### 3.3 차원 이동 만류귀종의 흡수 코드 (R5 적용)

R5가 검증한 4개 작품의 정수만 5% 이하씩 흡수, 본작 메커닉이 60%+를 차지한다.

| 작품 | 흡수 코드 | 본작 변주 |
|---|---|---|
| 화산귀환 | 정통 회귀무협 + 단계적 누적 성장 | 누적은 회차 사이 *지식·기억*만, *영약·신체는 매번 리셋* |
| 전지적 독자 시점 | 정보 우위 + 차원 너머 빌런 단체 | 도윤은 한 인격, 차원 이동으로 정보 누적 |
| 세이브 로드 라이프 | 능동적 회귀 능력 | 도윤은 *자살로 능동 발동* (SSS급 죽어야 사는 헌터 코드 결합) |
| 천마육성 | 다중 차원·시스템 통달 | 본작 만류귀종의 직접 선조 |
| 1억년 버튼 (라노벨) | 무한 시간 후 귀환 | 무한 회차 누적 정신력 코드만 흡수 |
| 해골병사는 던전을 지키지 못했다 | 장기 리셋 다회차 누적 | 누적 지식이 결국 게임 시스템 룰을 깨는 코드 |
| 게임 폐인의 리셋 라이프 | 회귀 후 권태 코드 | 자조 5~7%로 압축 흡수 |
| 회귀수선전 | 회귀 = 저주 톤 | "도윤마무"라는 *부모가 건 저주*로 직설화 |

### 3.4 자조 5~7% 마지노선 운용 방식 (★ 매우 중요)

R1·R5가 공통적으로 경고하는 *자조 톤이 사이다 부재의 변명이 되는 위험*을 다음 규칙으로 통제한다.

| 규칙 | 운용 |
|---|---|
| **분량 비율** | 자조 5~7% / 사이다 60% / 메커닉·미스터리 운용 35% |
| **5화 좌절 시 일시 7%까지 허용** | 그 외에는 5% 마지노선 |
| **자조→행동 1:5 패턴** | 자조 한 줄 등장 시 그 직후 5줄 이내 행동·결정·사이다로 전환 |
| **시그니처 자조 1화 1번** | 한 화당 도윤 시그니처 자조 대사 1번 (예: *"부모님, 거래의 무대가 좀 멀어집니다"*) |
| **자조 = 결단의 무게를 농담으로 덮기** | 영웅적 결의 톤 절대 금지. 그러나 행동은 영웅 |
| **자조 → 사이다 결합 강제** | 자조 직후 반드시 *준비된 카드* 노출 (박사학위·매수 명단·매수 자금 등) |

## 4. 세계관 (R3·R6 직접 이식)

### 4.1 한국 헌터 협회 3층 구조 (R3 §1.1)

```
[청와대 헌터 정책 수석실]                ← 대통령 직속
        ↓
[국가 헌터 협회] (산업통상자원부 외청)
   ├─ 등급 검사부 / 게이트 관리부 / 헌터 정책부
   ├─ 영약·아티팩트 거래 감독부
   ├─ 협회 직속 레이드팀 (S급 비공개)
   └─ 7개 지부 (서울·부산·인천·광주·대전·대구·광역)
        ↓
[헌터 길드 / 헌터 회사]
   ├─ 명문가 길드 (검가·마법명가 등)
   ├─ 신흥 대기업 길드 (삼성 헌터·현대 헌터 등)
   └─ 보조 회사 (★ 도윤 1차 사이클 모델)
        ↓
[★ 비공개·정부 직속 부서 3종]
   ├─ 차원 이주민 관리 부서 (★ 도윤 7화 명단 매수)
   ├─ 신급 아티팩트 추적팀 (★ 부모 한경수·윤다인 소속)
   └─ 이중 포털 추적반 (★ 도윤 10화 봉쇄 우회)
```

### 4.2 헌터 등급 시스템 (R3 §2.1)

| 등급 | 정의 | 처우 | 측정 |
|---|---|---|---|
| 미각성 | 마력 0 / 짐꾼·후방·보조직만 | 일당 30~50만원, 4대보험 X | 측정 불가 |
| F급 | 측정 최저 임계 | 월 200~300만원 | 마력 측정기 |
| E~D급 | 일반 헌터 데뷔 | 월 500~800만원 | 측정기 |
| C~B급 | 본격 헌터 / 대기업 임원급 수입 | 연 1~10억 | 측정기 |
| A급 | 상급 헌터 / 게이트 보스 단독 처리 | 연 30~100억+ | 측정기 한계 근방 |
| S급 | 국가 전략 자원 | 연 수백억~조 단위 | **측정 불능 + 자기 신고** |
| SS급 | 신화급 / 한국 0~3명 추정 | 비공개 | **측정 불능** |

**도윤의 등급 곡선**:
- 1회차 18~26세: **미각성** → 짐꾼
- 2회차(1차 사이클) 23~26세: 미각성 → 영약·아티팩트로 **A급** → 5화에 S급한테 박살
- 3회차(2차 사이클): 미각성 (지구) → 무협 차원에서 *무공 보유*. 회귀해도 *지식·기억은 영속*

### 4.3 영약·아티팩트 시장 (R3 §5)

| 카테고리 | 가격대 | 본작 활용 |
|---|---|---|
| 하급~중급 영약 | 100만~5억 | 도윤 회피 (위조 위험) |
| 상급 영약 | 5억~50억 | 1차 사이클 도윤 매수 (5억/주 페이스, 1년 반) |
| 신급 영약 | 50억~수천억 | 협회 경매 1년 1~2회. 도윤은 1차 사이클에 못 얻음 |
| 백억대 명검 | 100~500억 | 도윤 1차 사이클 4화 매수 → 5화에 S급 검에 부러짐 |
| **신급 아티팩트** | **천억+** | ★ 빌런 단체 회수 대상. 시간 신 아티팩트 = 부모가 발견·파괴 |
| **블랙마켓** | 변동 | 차원 이주민 비급·연단법·등급 매수 (★ 도윤 7~9화 통로) |

### 4.4 게이트·이중 포털 룰 (R3 §7)

**일반 게이트**: F~SS급. 권장 인원·보상 표준화. 도윤 1차 사이클은 짐꾼 등록 외 거의 무관.

**★ 이중 포털 (★ 본작 시그니처 룰)**:
- 정의: 게이트 내부에 또 하나의 포털이 생기는 1회성 균열
- 발생 빈도: 한국 기준 연 4~6회 추정 (협회 비공개)
- 진입 결과: **영구 차원 추방** (무협/판타지/선협/룬연금/검술 정점 등 무작위)
- 협회 정책: 24시간 안에 위치 봉쇄 + 진입 시도자 사살 권한
- **2036-04 봉쇄 강화 (24h → 12h)** ★ 도윤 10화 진입은 봉쇄 강화 *직전 6개월*
- ★ **도윤만의 거꾸로 활용**: 일반인은 *영구 추방 = 비극*. 도윤은 *죽으면 23세 신촌 반지하로 회귀* → *양방향 통행*

### 4.5 차원 이주민 관리 부서 (R3 §6)

- 정부 비공개 부서 / 청와대 직속
- 한국 정착 이주민 약 1,500명 (무협 850 / 판타지 380 / 선협 270 / 기타)
- 비공개 명단을 협회 임원 거액 매수로 부분 입수 가능 (★ 도윤 7화)
- 이주민 출신지: 무협 차원(가장 다수, 정보 풍부) / 판타지 차원 / 선협 차원 / 기타
- ★ 본작 활용: 도윤 7~9화 과외 풀(검술 사범·한의사·박사 지도교수·기관진식 사숙·블랙마켓 비급 출처)

### 4.6 신급 아티팩트와 빌런 단체 (R6 §6)

**신급 아티팩트 5종** (R4 §6):
- 달의 단편 (공간 신, 협회 보관)
- 천공의 사슬 (공간 신, 강탈됨)
- ★ **시간 신 아티팩트** (★ 본작 트리거 — 부모가 발견·파괴 → 도윤마무 발동)
- 인과의 거울 (인과 신, 부서짐 — 후반 떡밥)
- 운명의 실타래 (운명 신, 강탈됨)

**빌런 단체 「대회수자 (Grand Collector) 결사」** (R6 §6.2):
- 차원 너머 단체. 우두머리 = 대회수자 (무성·무연령·외형 가변, 신성 추정)
- 세력: 약 50~80명의 회수자 + 차원별 거점 + 한국 정착 이주민 일부 스파이
- 무기: 회수한 신급 아티팩트 효과 누적 (공간·운명 신 보유)
- 목적: 신급 아티팩트 전부 회수 → **차원 통합** (★ 매우 후반 폭로)
- 약점:
  - (a) 한 차원 도달 평균 3년 (★ 5화 S급 도착의 수치적 정당성)
  - (b) **도윤마무는 효과 회수 불가능한 유일한 저주** (다른 저주 면역)

### 4.7 사회 분위기 (도윤 빈곤 짐꾼 정당화)

- **짐꾼 정체성의 박탈감** (멸칭 + 일당 30~50만원 + 1년 미만 생존율)

- **부모의 마지막 카드를 받은 자식의 책임** (★ 매우 후반에 깨닫는 정서)
