# 복선 매트릭스 — 나에게만 스킬이 코드로 보인다

> 큰 설계(`/design-big`) 단계에서 본격 추가 (목표 25개 이상).
> 현재는 핵심 라인만. 등급 표기: S(메인)/A(보조)/B(소소)/C(분위기).

## 3. 회수 일정 (가이드)

```
EP001~005 ───┐
              │  배치 집중
EP006~014 ───┤
              │
EP015~020 ───┤  히로인 합류 + FS-003·FS-006 활성
              │
EP021~040 ───┤  사이다 + FS-001·FS-009 부분 회수 (능력 진화)
              │
EP041~060 ───┤  이상 던전 + FS-002·FS-005 자각 시작
              │
EP061~080 ───┤  시스템 농장 1차 자각 + 1부 결심
                FS-002·FS-009 메인 회수
                FS-004·FS-008 → 2부로 이월
```
