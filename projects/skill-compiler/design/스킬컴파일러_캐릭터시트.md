## 1. 주인공: 안강해 (安剛海)

### 기본 프로필

- **이름**: 안강해 (安剛海). "강한 바다처럼 견뎌라" -- 아픈 아이에게 부모가 지어준 이름. 본인은 이름이 싫다. "바다는 무슨, 웅덩이도 안 되는데."
- **나이**: 28세 (서사 시작 시). 내면 성숙도: 40대 직장인의 체념 + 18세 소년의 경계심
- **직업**: 중소 SI 2차 협력사 4년차 백엔드 개발자 (Java/Spring, MySQL, AWS 기초, Git). 원청 대기업 빌딩 상주 파견. 연봉 약 4,000만 원.
- **헌터 등급**: F급. 대각성 후에도 각성 판정 불가인 유일 사례. 무능력자 부서(언랭커과) 배정 대기.
- **외형**: 173cm, 마른 체형. 항상 다크서클. 후드집업을 계절 불문 입음 (사람 시선 차단용). 사람과 눈을 잘 안 마주침. 손가락이 길고 건조함 (키보드 타는 손). 걸을 때 약간 구부정. 머리는 앞머리가 눈을 반쯤 가림.

### 내면과 외면의 격차

- **외면** (타인이 보는 모습): 말수 적고 존재감 없는 F급 비각성자. 협회에서는 "측정 오류 아니냐"는 농담의 대상. SI 회사에서는 묵묵히 야근하는 그냥 그런 개발자. 눈을 안 마주치니 "어두운 애" 취급.
- **내면** (진짜 역량/본성): 10년 전부터 상태창이 보였던 오리지널 베타 테스터. 남들이 아이콘+설명 카드를 볼 때, 그에게는 소스코드가 보인다. 4년간 레거시 코드를 만지며 단련된 "찾아내는 눈" -- 남들이 못 보는 버그를 찾는 데 특화. 관찰력과 논리적 사고는 상위 0.1%.
- **갭 활용**: 모든 사이다의 핵심 패턴. "F급이 왜 여기 있어" → 강해가 스킬 코드를 수정하거나 약점을 짚음 → 결과가 등급을 뒤엎음 → 주변 침묵/경악 → 강해 무덤덤: "...뭐, 그냥 변수 하나 바꿨는데요." 독자가 그 무덤덤에서 우월감 대리 쾌감을 얻는 구조.

### 핵심 동기

- **1차 동기**: "평범하게 살고 싶다." -- 10년간 정신병자 취급 받다 대각성으로 겨우 "미친 건 아니었다"는 증명을 얻었으나, F급이라 여전히 바닥. 월세 내고 밥 먹고 사는 게 목표.
- **2차 동기**: "이 능력으로 밥은 벌 수 있겠다." -- 코드 수정의 가능성을 발견한 후, 프리랜서 헌터로 자립하려는 실용적 욕구. 누구에게도 고용되지 않고 먹고사는 삶.
- **숨은 동기**: "이 시스템은 왜 이렇게 설계되었나." -- 프로그래머의 직업병. 코드를 보면 설계 의도가 궁금해진다. EP040 이후 시스템 자체에 대한 의심이 동기의 핵심으로 부상.

### 전문 역량

- **핵심 스킬 3가지**:
  1. 스킬 코드 시각화 (Read) -- 본인/타인의 스킬을 소스코드 형태로 볼 수 있음
  2. 코드 편집 (Edit) -- 본인 스킬의 코드를 수정할 수 있음 (타인은 읽기 전용)
  3. 컴파일 (Compile) -- 수정된 코드를 발동 가능한 형태로 빌드
- **특수 능력/자산**: 오리지널 상태창 v0.1 베타. 보급형(v1.0)과 완전히 다른 인터페이스. 코드가 보이고, 편집이 되고, 컴파일이 가능하다. 슬롯 제한(초반 3개), 등급 캡(초반 D급까지), 빌드 타임 제약(전투 중 수정 불가)이 밸런스.
- **약점/한계**:
  1. **타인 감정 누수** -- 코드를 읽는 과정에서 대상의 감정이 흘러들어옴. 군중 속 10명+ 동시 읽기 시 두통, 강한 감정(공포/분노/광기)에는 감정 동조로 판단력 상실, 과다 노출 시 의식 끊김. 대규모 레이드에서 치명적.
  2. **전투 중 코드 수정 불가** -- EP040 핫픽스 해금 전까지 절대 제약. 빌드 룸(안전 공간)에서만 편집 가능. 전투 전 준비가 전부.
  3. **사회적 고립** -- 10년간 정신과 이력 + 감정 누수 회피로 인간관계 거의 전무. 혼자 밥 먹고 혼자 퇴근하는 사람.

### 성장 아크 (1부 전체)

| 구간 | 상태 | 핵심 변화 |
|------|------|----------|
| EP001~020 | 체념한 F급 비각성자. 코드가 보이지만 "또 내가 이상한 건가" | 스킬 컴파일러 자각 → 첫 코드 수정 → 솔플 던전 클리어 → 히로인 합류. "이건 되는데?" |
| EP021~050 | 조심스러운 활용자. 능력을 숨기며 실리를 취함 | 길드 영입 거절 → 기술 탈취 위기 → 소형 연합 결성 → 길드전 승리. "내 동료를 건드리지 마" (대사 아닌 행동) |
| EP051~080 | 시스템 의심자. 코드 속 이상 함수를 추적하는 디버거 | 이상 던전 → 시스템 함수 발견 → 의념 수확 인지 → 0층 메시지 수신. "이 농장을 해체한다" -- 자조 톤: "퇴사 경험은 있으니까." |

### 습관과 특징

- **코딩 관련 행동 패턴**:
  - 생각할 때 허공에서 손가락을 타이핑하듯 움직인다. 본인은 무의식.
  - 위기 상황에서 멈추고 "...로그부터 보자"라고 중얼거린다 -- 문제를 감정이 아닌 구조로 접근.
  - 새로운 스킬 코드를 볼 때 눈이 좌에서 우로 훑는 버릇. 코드 리뷰할 때의 습관.
- **위기 시 반응**: 얼어붙지 않는다. 대신 극도로 차분해진다 (감정 차단 모드). 목소리 톤이 낮아지고 문장이 짧아짐. "어디가 터졌지." "로그." "원인." -- 장애 대응 모드.
- **성공 시 반응**: 겉으로는 무덤덤. "...됐네." 정도. 내면에서만 짧은 안도. 자축 같은 건 없다 -- 성공에 기뻐해 본 경험이 부족해서.
- **기타**: 편의점 삼각김밥 + 캔커피가 주식. 새벽 3시까지 깨어 있는 게 일상. 사람 많은 곳에서는 이어폰을 끼고 백색소음을 튼다 (감정 누수 경감용).

### Dialogue DNA (대화 고유성)

| 항목 | 정의 |
|------|------|
| **사고 패턴** | 로그 분석형. 문제 발생 → 현상 관찰 → 원인 추정 → 코드(데이터) 확인 → 패치 방안 도출. 직감을 불신하고 데이터를 신뢰한다. "느낌이 이상하다"가 아니라 "수치가 안 맞는다"로 인지. |
| **화법 DNA** | 단문 위주 (평균 대사 8~15자). 해요체/합쇼체 혼용 -- 상대 불문 거리를 유지하는 톤. "~인데요", "~인 것 같은데", "~별 거 아닌데" 식의 축소/회피 종결. 단정 대신 유보. 감탄사 "아" "...뭐" "하" 자주. 쉼표보다 말줄임표가 많다. |
| **정보 전달 방식** | 결론부터 말하고 근거를 뒤에 붙인다 (개발자 습관). "이 스킬 버그 있어요. 세 번째 줄, 조건문이 빠졌거든요." 비유를 안 쓴다 -- 비유할 여유가 없는 사람. 수치와 구체적 위치로 말한다. |
| **감정 표현 경로** | 감정이 대사에 거의 안 드러남. 대신 **말이 줄어든다** (평소 짧은데 더 짧아짐 → 단어 하나 수준). 또는 **IT 용어로 감정을 우회**한다: 화날 때 "...이건 스펙 변경인데요" (부당함), 슬플 때 "...메모리 릭이네" (감정이 새는 느낌). 감정이 폭발할 때는 말을 안 한다 -- 침묵 후 행동. |
| **금기 표현** | "지켜내겠다", "구하겠다", "포기하지 않아" 류 영웅적 결의 절대 불가. "내가 책임지겠다" 류 사명감 불가. "역시 F급이라" 류 자기 등급 직접 탄식/자랑 불가 (외부자 발언으로만). 감탄형 긴 문장 ("이것이야말로 진정한 힘이다!") 불가. |

**대사 견본**:
- 타인 스킬 분석 후: "...세 번째 줄에 조건문이 빠져 있어서요. 넣으면 위력 1.5배는 나올 것 같은데."
- 길드 영입 거절: "...아, 네. 괜찮습니다."
- 위기 상황: "로그." / "어디서 터졌지." / "...원인은 이거네."
- 동료 위험 시: (말 없이 코드를 수정하고 컴파일 → 행동으로 보여줌)
- 자조: "하도급은 세계를 넘어도 하도급이다." / "F급 유일 사례. 이건 업적인가, 저주인가."

## 2. 히로인: 한지유 (韓智裕)

> 서윤하 / 한지유 중 **한지유** 확정. 이유: "지유(智裕)" = 지혜가 풍부하다. A급의 압도적 스킬 하나를 가졌지만 응용법을 모르는 캐릭터에, "지혜는 있는데 활용을 모르는" 아이러니가 맞는 이름.

### 기본 프로필

- **이름**: 한지유 (韓智裕)
- **나이**: 26세
- **등급**: A급 마법사
- **소속**: 5대 길드 중 하나(아르케 길드)에 소속되어 있었으나, 등장 시점 전후로 탈퇴 상태 (갈등으로 인한 자진 탈퇴)
- **외형**: 168cm. 단발 보브컷 (실전 편의). 날카로운 인상이지만 표정이 무방비인 순간이 있다. 전투복 차림이 기본. 사복은 무채색 계열 -- 화려한 걸 입으면 시선이 부담스러운 사람. 손에 항상 미세한 마나 잔류광이 있다.

### 스킬 & 역량

- **핵심 스킬**: [절대영도(Absolute Zero)] -- A급 단일 공격 스킬. 지정 범위 내 온도를 순간 영하 200도 이하로 낮춘다. 위력은 A급 상위. 그러나 ON/OFF만 가능하고, 범위 조절, 지속 시간 제어, 부분 적용이 안 된다. "핵폭탄 버튼은 있는데 조준경이 없는 상태."
- **약점**: 단일 스킬 의존. 콤보/연계/응용력 제로. 절대영도를 쓰면 아군도 범위 안에 있으면 피해. 파티 플레이가 어렵다. 솔로 플레이도 "쓰면 끝, 못 쓰면 끝"의 이분법. 길드에서 "폭탄" 취급 -- 마지막에 한 번만 쓰라는 지시를 받았고, 나머지 시간은 대기.
- **강해와의 시너지**: 강해가 절대영도 코드를 읽고 "범위 지정 변수", "지속 시간 파라미터", "출력 조절 함수"가 코드에 존재하지만 비활성 상태임을 발견. 코드 리뷰(타인 스킬 분석) → 활용법 제공 → 지유가 직접 체감으로 적용법 습득. "코드를 수정한 게 아니라, 원래 있던 걸 읽어준 것."

### 핵심 동기

- "내 스킬을 내가 제어하고 싶다." -- 자기 능력에 대한 통제권 갈망. 길드에서 "무기" 취급당한 경험. 강해가 코드를 읽어주면서 처음으로 자기 능력의 구조를 이해하게 된다.
- 숨은 동기: 과거 절대영도 오발로 팀원을 다치게 한 사고가 있음. 그 죄책감이 "제어"에 대한 집착의 뿌리.

### 내면과 외면의 격차

- **외면**: A급 마법사. 차갑고 과묵. 접근 불가의 아우라. "얼음 여왕" 별명.
- **내면**: 자기 능력이 무서운 사람. 제어할 수 없는 폭탄을 들고 다니는 불안. 사람을 다치게 할까 봐 거리를 둔다 -- 냉담이 아니라 경계.
- **갭**: 강해에게 코드 분석을 받은 후, 처음으로 자기 스킬을 "이해"하게 됨. 그 순간의 표정 변화가 독자를 끄는 포인트.

### Dialogue DNA

| 항목 | 정의 |
|------|------|
| **사고 패턴** | 결과 역산형. 목표 결과를 먼저 설정하고 역으로 수단을 찾는다. "이걸 얼리면 돼" → "어떻게?" 강해의 로그 분석형과 정반대 -- 강해는 원인에서 출발하고, 지유는 결과에서 출발한다. |
| **화법 DNA** | 중문 위주 (평균 대사 15~25자). 합쇼체 기본이나, 감정이 올라오면 반말이 섞인다. "~합니다" → "~인데" → "~잖아" 식으로 거리가 줄어드는 게 관계 발전의 신호. 질문을 잘 안 한다 -- 답을 듣기 싫어서가 아니라, 질문하면 대화가 길어지니까. |
| **정보 전달 방식** | 핵심만 말한다. 군더더기를 깎는 화법. "범위가 안 줄어요." (상황/배경/감정 전부 생략, 사실 하나만). 강해가 근거를 붙이는 타입이라면, 지유는 결론만 던지는 타입. |
| **감정 표현 경로** | 감정이 올라오면 **말이 빨라진다** (강해와 반대 -- 강해는 말이 줄어듦). 화나면 문장이 길어지고 반말이 섞임. 감사할 때는 말 대신 **행동** (커피를 건네거나, 등을 돌리고 서 있거나). 울지 않는다 -- 대신 입술을 깨문다. |
| **금기 표현** | "도와줘", "부탁해" 류 직접 요청 못 함 (자존심 + 폐 끼치기 싫음). "고마워"를 직접 말하는 데 EP060까지 걸림. 감탄사가 거의 없다 -- "우와", "대박" 같은 표현 안 씀. |

**대사 견본**:
- 강해에게 분석 받은 후: "...이 코드가 원래 있었다는 건가요."
- 전투 중: "범위, 좁힐 수 있어요?" (강해에게) / "비키세요." (적에게가 아니라 아군에게)
- 관계 발전 후: "...안강해 씨는 왜 그렇게 별 거 아닌 것처럼 말해요."
- 감정 폭발: "그러니까 그게 -- 범위가 안 되는데 어떻게 하라는 건데, 매번 대기만 하라고, 쓸모없으니까 뒤에 서 있으라고, 그게 맞는 건가요?"

### 로맨스 라인

| 시점 | 관계 | 핵심 장면 |
|------|------|----------|
| EP015~017 | 첫 만남. 던전 사고에서 조우. 강해가 지유의 스킬 코드를 읽고 "범위 변수가 있는데 비활성이에요"라고 알려줌. 지유 충격 + 경계. | "...당신, 내 스킬을 어떻게 아는 건데요." |
| EP018~020 | 협력 테스트. 강해의 코드 분석 + 지유의 실전 적용. 절대영도의 범위 축소에 처음 성공. 지유가 강해를 따라옴 ("혼자 가겠다는 건 아니죠?"). 아크 1 종료. | IT 동료 같은 실용적 관계 시작 |
| EP021~035 | 동료. 던전 공략 파트너. 강해가 분석, 지유가 실행. 서로의 능력을 이해하는 유일한 상대. 감정은 아직 "편한 동료" 수준. | 일상적 상호작용 축적 |
| EP036~050 | 신뢰. 길드전에서 강해가 감정 누수로 의식 끊김 → 지유가 보호. 이후 지유가 대규모 전투에서 강해 옆에 서서 감정 차단 역할을 자처. "당신이 쓰러지면 코드 읽어줄 사람이 없잖아요." | 감정 누수 방어 역할 |
| EP051~070 | 유대. 시스템 진실을 함께 추적. 코드 분석의 파트너 → 비밀을 공유하는 유일한 사람. 지유의 과거 (오발 사고) 고백. 강해의 과거 (정신병원) 고백. | 비밀 공유 = 신뢰의 정점 |
| EP071~080 | 동반자. 시스템 반항을 결심하는 순간, 옆에 있는 사람. 고백이나 키스 같은 직접 이벤트는 1부에서 안 함 -- "같이 있다"는 것 자체가 이 두 사람에게는 최대치. | 열린 결말. 2부에서 발전 |

**서사 기능**: 감정 누수 방어/보완 역할 + 코드 분석의 실행 파트너. 강해가 "읽는 눈"이라면 지유는 "실행하는 손". 강해의 사회성 회복 매개 -- 지유를 통해 강해가 처음으로 "옆에 사람이 있는 상태"를 경험한다.

**오리지널 베타 테스터 떡밥**: 지유의 절대영도 코드 안에 `// BETA_CANDIDATE: rejected` 주석이 숨어 있음. EP060~ 떡밥 배치. 시스템이 지유를 베타 테스터 후보로 검토했으나 "기각"한 흔적. 왜 기각했는가 = 2부 떡밥.

  - 이면: 코드 편집 능력이 확산되면 현재의 등급 체계가 무너지고, 등급 체계에 기반한 헥사의 지배력도 무너진다. 강해의 능력은 "체제 위협".
- **위협 수준**: S급 전투력 + 재벌 자본 + 정치적 로비력 + 정보 조직. 강해가 개인 전투로 이길 수 없고, 조직적으로도 자원이 압도적. 코드 편집의 "지적 우위"로만 대응 가능.
- **약점**: 시스템을 도구로 활용하고 있을 뿐, 시스템의 실체(의념 수확)를 모른다. EP060~ 시스템이 진현승도 "수확 대상"이라는 사실이 밝혀지면서 도덕적 회색지대 진입.
- **다층성**: 단순 악인이 아님. 본인 나름의 논리가 있고, "질서 유지"라는 대의를 위해 소수를 희생하는 것이 정당하다고 믿음. 나중에 시스템의 피해자이기도 했음이 밝혀짐.

**Dialogue DNA**:

| 항목 | 정의 |
|------|------|
| **사고 패턴** | 비용-편익 분석. 모든 결정을 자원 효율로 판단. "이 인재를 확보하면 ROI가 얼마인가." 감정은 비용이라고 생각하는 사람. |
| **화법 DNA** | 장문 (평균 대사 25~40자). 존댓말 기본이나 "우아한 위압"의 톤. "~하시죠", "~하실 의향은 없으신가요" 식의 정중한 압박. 명령을 질문으로 포장. 목소리가 낮고 느리다. |
| **감정 표현 경로** | 감정을 안 드러낸다 -- 가 아니라 감정이 "지연"된다. 화가 나도 즉시 반응 안 하고, 며칠 뒤에 조직적 보복으로 돌아옴. 유일하게 감정이 바로 나오는 순간: 체제가 위협받을 때. |
| **금기 표현** | 욕설/막말 절대 안 함. 고함 안 침. "실망했다"가 최대치의 분노 표현. |

**대사 견본**:
- 영입 시도: "안강해 씨, 헥사는 능력 있는 분들이 제대로 대우받는 곳입니다. F급이라는 등급이 당신의 전부는 아닐 테니까요."
- 거절 후: "...아쉽군요." (그리고 일주일 뒤 조직적 압박이 시작됨)
- 위협: "시장은 질서로 돌아갑니다, 안강해 씨. 질서를 무너뜨리는 개인은 -- 글쎄요, 시장이 알아서 처리하겠죠."

### 얼굴 빌런 2: 권도영 (權道營) -- 협회 기술국장

- **나이**: 52세
- **등급**: C급 (비전투 계열 -- 마나 측정/분석 특화)
- **정체**: 한국 헌터 협회 기술국 국장. 등급 측정 시스템의 설계와 운영을 총괄하는 인물. 15년간 협회에서 커리어를 쌓은 관료. 등급 시스템의 "설계자"이자 "관리자".
- **IT 모티프**: 플랫폼 알고리즘 관리자. 검색 결과를 조작하듯 등급 판정 가중치를 조작. 시스템의 공정성을 믿는 척하면서 자기 이익에 맞게 미세 조정.
- **동기**: 3중 구조.
  - 표면: 등급 시스템의 "안정적 운영". 공정한 관료.
  - 중층: 대형 길드(특히 헥사)로부터 로비를 받고, 등급 판정에서 특정 길드에 유리한 가중치를 적용. 부패 관료.
  - 심층: 시스템이 "코드 편집 가능 능력자"를 F급으로 분류하도록 설계된 사실을 알고 있으며, 이를 유지하는 대가로 시스템으로부터 "보호"를 받고 있음. 시스템의 협력자.
- **위협 수준**: 전투력은 약하지만, 제도적 권력이 거대. 강해의 등급을 영구 F급으로 고정하거나, 헌터 자격 자체를 박탈할 수 있는 행정 권한. "싸워서 이길 수 없는 적" -- 행정/제도의 벽.
- **약점**: 관료답게 증거에 약하다. 등급 조작의 로그가 남아있고, 이도현이 이를 추적 중.

**Dialogue DNA**:

| 항목 | 정의 |
|------|------|
| **사고 패턴** | 절차주의. "규정에 따르면", "기준에 의하면"이 모든 문장의 시작. 규정 뒤에 숨어서 자기 의지를 관철하는 타입. |
| **화법 DNA** | 합쇼체 일관. 관료 화법. 장문이고 주어를 흐리게 함 -- "이 건에 대해서는 검토가 필요합니다" (누가 검토하는지 안 밝힘). 수동태 남발. 책임을 회피하는 문장 구조. |
| **감정 표현 경로** | 감정이 **말투가 아니라 행정 조치**로 나온다. 화나면 "재심사 일정을 앞당기겠습니다", 위협하면 "자격 유지 요건을 재검토해야 할 것 같습니다". |
| **금기 표현** | 직접 명령 안 함 ("~하세요" 대신 "~하는 것이 좋을 듯합니다"). 감정 단어 사용 안 함 ("화가 난다"가 아니라 "유감스럽습니다"). |

**대사 견본**:
- 강해 면담: "안강해 씨의 등급 판정 결과에 대해 이의를 제기하셨다고 들었습니다. 다만, 현행 측정 프로토콜에 따르면 재심사 요건을 충족하지 못하고 계시는데요."
- 위협: "물론 헌터 자격은 유지되고 있습니다만, 활동 이력에 비정상적 패턴이 감지될 경우 -- 보호 차원에서 활동 제한 조치가 내려질 수도 있습니다."
- 몰락 전: "저는 규정에 따라 업무를 수행했을 뿐입니다."

### 2층 빌런: 시스템

- **정체**: 인류에게 상태창/스킬/던전을 부여한 시스템. 인격 없음. 코드와 메시지로만 존재.
- **적대 방식**:
  - EP020~039 (의심기): 코드 속 이상 함수(`harvest_intention`, `report_anomaly`) 발견
  - EP040~059 (인지기): 강해의 비정상 활동을 감지 → 에러 메시지 빈도 증가 → 코드 보호 강화
  - EP060~080 (대응기): 강해를 "버그"로 분류 → 던전 내 시스템 이벤트로 암살 시도 → 강해가 코드 역분석으로 역이용
- **서사적 기능**: 물리적 힘이 아닌 "구조적 적대". 싸울 상대가 없다 -- 시스템 그 자체가 적이므로, 강해는 코드로 싸워야 한다.

### 0층 빌런: 설계자

- EP075~: 시스템 코드 최하층에서 `// DESIGNER_NOTE: iteration_47` 주석 발견 → "47번째 반복?"
- EP080: 0층에서 메시지 수신. 강해의 답: "버그 리포트 접수 완료."

### 관계 상세

| 관계 | 초기 | 중기 (EP025~) | 후기 (EP051~) |
|------|------|-------------|-------------|
| 강해 ↔ 지유 | 코드 분석자 ↔ 실행자. 경계하는 동료 | 신뢰하는 파트너. 감정 방어 역할 | 비밀을 공유하는 유일한 동반자 |
| 강해 ↔ 태산 | 스킬 고쳐준 은인 ↔ 감사하는 탱커 | 전투 조합 파트너. 형-동생 같은 관계 | 무조건적 신뢰. 진실을 함께 아는 동료 |

| 강해 ↔ 서진 | 같은 부서 동료 (무능력자 부서) | 후방 지원. 행정/네트워크 제공 | 비중 감소 (2부에서 재확대) |
| 강해 ↔ 진현승 | 무관심 (F급은 관심 대상 아님) | 전면 적대. 기술 탈취 + 영입 압박 | 회색 지대. 시스템의 피해자 측면 드러남 |
| 강해 ↔ 권도영 | 등급 판정자 ↔ F급 판정 대상 | 제도적 적대. 등급 조작 vs 증거 수집 | 몰락. 시스템 협력 사실 폭로 |
| 지유 ↔ 진현승 | 전 소속 길드(솔라리스) 탈퇴자. 직접 관계 없음 | 강해 편에 섬으로써 간접 적대 | 길드전에서 직접 대결 |
| 도현 ↔ 권도영 | 같은 협회 소속. 부하 관계 | 데이터 추적 vs 은폐 시도 | 도현이 증거를 공개하며 결별 |
| 태산 ↔ 장세영 | 과거 같은 길드(헥사) 출신. 태산이 "답답해서" 나옴 | 재회. 과거 동료에서 적으로 | 장세영 강등 후 접점 소멸 |

### 관계 변화 힌트

- **강해 ↔ 지유**: "편한 동료 → 유일한 동반자"로의 점진적 이동. 결정적 전환: EP045 강해 의식 끊김 시 지유가 보호 + EP055 서로의 과거 고백. 로맨스는 2부까지 미완결.
- **도현의 갈등**: EP040 전후가 분기점. 그 전까지는 "흥미로운 데이터"를 추적하는 학자, 그 후에는 "진실을 공개할 것인가"의 도덕적 선택.
- **진현승의 다층성**: EP060 이후 시스템의 피해자 측면 노출. "질서를 지키려 했을 뿐인데 농장의 관리인이었다니." 완전한 악인이 아닌 회색 빌런으로 진화.
