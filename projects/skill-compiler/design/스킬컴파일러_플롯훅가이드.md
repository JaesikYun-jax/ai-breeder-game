## 1. 대주제 서사선

- **테마**: "하청인 남자가 시스템의 소스코드를 읽고, 농장의 진실을 발견한다."
- **메인 퀘스트**: 의념 수확 농장의 실체를 파악하고, 시스템을 리팩토링(해체가 아닌 재설계)할 결심에 이르기까지.
- **서사 공식**: F급 체념 → 코드 자각 → 솔플 증명 → 조직 대결 → 시스템 해부 → 농장 자각 → 결심

**독자 감정 경로**:
```
EP001~005  : 공감("나도 하청이다") → 호기심("코드가 보인다고?")
EP006~020  : 통쾌("F급이 A급을 이긴다") → 몰입("히로인 합류")
EP021~050  : 분노("길드가 기술을 탈취한다") → 카타르시스("길드전 승리")
EP051~070  : 충격("시스템이 농장이었다") → 긴장("암살 시도")
EP071~080  : 결의 대리 만족("이 농장을 해체한다 -- 퇴사 경험은 있으니까")
```

### 4-3. 역량 소진 대비

- **전환점 1 (EP025)**: 기초 역량(Read/Edit/Compile + 튜닝/디버깅/리팩토링)만으로는 A급 이상 적대에 한계. → API 연동 + 역컴파일 해금으로 "솔플 → 팀플" 전환
- **전환점 2 (EP040)**: 빌드 타임 제약이 전투의 병목. → 핫픽스 해금으로 전투 중 제한적 수정 가능
- **전환점 3 (EP055)**: 인간 빌런이 주적이 아닌 시스템 자체가 주적. → 시스템 코드 역분석 능력 해금
- **대체 동력**: 역량 해금이 아닌 "기존 역량의 창의적 조합"이 소진을 보완. 특히 EP040~050에서 핫픽스 + API 연동 + 콤보의 복합 운용이 새로운 쾌감을 제공

### 5-2. EP024~025 이중 훅 상세

**EP024 (유료 전환 전전화)**:
- 강해의 대형 콤보가 협회 공식 레이드에서 폭발
- F급이 A급을 격파하는 장면 → 5대 길드 전원이 강해를 주목
- "이 F급은 뭐지?" 반응 폭발
- 절단: 진현승의 시선이 강해에게 고정되는 장면

**EP025 (유료 전환 직전화)**:
- **훅 1 (사이다)**: 진현승 직접 대면. "당신의 능력, 우리가 관리하겠습니다." 강해의 거절.
- **훅 2 (미스터리)**: 강해가 자기 스킬 코드 속에서 `harvest_intention` 함수를 최초 발견. "이 함수는...뭐지?" → 독자: "시스템에 뭔가 숨겨져 있다"
- 절단: 두 훅이 동시에 걸린 상태에서 끊기. 독자는 (1) 길드전 결과 + (2) 시스템 미스터리 두 가지를 알기 위해 유료 전환.

### 5-3. 유료 전환 직후 (EP026~030) 독자 유지 전략

| 원칙 | 적용 |
|------|------|
| 25화 떡밥 즉시 회수 | EP026: `harvest_intention` 샌드박스 분석. "이 함수는 스킬 발동 시마다 뭔가를 수집한다" -- 부분 해답 |

| 서사 단계 | 갈등 레벨 | 적대자 유형 | IT 모티프 | 독자 감정 | EP 범위 |
