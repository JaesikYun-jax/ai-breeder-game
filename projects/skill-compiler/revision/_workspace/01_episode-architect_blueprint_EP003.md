## 4. 감정 곡선

```
장면1(긴장 3 → 2 → 3) → 장면2(긴장 2 → 3 → 4) → 장면3(긴장 3 → 4 → 4) → 장면4(긴장 4 → 5 → 5)
   잔상 + 행정 권태       부서 정착 + 갈취 + 위화감     본인 코드 구조 파악         자각 + 갈등 + 절단
```

**곡선 설계 근거**: EP002는 W자형 (5→3→4→5)였다. EP003은 **점진적 상승 곡선** — 출발(3)에서 갈취 사건(4) → 본인 코드 인식(4) → 자각(5)로 차곡차곡 누적된다. 이는 EP004 첫 시도(긴장 6) → EP005 첫 사이다(긴장 7~8)로 가기 위한 **자각의 단단한 발판**이다. 사이다 누적 누계: EP001 = 고구마 80% / EP002 = 고구마 70% + 미스터리 25% + 사이다 5% / **EP003 = 고구마 50% + 미스터리 30% + 사이다 20%** — 자각 자체가 작은 사이다 (오케스트레이터 명시). 단 시도 X로 끊겨야 EP004 절단신공 효과 유지.
