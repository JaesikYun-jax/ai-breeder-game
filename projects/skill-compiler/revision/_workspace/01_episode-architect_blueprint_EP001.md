## 4. 감정 곡선

```
장면1(긴장 1) → 장면2(긴장 2) → 장면3(긴장 4) → 장면4(긴장 5)
   체념          가벼운 자극         돌발 위기      부조리한 정지
```

**곡선 설계 근거**: 도입화는 후크 박기 + 세계관 진입을 동시 처리해야 한다. 1·2장면을 낮은 긴장(자조 톤 정착) → 3장면 급격한 상승(게이트) → 4장면 절정 직전 정지(상태창 갱신)로 가야 클리프행어가 살아난다. 4장면을 6이 아닌 5로 두는 이유: 6은 능력 자각 직후의 대형 사이다용으로 EP005·EP010에 비축한다.

## 5. 타임라인 락

- **이 EP의 절대 날짜**: 대각성 1년 1주년 시점 (계절·요일은 작가 재량 — 저녁 골목·짧아진 일조 시간을 고려하면 늦가을 평일 권장)
- **장면별 시간**:
  - 장면 1: 새벽 0시~1시 (자취방)
  - 장면 2: 정오 직전 (원청 빌딩 9층)
  - 장면 3: 오후 5~6시 (협회 측정실 → 퇴근길)
  - 장면 4: 오후 6시 직후 (막다른 골목, 1초의 정지)
- **허용 상대시간**: "오늘", "어제", "1년 전(=대각성 직후)", "10년 전(=18세, 첫 상태창)", "5년 전(=첫 게이트)" — 그 외 미래 상대시간 사용 금지 (회귀 X / 미래 앎 X 가드레일)

- **인물 나이**: 안강해 28세 (출생년도 직접 노출 X — "스물여덟"이라는 단어로만 1회 등장 가능)

## 7. 훅 전략

- **플롯가이드 지정 훅 타입**: 미스터리 ("왜 나만 다른 상태창이지") + 코드 발견 (장면 4 클리프행어) — plot L46 / plot §9-3
- **최소 훅 강도**: **오프닝 4 / 엔딩 5** (오케스트레이터 지시. 무료 구간 1화는 유료 전환 구간 아니나, 첫 화는 독자 정착의 절대 분기점이라 엔딩 5)
- **오프닝 훅 (강도 4)**:
  - 첫 줄 견본: **"모두에게 상태창이 열린 날, 나는 그게 새삼스럽지 않았다."** (plot §2-2 명시)
  - 또는 변형: **"대각성 1주년이 새벽 0시를 넘어가던 그때, 나는 협력사 빌드 콘솔 앞에서 java.lang.NullPointerException을 보고 있었다."**
  - 왜 독자가 멈추지 못하는가: "새삼스럽지 않았다" 단 한 단어로 (1) 1인칭 자조 톤 (2) "이 화자에게 다른 사연이 있다"는 호기심 (3) 대각성을 후일담처럼 다루는 신선함 — 3가지가 한 번에 박힘
- **중반 훅 (장면 3 진입)**:
