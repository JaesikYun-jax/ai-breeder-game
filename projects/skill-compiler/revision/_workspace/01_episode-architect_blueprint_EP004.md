## 4. 감정 곡선

```
장면1(긴장 4 → 3 → 4) → 장면2(긴장 4 → 4 → 5) → 장면3(긴장 5 → 6 → 5) → 장면4(긴장 4 → 4 → 5)
   자각 잔류 + 일상 흐름     자취방 결심 형성       첫 컴파일 SUCCESS + 부작용  사이다 + 자조 + 도화선
```

**곡선 설계 근거**: EP003은 점진적 상승 곡선 (3→3→4→5)이었다. EP004는 **W자형의 변형** — 자각 잔류(4) → 일상(3) → 결심 직전 압박(5) → 시도 트리거(6) → 안도(5) → 도화선(5). 첫 컴파일 SUCCESS 순간(긴장 6)이 정점이고, 그 직후 부작용으로 5에서 멈춰야 EP005 첫 사이다로 7~8까지 끌어올릴 여지가 남는다. 사이다 누적 누계: EP001 = 고구마 80% / EP002 = 고구마 70% + 미스터리 25% + 사이다 5% / EP003 = 고구마 50% + 미스터리 30% + 사이다 20% / **EP004 = 사이다 30% + 미스터리 20% + 자조 50%** (오케스트레이터 명시). 시도 성공이지만 위력 2배 정도 + 부작용 동반 = 작은 사이다 + 자조 우위. 본격 사이다는 EP005.
