## 4. 감정 곡선

```
장면1(긴장 4 → 4 → 5) → 장면2(긴장 4 → 6 → 5) → 장면3(긴장 5 → 7 → 6) → 장면4(긴장 5 → 6 → 6)
   잔류 부작용 + 갑질 임무   사고/분리 + 첫 시도   디버그 발화 + 정점 격파  외부 침묵 + 새 매듭
```

**곡선 설계 근거**: EP004는 W자형 (4→3→4 / 4→4→5 / 5→6→5 / 4→4→5)이었다. EP005는 **상승+정점+하강의 산형**으로 도입부 5화의 카타르시스 결착 — 잔류 부작용(4) → 갑질 압박(5) → 사고 분리(6) → 첫 시도(5) → 디버그 발화(5) → 정점 격파(7) → 안도+부작용(6) → 외부 침묵(6) → 새 매듭(6). **장면 3-3 정점(긴장 7) = 도입부 마지막 화의 카타르시스 정점** — 단 영웅적 결의 X / 무덤덤 안도. 사이다 누적 누계: EP001 = 고구마 80% / EP002 = 고구마 70% + 미스터리 25% + 사이다 5% / EP003 = 고구마 50% + 미스터리 30% + 사이다 20% / EP004 = 사이다 30% + 미스터리 20% + 자조 50% / **EP005 = 사이다 50% + 미스터리 20% + 자조 30%** (오케스트레이터 명시 — 첫 사이다 강도 7~8). plot §3-2 EP005 강도 = 4/10 (등급 역전이 아닌 첫 솔플). 단 오케스트레이터의 "첫 사이다 강도 7~8"은 **도입부 5화 누계 카타르시스 곡선 측면**이며, plot §3-2의 4/10은 **80화 전체 곡선 측면**. 본 설계도는 양쪽 모두를 고려해 장면 3-3 정점을 긴장 7로 설계 (영웅적 결의 X / 무덤덤 안도가 지배 — 외부 시선의 침묵으로 사이다 강화).

- EP006 첫 비트 (plot §2-3 EP006): "솔플 던전 반복 클리어. 타인 스킬 코드가 읽히기 시작 — 군중 속 두통 첫 경험" → 강해의 약점(타인 감정 누수) 첫 노출
- EP005 → EP006 연결의 핵심: **첫 던전 솔플의 작은 사이다 + 외부 침묵 + FS-009 강화 → EP006에서 솔플 반복으로 능력의 응용 패턴 정착 + 약점(타인 감정 누수) 첫 노출의 균형**. 강해의 4년차 디버그 본능이 단순 변수 수정에서 한 단계 더 나아가, 첫 던전에서 즉각적 응용 (단, EP006에서 새 응용은 신중히 — 솔플 반복 + 첫 약점 노출이 핵심)
- EP005 끝에서 EP006으로 이어지는 핵심 정보: (1) 슬롯 1/3 유지 + 부작용 누적 패턴 (2) FS-009 새 코드/메시지 단편 → EP006에서 의미 추적은 신중히 (본격 추적은 EP020 이후) (3) FS-007 헥사 갈취 + 도주 + 진현승 이름 1줄 → EP006~010에서 헥사 측 주목 누적 → EP011 장세영 직접 등장의 도화선
