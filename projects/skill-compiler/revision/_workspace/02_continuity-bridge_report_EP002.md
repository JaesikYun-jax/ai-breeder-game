## 1. 타임라인 위치

### 1-1. EP001 끝 → EP002 시작 시간 간격

**EP001 종료 시점**: 대각성 1주년 평일 늦가을, 오후 6시 직후 (협회 1층 빠져나온 직후 골목, 11월 서울 기준 골목이 어두워진 시간대 — EP001 line 96 인용: "11월의 서울은 오후 다섯 시면 골목이 어두워진다")

**EP002 권장 시간 간격**: **수 시간 후 ~ 익일 오전** (의식 잃음 후 깨어난 시점)

근거:
- 플롯훅가이드 §2-2 EP002 핵심: "협회 등급 측정 — 또 F급. 보급형 v1.0과 자기 v0.1의 차이를 처음 인식. 스킬 코드가 '텍스트'로 보이기 시작" → EP001에서 측정은 이미 끝났으므로 EP002 측정은 **재측정** 또는 **이상 신고로 인한 추가 검사** 맥락이어야 함. 또는 회상 형식.
- EP001 마지막 장면(막다른 골목)은 강해의 자각 직전 상태로 끝남. **자각은 EP003에서**(설계도 §1 장면4 + 부트스트랩 §3-2 트리거 명시: "각성 → 스킬 컴파일러 각성"이 EP003)
- 따라서 EP002는 "막다른 골목에서 어떻게 살아남았나"의 답 + 의식 회복 + 익숙한 환경 복귀 + v0.1 인지 시작의 흐름이 자연스러움.

### 1-2. EP001 막다른 골목 — 어떻게 살아남았나? (★ 핵심 결정 사항)

권장 해소 방식 (오케스트레이터 지시 그대로 채택): **"정신 잃었다 깨어나니 살아 있더라" 형태**

채택 근거:
- **강해 능력 자각 X 단계 보존** (가드레일 §능력단계 + 부트스트랩 §3-2: 자각은 EP003)
- 시스템 자동 발동·게이트 자체 정리 가능성은 **암시만**, 강해는 의미 모름
- 다른 헌터 개입 시나리오 가능 — 단 인물 등장 부담을 최소화 (1회성 NPC로 처리, 이름 X)

권장 시나리오 3안:

| 안 | 시나리오 | 장단점 |
|----|---------|-------|
| **A안 (권장)** | 시야에 떠오른 코드 다섯 줄을 본 직후 의식 차단 → 깨어나니 골목에 아직 누워 있음 / 또는 협회 측 의료실 / 게이트는 이미 정리됨. 정리한 주체는 알 수 없음 (지나가던 헌터 / 협회 출동대 / 게이트 자가 소멸 — 강해는 모름) | 자각 X 보존 + 미스터리 강화 + 인물 등장 0 |
| B안 | 의식 잃기 직전, 시스템이 강해의 마나(?)를 임의로 사용해 몬스터 회피 — 강해는 "그냥 운이 좋았다"고만 인식. 코드 다섯 줄은 그대로 떠 있음 | 시스템 능동성 떡밥 강화. 단 강해가 "내가 살아난 것 같은데 모르겠다" 정도의 자조 필요 |
| C안 | 정체 모를 검은 실루엣(상위 헌터)이 1초 만에 게이트와 몬스터 처리 후 사라짐. 강해는 얼굴도 못 봄 | 0층/베타 테스터 떡밥의 1차 시드 (FS-004). 단 EP002 분량에서 묘사 부담 |

**권장: A안.** B/C는 EP002 후반 또는 EP003~005에서 더 큰 효과로 사용 가능. EP002의 핵심은 **v0.1 인지 + 협회 재처리 절차**여야 한다.

### 1-3. EP002 시점 안강해 위치

권장 동선:
1. **장면 1**: 깨어남 — 골목 / 또는 응급 의료 부스 / 또는 협회 1층 임시 보호실
2. **장면 2**: 협회로 인계 또는 자체 보고 — F급 비각성자가 E급 게이트 인근에서 발견됨 = 행정 절차 발생 (사고 보고서, 추가 측정 권고)
3. **장면 3**: 협회 재측정실 또는 의료부 — 여기서 "v0.1" 단어가 처음 강해의 시야에 인지됨 (남들 측정기와 자기 화면 차이 자각)
4. **장면 4**: 자취방 복귀 — 새벽 또는 익일 오전. 코드 다섯 줄이 **사라지지 않고 시야에 잔류**한 상태로 EP002 종료. 또는 명멸하며 사라졌다가 잔상만 남음

장소 시퀀스: **막다른 골목 → 협회 1층 → (선택: 의료부) → 자취방**
