## 1. IT 업계 갑질 사건 사례 (현실 모티프)

### 1-1. 협력사 직원 차별 사례 (보도/판례 학습 데이터 기반)

| 유형 | 사례 패턴 | 강해 일상 활용 |
|---|---|---|
| 식사 차별 | 정직원 사내식당 입장 카드 비활성화. 협력사 점심값 자비. | EP002, EP007: 사내식당 입구에서 카드 안 됨 → 1층 김밥 |
| 화장실/엘리베이터 분리 | 보안 명목으로 협력사 동선 분리. 같은 층 다른 화장실. | EP007 후반: 강해 머릿속 자조 "차별은 차원을 초월한다" |
| 회식 강제 + 분담 | 정직원 행사에 협력사 참여 강제, 회비는 동률 분담 | EP013 또는 EP021 회식 신 |

| 회의실 자리 배치 | 발주처 상석, 협력사 입구 끝자리 | EP014 헥사 회의 → SI 데자뷔 |

### 1-2. 요구사항 변경/일정 압박 사례

- **계약 외 추가 작업 강요** [학습 데이터 기반]: 발주처 PM이 스프린트 종료 직후 "이건 원래 포함된 거였어요" 신규 기능 통보 → 협력사가 무상 야근으로 처리. 공정거래위원회 조사 사례 다수.
- **금요일 17시 통보**: 다음 주 월요일까지 신규 기능 요청. 주말 근무 강제. 추가 비용 미지급.
- **요구사항 변경 횟수 제한 위반**: 표준 SI 계약은 변경 횟수를 제한하지만, 실무에서 발주처가 무시하고 무한 변경.
- **테스트 책임 전가**: 발주처가 QA 인력을 빼고 협력사에 테스트까지 떠넘김.

**강해 활용**:
- EP001 도입: 자취방 벽시계 새벽 1시. 노트북 앞 강해. 슬랙 알림 "내일 아침까지 부탁드려요". 자조 "내일 아침이 4시간 뒤인데."
- EP013: 회사 야근 중 무능력자 부서 호출. 양쪽 동시 마감. "이게 헌터 사회의 마법인가, 아니면 SI의 마법인가."

### 1-3. 코드 소유권/지적재산권 분쟁

- **소스 귀속**: 표준 도급 계약상 소스 코드는 발주처 소유. 협력사 개인이 작성한 코드도 회사·발주처 자산.
- **퇴사 후 사용 금지**: NDA + 경업금지 조항으로 퇴사자가 동일 도메인 취업 어려움.
- **오픈소스 무단 도입 후 책임 전가**: 발주처가 시간 압박으로 오픈소스 도입 → 라이선스 위반 발견 → 협력사 책임으로 전가.

**강해 활용 (작품 IT-헌터 동형)**:
- EP020 이후 진현승의 "당신의 능력은 헥사 자산으로 등록하시죠" = SI 코드 소유권 모티프 직역.
- EP021 "내가 짠 코드인데 네 거라고?" 자조 그대로 사용.

### 1-4. 야근/주말 근무 강제

- **포괄임금제**: 월 60시간 야근까지 무제한 무상. 4년차 강해 연봉 4천 = 시급으로 환산하면 일반 알바보다 낮은 시즌 빈번.
- **야근비 미지급 관행**: 근로기준법상 1.5배 가산이지만 SI 협력사는 미지급 다수.
- **회식 = 근무 연장**: "친목"으로 분류해 근무시간 인정 X.

### 1-5. 미세한 갑을 디테일 (작품 톤 적용 사례)

- **점심 호출**: 발주처 PM이 "협력사분들 같이 점심 어떠세요?" → 거절 어려움. 식사비는 협력사 자비.
- **회의 발언 순서**: 발주처가 먼저 발언, 협력사는 마지막. 강해는 보통 침묵.
- **출장 좌석**: 발주처 KTX 특실, 협력사 일반실.
- **명함**: 협력사 명함에 발주처 빌딩 주소 + "OO기업 파견" 병기. 본인 회사 표시 작게.
- **연차 사용**: 발주처 일정에 맞추기 위해 협력사가 연차 양보.
- **건강검진**: 정직원 종합검진 vs 협력사 기본검진.

## 2. 2025~2026 IT 산업 컨텍스트

### 2-1. AI 코딩 도구 보편화 → 주니어/중니어 위기감

[학습 데이터 ~2025.05 기반, 일부 트렌드 추정]
- **GitHub Copilot, Cursor, Claude Code** 대중화 (2024~2025).
- **2025년 글로벌 빅테크 채용 축소**: Meta, Google, MS 등 대규모 구조조정 이력 학습 데이터 내 확인. 한국 IT 시장도 2024 하반기 ~ 2025 채용 축소 분위기.
- 4년차 강해의 위기감: "AI가 코드 짜는 시대에, 4년차 협력사 백엔드는 무슨 가치가 있지." 자조의 시대적 근거.
- 동시에 **"AI는 디버깅·레거시 분석에 약함"**이 업계 공감대 → 강해의 "찾아내는 눈"의 시대적 가치 부상. 작품의 사이다 동력과 정합.

### 2-2. 빅테크 구조조정 분위기 + 개발자 연봉 하향 안정화

- 2022~2023 개발자 연봉 버블 → 2024~2025 안정화/하락. 신입 초봉 하향, 협력사는 더 심함.
- 강해 4천만 원 = 4년차 협력사로는 평균 ~ 다소 아래. 사실적.
- 직장인 익명 커뮤니티(블라인드, 잡플래닛 등)에서 "이직해야 하나 vs 버텨야 하나" 일상 화제.

### 2-3. 강해가 28세 4년차로서 느끼는 시대적 압박 (작품 톤 근거)

- 또래 친구들이 "퇴사 후 부트캠프", "스타트업 이직", "프리랜서 전환" 트렌드. 강해는 정신과 이력 + F급으로 이직 시장에서도 약자.
- "이직 면접에서 정신과 약물 복용 이력 질문" → 학습 데이터상 실제 차별 사례 보고 다수. 강해 사회적 고립의 현실 근거.
- IT 업계의 "테크 트위터", "개발자 컨퍼런스", "오픈소스 기여" 같은 자기 PR 문화에서 강해는 완전 격리. 후드집업 + 1인 점심.

### 2-4. SI 협력사의 비주류 위치 (구체)

- 한국 SI 시장: 삼성SDS, LG CNS, SK C&C 등 대기업 SI가 원청. 1차 협력사(대형 IT 서비스 기업)는 자체 인력. **2차 협력사는 사실상 인력 파견업체** → 강해의 위치.
- 2차 협력사 4년차 = 시장에서 "정직원 전환 가능성 거의 없음, 다른 2차로 이직 가능" 위치. 사회적 인정 척도 약함.
- 부모 세대에게는 "대기업 IT 회사 다닌다"고 말할 수 있지만, 본인은 알고 있다. **자조의 핵심 동력**.

### 3-2. 5대 길드 형성 과정 (간략)

- **헥사**: 현승그룹(재벌)이 대각성 직후 6개월간 1억 원대 사이닝으로 S~A급 흡수. 진현승이 마스터 자처.
- **솔라리스**: 윤세하(전직 IT 스타트업 CEO, S급 각성)가 30명 핵심 인력으로 출범. 실력주의 광고.
- **뱅가드**: 정부가 "공공 안보" 명분으로 공무원 각성자 흡수. 임금 낮으나 안정.
- **크로스보더**: 미국·유럽 합자. 한국 던전 자원 일부 해외 반출 의혹. 국정감사 단골.
- **넥스트**: 강민준(28세 S급 미만이지만 A급) 주도. 젊은 각성자 SNS 모집.

### 3-3. 협회의 등급 측정 시스템 도입과 정착

- 측정 알고리즘은 협회 기술국 권도영 국장이 설계 주도.
- 표면: "공정한 6등급 분류".
- 이면: 가중치 조정 권한 = 등급 인플레/디플레 가능. 헥사 등 대형 길드와 결탁한 미세 조작.
- 강해의 F급 영구 고정 = 시스템 자체가 코드 편집자를 위협으로 분류 → 권도영도 알면서 협조.

### 3-4. 무능력자 부서 신설 배경

- 대각성 1년 +9개월: F급 9만 명 (전체 인구 0.18%) 등록. 사회적 격차 심화 + 자살률 급증.
- 정부가 "F급 사회 통합"을 명분으로 협회에 무능력자관리과 신설 강제.
- 실제 운영: 행정 보조, 던전 잔해 청소, 통계 작성 같은 단순 업무 위탁. 사실상 격리·관리 시설.
- 강해는 EP002 부근 무능력자 부서 출근 첫날 → 최서진 만남.

### 3-5. F급 차별의 사회적 정착 (EP001~025 배경 분위기)

## 4. EP025 시점 시대적 사건 모티프 (현실+허구 융합)

### 4-1. 2025~2026 IT 사건 모티프 (시점 거울)

[학습 데이터 ~2025.05, 일부 추정 `[미확인]`]:
- **AI 학습 데이터 무단 수집 논란**: OpenAI, Meta 등 저작권 분쟁. → 작품의 "harvest_intention" = 시스템이 인간 의념을 동의 없이 수확 = 직접 거울.
- **빅테크 알고리즘 편향성 폭로**: 검색·SNS 알고리즘이 특정 그룹 차별. → 협회 등급 측정 알고리즘 편향(권도영 부패) = 직접 거울.
- **개인정보 유출 사고**: 통신사·금융사 대형 유출 사건 다수. → 협회 측정 데이터 유출 가능성, 이도현이 자료 추적하다 발견하는 모티프 가능.
- **플랫폼 갑질 공정위 조사**: 카카오·쿠팡 등 플랫폼이 입점업체 차별. → 헥사가 소형 길드/F급 헌터 자원 착취 = 거울.
- **노조 파업·근로기준법 위반 보도**: SI 야근 강요 보도. → 강해의 일상 자조 톤 사회적 근거.

### 4-2. EP025 시점 사회적 뉴스 (작품 내 분위기)

- **"F급 헌터 콤보 영상 익명 유출"** (EP024~025): 헌갤·유튜브 클립 확산. "F급이 A급 헥사 공략대장 격파". 5대 길드 동시 주목.
- **"헥사 vs 무명 헌터, 길드전 신청"** (EP025): 협회 공식 분쟁 등록. 1년 만에 처음 있는 "F급 vs S급 진영" 공식 대결. 사회적 화제.
- **"등급 측정 시스템 의혹"**: 이도현이 EP020~023 사이 익명으로 "측정 알고리즘 편향성" 데이터 흘림. 트위터·헌갤에서 토론.
- **"길드 자본 외국 유출 의혹"** (크로스보더 관련): 국정감사 일자 임박. 정치권 화제. 작품 배경 분위기.

### 4-3. 길드/협회의 분쟁이 사회적 뉴스가 되는 정도

- 길드전 = 사회 1면 뉴스. 시청률 올림픽급. 강해의 EP025 길드전은 작품 내 가상 KBS·MBC 실시간 보도 수준.
- 협회는 길드전 중계 수익을 가져감. 이 수익 구조가 협회의 부패 동력 중 하나.
- 일반 시민은 길드전을 스포츠처럼 소비. F급 = 언더독 = 베팅 배당 100배. 강해의 콤보가 터지는 순간 베팅 시장 폭발.
