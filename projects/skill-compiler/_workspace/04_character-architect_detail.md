**리듬 원칙**:
- 같은 감정 3화 연속 금지 → EP022~023 분노 2연속 후 EP024 긴장 전환
- 통쾌(대형) 직후 1~2화 이완: EP005 직후 EP006 자조, EP010 직후 EP011 경계, EP020 직후 EP021 자조
- 트라우마(F급/감정 누수/정신과)는 누적 노출. 25화 동안 정신과 직접 언급 2회만(EP001 도입, EP003 자조), 감정 누수 약점은 EP006 첫 노출 → EP012(태산) → EP015(지유) → EP024(군중)으로 점진 강화

## 3. 신규 캐릭터 EP별 등장 디테일

### 3-1. 이도현 (李道現, 30세) — EP008 첫 등장

#### 첫 등장 신 (EP008)

**무대**: 협회 본부 5층 데이터 분석 1팀 사무실. 강해가 무능력자 부서 행정 보조로 자료를 전달하러 들름. 다른 직원들은 다 퇴근했고, 도현 혼자 야근 중. 모니터 3개에 차트와 데이터.

**서사 기능**: 강해를 "흥미로운 데이터"로 발견 → 강해의 능력 정체를 추적자 시점에서 점진적으로 좁혀 들어감.

**첫 대화 다이얼로그 (5~10턴)**:

> 도현: 안강해 씨. 잠시만요.
> 강해: ...네.
> 도현: 이거 한번 보세요. (모니터를 강해 쪽으로 돌림)
> 강해: ...뭡니까.
> 도현: F급으로 등록되어 계시죠. 근데 최근 3주간 솔로 클리어가 7건이에요. E급 던전 4건, D급 던전 3건. D급 헌터 평균 클리어 수가 월 5건이에요. 안강해 씨는 3주 만에 7건이고요. 이게 -- 통계학적으로 자연 발생 확률이 0.003% 이하예요.
> 강해: ...운이 좋았어요.
> 도현: (안경 살짝 올리며) 이게 운이라면, 안강해 씨는 지금 로또 1등을 한 사람인데요. 7번 연속으로요.
> 강해: ...뭐, 그래도 운이라고 보고하시면 안 됩니까.
> 도현: 보고를 한다, 안 한다 -- 이게 지금 제 고민이에요. 보고를 하면 분석팀 재조사 들어갈 거고, 그러면 안강해 씨 데이터가 노출됩니다. 안 하면 -- 데이터를 숨기는 게 제 일은 아닌데요.
> 강해: ...아, 네.
> 도현: 일주일만 시간 주세요. 제가 패턴을 더 모아볼게요. 그동안 솔로 클리어 횟수, 줄여주실 수 있을까요.
> 강해: ...일주일이요.

**Dialogue DNA 발화 디테일**:
- 통계 수치를 대사에 자연스럽게 섞음 ("0.003% 이하", "7번 연속")
- 흥분 시 괄호 삽입 ("이게 -- 통계학적으로")
- 근거 → 결론 순서 (강해의 결론 → 근거와 정반대)
- 안경 살짝 올림 = 핵심 비언어 1번

#### EP008~025 동안 강해와의 관계 변화

| EP | 관계 단계 | 핵심 사건 | 도현의 화법 변화 |
|----|---------|---------|--------------|
| EP008 | 추적자 → 정보 동맹 직전 | 첫 만남, 일주일 유예 | 합쇼체 + 통계 수치 강조 |
| EP009 | 정보 동맹 직전 → 동맹 첫 합 | 강해가 던전 데이터 일부 공유, 도현이 등급 측정 가중치 자료 일부 제공 | 합쇼체 유지 + "확률" "분포" 빈출 |
| EP011 | 동맹 첫 합 → 보호자(데이터 측) | EP010 콤보 후 헌갤 폭발, 도현이 익명으로 "측정 알고리즘 편향" 데이터 흘림 | 흥분 시 문장 길어짐 |
| EP016 | 보호자 → 적극 협력자 | 강해의 콤보 데이터를 추가 분석, 5대 길드 정보망 동향 공유 | 합쇼체 + "유의수준 0.01" 같은 장치 |
| EP018~020 | 적극 협력자 (장세영 격파 정보전 보조) | 헌갤·SNS 동향 실시간 공유, 헥사 내부 강등 보도 분석 | 흥분 시 빨라지는 말 첫 등장 ("이거 보셨어요? 이거! 이거요!" 단편 첫 등장) |
| EP021~023 | 내부 갈등 본격화 직전 | 진현승 영입 제안 + 권도영 등급 재심사 거부의 맥락 분석. 도현이 권도영 직속 부하임이 노출됨 | 합쇼체 유지하되 강해 앞에서만 미세하게 개인적 톤 |
| EP024~025 | 정보전 핵심 | 길드전 빌드업, 등급 측정 가중치 조작의 로그 추적 시작 | 강해의 무덤덤에 도현이 답답해함 ("좀 -- 이게 얼마나 큰 건지 아세요?") |

**EP025 직전 내부 갈등의 씨앗**: EP023에서 권도영이 강해 등급 재심사를 직접 거부하는데, 도현이 그 회의에 배석. 도현은 권도영의 부하 → 보고서를 올릴 의무 vs 강해 보호 사이의 갈등 시작. EP028~ 본격화의 복선 (큰 설계 §3 도현 갈등 포인트와 일치).

### 3-2. 박태산 (朴泰山, 32세) — EP012 첫 등장

#### 첫 등장 신 (EP012)

**무대**: 서울 남부 D급 던전 "지하주차장 균열". 강해가 솔로 클리어 도중, 같은 던전에 진입한 솔로 헌터 박태산과 우연 조우. 워그 무리에 둘러싸인 상황. 태산이 탱킹 + 강해가 후방.

**서사 기능**: 강해의 첫 인간적 동료. 코드 분석으로 태산 스킬의 버그를 발견 → 패치 → 태산이 감동 → 의리로 합류. 강해의 "찾아내는 눈"이 타인을 위해 처음 작동하는 신.

**첫 대화 다이얼로그 (5~10턴)**:

> 태산: (워그 5마리에 둘러싸인 상황) 야! 거기 너! 거기 있으면 뒤져!
> 강해: ...네.
> 태산: 일로 와! 내 뒤로! 빨리!
> 강해: (조용히 태산 등 뒤로 이동) ...아.
> 태산: (방패 들고 워그 정면 돌격) [방어_벽] -- 야 너 등급 뭐야? F급이야? 미친놈, F급이 D급 던전을 왜 와!
> 강해: ...네, F급 맞습니다.
> 태산: 일단 살리고 본다! [그라운드_슬램]! ...아 이거 또 미스. (스킬 발동 실패)
> 강해: (태산의 스킬 코드를 시야에 시각화) ...저, 형님.
> 태산: 형님? (잠깐 멍) 어, 그래, 형님!
> 강해: ...그라운드 슬램, 발동 조건에 `if (target.count >= 3)` 들어 있어요. 두 마리 이하면 안 나가요.
> 태산: 뭐? 뭔 소리야!
> 강해: 적이 세 마리 이상이어야 발동인데, 지금 한 마리씩 떨어져서 오니까 트리거가 안 걸려요. ...등 돌리시고 모아주세요. 한 번에.
> 태산: ...야 너 그거 어떻게 본 거야! 아니 진짜로! 나 그 스킬 3년 썼는데 몰랐거든!
> 강해: ...일단 모으세요.
> 태산: 어! 어어!! (워그 3마리 한쪽으로 몰아넣음)
> 태산: [그라운드_슬램]! 와 -- 됐다! 됐어!! 야 너 진짜야 너!

**Dialogue DNA 발화 디테일**:
- 짧은 문장, 감탄사 폭발 ("미친놈", "와", "야 너 진짜야 너")
- 반말 즉시 돌입 (존댓말은 "형님?" 한 번만)
- 의성어/의태어: "어! 어어!!"
- 사고 패턴: 직감 → 행동 → 사후 합리화. "일단 살리고 본다"
- 어깨 두드리기 + 호탕한 웃음 = 핵심 비언어

#### EP012~025 동안 강해와의 동료화 과정

| EP | 관계 단계 | 핵심 사건 | 태산의 변화 |
|----|---------|---------|---------|

| EP025 | 길드전 핵심 전력 | 길드전 콤보 폭발 신. 태산이 정면 탱킹 + 강해의 콤보가 헥사 간부를 격파 | 환호 폭발 ("이야!! 봤지 형이 뭐랬어!!") |

**EP025 시점 태산의 위치**: 강해를 "동생" 또는 "친구"로 인지. 의리 핵심. 강해의 능력 정체(코드 편집)를 어렴풋이 알지만 깊이 안 따져 묻는 캐릭터. "어려운 건 모르고, 같이 싸우면 동료" 노선 유지.

### 3-3. 한지유 (韓智裕, 26세) — EP015 첫 등장

#### 첫 등장 신 (EP015)

**무대**: 서울 동부 D급 던전 "잠실 구상가". 협회 공식 합동 클리어 작전 중 사고 발생. 한지유(아르케 길드 소속, EP015 시점에는 아직 탈퇴 직전 상태)가 절대영도를 발동했는데 범위가 통제 불가 → 아군 헌터 3명이 범위 안에 갇힘 → 사망 위기. 강해는 마침 협회 행정 보조로 던전 입구 보고대에 있다가 사고 보고를 듣고 진입.

**서사 기능**: 히로인 첫 만남. 강해가 지유의 절대영도 코드를 읽고 "비활성 변수"의 존재를 즉시 발견. 지유는 충격과 경계를 동시에. "당신, 내 스킬을 어떻게 아는 건데요."

**첫 대화 다이얼로그 (5~10턴)**:

> (사고 직후, 던전 내부. 절대영도 범위가 풀리지 않아 아군 3명이 영하 200도에 노출. 지유는 범위 해제 시도 중이지만 실패 반복.)
> 지유: 비키세요. 다 비키세요.
> 헌터A: 한지유! 풀어!! 우리 죽어!!
> 지유: ...풀려고 하는 중이에요. (입술 깨물고 시선 회피)
> 강해: (멀리서 지유의 절대영도 코드를 시야에 시각화. 회색 점선 박스의 비활성 변수 발견) ...
> 강해: (지유에게 다가감) ...저.
> 지유: 누구.
> 강해: ...범위 변수가 있는데, 사용자 입력으로 안 묶여 있어요. 비활성 상태라서요.
> 지유: ...뭐?
> 강해: 코드를 보고 말씀드리는 건데요. `// double user_radius_m = ?` 라인이 회색이에요. 활성화되면 직접 반경 입력이 가능할 것 같은데.
> 지유: ...당신, 내 스킬을 어떻게 아는 건데요.
> 강해: ...설명은 나중에. 지금은 -- 범위를 의식해서 좁힌다고 생각하지 마시고, "반경 5미터" 이렇게 수치로 떠올려 보세요.
> 지유: ...그게 무슨.
> 강해: ...일단요. 부탁드릴게요. (3명이 범위 안에 갇혀 있음)
> 지유: (잠깐의 정적, 시선 강해에게 고정) ...반경 5미터.
> 지유: (의식 집중. 범위가 확연히 줄어듦. 헌터 3명 노출 면적 감소 → 회복 가능 영역으로 이동) ...됐어요.
> 강해: ...됐네요.
> 지유: ...당신 누구예요.
> 강해: ...안강해입니다.

**Dialogue DNA 발화 디테일**:
- 결과 역산형: "반경 5미터" (목표 결과) → 의식 집중 → 적용
- 핵심만 말함: "비키세요. 다 비키세요." (상황/배경/감정 생략)
- 감정 표현: 입술 깨물기, 시선 회피 = 핵심 비언어
- 합쇼체 기본 ("비키세요", "풀려고 하는 중이에요")
- 강해의 무덤덤 "...됐네요"에 지유가 미세하게 흔들리는 첫 신

#### EP015~025 동안 관계 변화

| EP | 관계 단계 | 핵심 사건 | 지유의 화법 변화 |
|----|---------|---------|------------|
| EP015 | 첫 만남 → 충격 + 경계 | 절대영도 사고. 강해 코드 분석으로 1차 해소 | 합쇼체 유지, "당신" 호칭 |
| EP016 | 검증 모드 | 지유가 강해의 말을 검증. 다른 조용한 던전에서 절대영도 재시도. 부분 성공 | 짧은 단답 ("...진짜네요") |
| EP017 | 협력 시작 | 범위 축소 + 지속 시간 조절 첫 성공. 강해의 자조 무덤덤에 지유가 미세 반응 ("...왜 그렇게 별 거 아닌 것처럼 말해요") | "안강해 씨" 호칭 시작. 합쇼체 유지하나 끝에 미세한 호기심 |
| EP018 | 동행 시작 | 헥사 측이 강해+지유의 결합을 감지. 영입 압박 신에 지유가 강해 옆에 서 있음. 한 마디 안 하고 분위기로 압박 차단 | 거의 무발화. 시선과 자세로만 |
| EP019 | 첫 3인 합 | 장세영 직접 전투에서 강해+태산+지유 합. 지유의 축소 절대영도가 결정타 | 전투 중 단답 ("범위 좁힐게요") |
| EP020 | 협력자 → 동료 | 장세영 격파 후 지유가 자진 합류 의사. "혼자 가겠다는 건 아니죠?" 첫 길게 말한 대사 | 합쇼체에서 짧게 반말 섞임 ("...가요") 시작 |
| EP021 | 동료 (정착) | 진현승 영입 제안 신에 지유 부재. 강해가 거절 보고 시 지유가 짧게 "잘했어요" | "잘했어요" = 지유의 첫 칭찬 |
| EP022 | 동료 + 보호자 | 헥사 보복 신. 지유가 강해 옆에 점점 더 자주. 같이 점심 첫 신 (편의점 삼각김밥 + 캔커피) | 점심 자리에서 첫 침묵 5분. 지유 먼저 깸 ("...별 거 안 시키네요, 점심.") |
| EP023 | 동료 + 코드 분석 파트너 | 권도영 등급 재심사 거부 신에 지유가 동행. 행정실 분위기에서 지유가 강해 대신 한마디 ("...규정대로 처리하시는 거 아닌 거 같은데요") | 짧은 단답이 처음으로 강해를 위해 발화됨 |
| EP024 | 코드 분석 + 실행 파트너 | 길드전 콤보 빌드업. 지유가 강해의 분석 노트를 옆에서 같이 본다. 코드를 못 읽지만 강해의 설명을 들음 | "이건 어떻게 되는 건가요" 같은 능동적 질문 첫 등장 |
| EP025 | 결정적 파트너 | 길드전 콤보 폭발 신. 지유의 절대영도 축소가 콤보의 핵심 한 축. 콤보 후 강해가 harvest_intention 발견 → 지유에게만 짧게 "...뭔가 이상한 게 보였어요" | 강해가 미스터리를 처음 공유한 인물. "...같이 보세요" |

#### 절대영도 코드 분석 신의 대사 5~10턴 (EP016, EP015 다음 화)

**무대**: 협회 외곽 폐쇄 D급 던전 "수도 정수장 균열" (낮은 위험도, 측정용으로 보존). 지유가 강해의 분석을 검증하기 위해 단독 방문 요청. 강해가 빌드 룸 환각으로 코드를 펼침.

> 지유: (방어 자세) ...안강해 씨.
> 강해: ...네.
> 지유: 한 번만 더, 정확히 보여주세요. 지난번 그 -- 코드라는 거.
> 강해: ...네. (시선 집중. 코드 시각화)
> 강해: 절대영도. A급. 지금 활성된 건 세 줄이에요. `freeze_temperature = -200.0`, `apply_freeze(target, MAX_RADIUS, INFINITE_DURATION)`, 그리고 보호 영역 내부에 또 뭐가 있는데 그건 잠겨 있고요.
> 지유: ...그게 잠겨 있다는 게 무슨 뜻인데요.
> 강해: ...읽을 수는 있는데 수정은 못 한다는 거예요. PROTECTED라는 표시가 붙어 있어서요.
> 지유: ...
> 강해: 근데 그 위에, 활성된 게 아니라 회색으로 죽어 있는 줄이 세 개 더 있어요. `user_radius_m`, `user_duration_ms`, `user_intensity`. 다 비활성. 사용자 -- 그러니까 한지유 씨가, 직접 입력할 수 있는 자리인데 그게 비어 있는 거예요.
> 지유: ...애초에 안 만들어진 게 아니고요.
> 강해: ...네. 만들어져 있는데 비활성. 슬롯이 비어 있는 거예요.
> 지유: ...누가 비워둔 건데요.
> 강해: ...그건 모르겠어요. 시스템이 그렇게 출고한 거 같은데요.
> 지유: ...그게 무슨 -- 차를 샀는데 핸들이 빠져 있는 거랑 비슷한 건가요.
> 강해: ...아, 비유는 좀 어렵고요. 그냥 -- 입력 자리가 있는데 안 묶여 있는 거예요.
> 지유: ...이 코드가 원래 있었다는 건가요.
> 강해: ...네.
> 지유: ...
> 지유: 다시 한 번 시도해볼게요. 반경 3미터. 지속 시간 5초.
> 강해: ...해보세요.
> 지유: (의식 집중. 절대영도 발동. 범위 3미터, 지속 5초로 정확히 적용됨) ...
> 지유: ...
> 강해: ...됐네요.
> 지유: ...코드를 수정한 게 아니라, 원래 있던 걸 읽어준 거네요.
> 강해: ...네, 뭐, 그렇죠.
> 지유: ...

**이 신의 의미**:
- 강해의 정체성("찾아내는 눈"): 비활성 변수를 코드 리뷰로 발견하는 능력
- 지유의 첫 능력 통제: 자기 폭탄을 처음으로 "다룰 수 있게 됨"
- 두 사람의 첫 협력: 강해는 읽고, 지유는 실행. 큰 설계의 "코드 분석 ↔ 실행 파트너" 첫 합

### 3-4. 장세영 (張世映, A급, 헥사 공략대장) — EP011 직접 등장

#### 첫 등장 신 (EP011)

**무대**: 강남 헥사 길드 본부 30층 회의실 "팔라딘 룸". 강해의 EP010 D+D+D 콤보 영상이 헌갤·유튜브에 익명으로 유출 → 헥사 정보망이 강해의 신원을 24시간 만에 특정. 헥사 공략대장 장세영이 강해를 호출. 강해는 거절할 수 없는 분위기에서 호출에 응함 (협회 공식 합동 작전 명분).

**서사 기능**: 아크1 중간 보스. F급을 무시하다 콤보 성과에 놀라, "이상한 놈이네" → 영입 압박 → 거절 시 직접 위협 → EP019 격파.

**첫 대화 다이얼로그 (5~10턴)**:

> (강해, 회의실 입구 자리. 장세영, 상석. 헥사 분석팀 직원 김선우가 옆에서 태블릿 들고 대기.)

> 강해: ...네.

> 강해: ...아.
> 장세영: 그게 -- 운이라고 보기엔, 너무 깔끔하더군요. 콤보 동기화 변수까지. (김선우 쪽을 살짝 돌아봄)

> 강해: ...아, 네. 괜찮습니다.
> 장세영: ...괜찮으시다는 게.
> 강해: ...영입 제안이라면, 거절하는 거예요.
> 장세영: ...
> 장세영: (잠깐의 침묵) ...안강해 씨. 정중하게 한 번 더 말씀드릴게요. 헥사는 능력 있는 분이 정당한 대우를 받는 곳입니다. 거기 -- 무능력자 부서? 행정 보조 일이 적성에 맞으세요?

> 장세영: ...

### 3-5. 최서진 (崔書珍, F급, 27세) — EP002 첫 등장

#### 첫 등장 신 (EP002)

**무대**: 한국 헌터 협회 본부 6층 무능력자관리과. 햇빛 안 드는 사무실. 형광등. 책상 30개. 박스 더미. 강해가 첫 출근.

### 3-6. 진현승 (陳現承, S급, 헥사 길드 마스터, 45세) — EP021 이름 / EP023 직접

#### 직접 등장 신 (EP023)

**무대**: 헥사 길드 본부 최상층 30층 마스터 룸. 통유리창 너머 강남 야경. 진현승 단독 면담. 강해가 정중한 호출에 응함 (거절하면 헥사 측 보복이 더 거세질 분위기). 동행: 한지유 (강해 옆 자리, 무발화).

**서사 기능**: 1층 얼굴 빌런 본격 진입. 정중한 영입 → 거절 → 일주일 뒤 조직적 보복 본격화의 시발점.

**첫 대화 다이얼로그 (5~10턴)**:

> 진현승: 안강해 씨, 한지유 씨. 와주셔서 감사합니다.
> 강해: ...네.
> 지유: ...
> 진현승: (앉으며) 단도직입적으로 말씀드릴게요. 헥사는 안강해 씨의 능력에 관심이 있습니다. 정확히 말하면 -- "스킬 최적화"라는 새로운 영역에 관심이 있고요.
> 강해: ...네.
> 진현승: 안강해 씨가 가진 능력은 -- 글쎄요, 본인의 자산일 수도 있고, 사회적 자산일 수도 있고, 또는 헥사의 자산이 될 수도 있죠. 어느 쪽이든, 제대로 관리하지 않으면 위험합니다.
> 강해: ...어떤 위험.
> 진현승: 능력자 보호 차원에서요. 안강해 씨가 모르시는 사이에 다른 길드나 외국 자본이 능력을 -- 제가 좋아하는 표현으로는 "수확"이라고 합니다만 -- 그러기 전에, 헥사가 보호하는 게 안전합니다.
> 강해: (자주색 코드 한순간 떠오름. EP025의 'harvest_intention'과 연관 어휘 '수확'이 진현승 입에서 미리 등장 — 복선) ...아, 네.
> 진현승: 헥사는 능력 있는 분들이 제대로 대우받는 곳입니다. F급이라는 등급이 당신의 전부는 아닐 테니까요. (잠깐의 정적) 같이 일해보시는 건 어떻습니까.
> 강해: ...아, 네. 괜찮습니다.
> 진현승: ...괜찮으시다는 게.
> 강해: ...거절입니다.
> 진현승: ...아쉽군요.
> 진현승: (지유 쪽으로 시선) 한지유 씨도 같이 거절하시는 거죠.
> 지유: ...네. 같이 거절합니다.
> 진현승: ...
> 진현승: 알겠습니다. 시장은 질서로 돌아갑니다, 안강해 씨. 질서를 무너뜨리는 개인은 -- 글쎄요, 시장이 알아서 처리하겠죠.
> 강해: ...네.
> 진현승: 오늘은 여기까지 하시죠.

**Dialogue DNA 발화 디테일 (진현승)**:
- 합쇼체 + 우아한 위압
- 정중한 압박: "어떻습니까", "안전합니다"
- 비용-편익 사고: "자산", "자원", "관리", "보호"
- 감정 지연. 거절 후에도 즉시 분노 X. "...아쉽군요" 한 줄
- 위협 발화: "시장이 알아서 처리하겠죠" (직접 명령 X)
- 최대 분노 표현: "실망했다" 류 (EP025에서 후속 등장)
- 핵심 비언어: 통유리창 야경 등지고 앉기. 손깍지. 종이컵 안 쓰고 머그잔.

**복선**: 진현승의 입에서 "수확"이라는 단어가 EP023에서 먼저 나옴. EP025 강해가 'harvest_intention' 함수를 발견했을 때 "...수확"이라는 단어가 머릿속에서 진현승의 발화와 겹침. 진현승이 시스템과 어떤 관련이 있는가 = 큰 설계의 EP060 "진현승도 수확 대상" 복선의 첫 씨앗.

| EP | 단계 | 진현승의 행동 | 강해 측 영향 |
|----|------|----------|---------|
| EP021 | 이름 언급 + 그림자 | 헥사 분석팀에 강해 영상 정밀 분석 지시 | 강해 모름 |

| EP023 | 직접 영입 → 거절 | 위 신 | 거절 후 일주일 유예 |
| EP024 | 보복 본격 | 보험 거부, 던전 출입 제한, 헌갤 비방 사주, 솔라리스(윤세하) 측 접근 차단 시도 | 강해 일상 봉쇄 |

### 3-7. 권도영 (權道營, C급, 협회 기술국장, 52세) — EP021 이름 / EP023 직접

#### 직접 등장 신 (EP023)

**무대**: 협회 본부 4층 기술국 국장실. 사무용 책상, 정장. 강해 호출. 권도영 단독.

**첫 대화 다이얼로그 (5~10턴)**:

> 강해: ...네.
> 권도영: 다만, 현행 측정 프로토콜에 따르면 재심사 요건을 충족하지 못하고 계시는데요.
> 강해: ...어떤 요건이.

> 강해: ...최근 던전 클리어 7건은요.
> 권도영: 그것은 측정 프로토콜과는 별개의 기록입니다. 클리어 기록은 헌터 활동 이력이고, 등급은 본질적인 마나 잔류 신호 기준입니다. 본질이 다르죠.
> 강해: ...
> 권도영: 물론 헌터 자격은 유지되고 있습니다만, 활동 이력에 비정상적 패턴이 감지될 경우 -- 보호 차원에서 활동 제한 조치가 내려질 수도 있습니다.
> 강해: ...
> 권도영: 안강해 씨를 위해 드리는 말씀입니다. 무리한 활동은 본인 안전을 해칠 수 있어요.
> 강해: ...아, 네. 알겠습니다.
> 권도영: 필요하시면 추가 면담 일정을 잡으시죠.
> 강해: ...

**Dialogue DNA 발화 디테일 (권도영)**:
- 합쇼체. 관료 화법

### 3-8. 윤세하 (尹世河, S급, 솔라리스 마스터, 33세) — EP021 간략 등장

**서사 기능**: "호의적이지만 의도 있는" 잠재 동맹. 진현승의 안티테제. 헥사 = 원청, 솔라리스 = 빅테크 모티프 대비.

#### 등장 신 (EP021, 간략)

**무대**: 판교 솔라리스 사옥 1층 라운지 카페. 진현승의 영입 제안 직전. 윤세하가 "잠깐 차 한잔" 명목으로 강해 자취방 근처 카페로 직접 와서 만남.

**대사 견본 (3~5턴)**:

> 윤세하: 안강해 씨. 시간 내주셔서 감사합니다.
> 강해: ...네.
> 윤세하: (캐주얼 후드 + 청바지) 솔라리스의 윤세하입니다. 헥사보다 먼저 만나뵙고 싶어서요.
> 강해: ...네.
> 윤세하: 결론부터 말씀드릴게요. 솔라리스는 안강해 씨를 영입하지 않습니다. 다만 -- 솔라리스가 파트너십 이외의 형태로 협력할 수 있을지, 그게 궁금했어요.
> 강해: ...파트너십.
> 윤세하: 정확히는, 안강해 씨가 솔플로 활동하시되, 솔라리스가 후방 지원만. 던전 정보, 보험, 행정. 그 정도. 강해 씨 능력은 강해 씨 거예요.
> 강해: ...아, 네.
> 윤세하: 지금 답 안 주셔도 됩니다. 헥사 만나신 다음에, 차이를 비교하시고 알려주세요. 저는 안강해 씨가 어떤 결정을 내려도 응원합니다.
> 강해: ...네, 알겠습니다.

**Dialogue DNA 발화 디테일 (윤세하)**:
- 해요체 + 캐주얼
- 짧고 명료한 결론. "파트너십 이외의 형태"
- 비용-편익 X, 가치 정렬 O ("강해 씨 능력은 강해 씨 거예요")
- 핵심 비언어: 후드티, 청바지, 직접 1층 라운지로 내려옴
- 진현승의 정중한 위압과 정확히 반대 톤

**EP025 시점 윤세하 위치**: 강해와 직접 손잡지는 않음. 길드전에서 솔라리스는 중립. 그러나 헥사의 보복에 대해 "솔라리스 시설은 자유 출입 가능"이라는 미세한 우호 조치. EP029에서 본격 재등장 예정 (이번 25화 구간 외).

## 4. 적대자 활동 타임라인 (EP001~EP025)

### 4-1. 장세영 (아크1 중간 보스) 곡선

```
EP001~009 : 부재. 강해는 적대자가 없는 상태에서 능력 자각
EP010    : ▲ 이름 언급 (헥사 정보망 보고서)
EP011    : ● 첫 직접 대면 → 영입 제안 → 거절
EP012~013: ◯ 간접 압박 (강해 협회 보고서에 "정신 건강 재검진" 의견 시도)
EP014    : ● 헥사 회의실 재호출 → 거절 → "실망스럽군요"
EP015~017: 부재 (지유 신 + 협력 검증)
EP018    : ● 직접 도발 (던전 진입로 차단 + "이번 기회에 한번 보여주시죠")
EP019    : ● 직접 전투 → 격파
EP020    : ▲ 헥사 강등 보도 + 헌갤 폭발
EP021~  : 부재. 진현승 명령으로 후속 아크에서 재등장
```

**격파 메커니즘 (EP019)**:
1. 강해의 코드 분석 → 장세영 스킬 [심판의 일격, A급]에 발동 조건 `if (caster.morale > 0.7)` 발견. 심리적 안정 상태 의존
2. 태산이 정면 탱킹 + 도발로 장세영의 평정심 흔들기
3. 지유의 절대영도 축소 적용으로 장세영 발 묶기 (3미터 반경, 5초 지속)
4. 장세영의 morale이 0.7 이하로 떨어지는 순간 [심판의 일격]이 발동 실패 → 강해의 D+D+D 콤보 폭발 → 격파

### 4-2. 진현승 (헥사 마스터) 곡선

```
EP001~020 : 부재
EP021    : ▲ 이름 언급 (영입 제안 그림자)
EP022    : ◯ 측면 정찰 (강해 SI 회사 압박)
EP023    : ● 직접 영입 제안 → 거절 → "...아쉽군요"
EP024    : ● 본격 보복 (보험 거부, 출입 제한, 비방 사주)
EP025    : ● 길드전 공식 신청 → 콤보에 1차 패배
```

**EP022 측면 정찰 디테일**: 진현승 측이 강해의 SI 회사 발주처(원청)에 헥사가 영향력 행사. 발주처 PM 송영주를 통해 강해의 회사에 "안강해라는 직원의 신용도가 의심된다"는 비공식 정보 흘림. 강해 회사에서 강해에게 추가 야근 + "이상한 사적 활동 자제하라" 압박. 두 세계 갑질의 동기화. 강해는 처음에는 우연인 줄 알다가 EP024쯤 인지.

### 4-3. 권도영 (협회 국장) 곡선

```
EP001~020 : 부재
EP021    : (배경) 헥사 로비 수령
EP023    : ● 직접 등장 → 등급 재심사 거부
EP024    : ◯ "활동 제한 조치" 검토 시작
EP025    : ● 길드전 공식 등록 지연 시도 → 실패
```

### 4-4. 시스템 (2층 빌런) — EP025 첫 시그널

```
EP001~024 : 부재 (코드 시각화는 작동하지만 "시스템"으로서의 적대 시그널 없음)
EP025    : ● 'harvest_intention' 자주색 함수 한순간 깜빡임. 보호 영역 안쪽에서 잠깐 노출 후 사라짐. 강해가 다시 보려 해도 안 보임. "...분명히 봤는데."
```

EP025의 시그널은 "시스템이 강해를 인지하기 시작했다"의 첫 신호이자, 큰 설계의 EP040~ "시스템 적대 시작" 복선의 첫 씨앗.
