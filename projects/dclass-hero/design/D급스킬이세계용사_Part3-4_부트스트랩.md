## 1. 작품 컨셉 (재확정)

### 1-1. 한 줄 로그라인 (Part 3·4 확장판)

> 사상 최초로 살아남은 용사 강지호가 부왕(夫王)으로서 9기둥의 마지막 7개를 치유해 가는 도중, 무협 차원에서 강림한 세 인물과 갓 태어난 아들 우주를 만나면서, 본인의 수명이 다하기 전에 아들에게 9기둥의 책임을 어떻게 넘겨줄지를 두고 부자(父子)의 길을 함께 걷는 이야기.

### 1-2. 장르 태그

`이세계 선형 멸망 저지물` × `정통 판타지 성장물` × `무협 차원 융합` × `세대 교체형 부자 서사` × `1인칭 자조` × `회귀 X / 루프 X / 영생 X`

### 1-3. 핵심 셀링포인트 (Part 3·4 신규)

1. **세대 교체형 9기둥 완결** — Part 1·2(EP001~058)는 강지호 단독 무쌍이었지만, Part 3 중반(EP120 전후)부터 아들 우주가 후계자로 합류한다. 한국 웹소설에서 드문 **"부자 공투"** 구조.
2. **무협 차원 융합** — EP058에서 강림한 카에데(무녀)·시즈루(여무사)·노인(은거 고수)이 정통 판타지 무대에 동양풍을 더한다. 우주천마 3077·무림 속 외노자 등 검증된 메가트렌드의 변형.
3. **결말 톤 = 미세 노화 + 평화 + 곁에 있음** — 강지호 죽음 X, 영생 X. 7기둥 완결 후 D급 재생이 점진 약화되며 자연 노화 진입. 자조 톤 유지.
4. **3 페이즈 페이싱** — Phase A(아빠 메인) → Phase B(동반자, 위험 구간) → Phase C(아들 메인, 아빠 후견인)의 명시적 전환 구조.

### 1-4. 기존 차별점 유지 (EP001~058 정본)

- D급 재생 + 무재능 = 다중 체계 흡수 (정령술 + 문신 코딩 + 기후 프로그래밍)
- 사상 최초로 기둥에 흡수되고도 살아남은 용사
- 1인칭 자조, 영웅적 결의 0회
- 빌런(천인 평의회·검은 로브) 의도적 약화 → 미스터리 라인은 부차

## 3. 무협 차원 캐논 (신규)

### 3-1. 위치·관계

- **명칭**: 잠정 **"동천(東天)"** — 본문 노출 시 노인이 한 번만 발화 ("나는 동천에서 왔노라" 식). 강지호 1인칭에서는 "노인의 고향" / "차원 저편"으로만 지칭.
- **위치**: 아젤리아·솔라리스·카이젤 차원과 **별개 차원**. 단, **용화국(동방 선계, 5번 지역)과 좁은 통로로 연결**된 듯한 설정. 우주가 자라면서 용화국·동천의 양쪽 인연을 잇는 매개체가 됨. 정확한 통로 위치·기원은 Arc 13(용 기둥) 진입 시 plot-hook-engineer가 회수.
- **EP058 강림 경로**: 카에데가 "오래도록 기다려온 그 존재"(=우주)의 출생 기운을 감지하고 노인·시즈루를 데리고 차원의 균열을 통해 강림. **균열은 일회성, 자유 왕복 불가**.

### 3-2. 사문 / 위계

- **사문 명칭**: **"청월문(靑月門)"** — 카에데·시즈루·노인이 함께 속한 은거 문파. 동천의 정파 중 하나로 추정. 노인이 장문인급 노고수, 카에데가 신녀 직위, 시즈루가 검녀(수석 제자) 직위. **풀이 1줄**: "청월문은 동천의 은거 문파다. 별을 읽고 검을 단련한다. 그 정도면 됐다." (강지호 자조 톤으로 풀이)
- **위계 용어 차용 금지** (R-A 권장): 사숙·사조·장문인 등은 본문 1회 등장 시마다 1줄 풀이. 평균 EP당 무협 위계 용어 1개 이하.
- **사문 운명**: 청월문은 우주의 출생을 1세대 전부터 예언한 상태. EP058 강림은 사문의 마지막 운명. 즉 **세 명이 동천에 돌아갈 가능성은 낮음** — 본 작품 결말까지 아젤리아에 머물러 강지호·우주의 곁을 지킨다.

### 3-3. 내공·축기·검술 용어 차용 범위

| 영역 | 차용 OK | 차용 시 풀이 필수 | 금지 |
|---|---|---|---|
| 무공 용어 | 내공, 단전, 축기, 검기, 검강, 심법, 운기조식, 비급 | 화경/현경 (단계명만 유지, 의미는 풀어 설명) | 사문(師門), 사숙, 장문인, 천기누설 등 위계·서사 클리셰 |
| 인물 화법 | 노인 = 풀 고풍체 / 카에데 = 짧은 한국어 + 신녀 풍 / 시즈루 = 현대 한국어 + 검술 시그니처 | — | 시즈루·카에데 고풍체 |

**평균 한도**:
- EP당 고풍체 대사 5줄 이하
- EP당 무협 위계 용어 1개 이하
- 무공 시그니처 비트(축기·운기조식 등) EP당 1~2회

### 3-4. 무협 차원이 EP059~EP158에 미치는 영향

| 영역 | 영향 |
|---|---|
| **강지호 회복 (EP059~075)** | 노인의 응급 처치 + 운기조식 보조 + 동천의 약초/단약 사용. D급 재생만으로는 회복 불가했던 쇳독을 무협 의술이 보완. **단, 강지호가 무공을 배우는 일은 없음** — 노인이 잠시 도울 뿐. |
| **우주 멘토링 (EP080~)** | 카에데가 우주에게 **무협식 호흡·자세** 수련을 가르침. 가치관·정체성은 강지호 영역, 신체 단련은 카에데 영역. |
| **시즈루의 검술 (EP076~)** | 시즈루가 강지호의 동료 겸 호위로 7기둥 마라톤에 동행. 검술 vs 코드의 대비. 강철·빙·용 기둥 치유에서 검기 합공 1~2회. |
| **노인의 결정적 조언 (Arc당 1~2씬)** | 곰방대 + 술잔 + 코미디 비트. 큰 결단의 순간에만 짧은 명언. |
| **카르무스(흑철 제국 잔당)와의 관계** | 무협 차원 인물이 카이젤 보호자에게 어떻게 보이는가? — 카이젤 시스템 관리자 입장에서 청월문 3인의 강림은 **변수**. Arc 11 진입 시 활용. |

**경고 (R-A 회피 함정)**:
- 무협 차원 백스토리 과다 노출 X. EP059~080 사이에 사연을 다 풀면 우주 성장 축이 묻힘. **백스토리는 Arc 13(용 기둥) 부근에서 점진 노출.**
- 무협 우월주의 톤 X. 강지호 코드 능력과 시즈루 검술은 **"다른 길"**.

## 4. 우주(아들) 캐논 (신규)

### 4-1. 출생 시점 능력 떡밥 (EP058 노출분)

- **EP058 카에데와 눈맞춤**: "갓 태어난 아기라기엔 지나치게 깊고 고요한 우주의 눈동자가 카에데를 빤히 응시했다. 카에데의 눈동자 깊은 곳에서 은빛 안개가 소용돌이쳤다." → 정본 확정.
- **떡밥 해석 (Part 3·4에서 회수)**:
  1. 카에데가 1세대 전부터 예언한 "오래도록 기다려온 그 존재" = 우주.
  2. 우주는 **3중 혈통의 첫 사례**:
     - 父 강지호의 D급 재생 + 코드 능력 (외부 시스템 디버깅)
     - 母 에리나의 정령 감응 + 왕족 혈통 (정치적 인계 권한)
     - 청월문이 예언한 동천의 기운 (무협식 신체 + 정령 감응의 통합)
  3. 우주가 9기둥 인계의 적임자인 이유 = **시스템 관리자(모르데우스 류)도 아니고, 외부인(강지호 류)도 아닌, 인계 권한을 자연적으로 갖춘 자**.
  4. 카에데는 우주의 능력 본질을 EP058에서 이미 알아보았지만, **본인 입으로 명시하지 않음**. Phase B(EP100 전후)에서 점진 노출.

### 4-2. 0세~10세 성장 시간선

| 페이즈 | EP 범위 | 우주 나이 | 능력 단계 |
|---|---|---|---|
| **A1** | EP059~EP070 | 0~1세 | 정상 갓난아기 + 미세 이질감 1~2 비트/EP (옹알이가 정령에게만 반응 등) |
| **A2** | EP071~EP080 | 2~3세 | 짧은 어림 한국어 ("아빠 가" / "검 빛나"). 카에데와 비밀 의례 시작 |
| **B1** | EP081~EP100 | 4~6세 | 부모 모방, 능력 자각 시작. 강지호 코드 흉내 → 실패 코미디. 카에데의 호흡 수련 본격화 |
| **B2** | EP101~EP130 | 7~8세 | 자기 능력 결 자각: **"무공+정령+코드 융합체"**. 짧은 활약(EP당 5~15%) |
| **C** | EP131~EP158 | 9~10세 | 메인 활약(EP당 50~70%). 자조 톤 흡수 ("내가 이렇게까지 해야 돼? ...해야 되겠지") |

**시간 압축 챕터 3회 (R-C 권장)**:
- **EP080~EP081**: 0~3세 → 4세 도약 (몽타주, 1화)
- **EP110~EP111**: 4~6세 → 7세 도약 (몽타주, 1화)
- **EP130~EP131**: 7세 → 10세 가까이 도약 (몽타주, 1화)

각 압축 챕터 의무 포함:
- 일상 비트 3~5개
- 외부 사건 1~2개 (다른 인물 시점)
- 능력 자각 미세 비트 1~2개
- **사이다 비트 1개 (의무)**
- **부자 정서 비트 1개 (의무)**
- 강지호 자조 해설 1~2개

### 4-3. 우주의 핵심 차별점 — "무공+정령+코드 융합체"

| 영역 | 강지호 (父) | 우주 (子) |
|---|---|---|
| **본질** | 외부 시스템을 디버깅(편집) | **두 체계의 통역 + 새 규칙 작성** |
| **트리거** | 손바닥 + 정령 인장 + 문신 코딩 | **시선 + 호흡** (가장 자연스러움) |
| **한계** | 코드로 풀 수 없는 영역 존재 | **너무 어림** — 신체·집중력 한계 |
| **성장 곡선** | EP058 시점 거의 정점 (이후 점진 약화) | EP059 0세 → EP158 10세 점진 상승 |
| **에피소드별 시연** | EP019 빛 / EP021 태양 / EP055~56 흑철 | EP110 첫 단독 사이다 / EP120 카에데 인정 / Phase C 7기둥 합공 |

**아젤리아 혁신 가드레일 §13 적용**: 우주의 능력도 **프로그래머 가능 범위 + 호흡 수련 영역**에 한정. 증기·화약·야금·전기 자체 발전 절대 금지.

### 4-4. 우주의 결말 (EP158)

- 우주 = 9기둥 중 마지막 1~2개를 본인 손으로 풀이.
- 강지호 = 결정적 순간에 곁에서 자조하며 미소. D급 재생은 점진 약화로 노화 진입 중.

## 10. character-architect / plot-hook-engineer에게 전달할 핵심

### 10-1. character-architect에게

1. **우주의 0세~10세 능력 떡밥**: 3중 혈통(父 코드 + 母 정령 감응 + 청월문 동천 기운). EP058 카에데와 눈맞춤 = 1세대 전 예언 회수. 능력 본질 = "무공+정령+코드 융합체". 카에데·강지호만 알아봄.
2. **무협 3인의 동기·관계**: 카에데(중비중 우주 멘토, 신녀, 1세대 전 예언자), 시즈루(고비중 강지호 동료/라이벌, 검녀), 노인(저비중 사부, 청월문 장문인급, EP155 자연사). 셋 다 청월문 소속. 동천으로 돌아갈 가능성 낮음.
3. **4자 관계도(지호·에리나·우주·카에데)**: 지호 = 부친 자조 + 결정적 펀치 / 에리나 = 여왕 + 정치 인계 권한 / 우주 = 새 세대 / 카에데 = 우주의 멘토(가치관 X, 신체 영역만). 부자 갈등 X, 부부 갈등 X.
