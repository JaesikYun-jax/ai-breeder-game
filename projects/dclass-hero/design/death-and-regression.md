## 1. 재생 스킬 (D급)

### 본질
- 강지호가 1화 사이 공간(에 해당하는 어둠)에서 선택한 D급 스킬. 등급은 1인칭 본문에서 자체 인식하지 않고, 제3자/시스템 언급 시에만 표면화.
- "안 죽는다"가 아니라 "끝없이 손상되고 끝없이 복구된다"는 메커닉.

### 작동 원리
| 항목 | 내용 |
|------|------|
| 발동 | 자동/상시. 본인 의지로 ON/OFF 불가 |
| 복구 대상 | 신체 손상 — 피부, 근육, 장기, 신경, 골수 |
| 복구 속도 | 손상 종류에 비례. 표면 화상 ~15분, 혈관 내부 손상 기능 복구 ~5분, 근섬유 ~15분, 갈비뼈 골절은 전투 중 부분 회복 |
| 통증 처리 | **복구 못 함**. 신경이 새것으로 재생되므로 매번 첫 번째와 동일한 강도의 고통 |
| 정신 | 의식이 끊기지 않음. 다른 사람이라면 즉사·기절할 상황도 의식 유지된 채 통과 |

### 재생이 못 하는 것 (핵심 한계)
1. **고통 기억**: 신경이 재생되니까 통증 내성이 누적되지 않음
2. **피로**: 세포 단위 에너지 소진은 복구 안 됨. 폭식으로만 보충
3. **수명 소모**: 정령 계약이 부과하는 수명 대가는 재생 영역 밖 (※ 후술)
4. **정령 인장**: 화염 계약 자국, 번개 리히텐베르크 문양은 정령의 표식이라 재생이 건드리지 않음
5. **문신 코딩 잉크**: 정령 잉크가 충분히 깊이 스며들면 정상 자산으로 인식되어 재생이 공격하지 않음

### 본문 활용 사례
| 화 | 활용 |
|----|------|
| 5화 | 빛 기둥의 분해-재생 루프를 6사이클 견딤. 죽지 않고 기둥에서 뱉어짐 |
| 8~10화 | 화염 정령 계약. 일반 술사는 화상이 누적되어 죽지만 지호는 매 시도 후 화상이 소멸 → 야간 반복 훈련 ("남들의 일주일을 하룻밤") |
| 13화 | 열사병 증상이 1~5초 단위로 왔다가 사라짐. 외부에서는 "안 쓰러지는 사람"으로 보이지만 내부에서는 끊임없는 재발-복구 |
| 15화 | "벼락 마당"에서 21회 감전·심장 정지. 매번 재생 후 다시 일어남 → 역사상 최초의 번개 정령 계약 |
| 16화 | 번개로 근섬유를 의도적으로 파괴 → 재생 = 운동 초과 회복의 압축판. 스쿼트 3개 → 100개 |
| 19화 | 빛의 기둥 직접 접촉. 손바닥의 분해-재생 루프를 10사이클 이상 견디며 기둥 코드 리팩토링 |

## 2. 모래시계 (재정의)

### 정체
- **세상 모든 기둥의 잔여 시간을 가리키는 카운트다운 지표.**
- 마르코가 4화에서 깨진 유리 팔찌 형태로 지호에게 떠넘겼지만, 이는 마르코가 "주인을 정해 보낸 것"에 가깝다 (마르코 정체는 미공개).
- 지호가 능동적으로 발동하는 도구가 **아니다**. 시간정지 효과를 의지로 사용한 적이 없다.

### 본문 사실
| 시점 | 모래량 | 설명 |
|------|--------|------|
| 4화 (입수) | 절반 | 마르코에게 받음. 깨진 유리, 안에 모래 반쯤 |
| 5화 (기둥 흡수 중) | 일순간 은빛 발광 | 기둥의 분해-재생 루프 도중 자동 발동처럼 보이는 0.3초의 정지가 일어났음. 이는 "지호의 시간정지 도구 작동"이 아니라 "기둥이 자기 카운트다운에 닿은 순간, 모래시계가 그 동기화를 가시화한 일회성 현상"으로 재해석 |
| 5화 (기둥에서 뱉어진 직후) | 약 1/4 | 솔라리스 도착 직후 확인. 절반에서 1/4 정도로 감소 |
| 13화 | (감지) "용량이 줄어든 느낌" | 체내 순환 훈련 후 모호한 감각. 수치 미확인 |
| 19화 (빛 기둥 리팩토링) | 은빛 0.1초 발광 | 기둥 내부 일별. 갇혀 있던 영혼들의 표정이 달라 보임 |
| 20화 | 약 1/4 잔존 | "한때 절반이었던 모래가 이제 4분의 1 정도만 남아 있다" |
| 21화 (Part 2 시작) | 1/4 (잔량 동일) | 두 번째 태양 기둥 치유 직후 두 달 경과. 평소 다섯을 세면 한 알의 박자가, **반 박자 빠르게 한 알 떨어짐**. 잔량 변화 X, 박자만 어긋남. "그게 더 무섭다" |
| 22화 | 1/4 | 마르코의 정체 확인 인계 — "9개 기둥 전체의 카운트다운". 들여다보는 동안 박자는 어제와 같았으나 손바닥이 차가워짐 |
| 24화 | 1/4 | 박자 어긋남 폭 확대. 첫 알·둘째 알 사이 짧음 → 셋째 알이 둘째에 거의 붙어 떨어짐 → 두 알이 한 박자에 떨어진 것에 가까움. **잔량 그대로인데 박자만 빨라지는 게 더 무서움** |
| 25화 (솔라리스 출발 전날 밤) | 1/4 (한 알 감소) | **위쪽 무더기에서 한 알이 위에서 아래로 떨어지는 게 보임.** 한 알이 떨어지는 데 작은 소리 한 번. 어제까지의 결과 한 알 차이. "세계의 시간이 줄었다. 한 알만큼." 명시적 첫 감소 |
| 28화 (솔라리스 의례 종료 후) | 1/4 (또 한 알 감소 추정) | 두 번째 어긋남. "박자가 한 호흡 어긋났다는 확신". 위쪽이 한 알 분만큼 내려앉은 색, 아래쪽이 그만큼 올라앉은 색. 알 사라졌다는 확신은 아니었으나 박자 어긋남 확정 |

### 의미
- **잔량 감소 = 세상의 시간이 줄어드는 것** (지호 개인 수명과는 별개).
- 9개 기둥 중 빛의 기둥 1개를 정상화했지만 모래는 더 이상 줄지 않고 (혹은 미세하게만) 머무름.
- 8개 기둥이 남았고, 모래가 다 떨어지면 어떻게 되는지는 본문에서 미공개 = 후속 떡밥.

### 무엇이 **아닌가**
- 지호의 의지로 발동하는 시간정지 스킬이 아니다.
- 기둥 통과 횟수의 카운터가 아니다.
- 회귀 횟수 표시기가 아니다 (회귀 자체가 없다).

## 3. 기둥 흡수 = 일회성 사고

### 5화 사건의 정확한 메커닉
1. 모르데우스가 의식 발동 → 빛의 기둥이 지호의 영혼/마나를 흡수 시도
2. 일반인이라면 **수 초 내 의식 소멸 → 기둥에 영혼이 갇힘** (역대 7명의 갇힌 용사처럼)
3. 지호는 D급 재생 때문에 분해 속도와 재생 속도가 길항 → 분해-재생 루프가 6사이클
4. 기둥이 "흡수도 소멸도 안 되는 이물질" 처리 실패 → **물리적으로 뱉어냄**
5. 뱉어진 위치가 솔라리스 사막 (공간 점프 같은 명확한 기제는 본문에서 설명 안 함, "뱉어졌고 깨어보니 사막"의 모호함 유지)

### 19화 (재접촉)
- 같은 빛의 기둥에 의도적으로 손바닥을 댐
- 기둥의 결계가 "이물질"인 지호를 거부 → 완전 흡수는 일어나지 않음
- 접촉부에서 분해-재생 루프만 반복 → 그 사이 문신 코딩의 감지 함수가 기둥 내부 코드를 읽음
- "사람의 생명력 흡수" 라인이 **300년 전 누군가 심어놓은 기생 코드**임을 발견 → 우회 경로를 정령 잉크로 새겨 리팩토링

### 결론
기둥 흡수는 **반복 메커닉이 아니라** 5화의 일회성 사고 + 19화의 의도적 단발 접촉, 두 사례뿐이다.
"솔라리스 기둥은 어떻게 흡수되나" 같은 지역별 흡수 양상 카탈로그는 폐기.

## 4. 정령 계약의 수명 대가 (RETCON-C 본진)

### 일반 정령술사
- 머리카락 탈색이 즉시 시작 (나이라: 14세 계약 → 24세 시점에 앞머리 1줄기 백화 → 폭풍 방벽 무리한 직후 1/3 백화)
- 화상 흉터가 사용 시마다 번짐. 어깨를 넘으면 심장에 닿음 → "정령의 곁으로 간다" (사망)
- 솔라리스 정령술사 평균 수명 **35세 전후**. 일찍 계약한 자는 20대 사망.
- 호위 정령술사 다수가 30~40대에 전사하거나 화상으로 은퇴.

### 강지호의 비대칭
- D급 재생이 **표면 화상은 즉시 복구**하지만, 수명 침식 자체는 모래시계와 별개의 깊은 층위에 누적되는 것으로 추정.
- 13화에서 체내 순환 후 "용량이 줄어든 느낌" — 수명 대가가 들어오기는 한다는 암시.
- 화염(8화) + 번개(15화) 두 정령을 동시 보유 가능 — 일반 술사는 두 체 계약 시 수명이 두 배로 깎이지만 지호는 재생 덕분에 표면 부담을 우회.
- **단**, 19화 기둥 치유에는 본인 수명을 직접 지불하는 정황 (※Part 3 회수 라인. 본문상 명시는 모호하게 유지).

### 문신 코딩 (17화) — 솔라리스 전통 + 프로그래머 논리
- 정령 잉크 + 뼈 바늘로 피부에 회로를 새김.
- 일반인은 한 번 찌르면 정착되지만 지호는 재생이 잉크를 이물질로 밀어냄 → **다섯 번 같은 자리** 찔러야 정착.
- 6시간 시술 = 매 획마다 첫 번째와 같은 강도의 고통이 수만 번.
- 결과: 5개 함수 (방어/경로/감지/보온/차단) 자동 발동. 번개 능력 안정성 60%→90%.

## 5. 기둥의 진짜 정체 (RETCON-D)

### 9개 기둥 = 이계 차단 배리어
- 본문 19화에서 지호가 빛 기둥 코드를 읽으며 확인:
  - **원래 설계**: 세계 마나만으로 자기 순환하며 이세계로부터 아르카디아를 차단하는 배리어
  - **기생 코드**: 300년 전 누군가가 "사람의 생명력 흡수" 라인을 처리 과정에 삽입 → 이후로 용사가 정기적으로 잡아먹힘
- 모르데우스는 이 기생 코드를 "원래 그런 것"으로 믿고 60년간 시스템 관리자 역할.
- "용사" 호칭은 사실 배리어 유지용 소모품 시스템 (300년간).

### 빌런 진영
- 19화 시점에서 **누가 기생 코드를 심었는지는 미공개**. 모르데우스도 피해자에 가까움.
- 8개 기둥에 동일한 기생 코드가 심어져 있을 것으로 추정 → Part 4 이후의 떡밥.
- **Arc 4 노출 (ch024)**: 빌런 진영 = **검은 로브 네트워크** (`worldbuilding.md` 참조). 익명·추적 불가. 표식 = 깨진 모래시계 모양. 모르데우스 인계 — "300년 전 카이젤과 약속을 했어요. 같이 시간을 나눠 쓰기로." 두 기둥 보호자 사이의 시간 거래 시스템 존재.

### 기둥 보호자 시스템 (Arc 4 ch024 노출)
- 모르데우스만이 시스템 관리자였던 게 아니다. **각 지역 기둥마다 보호자 한 명씩** — 카이젤·프로스트헬·용화국에 동급 또는 더 노련한 시스템 관리자 존재
- 보호자는 기둥과 결합되어 있는 동안 본인 시간이 멎음 ("기둥과 묶여 있을 땐, 시간이… 안 흘렀어요"). 모르데우스가 폐위 후 갑자기 노쇠·말이 늘어난 이유.
- 보호자들 사이의 거래 시스템: "한쪽이 빌리고 한쪽이 갚고, 다음엔 반대로." 누가 빌렸는지·갚을 차례인지 잊혀짐.
- Arc 6~8에서 카이젤·프로스트헬·용화국 보호자 순차 노출 예정.

### 결과
- 19화에서 빛의 기둥 1개 정상화 → 기둥 빛이 금빛에서 금빛+은빛 부드러운 맥동으로 변화
- 기둥의 "축복" 기능은 사라지지 않음. **사람을 먹지 않는 빛**으로 바뀐 것.
- 갇힌 영혼들의 운명은 본문에서 명시 안 됨 (0.1초의 잠깐 일별로 표정만 살짝 달라 보였다).

## 6. 사망·고통 묘사 프로토콜 (유지 항목)

### 분해-재생 루프 6단계 구조 (5화 / 19화에서만 사용)
1. **전조**: 기둥이 반응 (공기 밀도, 온도, 빛 변화)
2. **흡수 시작**: "처음에는 따뜻했다" 패턴
3. **분해-재생 루프**: 안에서부터 타고, 재생, 다시 탐. try-catch 없는 무한 루프 비유
4. **임계점**: 뇌가 사고 포기, 본능과 공포만 남음. 문장 파편화
5. **모래시계 0.1~0.3초 발광**: 기둥 내부 일별 (※지호의 능동 발동이 아닌 동기화 현상)
6. **종료**: 5화는 기둥이 뱉어내고 솔라리스 사막에서 깨어남 / 19화는 코드 리팩토링 완료 후 자발적 손 떼기

### 감각 묘사 우선순위 (변경 없음)
| 감각 | 빈도 | 용도 |
|------|------|------|
| 촉감/온도 | ★★★★★ | 분해-재생 고통의 핵심 채널 |
| 청각 | ★★★★☆ | 자기 비명, 기둥 내부의 갇힌 영혼들 |
| 시각 | ★★★☆☆ | 모래시계 발광 시 기둥 내부 관찰 |
| 후각 | ★★☆☆☆ | 살이 타는 냄새, 오존 냄새 (번개) |
| 미각 | ★☆☆☆☆ | 입 안의 피 맛, 철 맛 |

### 감정 마모 (유지)
- 1~2회: 공포 100%
- 5화 이후: 재생이 고통 강도를 매번 새것으로 유지하므로 **마음은 무뎌져도 몸은 매번 비명**
- 인간성 앵커: 에리나, 나이라, 아이들 (루비), 나중에 결혼한 에리나가 return 문 역할
