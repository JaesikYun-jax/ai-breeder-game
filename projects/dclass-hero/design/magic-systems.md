# 마법 체계 — 기둥의 힘과 정령

> 「D급 스킬 이세계 용사」 20화 완결판 정본 마법 체계.
> 본문 ch001~020에서 확정된 것은 **성광술(아젤리아)** + **정령술(솔라리스, 화염·번개)** + **문신 코딩(솔라리스 전통+프로그래머 논리)**.
> 나머지 7개 지역 마법은 게임 시절 매크로 설정 보존, 본문 확정까지 보류.

---

## 공통 원칙

1. **기둥 의존**: 모든 마법은 해당 지역 기둥의 힘에서 파생. 기둥에서 멀어지면 약해짐.
2. **대가 존재**: 강한 마법은 반드시 수명/감정/체력 등의 대가를 요구.
3. **상극/상생**: 기둥 간 상성이 존재.
4. **강지호의 예외**:
   - D급 재생이 표면 손상을 즉시 복구 → 일반 술사가 대가를 견딜 수 없는 영역까지 반복 가능
   - 이 비대칭 덕에 화염 + 번개 두 정령을 동시 보유. 일반 술사는 두 체 계약 시 수명 침식이 두 배지만 지호는 표면적으로는 멀쩡 (단, 깊은 층위의 수명 소모는 모래시계와 별도로 누적되는 것으로 추정 — Part 3 회수)

## 1. 성광술 (아젤리아 — 빛의 기둥)

**원리**: 빛의 기둥에서 흘러나오는 "성광"을 시전자의 몸에 순환시켜 발현.

| 등급 | 마법 | 효과 | 본문 등장 |
|------|------|------|----------|
| 초급 | 치유의 빛 | 가벼운 상처 회복 | (간접 언급) |
| 중급 | 빛의 방패 | 물리/마법 방어 | 19화 모르데우스 사용 |
| 상급 | 성스러운 심판 | 광역 정화/공격 | 19화 모르데우스 사용, 지호 방어 문양과 충돌 |
| 금기 | 영혼 소환 (=용사 소환) | 타 세계 존재 소환 | 1화 지호 소환에 사용 |

- **시전 조건**: 알려진 바는 "신앙심" / 실제로는 기둥 공명도.
- **모르데우스의 비밀 (19화 폭로)**:
  - 기둥과 직접 연결되어 사실상 무한 마나 보급. 기둥 연결이 끊기는 순간 60대 노인의 본모습으로 돌아감.
  - 그가 60년간 사용한 모든 성광술은 사실 **기생 코드가 흡수한 사람들의 생명력**으로 작동했음.
- **강지호의 감응**: 5화/19화에서 빛 기둥에 직접 접촉 가능. 단, "흡수도 소멸도 안 되는 이물질"로 처리되어 분해-재생 루프만 반복됨.

## 2. 정령술 (솔라리스 — 화염의 기둥)

### 기본 원리
- 기둥에서 떨어진 파편(=정령)과 감정적/생리적 교감을 통해 정령의 힘을 빌려 씀.
- 정령은 기둥의 일부이므로, 정령술 사용 = 기둥 약화 (장기적으로).

### 화염 정령 계약 (8화 절차)
- 열기둥(오아시스 가장자리의 증기 균열) 안에 손을 넣고 정령의 시험을 견딤
- 첫 시험: 5~10초간 정령이 "디컴파일"로 시전자 내면 전체를 읽음
- 견디면 손바닥에 동전 크기의 화상 인장이 새겨짐 → 계약 성립
- 이후 사용 시마다 화상 흉터가 번짐. 어깨를 넘으면 심장에 닿음 = 사망 ("태양에 닿은 자")

### 단계별 출력
| 단계 | 출력 비유 | 일반 술사 도달 시간 |
|------|----------|--------------------|
| 라이터 | 손가락 끝 작은 불꽃 | 계약 직후 |
| 성냥불 | 손바닥 위 작은 불꽃 | 1~3개월 |
| 촛불 | 안정적인 불꽃 유지 | 6개월 |
| 횃불 | 전투 활용 가능 | 2년 |
| 모닥불 | 야영용 광역 발열 | 3~5년 |
| 방벽 (유리화) | 모래를 녹여 유리벽 생성 — 폭풍 방어 | 5년+ |

### 강지호의 화염 정령 "스파크" (가칭)
- 8화 계약. 일반 술사보다 정령 디컴파일 시간 2배(10초). 정령이 "재밌다"는 호기심 표명
- 야간 단독 훈련 가속: 재생이 화상을 15분에 복구 → 남들의 일주일을 하룻밤에
- 9~10화: 라이터 → 성냥불
- 12~13화: 횃불, 체내 순환(혈관에 불을 흘려보내는 고급 기법) 15초 달성
- 17화: 문신 코딩 보온 함수에 자동 연계
- 19~20화: 양손 동시 운용, 번개와 함께 정밀 화공으로 호위병 무력화

### 일반 술사의 비대칭 (RETCON 명시)
- 일반 정령술사는 **머리카락 탈색·수명 단축이 즉시 가시화**됨 (나이라: 14세 계약 → 24세 시점 백화 1줄기 → 17화 시점 1/3, 깊은 기침)
- 강지호는 D급 재생이 침식의 표면 흔적을 모두 복구 → 다중 정령 계약 가능
- 단, **기둥 직접 치유**(19화)에는 본인 수명을 직접 지불하는 것으로 추정 (Part 3 회수)

### 기둥 치유 메커닉 (확정 2026-04-20)
- **3단 조건 충족 + 외부 조력자 필수 + 매 지역마다 새 풀이**
- "손만 대면 해결"은 X
- 1단: 물리 접촉(손바닥 또는 정령 인장) → 모래시계 0.3초 정지 발동 (1회성, 지호 의지 X)
- 2단: 그 0.3초 동안 기생 코드 시각화 → 디버깅 미니 시퀀스 (정령술 + 문신 코딩 + 외부 조력자의 지역 지식)
- 3단: 외부 조력자의 결정적 협력 (각 지역 출신의 정치·혈통·정령술·고대 지식)
- **대가**: 매 치유마다 지호 본인 수명 미세 지불 → Arc 7 흰머리 1가닥 → Part 3 노화 가속·죽음 (회수)
- 상세 풀이는 `story-framework-21-70.md §5-2` 표 참조
- **본문 회수**:
  - **ch019 빛의 기둥 (아젤리아)**: 첫 치유. 빛의 대성당 수정 기둥에 손바닥 접촉 → 분해-재생 루프 10사이클+ → 문신 코딩 감지 함수가 기생 코드 라인 시각화 → 정령 잉크로 우회 경로 새겨 리팩토링. 외부 조력자: 에리나(저항 세력 동원·축복 의식 폐지 정치 처리), 모르데우스 무력화 후 단독 작업. 본문 시간: 길게 다뤄짐 (대성당 그날 "무릎이 꺾이고 갈비뼈가 부러지는 식").
  - **ch021 두 번째 태양 기둥 (솔라리스, 회상)**: 두 번째 치유. ch021 회상으로 압축 처리 — "캐러밴이 멈추는 짧은 정비 시간 동안, 모래 아래 묻힌 작은 결정 가지에 손을 한 번 얹었을 뿐이다. 끝나고 한참 뒤에야 끝났다는 걸 알았다." 결재함 통과처럼 가벼웠음. 외부 조력자: 하산·캐러밴 노장의 정비 일정 + 두 달간 사막 적응으로 쌓인 정령 친화도. 결과: 모래폭풍 빈도 -75%, 두 번째 태양 박자 정상화, 일사병 사망 급감.
  - **대가의 가시화 (ch021~028 추적)**: 모래시계의 박자가 어긋나기 시작. ch021 첫 어긋남(반 박자 빠른 한 알), ch022 마르코 인계 후 손바닥 차가움, ch024 박자 일정하지 않음·어긋남 폭 확대, ch025 모래 한 알이 위에서 아래로 떨어짐 (잔량 1/4 그대로지만 박자만 빨라짐 — "박자가 빨라졌다. 잔량은 그대로였다. 잔량이 줄지 않았는데 박자만 빨라지는 게, 더 무서웠다"), ch028 두 번째 어긋남.

### 지역 한정 제약 (Part 2 RETCON)
- **정령 계약은 솔라리스 사막 한정** — 정령은 솔라리스 화염 기둥의 파편이므로, 계약 의식은 사막의 열기둥·벼락 마당 등 솔라리스 영토 안에서만 가능
- **외지인이 솔라리스 와서 계약하는 것은 가능하나 극히 어려움**: 사막 환경 적응 + 부족장 허락 + 정령의 시험 통과 — 외지인이 통과한 사례는 강지호가 사실상 최초의 외부인 계약자 (15화 번개 정령은 역사상 최초)
- **계약된 정령은 외지로 가져갈 수 있으나 사용 시 페널티**: 솔라리스 밖에서 정령술 사용 시 출력 감소 + 수명 침식 가속(일반 술사 기준 1.5~2배) — 강지호는 비대칭 덕에 페널티 미미
- **카이젤 등 다른 강대국이 정령술을 못 가진 이유**: 의식 자체가 솔라리스 한정이고, 가져가도 본인 생명력을 직접 태우는 방식이라 자국민에게 강요/장려할 수 없음 (왕정·귀족 정치 환경에서 신민의 수명을 단축하는 마법은 정치적으로 부담)
- **서사적 함의**: 카이젤이 "솔라리스 정령술 기술 이전"을 외교적으로 요구하는 것은 개연성이 낮음. Part 2의 카이젤은 다른 목적(예: 살아남아서 부왕이 된 강지호 본인에 대한 정보 수집)으로 접근한다.

### 정령석 활용 (Arc 5~6 비장의 무기)
- **정령석**: 솔라리스 사막에서 채취되는 화염 정령의 잔열이 굳은 결정. 나이라가 ch020에서 지호에게 양도
- **단순 등불 (ch023 시연 — 본문 회수)**: 화염 정령 인장 접촉으로 정령석에 미세 에너지 주입 → 일주일 충전. **초기 시제품·외교 의례용**. ch023 만찬에서 다섯 개 시연 + 다마르 공작에게 한 개 외교 선물 전달. 카이젤로 가져가도 충전이 솔라리스 한정이라 일주일 후 "예쁜 돌"이 되는 함정 동봉.
- **고가 사치품 세공 (ch028 본격 시연 — 본문 회수)**:
  - **전기 + 불 정령 힘을 정령석에 동시 봉인** (지호 손목 번개 인장 + 손바닥 화염 인장 동시 접촉 + 5함수 분기 알람)
  - **삼중 봉인 결과 (ch028)**: ① 빛 — 중심부 한 점 켜짐, 느림·느림·빠름 박자로 깜빡임 ② 온도 — 한쪽 면만 발열, 반대쪽은 차가워지는 선택적 발열 ③ 진동 — 짧·짧·긺의 미세 박자가 손가락 끝에서만 감지. 세 박자가 겹치지 않고 동시 발현
  - **외형 분업 (ch028)**: 솔라리스 부족 장인이 카이젤 취향(직선 6줄 + 방패 모티프)으로 가공. 내부 코어 봉인은 지호 단독, 외형은 장인 단독.
  - **모방 불가 함정**: 화염 + 번개 정령 동시 계약자가 지호뿐 + 회로 사고 모듈 외부 모방 불가 + 충전이 솔라리스 한정. 카이젤 황실이 견본을 뜯어도 안쪽은 빛나지 않는 돌이 됨.
  - 시장: 카이젤 황실, 천인, 각국 귀족
  - 서사적 함의: 아젤리아·솔라리스 양국 동시 부강의 핵심 수익원. ch028 카이젤 황실 외교 선물 견본 1점 완성.
- **양산 시연 (ch031 캐논 회수)**:
  - 일곱 번째 오아시스 작업장에서 5점 시도 → **4점 성공 / 1점 실패 (80% 성공률)**. 솔라리스 부족 장인 4명이 연마·세팅 분담, 지호 봉인 단독, 나이라 화염 잔열 보조.
  - **봉인 절차 (ch031 캐논)**: ① 화염 먼저, 번개 뒤. 순서 바뀌면 결정 내부 결을 따라 힘이 튀어 나감 ② 들어가는 각도는 수직 아닌 비스듬 — 결의 세로줄 따라 흐르되 결을 건드리지 않음 ③ 봉인 = 새기는 일이 아니라 가둬 두되, 갇혀 있다는 걸 갇힌 쪽이 모르게 ④ 인장 식는 시간 평소 3호흡 → 한 점당 4~5호흡으로 늘어남 (피로 누적 신호)
  - **실패 패턴 (ch031)**: 원석 표면의 금 한 줄이 손가락 끝에 걸려 각도 잡기 반 박자 지연 → 화염 첫 파동이 결의 가로에 닿음 → 원석 풀어지며 손바닥 화상 (D급 재생 5초). 결의 세로/가로 식별이 봉인 성공의 임계 조건.
  - **유통 분기 (ch031 부족장 회의)**: 5점 중 1점 솔라리스 자체 보관 + 2점 카이젤 시장 시드 + 2점 각국 귀족 시장 프리뷰. 다마르 공작 경로와 부족 자체 경로를 분리하면 가격 상승. **카이젤 시장 시드 = ch028 견본 1점 + 양산 4점으로 본격 확립**.
- **천막 기술·세공 도구 인계 (ch034 캐논)**: ch034 작별 의례에서 천막 기술 양산 + 정령석 세공 도구가 솔라리스 이름으로 묶여 인계됨. 이후 북쪽에서 오는 질문은 솔라리스 부족 장인에게 향함 (지호 단독 의존 끊기 시작).

### 천막 기술 양산 (Arc 5 솔라리스 비장의 무기 — ch029~030, ch034 캐논)
- **4요소 결합 (ch029)**: ① 이중 천막(바깥/안 벽 사이 손 한 뼘 공기층 = 1차 단열) ② 은박 반사(바깥 벽 외면에 카이젤 수입 금속박 가죽 부착, 화염 정령 잔열로 표면 길들이기 → 반사각 45도→40도로 낮춤) ③ 통풍 굴뚝(지붕 꼭대기, 풍 정령 약한 상승 기류) ④ 경량 폴(나무 기둥 대신 사막 갈대 묶음 = 캐러밴 빠른 접고 풀기)
- **풍·화염 보강**: 굴뚝 내 풍 정령 미세 상승 기류 + 은박 외면 화염 정령 잔열 표면 길들이기. **계약 없이 표면만 길들이는 수준** — 일반 술사도 흉내 가능하지만 은박 각도 잡기는 지호 전용(ch029 나이라 "이 각도, 내가 못 해. 저 사람들도 못 해")
- **현장 보정 (ch029)**: 이중 벽 간격 한 뼘 → 한 뼘 반(붉은 자국 장인 30년 경험 — 모래 바람 쌓임 방지). 굴뚝 경첩 단순화(자말). 갈대 폴 5개 → 7개(녹색 자국 장인 — 태풍 잔여기). 시제품은 설계자가 덜 말하고 현장이 더 말한다는 솔라리스 협업 원칙.
- **체감 효과 (ch029)**: 외부 대비 내부 -15도. 사막 한낮과 그늘 차이보다 큰 폭. 굴뚝 경첩은 시제품 단계에서 두 종류(단순화·구버전) 혼재.
- **양산 (ch030)**: 자말과 장인 셋이 이틀 만에 11동 양산. 갈대 폴=다발, 이중 벽=접힘, 은박 조각=겹말이로 낙타 짐바구니 분산 적재.
- **인계 완료 (ch034)**: 천막 기술 + 정령석 세공 도구 솔라리스 부족 이름으로 묶임. 향후 외부 질문은 부족 장인에게 향하고 지호는 우회됨.

### 풍 정령 새 활용 — 모래 결 분석 (ch032, ch033 캐논)
- **추적 용도 (ch032)**: 협곡 입구 잠복에서 풍을 모래 위에 얇게 얹어 결이 지워지는 속도를 늦춤. **방금 지워지는 중 / 이미 지워진 / 아직 지워지지 않은 자취**를 세기 차이로 구별. 엊그제·어젯밤·오늘 아침 세 자취 시간 분리 가능. 추격이 아닌 "지나가는 걸 봐서 세는" 관측 도구.
- **숨결 가림 (ch032)**: 잠복 시 본인·동료 숨결을 앞쪽으로 흘려 보내 표적 시야에 닿지 않게 우회. 바람 세기 반 단계 정밀 제어.
- **신체 신호 (ch033)**: 발목 풍 인장 발동이 화염·번개보다 반 박자 늦음(평소). ch033에서 반 박자 빠르게 올라옴 — 몸이 기억한 길 단축. 동시에 번개 인장 발동이 한 박자 늦음 — 신호의 비대칭이 5함수 분기 알람으로 자동 보정.

### 3정령 통합 운용 — 풍·화염·번개 동시 호출 (ch033 협곡 클라이맥스 캐논)
- **5함수 자동 분기**: 풍 결 분석 / 화염 차단선 / 번개 정밀 타격 / 접합선 열 보정 / 동기 타이밍. 함수 다섯이 자기들끼리 순서를 자동 배치. 어제까지 머리가 먼저, 손이 나중. ch033부터 손이 먼저, 머리가 뒤따름 → 사후 의식.
- **기후 프로그래밍 발현 (ch033 — FS-A11 첫 가시 단계)**: 마지막 호출에서 묶음 순서를 의도적으로 변주(풍 가장 바깥 / 번개 전하 가운데 / 화염 열 사이) → 협곡 한 구간의 기류 전체 역전. 아래→위 상승 바람이 위→아래 하강. 적이 유도한 모래폭풍 잔열이 적 진영으로 되돌아 밀림. **예상 밖** 결과 — 지호 본인도 "내가 한 건가" 되물음. 할리마·나이라 확인 필요했음.
- **함의 (ch033 지호 내적 메모)**: "세 개를 한 번에 묶으면 날씨 자체를 함수처럼 쓸 수도 있겠다 — 무서운 생각이었다." FS-A11 기후 프로그래밍 능력의 첫 우연 시연. 결심·결의 아닌 메모 단계로 머무름. 본격 시연은 후속 아크 보류.
- **외부 조력자 (ch033)**: 할리마(전통 문양으로 협곡 상공 기류 결 시각화 — 은폐 사각 노출), 나이라(결정 가지로 풍 결 거점 고정), 하산(동선 좌표), 로헨(검 처리). 지호 단독이 아닌 4인 협업 구조 명시.
- **할리마의 신체 한계 (ch033)**: 70년 정령술 안 쓰고 산 노인이 협곡 바닥에 무릎 꿇어 전통 문양을 큰 규모로 그림 → 첫 긴 호흡 후 벽에 등 붙이고 길게 앉음. "70년 동안 본 적 있는 일. 손이 기억해. 그런데 오늘은 손보다 눈이 먼저 피곤해." 다음 전투 참여 여부는 미정.

### 문신 코딩 5함수 보강 (ch033 캐논)
- **자동 분기 패턴 명시**: 5함수가 더 이상 의식적 호출이 아닌 자동 분기. 풍·화염·번개 3정령 인장 신호 차이를 함수가 자기들끼리 잡음 보정.
- **기존 5함수**(방어/경로/감지/보온/차단)에 운용 측면에서 새 함수 차원 추가 — ch033 마지막 호출에서 "함수 하나가 더 열렸다"는 메모. 6번째 명시 함수는 아직 정착 안 됨, 메모 단계.

### 모래바람 진정 메커닉 (Arc 5 솔라리스 비장의 무기)
- **풍 정령 + 화염 정령 + 문신 코딩 5함수 결합**
- 풍 정령으로 회오리 코어를 미세 제어 + 화염 정령으로 열압 조절 → 모래폭풍 형성을 협곡·캐러밴 루트 부근에서 부분 차단
- 전체 사막 기상은 못 바꿈. **국지적 안정화** 한정
- 결과: 솔라리스 핵심 교역로(아젤리아·카이젤·프로스트헬 방향) 안전성 회복 → **캐러밴 손실률 -70% (ch025 보고 캐논 회수: "직전 분기 대비, 칠할 가까이 회복" + "모래폭풍 빈도 한 달 평균 사 회 → 일 회" + "일사병 캐러밴 사망 거의 없음")**
- 외부 조력자: 나이라(부족장 라인 정치 협상), 하산(캐러밴 네트워크 활용), 할리마(전통 문양 자문)
- **번개 모스 통신망**과 결합하면 캐러밴 → 아젤리아 비밀 정보 라인 구축 가능 (Arc 6~7 외교 우위)
- **본문 첫 시연 (ch027 협곡 호송 — 캐논 회수)**:
  - 협곡 한가운데 한 달째 자리 잡은 폭풍 코어. 자연 회오리치고 너무 규칙적인 결 → 코어 안쪽 응축점 2개가 회오리를 자가 유지시키는 골격
  - 풍 정령으로 회오리 안쪽 결 시각화 → 결정 가지(협곡 벽 30보·60보 자리)에 정령석 박아 응축점 2개를 어긋나게 끌어냄
  - 그 어긋남의 한 박자에 풍(외곽 압력 분산) + 화염(내쪽 열대류 끌어내림) 동시 발동 → 5함수 분기 알람이 두 신호 사이에 자동 잡음
  - 결과: 회오리가 거꾸로 솟구쳐 양옆 절벽으로 모래의 비로 분산. 코어 소멸. 정령석 마차 둘 안전 통과.
  - **시도 자체가 처음** — 풍·화염·5함수 셋 동시 운용은 ch027이 첫 시연. "안 되면 모래에 묻히겠지"의 도박이었음.

## 3. 번개 정령 (솔라리스 — 화염 기둥의 부속 현상)

### 본문 등장 (15화 — 사상 최초의 계약자)
- 솔라리스 서쪽 "벼락 마당"에서 건조 번개 안에 깃든 정령
- 70년간 4명이 계약 시도 → **전원 감전사**. 심장이 정령 접촉 전에 멈춤
- 강지호는 D급 재생으로 심장 정지 → 자동 재시작을 21회 반복하며 0.1~0.3초 접촉 시간을 누적
- 21번째 시도에서 정령이 "왜 죽지 않지"라는 호기심으로 응답 → 손목에 리히텐베르크 패턴(흰색 나뭇가지 문양) 인장 새김

- 동기식 접촉 불가능, 비동기 콜백 패턴 (체내 순환 경로를 미리 열어둔 상태에서 번개가 흐르면 그 경로로 따라 흐름)

### 번개 모스 통신 (확정 2026-04-23)
- **원리**: 번개 인장 가장자리에서 미세 방전을 점(짧은 튐)·선(긴 튐) 두 박자로 제어. 외부에 불꽃이 튀지는 않고 문양 안쪽에서만 리듬을 탐 → 시각·청각으로는 거의 감지 안 됨, 동조 가능한 수신자(번개 정령 잔향 또는 사전 약속한 신호 거점)만 받음
- **수신 거점**: 솔라리스 캐러밴 노장(하산 등)이 사전에 약속한 좌표·인원 신호만 받는 구조. 정령술사가 아니어도 약속된 거점에 잔향 감지 장치(번개 인장 잉크로 표시된 천)를 두면 수신 가능 추정 — 본문 명시 보류
- **한계**: ① 단방향 (송신만) ② 사전 약속한 신호 코드만 ③ 지호 본인의 손목 인장에서만 가능 (역사상 유일한 번개 정령 계약자) ④ 송신 후 손목 잔열 회복까지 짧은 휴지
- **서사적 함의**: 아젤리아 ↔ 솔라리스 비밀 통신 라인. 캐러밴 합류·외교 사절 출발 등 주요 이동 신호를 외교 파발보다 빠르게 송신 가능. Arc 6 카이젤 진입 후 외교 우위의 한 축.

## 4. 문신 코딩 (Arc 3 핵심 — 솔라리스 전통 + 프로그래머 논리)

### 솔라리스 전통 문양의 한계
- 정령 잉크(불의 정령 에너지가 깃든 적갈색 안료) + 뼈 바늘로 피부에 회로 새김
- 곡선 반복 + 대칭 구조 = 정령술의 출력만 증폭하는 "볼륨 조절기"
- 어디로 보낼지·언제 멈출지는 술사가 매번 의식적으로 제어해야 함

### 강지호의 확장 (17화)
프로그래머 사고로 분기점·종결점·조건문을 추가 → 정령이 읽고 자동 실행하는 코드:

| # | 함수 | 위치 | 작동 |
|---|------|------|------|
| 1 | **방어 함수** | 등 상부 | 외부 충격 감지 시 번개 반원형 방벽 자동 전개 |
| 2 | **경로 함수** | 양팔 | 번개 흐름을 지정 경로로 자동 유도. 안정성 60→90% |
| 3 | **감지 함수** | 목 뒤~척추 | 위험 요소 미세 전류로 사전 경고 (0.3~0.5초 전) |
| 4 | **보온 함수** | 가슴 | 화염 정령 에너지를 자동 체온 유지에 사용 |
| 5 | **차단 함수** | 양 손목 | 과부하 임계치 초과 시 전류 강제 차단 |

### 시술 (17화)
- 시술자: 나이라 (할리마는 손이 떨림). 6시간 소요
- 일반인은 한 번 찌르면 정착, 지호는 재생이 잉크를 이물질로 밀어냄 → **다섯 번 같은 자리** 찔러야 정착
- 매 획마다 첫 번째와 같은 강도의 고통이 수만 번 (재생이 신경을 매번 새것으로 복구)
- 정착된 후에는 정령 인장과 동일하게 재생이 건드리지 않음

### 의미
- "프로그래머가 이세계에서 마법사가 되는" 컨셉의 정점
- 19화 빛 기둥 리팩토링도 이 문신 코딩의 감지/경로 함수를 기둥 내부로 확장 적용한 결과

## 상성 관계 (게임 시절 설정 — 본문 확정 전 보류)

```
빛 ↔ 심연
철 ↔ 바람
화염 ↔ 얼음
기 ↔ 천공
대지 = 모든 기둥의 기반
```

본문에서는 화염↔번개가 "다른 원소이지만 둘 다 에너지 계열이라 공명 가능"하다는 것이 15화/19화에서 확정됨. 다른 상성은 후속 아크에서 확정 예정.

## 강지호의 마법 성장 표 (수정판)

게임 시절 "루프별 성장 표"는 폐기 (회귀 자체가 없음). 본문 기준 단일 성장:

| 시점 | 능력 |
|------|------|
| 1~5화 (Arc 1) | 마법 사용 불가. D급 재생만 보유. 빛 기둥에서 분해-재생 6사이클 견딤 |
| 6~13화 (Arc 2 전반) | 화염 정령 계약, 라이터→횃불, 체내 순환 15초 |
| 14~17화 (Arc 2~3 전환) | 번개 정령 계약(사상 최초), 신경 가속, 근육 운동 압축, 문신 코딩 5함수 |
| 18~20화 (Arc 3) | 화염+번개 동시 운용, 정밀 방전·정밀 화공, 빛 기둥 코드 리팩토링 (감지 함수를 기둥 내부 확장) |

**모래시계 잔량은 절반 → 1/4로 감소** (5화 흡수 + 19화 기둥 치유의 누적 추정). 8개 기둥이 남았으므로 모래 소진 vs 기둥 정상화 레이스가 Part 4 이후 핵심 긴장.
