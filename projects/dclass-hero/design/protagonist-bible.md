# 주인공 바이블 — 강지호 (姜志浩)

> 이 소설의 모든 문장은 강지호의 눈과 입을 통해 나간다.
> 이 문서는 그의 목소리, 감정, 한계, 성장을 정의하는 최종 레퍼런스다.

## 1-1. 재생 스킬 — "죽지 못하는 저주"

### 기본 성능
| 항목 | 수치 |
|------|------|
| 등급 | D급 (사이 공간에서 선택 가능한 최저 티어) |
| 얕은 베임 | 10~15분 회복 |
| 깊은 상처 | 수 시간 회복 |
| 골절 | 반나절~하루 |
| 절단 | 잘 붙여놓으면 서서히 수복. 분리되면 불가 |
| 장기 손상 | 수 시간~수일 (심장 관통 등 치명상도 시간이 걸리지만 재생) |
| 미세 신경 (손가락 끝) | **약 10분** (ch016 검증) |
| 회복 불가 | 머리 절단 후 분리, 신체 완전 파괴 (소각, 산 등) |

### 핵심 저주: 신경세포 재생
- 모든 신경세포가 완벽하게 재생됨
- 결과: 고통에 **절대** 익숙해질 수 없음
- 100번째 화상이 첫 번째와 동일한 강도로 아픔
- 이세계물의 "고통 내성" 클리셰가 적용되지 않는 유일한 주인공
- 이것이 나중에 "고통참기의 기술"을 필수로 만드는 원인

### 재생의 천장 (ch016에서 자각)
- 무한 반복이 아니다. **재생에는 속도가 있고, 속도에는 한계가 있다.**
- 파괴 속도가 재생 속도를 지속적으로 초과하면 **신경 순감소 → 영구 소실 가능**.
- D급 재생은 만능이 아니다 — 기능 복구와 완전 복구 사이에 격차가 존재.
- "무한 루프가 아니라 카운트다운이다." (ch016)

### 서사적 의미
- **축복**: 무한 시도 가능. 정령술 화상이 회복되므로 남들보다 더 많이 연습 가능.
- **저주**: 매번의 고통이 처음 그대로. 정신이 버틸 수 있는 한계가 성장의 천장.
- **조건부 불사**: 사실상 죽지 않지만, 머리 분리·신체 완파에서는 사망. 고통 면역 없음.
- **기둥과의 상호작용**: 기둥 흡수 시 분해→재생 루프로 의식 유지. 다른 사람은 즉사하지만 지호는 오랜 시간 고통받음. ch001(아젤리아 의식)에서 6회 루프 후 솔라리스로 방출됨. ch021 회상에서 솔라리스 태양 기둥은 **분해-재생 루프 없이 결재 한 장 통과식**으로 처리됨 — 같은 흡수 메커니즘이 두 번째에는 다르게 동작 (이미 한 번 한 자에게는 비용이 다름).

### 성장 공식 (최종)
> 재생(무한 시도) + 무재능(다중 체계 흡수) + 코드 사고(체계화) = 유일무이한 성장 경로

## 2. 현대 생활 — 회상/비유의 소재 풀

지호의 독백은 현대 한국의 구체적인 경험에서 비유를 끌어온다. 이세계 묘사를 할 때도 반드시 현대적 비유가 하나 이상 들어간다.

### 일상 소재
| 소재 | 활용 예시 (실제 본편 기반) |
|------|--------------------------|
| **편의점** | "이 세계에도 편의점은 없다" (4화 제목 자체), 가격 비교의 기준점 |
| **웹소설 앱** | 무료 코인 줍기, 《무한회귀 천재검귀》 437화 정주행 중 트럭에 치임 |
| **사무실** | "개발자 직업병" — 이세계 상황을 디버깅하듯 분석 ("이건 버그야") |
| **월급** | 왕궁 옷을 보고 "서울 월세 몇 달치" 환산 |
| **아메리카노** | "씁쓸한 거 알면서 먹는 아메리카노" 같은 체념 비유 |
| **대중교통** | 횡단보도, 신호등, 퇴근길 — 1화 도입부의 공간 |
| **호텔** | 왕궁 목욕탕을 "5성급 스파"에 비유 |
| **원룸** | 왕궁 침실 크기를 "서울 원룸 다섯 개" 단위로 측정 |
| **스마트스토어/SI** | 마르코 같은 상인을 "네이버 스마트스토어 하면 대박" |
| **CI/CD** | 매일 밤 30회 정령술 반복을 "프로덕션 서버가 폭발하는 빌드 파이프라인" |

### 비유 규칙
- 이세계 사물 → 현대 한국 등가물로 번역 (독자 공감 + 지호 캐릭터성)
- 감정 → 신체 감각 + 일상 비유 ("비명이 물속에서 들리는 것처럼 왜곡되어 들렸다")
- 돈 관련 → 반드시 구체적 숫자 ("쿠론은 대략 500원")
- 이세계 클리셰 → 구체적 작품명 언급 ("무한회귀 천재검귀" 같은 가상 제목)
- 프로그래머적 비유 → 상황을 코드/시스템 용어로 해석 ("무한루프에 빠졌다", "예외처리가 안 된 거야", "이 세계 런타임 에러 아닌가", "DDoS는 안 되니까 SQL 인젝션", "그리드 서치", "베이지안 최적화")

## 3. 이세계 지식 DB — "트로프 박사"

지호가 500작 이상의 웹소설에서 축적한 장르 지식. 이것이 그의 최대 무기이자 최대 약점.

### 인식하는 클리셰 목록
| 클리셰 | 지호의 반응 | 실제 결과 |
|--------|-----------|----------|
| 트럭에 치여 이세계 | "이거 그거잖아" — 즉시 인식 | 맞음, 하지만 고통이 100% 리얼 |
| 소환진 등장 | "소설에서 봤어" | 맞음 |
| 공주가 마중 | "데레 캐릭터 확정" | 실제로는 절박한 내부고발자 |
| 히든 스킬 각성 | "이제 곧 폭발할 거야" | 아무 일도 일어나지 않음 |
| 금빛 로브 사제 | "악역 확정" | 맞음 (모르데우스) |
| 스탯 창/시스템 | 허공에 손 휘저어봄 | 없음. 시스템 자체가 존재하지 않음 |
| 축복 의식 | "파워업 이벤트겠지" | 영혼 추출 제의 |
| 위기의 순간에 각성 | "이제 숨겨진 힘이..." | D급 재생뿐. 사기캐 각성 없음 |
| 재생 = 사기캐 | "잠재력 S급이라며?" | D급은 D급. 15분에 베인 상처 하나. |

### 지식이 무기일 때
- 에리나의 표정 변화를 "경고 3박자 패턴"으로 분석 (4화)
- 모르데우스의 "꿀처럼 달콤한데 뒷맛이 씁쓸한" 어조에서 위험 감지
- 시장 경제를 프로그래머적 논리로 패턴 분석해 세계관 파악
- 솔라리스 정령술을 API 호출 모델로 추상화 (ch012)
- 모르데우스의 성광술을 "기둥 의존 보급 시스템"으로 진단, 본체 대신 보급선을 끊는 전략 (ch019)

### 지식이 독일 때
- "설마" — 위험 신호를 소설적 클리셰로 합리화 ("주인공은 안 죽으니까")
- 히든 파워를 기대하며 대비를 안 함
- 모래시계가 "그냥 깨진 모래시계일 뿐"이라고 판단 (5화, 의식 직전)
- **핵심 자각**: "여기는 소설이 아니다" (4화 마지막)

## 4. 보이스 규칙 — 강지호의 말과 글

### 독백 문체 (1인칭 내면)
- **기본 톤**: 반말 + 속어 + 자조
- **문장 길이**: 짧은 게 기본. 분석할 때만 길어짐
- **특징적 패턴**:
  - 상황 인식 → 클리셰 분류 → 현실과의 괴리 자각
  - 자기 비하 유머: "스쿼트 3개가 한계인 28세 직장인의 위엄"
  - 돈 환산: 자동 발동 ("이 옷 서울에서 얼마야")
  - **코드 비유**: 위기 상황을 시스템/코드 용어로 객관화 ("레이스 컨디션", "예외처리가 안 된 거야")

### 대사 문체 (실제로 입 밖에 내는 말)
| 상대 | 화법 |
|------|------|
| 왕족/권력자 | 어색한 존댓말 ("...감사합니다?") — 물음표 뉘앙스 |
| 에리나 (초기) | 경계 섞인 존댓말 ("그게 무슨 뜻이죠?") |
| 에리나 (결혼 후) | 반말 + 솔직한 토로 ("응") |
| 모르데우스 | 형식적 존댓말 + 내면에서 욕설 → ch019 결투 시 반말 |
| 로헨 | 군대식 존댓말 ("네, 알겠습니다") |
| 마르코 | 경계하면서도 호기심 있는 반존대 |
| 나이라 | 평이한 반말, 거리감 있는 친근함 |
| 하산 | 짧은 반존대 + 실리적 응답 |
| 혼잣말 (위기) | 순수 반말 + 욕설 ("아 씨, 뭐야 이게") |

### 유머 패턴
1. **메타 코멘트**: 이세계 상황을 웹소설 용어로 해석 ("악역 확정", "데레 캐릭터")
2. **자조**: 자기 능력의 처참함을 담담하게 서술 ("푸시업 7개. 로헨이 세는 걸 포기했다")
3. **돈미새 발동**: 부적절한 타이밍에 가격 계산 ("목숨이 위험한데 물가 분석을 하고 있다")
4. **여미새 발동**: 로맨스 상황 인식 후 자체 차단 → 결혼 후에도 잔존 ("UR 히로인" 등)
5. **직업병**: 삼류 개발자의 사고방식을 엉뚱한 데 적용 ("이 왕국 시스템 아키텍처가 레거시야")

### 금지 표현 (지호가 절대 하지 않는 것)
- 영웅적 각오 ("이 세계를 구하겠다!") — ch014~ch020에서도 "가만히 있을 수 없어" 정도가 한계
- 상대를 깔보는 발언 (자기 비하는 하지만 타인 비하 안 함)
- 감정 직접 명명 ("슬펐다", "분노했다" → 대신 신체 반응으로)
- 낭만적 독백 ("그녀의 은발이 달빛에..." → "은발이 보였다. 진짜 은색이었다. 렌즈 아니고?")
- 시스템/스킬 용어 사용 (이 세계에 시스템은 없다)

## 5. 감정 스펙트럼

### 공포
- **초기 반응**: 몸이 먼저 반응 (다리 떨림, 목소리 안 나옴, 소변 실금)
- **인식 지연**: "비명이 내 비명인지 인식하는 데 시간이 걸렸다" (5화)
- **분석 본능**: 공포 속에서도 상황을 분석하려 함 — 하지만 "분석해서 뭐하냐. 죽어가면서."
- **극한**: 아이처럼 울음. "엄마를 부르고 싶었는데 목에서 소리가 나지 않았다" (5화)
- **누적 공포**: 신경은 재생되어도 뇌의 기억은 누적 (ch010~). "이 손은 아플 것이다"를 몸이 학습.
- **문체 변화**: 문장이 파편화됨. 주어 탈락. 동사만 남음.

### 체념/피로
- **표현**: 담담한 서술 + 건조한 유머 ("두 번째 생이다")
- **사고 패턴**: 옵션을 나열하고 전부 불가능함을 확인 → 소거법적 절망
- **문체 변화**: 문장이 짧아지고 감정 형용사가 사라짐

### 분노
- **발현**: 느림. 지호는 분노보다 공포가 먼저 옴
- **방향**: 시스템/구조에 대한 분노 ("왕국이 300년간 해왔다고?")
- **표현**: 내면 독백이 짧고 날카로워짐
- **자기 분노**: "500편 읽고도 이걸 예측 못 했어?"
- **이를 갈다 (ch014)**: 모르데우스의 의식 재개 소식에 분노가 결심으로 굳음. "가만히 있을 수 없어."

### 안도/기쁨
- **드문 감정**: 1-5화에서 기쁨은 거의 음식에서만 발생
- **표현**: "이것만으로도 이세계 올 가치가 있다" (3화, 아침 식사)
- **돈미새 기쁨**: 공짜 의식주에 대한 본능적 만족
- **끝없는 환희 (ch020)**: 프러포즈/결혼식/첫날밤에서 세 번 자각. 우는데 슬픔이 아님.

### 호기심/분석
- **가장 빈번한 감정**: 지호의 디폴트 모드

## 6. 성장 마일스톤 (Part 1: ch001~ch020)

### Arc 1: 아젤리아 (ch001~ch005) — "트로프 박사에서 당사자로"
- **시작**: 이세계 상황을 소설 독자 시점에서 관찰. 트럭사 → 사이 공간 → D급 재생 선택.
- **아젤리아 왕궁**: 마법 적성 0, 검술 0, 체력 0. 마르코에게서 깨진 모래시계 팔찌 획득.
- **에리나의 경고**: "마력 적성이 없는 용사는 의식을 견디지 못한다." 도서관에서 200년 전 삽화 속 모르데우스 확인.
- **축복 의식 (ch005)**: 빛의 기둥에 흡수됨. 분해→재생 루프 6회. 모래시계가 0.3초 시간 정지를 일으켜 기둥 내부 관찰 — 갇힌 영혼들과 기둥 깊은 곳의 금빛 균열을 목격.
- **결과**: 기둥이 흡수도 소멸도 못 시키는 이물질로 처리하지 못해 **솔라리스 사막으로 토해냄**. 모래시계 모래가 절반 → 1/4로 감소.
- **핵심 변화**: 관찰자 → 당사자. "여기는 소설이 아니다" (ch004).

### Arc 2: 솔라리스 (ch006~ch013) — "코드 정령술사의 탄생"
- **캐러밴 합류 (ch006)**: 나이라가 모래폭풍 후 사막에서 발견. 하산 캐러밴(상인 60대)에 잡부로 합류. 정령술사·노예 노동·"태양에 닿은 자들" 부고판 목격.
- **정령술의 진실 (ch007)**: 솔라리스 정령술사 평균 수명 35세. 화상이 심장에 닿으면 사망. 할리마(70세, 비술사)와의 대화로 "오래 살기 vs 안전하게 살기"의 모순 인식.
- **화염 정령 계약 (ch008)**: 열기둥에 손 10초. 정령이 지호를 "재밌다"고 인식 (보통 5초인데 10초 검색). 손바닥에 동전 크기 화상 = 계약 인장. 첫 출력은 라이터 불꽃.
- **모래폭풍 (ch009)**: 나이라의 방벽에 자기 불꽃을 보냄 → "뜨겁지 않은 불"이 나이라의 대가를 줄임. 첫 정령 출력의 이질성 발견.
- **꺼지지 않는 불 (ch010)**: 재생 활용 야간 훈련 시작. 화상→재생→화상 무한 반복. 하룻밤 15~30회. 남들의 일주일치를 하룻밤에.
- **명예로운 노예들 (ch011)**: 술사 부고 장례 참관. "명예로운 노예" 자각. "내가 더 강해지면 나이라의 부담을 줄일 수 있다" 결심.
- **최적화 (ch012)**: 정령술을 API/파라미터 튜닝으로 추상화. 그리드 서치 → 양손 감싸기 발견(60초 유지). 라이터→횃불까지 3주 (정상은 2년). 나이라가 화상 회복을 눈치챔.
- **체내 순환 (ch013)**: 혈관에 정령 에너지 순환. 첫 시도 15초. 모래시계 모래가 미세하게 줄어드는 첫 신호 — "뭔가가 줄었다."
- **열사병 면역**: 재생이 실시간 세포 손상을 복구해 쓰러지지 않음 (ch013). 다만 증상 자체는 풀파워로 반복.

### Arc 3: 각성과 귀환 (ch014~ch020) — "코드 마법사의 귀환"
- **이를 갈다 (ch014)**: 행상인이 아젤리아 소식 전달 — 모르데우스 섭정, 축복 의식 3개월 내 재개, 에리나 행방불명. 강화 루트 4안 검토 → C안(번개 정령) 선택. 할리마에게 "벼락 마당" 좌표 받음.
- **번개 정령 계약 (ch015)**: 벼락 마당 4일 체류. **21회 감전-심장정지-재생 반복**. 역사상 계약자 0명이었던 영역. 정령이 "왜 죽지 않지"라는 호기심으로 응답. 21번째에서 번개가 관통이 아닌 표면을 타고 흐르며 손목에 나뭇가지 문양(리히텐베르크 패턴) = 두 번째 정령 인장. 머리카락이 정전기로 곤두선 채 캐러밴 복귀.
- **과부하 (ch016)**: 번개로 신경 가속(오버클럭) — 세계가 슬로모션. 근섬유 파괴-재생 루프로 근육 밀도 강제 증가 (스쿼트 3개 → 30개 → 50개). 폭식 시작 (한 끼 6인분). 부분 가속 통제 60%. **재생의 천장 자각** — 손가락 감각 10분 부재 후 복귀, 영구 신경 소실 가능성 인지.
- **코드를 새기다 (ch017, 문신 코딩)**: 할리마에게서 솔라리스 전통 문양 학습. 재생이 잉크를 밀어내는 문제 → 5회 반복 정착. 나이라가 6시간에 걸쳐 시술. 5개 함수 새김:
  - **방어 함수** (등 상부) — 충격 감지 시 자동 번개 방벽
  - **경로 함수** (양팔) — 번개 흐름 자동 제한, 통제 60% → 90%
  - **감지 함수** (목 뒤~척추) — 위험 사전 경고 미세 전류
  - **보온 함수** (가슴) — 화염 정령으로 자동 체온 유지
  - **차단 함수** (양 손목) — 임계치 초과 시 전류 강제 차단
  - "프로그래머가 코드를 쓴다. 정령이 읽는 언어로 쓰면 그건 마법이다."
- **빛의 왕국으로 (ch018)**: 솔라리스→아젤리아 도보 횡단 12일. 가슴 보온 문양으로 사막 밤 동사 면제. 김태호로 위장 입국. 마르코 표식 추정 양조장에서 에리나 저항군 합류 (50명). 작전 수립: 정면 돌파(DDoS) 불가 → 지하 수로 침투(SQL Injection).
- **심판 (ch019)**: A조 교란/B조 로헨 구출/C조 기둥 침투. 모르데우스의 성광 방패에 정면 공격 무력 → "본체가 아니라 보급선을 끊어야 한다" 깨달음. **기둥에 직접 접촉**, 손바닥의 분해-재생 루프를 견디며 문신 코딩의 감지+경로 함수를 기둥 내부로 주입. **300년 기생 코드(생명력 흡수 라인) 우회 → 자연 마나 순환으로 리팩토링.** 모래시계가 0.1초 시간 정지로 기둥 내부 영혼들이 웃는 모습을 보여줌. 모르데우스는 기둥 연결이 끊겨 60대 노인으로 회귀, 수감.
- **용사의 길 (ch020)**: 왕국 안정 후 에리나에게 즉흥 프러포즈 — "코드에 return 문이 없으면 프로그램이 어디로 가야 할지 몰라요. 당신이 return 문이에요." 광장 결혼식. **즉석 번개 반지** (장식 못을 번개 열로 구부려 만든 자주색 전기가 지직거리는 반지). 첫날밤에 미세 정전기 부산물 등장. 로헨과의 재대련에서 푸시업 100개·스쿼트 100개 달성. **남쪽 솔라리스로 출발**.

## 6-A. Part 2 성장 마일스톤 (Arc 4: ch021~ch028) — "부왕은 부왕이 아니다"

### Arc 4: 내정의 해 (ch021~ch028) — "결재함과 사막 사이의 외주 분업"

- **부왕 정착 (ch021)**: 결혼 후 3개월. "부왕은 부왕이 아니다" 자조를 하루 세 번. 좁은 동관 집무실, 4단 결재함. **카드 색인 시스템 도입** — 가로 5칸×세로 4칸 칸상자에 분뇨/곡물/북문/의약/기타 라벨. "세상의 절반은 기타 칸에 들어가요." 평민 출신 사관(이름 미공개) 캐릭터 곡선의 시작점. 로헨이 푸시업 200개 지시 (한 달간 100개 → 상향).
- **태양 기둥 치유 회상 (ch021 도입부)**: 두 달 전, 캐러밴 끝자리. 솔라리스 사막 새벽에 모래 아래 묻힌 작은 결정 가지에 손바닥 한 번 얹음. **분해-재생 루프 없이 결재 한 장 통과시키듯** 처리됨. 사막 사람들은 "신의 노여움이 풀렸다"로만 인식. 의식 직후 가슴께에서 **깨진 유리 팔찌 한 조각이 떨어짐 — 모래시계와 같은 종류의 무엇** (책상 가죽 주머니에 보관 중).
- **풍 정령 계약 (ch021 회상)**: 사막 끝 협곡, 풍속 미친 회오리에 9초 갇힘. 두 가지 생각만 — "죽을 수도 있다 / 또 죽을 거 알면서 왜 손을 내미는 거냐." **발목 안쪽 동전 절반만 한 옅은 자국 = 세 번째 정령 인장**. 왼쪽 관자놀이 머리카락 한 가닥 끝부분이 빛으로 살짝 바램.
- **모래시계 박자 어긋남 시작 (ch021)**: 잔량은 1/4 그대로. **떨어지는 한 알의 박자가 반 박자 빠르게 어긋남**. "잔량이 줄지 않았는데 박자만 빨라지는 게 더 무섭다."
- **카이젤 사절단 도착 (ch021 말)**: 검은 강철 마차. 다마르 공작 (50대, 황제 외종) 사절단장 입장.
- **마르코 재회 + 진실 인계 (ch022)**: 지하 문서고에서 만남. 다섯 달 만이고 첫 정식 재회. **모래시계의 정체 노출 — "9개 기둥 전체의 카운트다운"**. "300년 전부터 누군가가 기둥을 약화시켜 왔어. 너 같은 '용사'는 그걸 잠깐 늦추는 소모품으로 소환됐던 거고." 마르코 자기 정체 = "안내자". "깨진 모래시계 목걸이를 한 여자가 다음에 어떤 얼굴로 올지는 나도 몰라" 떡밥 추가.
- **약한 세계 위장 외교 (ch023)**: 다마르와 정식 회담. 에리나의 외교 코칭 — "두 번 끄덕이지 마라. 광물 얘기 꺼내지 마라." 카이젤 첫 사담에 "솔라리스에서 오신 길" 정보 노출. 지호의 위장 카드 **"마법 없는 약한 세계 + 잘 안 죽는 체질"** 정식 데뷔 (Part 2~3 외교 도구의 원본). 다마르가 가슴팍 셔츠 안쪽 모래시계 위치를 두 번 흘낏 — 정보망 두께 시연.
- **부강 시도 실패 회상 (ch023 중반)**: 한 달 전쯤 시도들. **비누(이미 존재) / 향수(라벤더+장미 끓여 술 비슷한 것) / 화장품(귀족 부인 후기 "한 번 발라보고 다음엔 안 사겠더라")** 모두 실패. "이세계물 500편 봤는데 정작 만든 건 다 망했다." 캠핑 장비 + 아웃도어 브랜드 리뷰 500편이 Arc 5 부강 카드로 남음.
- **정령석 등불 시제품 데뷔 (ch023 만찬)**: 손바닥만 한 유리관 + 붉은 정령석 1알. 빛이 흔들리지 않고 기름 냄새 없음. 다마르에게 1점 외교 선물 증여. **함정 — 카이젤 충전 불가 (사람 수명 태워야 하므로)**. "가져가면 일주일 후엔 그저 예쁜 돌." 호기심에 카이젤이 솔라리스에 사람 보내면 나이라 정보망 + 1.
- **첫 잠입자 검거 + 모르데우스 계열 표식 (ch023 말)**: 사절단 시종으로 위장. 사절단 비서가 다섯 자리 비운 채 묵인 출발. 잡힌 자 목 뒤에 **모르데우스 장신구와 같은 계열 무늬**. "한 명 보내고 한 명 잡았다. 잘했나? 모르겠다."
- **모르데우스 재면담 (ch024)**: 지하 별감방. 60대 노인으로 회귀한 모르데우스. 호칭은 끝까지 회피. 진실 추가 인계 — **"카이젤과 우리는 약속을 했어요. 같이 시간을 나눠 쓰기로. 기둥의 시간 말이에요."** "검은 로브는 이름이 없어요. 부르려고 하면 사라지지요." 잠입자 무늬는 "같은 계열일 뿐, 같지는 않을 겁니다" — 정본은 다른 자리. **"자네는 나와 다르지 않게 될 거예요. 다르게 시작했을 뿐이지요."** 마지막 한 줄. 영웅적 결의 0회. "이제 시간이, 있거든요" — 모르데우스의 가벼운 말투 + 손 떨림.
- **에리나 첫 어긋남 (ch024 말)**: 침소 침대에서 한 사람 분 자리가 비워짐. 둘 다 호칭 안 씀. 모래시계 박자 더 어긋남 — 두 알이 한 박자에, 잔량은 그대로. **"잔량이 줄지 않았는데 박자만 빨라지는 게 더 무서웠다."** 변하지 않음.
- **부부 회복 + 솔라리스 외교사절 결정 (ch025)**: 다실 대화. 에리나 "미안하다는 말은 안 받을게요. 받으면 당신 잘못으로 두는 게 되니까. 대신 가기 전에 한 번 말해줘요." 손등 위에 손 얹는 회복. 회의에서 솔라리스 동맹 외교사절을 지호 본인이 가기로 결정. **로헨이 회의 중 푸시업 200회 평균을 공식 기록** — 회의실 유일한 코미디. **출발 전날 밤, 모래시계에서 첫 한 알이 모래 위쪽 결을 바꾸며 떨어짐.** "세계의 시간이 줄었다. 한 알만큼." **에리나가 잠옷 위 배 부위에 오른손을 한 번 얹었다 내림** — 지호는 등 돌리고 못 봄 (임신 복선 1차).
- **솔라리스 가는 길 (ch026)**: 호송대 30명 + 마차 2 + 짐수레 1. 사흘째 사막 경계 야영지에서 **자객 첫 침입**. 옆구리·팔뚝 베임. 5초 안에 피 멎음. **로헨이 단독 목격 — 안 죽는 체질 외부 노출 1차 (Part 2 가장 가까운 군사적 동지에게)**. 로헨 시그니처 3어절 → "…부왕 각하" 두 어절로 깨짐 (훈련된 침묵의 균열). 자객 목 뒤에 모르데우스 계열 표식. 지호 자조 — "편의점 알바보다는 낫죠."
- **임신 친서 수령 (ch026)**: 오후 모래 위에서 에리나 친서 4줄 — *"사흘간 고민하다 부친 씁니다. 아이가 생겼어요. 놀라지 말고, 돌아오세요."* 식은 차·삼킴·반지·배 네 기억이 한 박자에 꿰임. 자조 한 줄 — **"내가 뭐라고 아빠가."** 28년 산 한국 IT 야근러가 사막에서 아빠가 되는 중. 친서를 셔츠 안쪽 주머니에 보관. 모래시계 자리는 한 발 옆, 만지지 않음.
- **번개 모스 통신 양념 시연 (ch026 말)**: 야영지 바위 등 기댄 채 손목 번개 인장으로 **모스 부호 송신** — 점·점·선·점 / 선·점·선·선. 수신인 = 하산. 합류 좌표. 한국 28년차 야근러가 고등학생 때 취미로 외운 모스 리듬을 사막에서 처음 활용. 송신만 가능, 수신 확인 불가.
- **솔라리스 도착 + 동맹 외교 (ch027)**: 닷새 만에 천막군 도착. 나이라 재회 — **"또 안 죽었네." / "또 안 죽었어." 콜백 의식**. 부족장이 동맹 답을 협곡 호송 성공 조건으로 미룸 (사막식 시험).
- **모래폭풍 코어 진정 — 풍+화염+5함수 동시 실행 (ch027)**: 한 달째 자리 잡은 폭풍 코어. **풍 정령 시야로 코어 안쪽 응축점 2개 + 회로 결 인식**. 화염은 안쪽 열대류만 제어. 등 5함수가 두 정령 사이에 분기 알람. **양 발목·양 손바닥 4자리 동시 발열**. 정령석 30보·60보 자리에 박힘 → 응축점 끌림 → 풍·화염 동시 신호 → 회오리 거꾸로 솟구쳐 풀림. **외부 조력자(나이라 결정 가지 좌표 / 하산 동선 감각 / 부족 사람 정령석 박기)가 결정적**. 첫 다중 정령 동시 운용 성공.
- **두 번째 자객 + 깨진 모래시계 표식 (ch027 말)**: 협곡 중간에서 두 번째 자객. 로헨이 베어내고 도주. 떨어진 장신구 = **깨진 모래시계 모양**, 안쪽 색이 다른 금속 알갱이. 모르데우스 계열의 다른 가지.
- **솔라리스 동맹 정식 체결 (ch028)**: 사막식 + 아젤리아식 양식. 사관이 매뉴얼 외 한 줄 직접 작성 — *"이 문서는 외지인 부왕의 말과 사막 부족장의 침묵 사이의 약속을 옮겼다."* 평민 사관 캐릭터 곡선의 클라이맥스. 캐러밴 노선 3, 월 2회 시작. 나이라·하산 동맹 파트너 정식화.
- **정령석 사치품 본격 세공 (ch028)**: 부족 장인의 작업장. 주먹 크기 원석에 **화염+번개 양손 동시 봉인**. 5함수가 화염 온도 한계 + 번개 전압 한계 + 분기 알람. 결과 = **빛·온도·진동 세 결**이 외부 자극에 반응. 외형은 카이젤 취향 직선+방패 모티프 (장인 담당). **분업 구조 — "본사는 코어만 쥐고 외주에 외형을 맡기고 시장에는 안 뜯기는 구조."** 카이젤이 견본 뜯어도 모방 불가 (화염=솔라리스 한정 / 번개=지호뿐 / 5함수=등 위쪽). "이게 내 직업인 모양이다." 카이젤 황실 외교 선물 견본 1점 완성.
- **마르코 3차 등장 (ch028)**: 천막군 외곽 그늘. **"또 보네"** (변주된 4음절). 두 번째 자객 깨진 모래시계 장신구 회수 (로헨이 언제 넘겼는지 불명). "그 표식, 본 적 있어." + **"다음은 북쪽일 거야. 강철 색이지."** 카이젤 = Arc 6 예약. "네 사람이 있어?" 질문에 답 거부 — "그건 가서 봐."
- **에리나 답장 + 모래시계 두 번째 어긋남 (ch028 말)**: 답장 친서 — "지호에게" 사적 첫 줄. 출산 한 달 남음. 약지 반지 헐거워짐. 새벽 배 위 아이 발. "돌아오는 길은 급하지 않게." **모래시계 두 번째 어긋남 / 두 번째 한 알 분만큼 위쪽 결 내려앉음**. 흔들 자격 없고 흔들 방법 없음. "그런 걸 알고도 앉아 있는 게 부왕의 일인 모양이다."

### 성장의 핵심 공식
> **"찌질한데 버틴다."**
>
> 성격은 안 나아진다. 돈미새, 여미새, 겁쟁이 — 기본 장착.
> **이세계에 와도 안 고쳐진다.** 결혼해도 안 고쳐진다. 그게 강지호다.
> 대신 **반복 횟수와 코드 사고**가 쌓인다.
> 재능이 없으니까 모든 체계를 조금씩 흡수해서 합칠 수 있다 — 무재능의 역설.
> 매번 처음과 같은 강도의 고통을 견딜 수 있는 한, 이 사람은 계속 강해진다.

### 의지의 주인공
- 이세계 지식도, 전략도 아닌 "그럼에도 불구하고"의 의지

- 자기 한계가 만든 깨달음으로, 남들이 못 보는 것을 본다 — 시스템의 버그
- 강지호만의 자격: "아무것도 아닌 자의 시선" — 특별하지 않기에 모든 것을 볼 수 있는 자

## 6-B. Part 2 성장 마일스톤 (Arc 5: ch029~ch034) — "사막의 부강 설계자"

### Arc 5: 사막의 캐러밴 (ch029~ch034) — "이름을 넘기고 북쪽으로"

- **이중 천막 4요소 설계 (ch029)**: 부족 장인 4인(자말 외)에게 매뉴얼 인계. ① 이중 벽(손 한 뼘 간격) ② 카이젤 은박 가죽 반사판 ③ 통풍 굴뚝 ④ 갈대 묶음 경량 폴. **풍 정령**으로 굴뚝 안 미세 상승기류 + **화염 정령 잔열**로 은박 반사각 45→40도 보정. 캠핑 리뷰 500편 본 한국 IT의 첫 사막 부강 카드. 시제품 1동 완성 시 실내 외부 대비 -15도. 하산 인부 한 명이 이중 벽에 이마 대고 눈 감음. **현장 피드백으로 이중 벽 한 뼘→한 뼘 반·갈대 5→7가닥·경첩 단순화** 모두 수용. 나이라 첫 평가 "너 죽이면 안 되는 이유가 하나 더 생겼네."
- **모래 위 첫 깨진 모래시계 표식 (ch029 새벽)**: 호위 야영지 외곽. 새끼손가락 한 마디 길이, 허리 사선 가른 윤곽. 발자국 없음. 보고하지 않고 셔츠 안쪽 셋째 주머니 손이 가다 멈춤. "한 명이 아니거나, 한 무리가 더 있거나."
- **일곱 번째 오아시스 협곡 통과 (ch030)**: 천막 11동 운반. 협곡 모래 나선 시동 단계에서 **풍+화염+5함수 동시 차단 — 지난번(ch027) 절반 시간**. 나이라가 위치를 결정 가지로 한 단어 ("여기")만 짚음. 신뢰 누적 1단. 야간 미식별 공중체 1회 목격 — 새 아닌 직선 궤적, 한 호흡 만에 시야 밖. 보고하지 않음 ("말하면 책임져야 할 정보의 양이 감당 안 된다").
- **정령석 사치품 양산 시작 (ch031)**: 오아시스 야자수 그늘 작업장. **하루 5점 목표 시도 → 3번째 원석 결 가로축에 화염 첫 파동 닿아 쪼개짐 = 첫 실패. 손바닥 5초 화상 → D급 즉시 회복.** 4·5점째 정상화. **80% 성공률 (4/5)**. "내가 짠 함수 중에 이 정도면 프로덕션 올려도 된다." 손목 인장 식는 호흡: 3→4→5→4 (과로 신호 누적). 부족장이 카이젤 분배 분리(공작 이름 / 부족 이름) 결정. 다마르 견본 + 본대 4점 → 카이젤 황실 손에 7주 내 도착 예정. **D-25 임신 친서 1차 수령** — 에리나 호흡이 가빠짐, 의원 정상 판정. "지호. 호흡이 자주 가빠집니다. 걱정 마세요." 답장 두 줄째에서 펜 멈춤.
- **두 번째 깨진 모래시계 표식 + 작업장 도주자 (ch031 자정)**: 대표 장인이 도주자 한 명을 쇠 집게로 격퇴. 모래 위에 같은 윤곽 표식 두 번째 발견. **로헨도 동일 윤곽 인지 — 정보 공유 지점.**
- **협곡 매복 + 검은 로브 단어 노출 (ch032)**: 풍 정령으로 모래 위 자취 감속 추적 — **결이 지워지는 속도를 늦춰 어제·엊그제·오늘 아침 자취 3종 식별**. 새 활용. 매복으로 1명 생포·1명 도주. 외투 안쪽 가죽에 **불로 지진 깨진 모래시계 표식 = 세 번째**. 심문에서 "검은 로브"만 4회 반복 → "반복이 늘면 단어가 가벼워진다"로 종료. **마르코 4차 등장** — "또 왔네"(4음절). **"그 표식 만든 자는 천 년 전부터 있어. 카이젤도, 너희 왕국도, 솔라리스도 다 그 그림자 안에서 자랐어."** "천 년" 정보 첫 인계. **모래시계 세 번째 한 알 떨어짐** — 잔량은 1/4 그대로, 박자도 어긋남도 없이 조용히 한 알. 검은 로브 매뉴얼 톤 첫 직접 인지.
- **본대 12~15명 전면전 + FS-A11 기후 프로그래밍 씨앗 시연 (ch033)**: 협곡 본대 매복. 적이 모래폭풍 결 유도. **풍 정령 시야로 결 반으로 쪼개 양 벽으로 흘림 → 화염 차단선 + 번개 무릎 신경 정밀 타격.** 5함수 자동 분기(풍 결/화염 차단/번개 정밀/접합 열 보정/동기 타이밍). 적 본대 은폐 사각 — **할리마가 무릎 꿇고 모래 위 전통 문양으로 협곡 기류 결 가시화 = 외부 조력자 결정타**. 나이라 결정 가지 + 하산 동선 + 로헨 검 12걸음. "내가 한 게 아니다." **마지막 함수 묶음에서 충동적으로 순서 변경 — 풍 바깥·번개 가운데·화염 사이로 묶자 협곡 한 구간 기류 전체 역전.** 적 진영에 역모래·역풍 동시 강타. 자각 첫 1줄 — **"세 개를 한 번에 묶으면 날씨 자체를 함수처럼 쓸 수도 있겠다"** (호기심+자조 반반, 결의 X). 신규 동작 = 어금니 한 번 물기, 귓바퀴 뒤 엄지 문지르기. 인부 1명 어깨 부상(뼈·힘줄 무사). **할리마 첫 노쇠 신호 — 협곡 벽에 등 붙이고 첫 긴 호흡 / "다음에 이 자리에 올 수 있을지 오늘 밤에 한 번 더 생각해 볼 일이야"**. 모래시계 박자 한 호흡 → 두 호흡 더 두꺼워짐, 잔량 1/4 그대로. 재자각 1줄 — "함수 하나가 더 열렸다."
- **솔라리스 출발 + 카이젤행 (ch034)**: 부족장과 네 번째 손바닥 마주 댐 의례 ("가는 사람이 받는 의례"). **천막 4요소 + 정령석 세공 도구 솔라리스 이름으로 정식 인계 — "이름이 넘어가면 질문도 넘어간다"**. 분업 구조 본격화. 나이라 "살아 있어. 이번엔 북쪽에서." 할리마 "가." 한 음절. **D-15 임신 친서 2차 수령** — *"여기 와서 같이 봤으면 좋겠어요."* 답장 두 줄 — *"잘 먹고 잘 자요. 같이 보러 가는 중."* **"지켜준다" 동사가 펜 끝에서 한 번 걸렸다가 그 자리에서 멎음**. 마르코 5차 등장 — "또 만났군"(다섯 번째). **"북쪽 길은 다섯이야. 그중 셋이 너를 알아."** 추가 인계 — **"바람·불·번개 — 세 개를 묶은 뒤에 네가 만나는 게 진짜야."** 어제 우연 시연을 마르코가 알고 있음. 카이젤 진입 임박. 지평선 강철 색 — 비유 아닌 실물. 모래시계 두 번째 박자 또 비켜 감, 잔량 그대로. "이번엔 한 기둥을 끝내러 가는 게 아니다. 한 무리를 끝낼 일이 있다."

### 모래시계 (마르코의 깨진 유리 모래시계)
- ch004에서 마르코가 "재고 정리"로 강제 증여. 마력 미감지 고물 판정으로 누구도 의심 안 함.
- ch005 의식 도중 자동 발동 — **0.3초 시간 정지** (1회성 현상으로 추정). 은빛으로 빛나며 기둥 내부의 갇힌 영혼들과 금빛 균열 목격.
- 의식 후 모래가 절반 → 1/4로 감소.
- ch013 체내 순환 후 "뭔가가 줄었다" 감각 — 모래시계 또는 수명 직접 차감의 첫 신호.
- ch019 기둥 작업 중 0.1초 재발동 — 갇힌 영혼들이 웃는 모습 잠시 목격.
- ch020 시점 모래 1/4 잔량. 다 떨어지면 어떻게 되는지 미확인.
- **ch022 마르코의 정체 노출**: 모래시계는 **9개 기둥 전체의 카운트다운**. "그건 세상의 모든 기둥에 남은 시간을 보여주는 거지." 1/4 잔량 = 9기둥 중 시간이 그만큼 남았다는 뜻. 0이 되면 기둥 = 차단 배리어 모두 무너짐 → 다른 곳의 침공. 300년 전부터 누군가가 기둥 약화를 가속화. 용사 = 잠깐 늦추는 소모품.
- **ch021 박자 어긋남 시작**: 잔량 변화 없이 떨어지는 한 알의 박자가 어긋남.
- **ch024 박자 두 알 동시 떨어짐**: 잔량은 그대로. 어긋나는 폭이 큼. "잔량이 줄지 않았는데 박자만 빨라지는 게 더 무서웠다."
- **ch025 첫 한 알 명시적 감소**: 출발 전날 밤. 위쪽 결이 한 알 분만큼 내려앉은 색. "세계의 시간이 줄었다. 한 알만큼."
- **ch028 두 번째 한 알 명시적 감소**: 사막 동맹 체결 후. 또 한 알 분만큼 위쪽 결 내려앉음. "두 번째 어긋남. 흔들 자격도 없고, 흔들 방법도 없었다."
- **ch032 세 번째 한 알 떨어짐**: 마르코 "천 년" 인계 직후. 잔량 1/4 그대로, 박자도 어긋남도 없이 위 → 아래 한 알이 조용히 자리 이동. "그 사이의 순간은 내가 보지 못했다."
- **ch033 박자 두 호흡 누적**: 협곡 기류 역전 시연 후. 한 호흡 어긋남이 두 호흡으로 두꺼워짐. "잔량은 그대로다. 박자만 더 빨라졌다. 박자가 빨라진다는 건 세계가 빨라진다는 뜻이다. 아니면 내가 느려졌다는 뜻이거나."
- **ch034 네 번째 박자 비켜 감**: 출발 직후. 두 번째 박자가 첫 번째보다 반 박자 앞서 먼저 나옴. 잔량 1/4 그대로 (세 알 변동 없음).
- **현재 확인된 것**: 극한 상황 자동 발동, 사용자 의지 무관, 9기둥 전체의 카운트다운, Part 2 시점 카운트다운 가속 시작.
- **미확인**: 의도적 발동 여부, 0이 되었을 때 정확한 결과, 누구의 약화 작업인지 (검은 로브 = 익명 거래자).

- **체력**: 솔라리스에서 번개 파괴-재생 루프로 근육 밀도 증가. ch020 시점 푸시업/스쿼트 각 100개. **ch021 푸시업 200개로 상향 (로헨 지시) — 근위대 평균과 동일**.
- **반사속도**: 번개 신경 가속 60% 통제 → 90% (문신 코딩 후).

- **인간관계 자산**: 에리나(아내·부왕 자격), 로헨, 세드릭 왕, 저항군 핵심 50명 / 솔라리스의 나이라·하산·할리마. **+ 평민 사관(이름 미공개, 카드 색인 동행자) / 다마르 공작(위장 외교 상대) / 솔라리스 부족장·부족 장인(정령석 사치품 분업 파트너)**.
- **다중 정령 동시 운용**: ch027에서 풍+화염+5함수 동시 실행 첫 성공. ch028에서 화염+번개 정령석 봉인 첫 성공.

### 재생의 한계 (ch016 이후 인지)
- 미세 신경 완전 복구까지 약 10분.
- 파괴 속도가 재생 속도를 지속 초과하면 영구 소실 가능.
- 피로와 수명 소모는 재생되지 않음.

## 9. 정령 계약 및 능력 (ch028 시점)

- **모스 부호 통신 (ch026)** — 손목 인장 안쪽에서만 점·선 리듬 방전. 송신만 가능, 수신 확인 불가. 한국 고등학생 시절 외운 모스 리듬 활용.

- **발목 안쪽 동전 절반만 한 옅은 자국 = 인장**. 사막 협곡 회오리 안에 9초 갇힌 직후 계약 체결.

- 정령에게 명령으로 외곽 압력 분산 가능.
- **굴뚝 안 미세 상승기류 양념** (ch029) — 천막 통풍 보조, 계약 외 표면 길들이기 수준.
- **모래 위 자취 감속 추적** (ch032) — 모래 위에 바람을 얇게 얹어 결이 지워지는 속도를 늦춰 어제·엊그제·오늘 자취 3종 식별. 새 활용.

### 문신 코딩 (ch017 시술)

- **ch033 3정령 + 5함수 동시 운용**: 풍 결 분석 / 화염 차단선 / 번개 정밀 타격 / 접합선 열 보정 / 동기 타이밍 — 5함수가 자동 분기. **마지막 함수 묶음에서 충동적으로 풍 바깥·번개 가운데·화염 사이 순서로 묶자 협곡 한 구간 기류 전체 역전(역모래·역풍 1회).** FS-A11 기후 프로그래밍 씨앗 시연 (자각 1줄 "세 개를 한 번에 묶으면 날씨 자체를 함수처럼 쓸 수도 있겠다" + 재자각 "함수 하나가 더 열렸다").

## 11. Part 2 예고 (Arc 4~10, 참고용)

> 이 섹션은 Part 1 종료 시점 기준의 **계획**이며, 본문 사실이 아니다. 작가/에이전트의 방향 고정용.

### 정령 다중 계약
- Arc 4~10에 걸쳐 **3~7마리 추가 정령 계약**. 각 지역 기둥 원소(물·바람·얼음·흙·금속·어둠·빛 등)에 따라 분기.
- 일반 술사는 다중 계약 시 침식이 가속되어 즉사하지만, 지호는 D급 재생 + 무재능 체질로 즉시 회복 → 다중 계약 가능.
- 단, **기둥 직접 치유에는 본인 수명을 직접 지불** (모래시계 차감 또는 노화). 정령 사용량과 별개의 비용 구조.

### 노화 신호
- **Arc 7부터 흰머리 시작**. 외형 노화의 시각적 마커.
- 에리나는 부왕의 아내로 외부에서는 변화 적음.

### 가족 변수
- **Arc 10: 아들 출산** → 아버지 정체성 추가. 단순한 기둥 수리 임무에 "다음 세대를 위해"라는 무게가 더해짐.

### Part 3 도착점
- **모든 기둥 치유 = 수명 전부 소진 → 죽음**.
- 모래시계 0이 되는 시점이 곧 결말.
- 결혼 + 부왕 + 아버지가 된 강지호가, 그래도 마지막 기둥까지 코드를 쓰러 간다.
