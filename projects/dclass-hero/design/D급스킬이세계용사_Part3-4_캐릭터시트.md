### 2-2. 능력 본질 — "무공+정령+코드 융합체"

| 영역 | 강지호 (父) | 우주 (子) |
|---|---|---|
| **본질** | 외부 시스템을 디버깅(편집) | **두 체계의 통역 + 새 규칙 작성** |
| **트리거** | 손바닥 + 정령 인장 + 문신 코딩 | **시선 + 호흡** (가장 자연스러움) |
| **한계** | 코드로 풀 수 없는 영역 존재 | **너무 어림** — 신체·집중력 한계 |

| **외부 인식** | 부왕·용사로 알려짐 | **카에데·강지호만 알아봄** (가드레일 §18). 외부는 일반 아이 |

**능력 발현 타이밍 (★ plot-hook-engineer 인계)**:
- **1차 (EP093, 만 4세)**: 정령 감응 자각. 모친 에리나의 결을 따라 정령(불·바람·물)에게 호흡으로 응답. **명시 X**, 미세 비트만.
- **2차 (EP120, 만 6세)**: 무공 자각. 카에데의 청월문 호흡법이 우주에게 가장 자연스럽게 흘러감. 카에데가 무협식으로 인정 — "오래 기다린 그 길이 여기 있구나."
- **3차 (EP140, 만 8세)**: 코드 직관 자각. 강지호의 코드를 시선만으로 통째로 읽어냄. 부자 공투 시작.

**아젤리아 혁신 가드레일 §13 적용**: 우주의 능력도 **프로그래머 가능 범위 + 호흡 수련 영역**에 한정. 증기·화약·야금·전기 자체 발전 X.

### 2-4. Dialogue DNA — 우주 (★ 핵심)

**구성 비율**: 강지호 식 자조 1/3 + 에리나 식 절제 1/3 + 카에데 식 신비 1/3.
- 카에데에게 배운 비유는 **동양풍** ("실타래" / "숨결" / "별의 결" 같은 어휘를 우주가 일상에 풀어씀)
- 강지호의 자조는 어휘는 아니고 **톤만** 흡수 (10세 아이가 "ㅋㅋ"하지 않음)
- 에리나의 절제는 **멈춤**과 **미완성 문장**

**나이대별 톤 진화**:

| 나이 | 톤 |
|---|---|
| 0~1세 (EP059~070) | 옹알이만. 대사 없음. 강지호 1인칭 해석으로만. 카에데가 안았을 때만 옹알이 결이 달라짐 |
| 2~3세 (EP071~080) | 짧은 어림 한국어. 발음 변형 X (한국 웹소설 거부감) — **짧은 평어로 어림 표현** |
| 4~6세 (EP081~100) | 짧은 문장 + 본질 짚는 1줄 (EP당 1줄 이하) |
| 7~8세 (EP101~130) | 어른 대사 + 감정적 미숙함 (위기엔 침착, 끝난 후 흔들림) |
| 9~10세 (EP131~158) | 강지호 어투 흡수 + 자조 첫 시도. 카에데 어휘 일상화 |

**대사 샘플 (나이별)**:

**0~1세** (옹알이만):
- 강지호 1인칭 해석: "녀석이 옹알이를 했다. 카에데를 향해. 내가 못 듣는 무언가가 있는 듯한 어조였다."
- 강지호 1인칭 해석: "에리나가 안았을 때보다, 카에데가 안았을 때 더 조용했다. 그게 거슬렸지만, 말은 안 했다."

**2~3세** (짧은 어림 한국어, 5개 샘플):

- "검 빛나."
- "할아버지 술 또?"
- "엄마, 별."
- "이거 — 따뜻해."

**4~6세** (본질 짚는 1줄, 5개 샘플):

- "이 빛은 슬픈 빛이야."
- "엄마 손, 따뜻해."
- "할아범, 안 졸려?"
- "시즈루 누나 — 안 무서워."

**7~8세** (어른 대사 + 미숙함, 5개 샘플):

- "...별이 안 와. 오늘은."
- "엄마, 안 울 거야. 약속."

**9~10세** (자조 첫 시도 + 카에데 어휘 일상화, 5개 샘플):

- "실타래 끝이 — 여기까지인 거 같아. 다음은 내가 잡아볼게."
- "카에데 누나, 오늘은 — 안 봐도 돼."

## 5. 적대자 계층 (Phase 3·4)

### 5-1. Arc 11 (EP076~092) — 카이젤 강철 기둥

- **메인 적대자**: **카이젤 황실 잔당 + 강철 기둥 시스템 관리자 1명** (이름 미상)
- **계층**: 카이젤 황실 (다마르 출신 가문 + 황제 직속) + 강철 기둥 보호자 (모르데우스급, 의도적 약화)
- **다마르**: ch022 정본 등록. EP080 부근 카이젤 입성 시 본격 재등장. **회색 인물** — 강지호와 협력하거나 적대하거나 양쪽 가능. 본 설계는 **부분 협력** 권장 (다마르가 카이젤 황실에 등을 돌리는 정치 비트).
- **위협 수준**: 중간. 카이젤 정규군 + 잔당 + 시스템 관리자. 모래시계와 결이 다른 강철 색 도구를 다룸.
- **합리적 동기**: 카이젤 황실은 천 년 전부터 강철 기둥에 의존. 강철 기둥이 무너지면 카이젤 자체가 무너짐. 강지호 = 카이젤 멸망의 트리거. 따라서 합리적 적대.
- **약점**: 강철 기둥 시스템 관리자의 도구화된 무력은 강지호의 코드 + 시즈루의 검술 + 정령 합공으로 분해 가능. 모르데우스보다는 강하나 절대 무적은 아님.

### 5-2. Arc 12 (EP093~110) — 빙 기둥 (프로스트헬)

- **메인 적대자**: **빙 기둥 시스템 관리자 1명** (이름 미상) + **검은 로브 네트워크 본격 합류** (EP100 첫 정면 충돌)
- **계층**: 프로스트헬 봉인지 수호 혈통(카이, 자리표) — 사실 적대자 X, 다만 빙 기둥 보호자가 다른 인물.
- **위협 수준**: 중상. 빙 기둥은 정령 감응을 요구하는 영역 → 강지호의 코드만으로는 불충분 → 우주의 정령 감응 1차 발현(EP093) 트리거.
- **합리적 동기**: 빙 기둥 시스템 관리자는 천 년간 봉인을 유지해온 자. 무너지면 본인의 존재 의미도 사라짐. 따라서 합리적 저항.
- **검은 로브 본격 노출**: EP100 부근 검은 로브 단원 1명이 정면 등장 — 단, 천 년 전부터 있어온 그림자 네트워크의 일원. 마르코의 ch032·ch034 인계 회수.

### 5-3. Arc 13 (EP111~128) — 용 기둥 + 동천 통로 회수

- **메인 적대자**: **용 기둥 시스템 관리자 1명** (이름 미상, 잠정 "현무진인" 자리표 활용 검토 — 단, 보호자가 적대자로 전환된 케이스로 재구성)
- **계층**: 용화국(동방 선계) — 무협 차원과의 좁은 통로가 있는 곳. 청월문 백스토리 점진 노출.
- **위협 수준**: 중. 무협 측 적대 ≠ 청월문(우호). 동천의 다른 문파 또는 강림 균열을 노린 적대 세력.
- **합리적 동기**: 무협 차원의 다른 정파/사파가 청월문의 강림을 추적. 카에데의 1세대 전 예언을 다른 해석으로 봄. 우주를 동천으로 데려가려는 시도.
- **회수 비트**: EP120 우주의 무공 자각 + EP125 카에데의 청월문 백스토리 1차 노출 + EP128 용 기둥 봉인 + 동천 통로 폐쇄 (자유 왕복 불가 재확인)
- **R-A 함정 2 회피**: 무협 차원 백스토리는 **여기서 점진 노출, 한 번에 다 풀지 않음**. 4부 떡밥으로 일부 잔류.

### 5-4. Arc 14 (EP129~145) — 바다 + 모래/대지

- **메인 적대자**: **바다 기둥 시스템 관리자 (리베르타 해상 해적 연합 배후) + 모래/대지 기둥 시스템 관리자 (카즈모르 드워프 광맥 배후)**
- **계층**: 두 보호자가 동시에 작동. **합본 처리** — 한 아크에서 두 기둥을 풀어내는 마라톤.
- **위협 수준**: 중. 점진적 강화 but 부차 (메인은 부자 공투 빌드업).
- **합리적 동기**: 바다 보호자는 군도의 자치권 유지. 모래 보호자는 드워프 광맥의 지속. 두 보호자 모두 강지호 = 자기 영역 멸망의 트리거로 인식.
- **부자 공투 시작 비트**: EP140 — 강지호 코드 분석 + 우주 직관 결단으로 두 기둥 중 하나 풀이.

### 5-5. Arc 15 (EP146~158) — 천 기둥 (셀레스티아) + 심해 통합

- **메인 적대자**: **천인 평의회 (셀레스티아 최종 적)** + **심해(아비살) 통합**
- **계층**: 천인 평의회 = 의도적 약화 유지 (가드레일 §17). 셀레스티아 함락 = 9기둥 완결.
- **심해 통합**: 카즈모르(지하) + 아비살(심해)을 **"지" 기둥 합본**으로 처리. 천 기둥과 동시에 마무리 가능.
- **위협 수준**: 외형은 중상, 실제는 중. 결말 톤(미세 노화 + 평화)에 맞춰 격렬한 전쟁보다 **정보·외교 무쌍 + 부자 공투 + 자조 톤 마무리**.
- **검은 로브 결말**: 일부 해체, 일부는 4부 떡밥으로 잔존 (concept-builder 명시). 본 작품에서는 완전 청산 X.

### 5-6. 마르코 / 검은 로브 (전 아크 부차)

- **빈도**: Arc당 0~1회 (가드레일 §7). 안 나와도 됨.
- **누적 등장**: EP058까지 5회. EP059~EP158 100화에 추가 5~7회 권장. EP145 즈음 미스터리 라인 부분 회수.

## 7. 보조 캐릭터 갱신 (EP059~EP158)

### 7-1. 나이라 (Naira) — 솔라리스 화염 정령술사

- **EP058 시점**: 머리 1/3 백화 유지, 폐 화상 진행 추정.

- **Phase B (EP076~130)**: Arc 11 카이젤 정복 동행 (EP080 부근 마지막 본격 활약), 이후 Arc 12·13 단편 비트만.
- **Phase C (EP131~158)**: **EP135 자연사 또는 폐 화상 사망 권장** (수명 35세 한계 도달, R-C 함정 6 회피 — 아이의 위기보다 어른 캐릭터의 죽음으로 정서 자산화). 작별 비트: "또 안 죽었네"의 마지막 콜백 후 침묵.

- 우주와의 관계: 짧은 친밀. 카에데와 다른 결의 정령 감응 멘토 — 단, 본격 멘토링 X.

### 7-2. 로헨 (Rohen) — 근위대장

- **EP058 시점**: 3어절 시그니처 + 갑옷 어깨 부딪치는 동선 정착.
- **Phase A (EP059~080)**: 강지호 회복 동안 근위대 통솔. 시즈루와의 미세 호위 분담.
- **Phase B (EP081~130)**: 카이젤·빙 기둥 동행. EP100 강지호 능력 한계 노출 시 로헨의 결정적 칼 한 합 권장 (1회 한정).
- **Phase C (EP131~158)**: 근위대장 직위 유지. EP155 노인 자연사 시 절차주의로 장례 동선 (3어절 시그니처).

- 우주와의 관계: 거리감 있는 보호자. EP130 시간 압축 챕터에서 우주에게 검 한 자루 (목검) 건넴 — "검을 잡아 본 적, 있습니까?" (1·20화 첫 마디 반복).

### 7-3. 피트 (Peat) — 의수 정령술사

- **EP058 시점**: 정본 등장 (지호를 지키는 자리). 의수에서 마력 끓어오름.
- **Phase A~C**: 보조 동행. 의수 정령술의 점진 진화. 우주의 정령 감응 1차 발현(EP093) 시 피트가 옆에서 함께 인지 — "이게 정령이 — 부르는 결입니까."

### 7-4. 마르코 (Marco)

- **빈도**: Arc당 0~1회 (가드레일 §7). 안 나와도 됨.

- **핵심 비트**: EP100 (검은 로브 본격 정면 등장 직전, 마르코의 단편 인계) / EP120 (우주 무공 자각 후 마르코의 미세 인정) / EP145 (마르코 정체 일부 노출, 4부 떡밥) / EP158 (마지막 1줄 자조 — "또 보네." 콜백 1회만)
