## 세계관 핵심 용어

| 용어 | 설명 | 상태 |
|------|------|------|
| 세계수 | 고대에 세계를 하나로 묶던 원초적 존재. 셀레스티아에 의해 9조각으로 분쇄됨 | 📋 |
| 기둥 (Pillar) | 세계수의 파편. 각 지역의 현실/마법/환경의 원천. 9개 존재 | ✅ 3화~ |
| 빛의 기둥 | 아젤리아의 기둥. 수정 기둥 형태. 영혼을 먹어 유지됨 | ✅ 3~5화 |
| 10번째 기둥 (시간의 기둥) | 세계수의 심장. 파괴됨. 파편이 지호 영혼에 박혀 있음 | 📋 |
| 축복의 의식 | 용사의 영혼을 빛의 기둥에 바치는 제의. 300년 역사 | ✅ 2~5화 |
| 소환진 | 다른 세계에서 용사를 불러오는 마법진 | ✅ 1화 |
| 결계 | 마법 방어막. 의식 전날 가동, 탈출 봉인 | ✅ 4~5화 |
| 회귀 (루프) | 사망 시 사이 공간을 거쳐 다른 지역에서 부활하는 현상 | ✅ 5화 |
| 사이 공간 | 사망과 부활 사이의 중간지대. 어둠. 존재의 목소리 | ✅ 1화, 5화 |
| 용사 | 다른 세계에서 소환된 인물. 아젤리아에서 33인 역사 | ✅ 1화~ |
| 대몰락 | 셀레스티아가 세계수를 의도적으로 분쇄한 사건 | 📋 |
| 천명 (天命) | 용화국의 운명 결정 시스템. 기의 기둥이 인위적으로 부여 | 📋 |
