## 1. 무협×이세계 융합 (R-A) — 검증된 흥행 영역

- 한국 웹소설은 **무협+다른 장르 융합**(우주천마 3077·무림서부·무림 속 외노자가 되었다 등)을 2020년대 메가트렌드로 확립. dclass-hero의 EP058 무협 차원 강림은 **안전 영역**.
- 톤 전환 처방: **무공 용어(내공·검기·축기·단전)는 차용 OK**, 사문/장문인/사숙 등 위계 용어는 풀이 1줄 필수. **고풍체는 노인 1인에게 몰아주기**.
- 무협 인물 3인 비중 차등: **노인 = 저비중 사부**(아크당 1~2씬) / **카에데 = 중비중 우주 멘토** / **시즈루 = 고비중 강지호 동료·라이벌**.
- 두 체계 우열 비교 X. **"다른 길"**로 처리, 결정적 위기에서만 합공.

## 2. 부자 서사 + 세대 교체 (R-B) — 메인 축, 가장 까다로움

- 회귀한 아빠는 10경 재벌이~(카카오 5.8M)·재벌집 막내아들·모험가가 되고 싶다며~ 등 **부모-자식 서사는 한국에서 검증된 흥행 동인**.
- **3 페이즈 전환 설계** (★ 핵심):
  - Phase A (EP059~EP080, 우주 0~3세): 강지호 메인, 우주 보호 객체
  - Phase B (EP081~EP130, 우주 4~8세): **위험 구간** — 사이다 공급원 전환의 가장 어려운 시점
  - Phase C (EP131~EP158, 우주 9~10세): 우주 메인, 강지호 후견인
- **Phase B 위기 처방**: 강지호 메인 사이다 유지하면서 우주 보조 활약 짧게. EP100 부근 강지호 능력 한계 → EP110 우주 첫 단독 사이다 → EP120 카에데가 우주를 무협식으로 인정.
- 결말 톤: **부모 죽음 X, 영생 X. "미세 노화 + 평화 + 가족"** 권장.
- 우주 능력 차별화: 강지호 코드와 "다른 결". **무협+정령+코드 융합체**가 우주만의 능력으로 권장.

## 3. 아동 성장 묘사 (R-C) — 실용 작법 노트

- 0세: 정상 갓난아기 + 결정적 1~2 비트의 이질감(카에데와 눈맞춤 등). 내면 직접 서술 X.
- 1~3세: 짧은 어림 한국어. 아기말 변형 최소 — **"아빠 가" / "검 빛나"** 식.
- 4~6세: 부모 모방 + 1화당 통찰 대사 1줄 이하.
- 7~10세: 강지호 자조 톤 흡수. Phase C 진입 시 1~2화 우주 1인칭 실험 가능.
- **시간 압축 챕터** EP080·110·130 3회 권장. 일상 컷 + 사이다 비트 1개 + 부자 정서 비트 1개 필수.
- POV: 강지호 1인칭 유지. 분리 씬은 EP당 1~2씬 이하.

1. **3 페이즈 전환을 큰 설계 단계에서 명시**: bootstrap·character-architect·plot-hook-engineer 모두 EP059~80 / EP81~130 / EP131~158의 3구간 구분을 기본 골격으로 사용.
2. **무협 인물 3인의 비중 차등 명문화**: character 시트에서 노인(저)·카에데(중)·시즈루(고) 비중 명시. 우주 멘토링은 카에데에 집중.
