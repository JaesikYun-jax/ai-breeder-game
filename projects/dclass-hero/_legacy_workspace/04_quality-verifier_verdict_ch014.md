### 축 6: HOOK — 훅 강도

```
오프닝 훅: 감각/분위기 / 강도: 3/5 / 목표: 3+
```
"눈을 떴을 때, 아젤리아 방향에서 바람이 불었다." — 아젤리아 방향이라는 복선적 오프닝. 바람 감각→일상→행상인 도착으로 자연스러운 전개. 목표 충족.

```
엔딩 훅: 클리프행어+시각적 이미지 / 강도: 5/5 / 목표: 4+
```
건조 번개 목격 → "저 안에 정령이 있다" → "아무도 살아서 계약한 적 없다" → "재생이 있으면 된다. 이론적으로는." → "내일 밤, 저기로 간다." + 그림자가 번개 방향으로 뻗는 시각적 마무리. 독자에게 "다음 화에서 번개를 맞을 것"이라는 기대+공포를 강력하게 심음. 목표 초과 달성.

```
포인트 장면 수: 3개 / 목표: 2~3
```
1. **플래시백** (L119~L143): 5화 축복 의식 고통 재현. 감정적 충격 장면.
2. **A~D안 분석** (L196~L276): 프로그래머 분석 모드. 캐릭터성 극대화.
3. **할리마 대화** (L359~L482): "죽으러 가는 거 아니지?" / "살려고 가는 거예요." 챕터 하이라이트.

```
코미디→고통 낙차: 있음 / 강도: 4/5
```
SI 비유(L37)+잡부 자조(L15~L16) → 행상인의 아젤리아 소식 → 플래시백. 일상에서 급격 전환.
