## 총평

6화는 Arc 2 솔라리스의 첫 챕터로, 나이라 3인칭 도입 → 지호 1인칭 합류의 이중 시점 구조가 잘 작동한다. 그러나 **`있었다` 계열 과용이 심각**하다. 전체 22회(순수 13회 + `고 있었다` 9회)는 상한의 4배 이상으로, 문장 종결의 단조로움을 유발하는 최우선 교정 대상이다.

**우선순위:**
1. **CRITICAL**: `있었다` 22회 → 5회 이하로 감축 (현재형 전환, 능동형 전환)
2. **MAJOR**: `고 있었다` 9회 → 3회 이하로 감축
3. **MAJOR**: MORPH `~다는 것을` 3회 반복 → 변주
4. **MINOR**: `이었다` 종결 14회 → 일부 변주
5. **MINOR**: AITONE "이때는 몰랐다" 패턴 2회 → 1회로 축소
6. **MINOR**: 지호 존칭→반말 전환 계기 보강
7. **참고**: 나이라/하산 voice-guide.md 미등록 — `/settings-sync` 필요
