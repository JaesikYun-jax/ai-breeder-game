| # | 지호 발화 | 존재 응답 | 메아리 여부 |

| 11 | (살아 있으면 후회도 할 수 있잖아) | "……." → 한숨 | PASS — 비언어 반응, 말하려다 관둠 |

**판정: PASS**

메아리 대사 0건. 사이 공간의 존재는 매 응답에서 새로운 정보, 역설적 관점, 또는 감정적 반응을 제공한다. 특히 지호의 메타적 예측("다음 대사는 뻔하다")을 충족시킨 후 즉시 반전("그래서 널 고른 거다")하는 패턴이 효과적. voice-guide.md의 "정보를 전부 주지 않음 — 단편적 설명 후 끊김" 보이스 규칙에도 부합.

**등장 조연 목록:**

| 조연 | 등장 분량 | 독자적 목표/동기 | 주인공과 별개 갈등 | 아크 진행 |

| 세드릭(왕) | 1줄 대사 | X — 배경 역할 | X | X |

**분석:**

1화는 구조적으로 "지호 vs 어둠 속 목소리" + "소환 직후 혼란"의 2파트로 나뉜다.

- **사이 공간의 존재**: 독자적 목표(세계 구원을 위한 용사 모집)와 지호에 대한 미안함이라는 내적 갈등이 암시된다. "……."(말하려다 관둠), 안도/연민의 복합 감정 등. 조연으로서의 긴장점이 존재한다. **PASS**.

- **소환실 인물들(왕, 에리나, 모르데우스, 기사들)**: 1화에서는 등장 순간일 뿐, 독자적 긴장점이 부재. 그러나 이는 1화 구조의 필연적 제약이며, 2화에서 에리나의 독립적 갈등("축복" 언급 시 눈 흔들림)이 즉시 배치된다.

**판정: MINOR (1건)**

소환실 씬(씬 5)에서 세드릭/에리나/모르데우스가 "배경 소품"으로만 기능한다. 1화 첫 등장이라는 구조적 한계를 감안하면 자연스러우나, 모르데우스의 코웃음이 2화에서도 후속 없이 묻힐 수 있으므로 추적 필요.

**수정 후보 (선택사항):**
- 소환실 씬에서 에리나 또는 모르데우스에게 1줄의 미세 행동을 추가하면 2화 전개와의 연결이 더 자연스러워질 수 있음. 예: 모르데우스가 소환진의 빛에 미소를 짓지만 미소가 0.5초 멈추는 묘사 (voice-guide의 "미소가 0.5초 멈추면 진짜 놀란 것" 활용).
- 단, 1화는 지호의 내면에 집중하는 구조이므로 과도한 조연 묘사는 시점 규칙(1인칭 내면 독백 중심) 위반 우려. **필수 수정은 아님.**

### ALIVE-4 관계 곡선

**1화 관계 변화 기록:**

#### 강지호 ↔ 사이 공간의 존재

| 항목 | 내용 |
|------|------|
| 관계 시작점 | 완전한 타인 (일방적 소환) |
| 관계 종착점 (1화 끝) | "약간 미안해하는 존재" — 묘한 유대의 씨앗 |
| 변동 유형 | 미세 진전 (+0.5) |
| 근거 | 존재의 연민("후회할 수도 있다"), 한숨, 말하려다 관두는 행동 → 지호에 대한 감정적 투자 암시 |
| 호칭 | 존재→지호: 호칭 없음 / 지호→존재: 내면에서 "목소리", "이 목소리" |

#### 강지호 ↔ 에리나

| 항목 | 내용 |
|------|------|
| 관계 시작점 | — (아직 대면 아님, 원거리 인지만) |
| 관계 종착점 (1화 끝) | 외모 인식 + 복잡한 표정 관찰 |
| 변동 유형 | 기선점(baseline) 설정 |
| 근거 | "예쁘다" 반사 반응 + "기대와 불안이 섞인 얼굴" 관찰 |
| 호칭 | 에리나→지호: (아직 대사 없음) / 지호→에리나: 내면에서 "은발 공주", "공주" |

#### 강지호 ↔ 세드릭(왕)

| 항목 | 내용 |
|------|------|
| 관계 시작점/종착점 | 배경 — 형식적 환영 |
| 변동 유형 | 기선점 설정 |
| 호칭 | 세드릭→지호: "용사여" / 지호→세드릭: 내면에서 "왕" |

#### 강지호 ↔ 모르데우스 추정 인물

| 항목 | 내용 |
|------|------|
| 관계 시작점/종착점 | 배경 — 불쾌한 인상 각인 |
| 변동 유형 | 기선점 설정 (부정적) |
| 근거 | "인상이 어딘가 불쾌했다" + 이세계물 레퍼런스로 "악역 확정" 레이블 |
| 호칭 | 지호→: 내면에서 "금색 로브 사제" |

**3화 연속 정체 체크**: 해당 없음 (1화 = 최초 기록).

**판정: PASS**

1화는 모든 관계의 기선점(baseline)을 설정하는 화이므로 "정체" 판정 자체가 불가. 사이 공간의 존재와의 관계에서 미세 진전(+0.5)이 자연스럽게 발생하며, 2화에서 에리나와의 본격 대면으로 이어지는 설계가 확인된다.

**2화 교차검증**: 2화에서 에리나와의 관계가 "먼 거리-경계 → 약간 가까워짐"으로 진행(alive-tracker 기존 기록과 일치). 1→2화 관계 곡선 흐름은 자연스러움.

## alive-tracker 갱신 내역

### 갱신 사항

1. **강지호 ↔ 사이 공간의 존재** 관계 쌍 신규 추가 (기존 tracker에 미등록)
2. **강지호 ↔ 에리나** ch001 행의 상세화 (기존 기록에 "소환 직후 만남"으로 있으나 세부 보강)
3. **강지호 ↔ 모르데우스** ch001 행의 상세화 (기존 기록에 "빛의 축복 설명"으로 있으나, 1화에서는 아직 대사 없음 — 2화 이후 사건으로 정정 필요 확인)
4. **비언어 메모리** 신규 섹션 추가 — "어둠 일렁임" 계열 3회 사용 기록

### 기존 tracker 데이터 정합성 확인

- 기존 alive-tracker의 `ch001 | 강지호 ↔ 모르데우스` 행에 `소환 후 "빛의 축복" 설명`이 기록되어 있으나, 1화 본문에서 모르데우스는 대사가 없고 코웃음만 침. "빛의 축복" 설명은 2~3화 이후 이벤트로 추정. → **정정 권고** (ch001 행은 "소환실 코웃음 — 불쾌한 첫인상"으로 수정 권장, 실제 축복 설명은 해당 화에 재배치)
