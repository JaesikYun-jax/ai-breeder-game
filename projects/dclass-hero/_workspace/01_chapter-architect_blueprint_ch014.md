## 4. 보이스 퀵레프 카드

### 강지호
- 종결어미: 반말 독백 + 상황별 존댓말 (나이라에게 반존대, 할리마에게 존댓말)
- 문장 길이: 기본 짧음. 분석 모드(강화 루트 검토)에서 길어짐
- 말버릇: 이세계 상황을 웹소설/IT 용어로 번역 ("이건 버그 리포트야"), 자조 유머, 돈 환산
- 감정 폭발 시: 욕설 섞인 내면 독백("그 개새끼가"), 문장 파편화, 신체 반응 선행
- 금지 패턴: 영웅적 각오("내가 세계를 구하겠다!"), 타인 비하, 감정 직접 명명("분노했다"), 시스템 용어
- 출처: protagonist_bible §4, tone_style §7

### 나이라
- 종결어미: 무심한 반말 기본 — "~거든", "~잖아", "~이야"
- 문장 길이: 짧고 직설적
- 말버릇: 감정 표현에 서투름, 돌려 말하기보다 행동으로 보여줌. "어차피 ~" 패턴
- 감정 폭발 시: 말이 끊기거나 침묵. 말 대신 행동(손을 잡거나 어깨를 침)
- 금지 패턴: 장문의 감정 토로, 눈물(쉽게 울지 않음), 달콤한 말
- 출처: plot_framework §Arc2 나이라 프로필, character_core §솔라리스 나이라

### 할리마
- 종결어미: 노인 특유의 편한 반말 — "~야", "~거다", "~이지"
- 문장 길이: 중간. 핵심을 짧은 경구로 압축하는 경향
- 말버릇: 비유와 경구가 많음 ("불을 빌려 쓰면 불이 네 안에서 탄다"), 직접적 경고보다 질문으로 깨달음 유도
- 감정 폭발 시: 감정을 드러내지 않음. 대신 한숨이나 긴 침묵으로 표현
- 금지 패턴: 흥분, 허둥지둥, 명령조
- 출처: plot_framework §Arc2 할리마 프로필 ("정령술 안 쓰고 70까지 산 것 자체가 전설")

### 행상인 (단역)
- 종결어미: 수다스러운 반존대 — "~요", "~라더라고요"
- 문장 길이: 길고 횡설수설. 핵심 정보가 잡담 사이에 묻혀 있음
- 말버릇: 과장, 전문 용어 모름("그 뭐냐, 대신관이라나"), 여러 지역 소식을 섞어서 전달
- 출처: [유추 — voice-guide에 미등록. 병사/민간인 집합 보이스 참조]

## 7. 코미디→고통 낙차 설계

- **코미디 구간** (씬 1 ~ 씬 3 전반):
  - 행상인의 수다스러운 과장 + 캐러밴 일상 유머
  - 지호의 잡부 역할 자조 ("오늘도 체력 담당. 월급도 없는데.")
  - A~D안 리스트업 시 프로그래머적 분석이 엉뚱한 상황과 결합 ("스프레드시트를 모래 위에 그리고 있다")
  - A안 검토: "현재 훈련 가속... 3년 걸려. 서울에서 3년 프로젝트면 회사가 먼저 망해."
  - D안 검토: "기둥에 직접 접근. 자살이잖아. 이건 git push --force 같은 거야."
  
- **전환점** (씬 1 → 씬 2):
  - 행상인의 입에서 "아젤리아"가 나오는 순간. 주변 소리가 멀어지고 지호의 초점이 좁아짐
  - "축복 의식 재개"라는 단어에 손이 떨림 → 밥그릇을 놓침

- **고통 구간** (씬 2 핵심):
  - 5화 축복 의식 플래시백 — 감각 중심: 빛, 세포 분해, 비명, "아이처럼 울었다"
  - "다음 용사에게는 재생이 없다. 수 초 만에 의식을 잃고, 기둥에 먹힌다. 그게 끝이다."
  - 분노가 공포를 덮는 순간: "그 개새끼가 또 사람을 먹이겠다고?"

- **낙차 강도**: **4/5** — 일상 코미디에서 급격히 분노/플래시백으로 전환. 5화급 극한 고통까지는 아니지만, 지호가 처음으로 능동적 분노를 드러내는 감정적 전환점

- 출처: tone_style §3-4, §9

## 8. 훅 전략

- **오프닝 훅**: **갈등** / 강도: **3/5**
  - 캐러밴 일상에서 시작하되, 지호의 내면에 "뭔가가 다가오고 있다"는 불안 깔기
  - 예: 아침에 눈을 떴는데 모래시계가 희미하게 빛나고 있었다 — 불길한 전조
  - 또는: 평범한 아침 작업 중 지호가 아젤리아 방향을 무의식적으로 바라보는 것

- **엔딩 훅**: **질문/클리프행어** / 강도: **4/5**
  - 할리마와의 대화 후, 지호가 사막의 밤하늘에서 건조 번개가 치는 것을 바라보는 장면
  - 내면 독백: 번개가 바위를 때리는 것을 보며 — "저 안에 정령이 있다. 아무도 살아서 계약한 적 없다."
  - 마지막 문장 후보: "재생이 있으면 된다. 이론적으로는." / "내일 밤, 저기로 간다."
  - 독자에게 "다음 화에서 번개를 맞을 것"이라는 기대와 공포를 동시에 심기

- **포인트 장면** (2~3개):
  1. **(씬 2) 플래시백** — 위치: 챕터 전반부. 유형: 감정/고통. 5화의 기둥 흡수 고통이 감각적으로 되살아남
  2. **(씬 3) A~D안 리스트업** — 위치: 챕터 중반. 유형: 긴장/분석. 판타지 고인물다운 계산적 접근이 지호의 캐릭터성을 드러냄
  3. **(씬 4) 할리마의 경고** — 위치: 챕터 후반. 유형: 미스터리/긴장. "아무도 살아서 계약한 적 없다"

1. **프로그래머 루트 분석**: A~D안을 리스트업할 때 스프레드시트/의사결정 트리 비유. "이건 기술 부채가 너무 크고, 이건 리소스가 부족하고..." 판타지 강화 루트를 프로젝트 계획처럼 분석
2. **git push --force**: D안(기둥 직접 접근)을 설명할 때. "이건 git push --force 같은 거야. 되돌릴 수 없고, 대부분 재앙으로 끝나."
3. **서버 리부팅**: 재생 스킬과 번개 계약의 조합을 설명할 때. "심장이 멈추면 재생이 리부팅한다. 서버 죽으면 자동 복구되는 거랑 같아. 문제는 매번 데이터 날아가는 것처럼 아프다는 것."

- **주의**: 3개 이내 유지. 위 3개 중 택 2~3.
- 출처: protagonist_bible §2 비유 규칙, novel-config §5 MODERN_REF

## 10. 가드레일 체크리스트

- [ ] **시점 제한**: 1인칭 중심 (강지호). 3인칭 전환은 이 챕터에서 불필요 — 행상인 시점 불요, 모든 씬을 지호 1인칭으로 처리 가능. 3인칭 전환 0%~15% 이내
- [ ] **재생 스킬 일관성**: D랭크. 신경 재생으로 고통 내성 불가. 플래시백에서 "고통은 매번 새롭다" 원칙 유지. 재생 등급을 지호 자신이 직접 언급하지 않음 (1인칭이므로 타인 시점에서만 D랭크 노출 가능 — 이 챕터에서는 등급 직접 노출 불필요)
- [ ] **기둥 흡수 규칙**: 이 챕터에서 기둥 흡수 없음. 플래시백으로만 언급. 5화 축복 의식 회상 시 death_regression §2 프로토콜 일관성 유지
- [ ] **모래시계 잔량 25%**: 5화 끝 기준. 이 챕터에서 감소 없음. 모래시계를 직접 확인하는 장면이 있다면 잔량이 "줄었다"고 표현하되 구체적 수치는 지호가 모르므로 표기 불요
- [ ] **톤 공식**: 코미디→고통 낙차 존재 (§7 참조). 일상 유머에서 플래시백/분노로 급전환
- [ ] **대사 교환 불가성**: 할리마의 경구체, 나이라의 직설 반말, 지호의 자조 독백이 뚜렷이 구분되어야 함. 보이스 카드(§4) 참조
- [ ] **IT 비유 3회 이내**: §9에서 3개 후보 중 택 2~3. 초과 금지
- [ ] **마르코 미등장**: 이 챕터에서 마르코 등장 없음 (plot_framework 확인 완료)
- [ ] **감정 직접 명명 금지**: "분노했다" (X) → 신체 반응 + 내면 독백으로 표현
- [ ] **영웅적 각오 금지**: "내가 아젤리아를 구하겠다!" (X) → "가만히 있을 수 없어. 그게 전부야." 식의 소극적 능동

## 11. 커스텀 축 체크리스트

- [ ] **REGRESSION**: 재생 스킬 서술 정합성
  - 플래시백에서 재생이 고통을 끝내주지 않는다는 핵심 메커닉 유지
  - "고통 내성이 생겼다" 류의 서술 금지
  - 재생 시간표(얕은 베임 15분, 깊은 상처 수 시간 등)에 모순되는 서술 금지
  - 번개 정령 계약 구상에서 "심장 정지 → 재생"의 논리가 death_regression 규칙에 부합하는지: 장기 손상 = 수 시간~수일이지만, 심장 관통도 재생 가능하므로 심장 정지 후 재생은 설정상 가능. 단, 즉각 재생은 아니므로 시간이 필요함을 암시해야 함
  - 출처: death_regression §1-2, protagonist_bible §1-1

- [ ] **FORESHADOW**: 배치/회수할 복선 목록
  - **배치할 복선**:
    - 번개 정령 복선: 할리마의 "건조 번개에 정령이 깃든다" + "아무도 살아서 계약한 적 없다" → 15화 회수
    - 아젤리아 상황: 모르데우스 섭정, 에리나 실종 → 18화 귀환에서 회수
    - 지호의 "독기": 첫 능동적 분노 → 이후 Arc 3 전체의 동력
  - **회수할 복선**:
    - 재생의 훈련 도구 활용 (8화에서 배치) → 이 챕터에서 "번개와 재생 조합"으로 확장 활용
    - 할리마의 관찰력 (7화에서 "네 몸은 축복이 아니라 족쇄" 발언) → 이번에도 지호의 의도를 직감
  - 출처: plot_framework §Arc3 14~15화

- [ ] **MODERN_REF**: 현대 지식 활용 범위 내
  - IT/프로그래머 비유 3회 이내 (§9에서 관리)
  - 프로그래머/웹소설 독자가 모를 전문 지식 시연 금지

  - 출처: novel-config §5 MODERN_REF, protagonist_bible §3
