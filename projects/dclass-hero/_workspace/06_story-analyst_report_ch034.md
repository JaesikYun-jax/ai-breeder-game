## B. 상세 포인트

### 1. 마르코 5회차 — "다섯 중 셋" 회수 (HOOK/FORESHADOW)
- ch028 "강철 색이지"·"다섯 중 하나" 예고 → ch034 "북쪽 길 다섯, 그중 셋이 너를 알아" 직접 수령.
- 명시 거부("다섯이 길인지 사람인지 보호자인지 문인지 — 명시하지 않았다") = 양면성 원칙 완벽.
- 반존대 "또 만났군" 5회차 유지, "다섯 번째의 문장을 아꼈다" 메타 서술로 콜백 패턴 의식적 변주. 패턴화 위험은 지호 관찰자 시점이 자체 메타로 풀어줌.

### 2. FS-A11 강화 1줄 적절성 — "바람·불·번개"
- **배치 타이밍**: ch033 협곡에서 지호가 우연히 묶은 직후 ch034에서 타자(마르코)가 인지 확인 → **타자 인지 = 복선 확대 신호**로 교과서적.
- **은유 유지**: 구체 3단어("바람·불·번개")만 나열 + "진짜"라는 추상명사로 닫음. "기상"·"기후"·"제어" 용어 **0회** — FS-A11 메모 준수.
- **결의 거부**: 지호가 "정리하면 결심이 된다" 명시하고 정리 보류 → §14 영웅적 결의 금지 + FS-A11 "의도 시연 X" 원칙 동시 충족.
- **양**: 딱 1줄 + 지호 독백 2줄 해석. 과잉 아님.

### 3. HOOK 평가 — Arc 5→6 브릿지
- 마지막 라인 "북쪽으로. 이번엔 한 가지가 아니라, 한 무리를 끝낼 일이 있다."
- **강도**: A (S급은 아님 — 대사/사건 반전이 아닌 독백 선언). 피날레 치고 차분하지만 Part 2 톤 공식(비장 5)에 정합.
- "한 무리" = 복수 타깃 암시 (강철 기둥 + 카이젤 황실 + 검은 로브 네트워크) → Arc 6~의 3중 빌런 라인과 호응. 이중 의미 작동.
- "강철 색 지평선" 직전 배치로 ch028 마르코 "강철 색이지" 콜백 회수까지 겸함.

### 4. REGRESSION — 모래시계 씬 1회 준수
- 씬 4 단 1회 등장, 잔량 **변동 없음** (ch028 두 알 감소 → ch033 밤에도 변동 없음 → ch034 변동 없음, 박자만 어긋남).
- 박자 진행: ch022 한 호흡 → ch028 → ch033 한 호흡 더 두꺼워짐 → ch034 박자 한 번 더 비켜감. **선형 누적** 일관성 유지.
- §11 준수 문장이 3중("흔들지 않았다. 돌리지 않았다. 들여다보는 것도 아니었다") — 도구화 금지 서사적 내면화.

### 5. PACING — 피날레 5,505자
- 하한 딱 맞춤. Arc 5 피날레치고는 얇으나, **배웅·회수·출발**의 3기능을 4씬에 분배하여 기능 과잉 없음.
- 씬 1(배웅, ~1,400자) > 씬 3(마르코, ~1,400자) > 씬 4(출발, ~1,200자) > 씬 2(친서, ~900자) — 정서 중심 씬 2개가 기능 씬보다 앞. 균형 OK.
- MINOR: 씬 2 친서가 D-15 숫자 계산에 기능 치우침. 다만 "같은 동작이었다. 보름 전에도 이 동작을 했다. 손이 먼저 기억하고 있었다" 1줄이 정서 앵커 역할 해서 기능 과잉 선은 안 넘음.

### 6. 직전 1화(ch033) 연속성
- 풍·화염·번개 묶음 (ch033 우연 시연) → ch034 마르코 확증 = 직접 연결.
- 할리마 "다음에도 올 수 있을지" 여운 → ch034엔 할리마 "가" 한 음절만 (가벼운 쪽 선택) = 직전 화 여운을 절제로 받음. 뛰어난 연속성.
- 나이라 반짝이는 백화 가닥 (ch033 → ch034 재등장 "오래 보지 않았다") = FS-B02 유지, 가속 없음 (§12 챕터당 1줄 이내).
- "북쪽 바람이 이미 시작됐다"(ch033 종결) → ch034 "오늘은 북쪽에서 나를 향해 정면으로 왔다" — 바람 방향 서사 직접 계승.
