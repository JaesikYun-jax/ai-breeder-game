## 1. 메아리(반복·자기인용) 회피 검사

### 1-1. 박자어 빈도 (본 화 톤 시그니처)
- **"한 결"**: 본문 내 약 28회. 본 화 시그니처 박자어로 기능 — "강철 색 도시에서 빛/그림자/소리/시선 모두 한 톤 더 단단해서 결로 끊긴다"는 지각 톤을 전화에 깔기 위함. 정당화 가능.
- **"한 호흡"**: 약 9회. 만찬의 단축 박자(말 대신 호흡 단위로 일이 정리되는 카이젤식 격식)를 받침.
- **"한 박자"**: 약 5회. 다마르 시그니처(한 박자 늦게 잔 들기 = canon-quickref §2 다마르 시그니처 5종 중 하나) 정합.

### 1-3. 식기 두드림 vs 에리나 두드림 라인 충돌 점검
- ch022 에리나 팔걸이 두드림(외교 신호) → ch023 누름(진화) → ch031 셔츠 단추 위 누름(원격 전이).
- ch036 지호 식기 두드림은 **에리나 신호 체계와 다른 라인**: 에리나 라인은 "팔걸이/잔 모서리/책상 모서리/셔츠 단추"의 손가락 끝 누름·두드림. ch036 식기 두드림은 "포크 끝을 손가락으로 한 번 두드려 보았다" — 도구(포크)를 매개로 한 자기 박자 점검 동작. 어휘는 인접하지만 **동작 대상·기능이 다름**(에리나 = 신호 발신 / 지호 = 자기 박자 점검).
- 본문 26행 "내가 만든 게 여기 와 있고, 옆에는 베끼다 만 게 있고, 나는 그걸 보면서 식기 끝을 한 번 더 두드렸다" — 두드림이 자기 호흡 정리 기능으로 명확히 분기됨. **PASS**.
- 단, ch037~ch038에서 에리나의 지호 친서 답장 회수 시 부부 두드림/누름 라인이 다시 활성화될 때, 지호의 "식기 두드림" 라인이 변주 자원으로 회수 가능 — 인계로 기록.

## 2. 침묵 → 비언어 환산 검사

### 2-1. 황제 발화 0회 처리 (요구 명시)
- 황제는 본 화 41~62행, 76행에서 등장. 발화 0회 (요구 충족).
- 황제 침묵 → 비언어 환산 5종:
  1. **단상 난간에 한 손가락 끝만 닿게 두기 (잡지 않음)** — 48행. "닿기만 했다"의 부재형. 권력의 절제 비언어. **신규 시그니처 — 강함**.
  2. **한 손을 단 한 번 들어 올렸다 내렸다 (만찬 시작 신호)** — 50행. 발화 대신 동작이 신호.
  3. **시선이 좌중을 한 번 훑음, 내 자리에 0.5초 머무름** — 60~62행. canon-quickref §2 다마르 "0.5초 더 길었다 / 0.3초 짧았다" 양적 시선 시그니처와 동질 어휘 — 황제는 더 짧고 더 무게 있는 라인으로 분기.
  4. **음식에 손을 거의 대지 않음** — 76행. 부재의 정중함.
  5. **한 손을 가볍게 들었다 내림 (만찬 종료 신호)** — 76행. 시작 신호의 거울. 시작·종료 양 끝을 같은 동작 1회씩으로 닫음 — 닫힌 구조.

→ **PASS (강)**. 황제의 권력 무게가 "잡지 않는 손가락 끝 / 0.5초 시선 / 두 번의 같은 손짓"으로 압축. 발화 0회가 캐릭터 결손이 아니라 **격식의 본질적 표현**으로 작동.

### 2-2. 다마르 정원 두 줄 ("이곳은 솔라리스가 아닙니다." / "한 손이 더 빠른 곳입니다.")
- 만찬 발화 0회 → 정원 두 줄로 응축. 그 사이 침묵을 받치는 비언어:
  - 콧수염 한 번 만짐 (98행)
  - 자갈 위 발끝 멈춤·한 호흡 (98행)
  - "단어를 입 안에서 굴리지 않음" (98행) — canon-quickref §2 다마르 시그니처 "단어 굴리기"의 **부재형 변주**. ch023~ch024 단어 굴림이 무기화 단계였다면, ch036에서는 굴리지 않음으로 톤 한 단계 낮춤 (요구 명시).
  - 분수 시선 한 번 (100행)
  - 콧수염 결을 거꾸로 쓸지 않음 (122행) — "결을 거꾸로 쓰면 비공식 자리"라는 ch023 정본 변주의 부재형 활용. **PASS — 시그니처 변주 정착**.

→ **PASS (강)**. 다마르 톤 한 단계 낮춤 요구 충족. 단어 수가 줄어든 만큼 시그니처 부재형(굴리지 않음·결 거꾸로 쓰지 않음)이 정확히 자리.

### 2-3. 마르코 한 줄 ("여기선 네가 누구한테 묻는지가 더 중요해.")
- 6회차 등장. canon-quickref §2 마르코 시그니처 "또 만났군"의 콜백 — 본 화에서는 음절이 아니라 **손짓으로 채움** (144행 "'또 만났군'의 자리를 그 손짓이 대신했다. 음절이 빠진 자리를 동작이 정확히 채웠고, 그 채워짐이 한 결 가벼웠다"). **시그니처 진화 — 음절→동작 치환**.
- 한 줄 후 "한 발 물러났다. 사라지지 않았다. 그림자 안쪽으로 한 결 더 깊게 자리를 옮겼을 뿐이다" — 사라짐의 부재형. 마르코 정체 양면성(canon-quickref §8 활성 복선) 유지.

→ **PASS**. 마르코 6회차 시그니처 갱신(손짓 치환). 단 가드레일 §7 마르코 최소 등장 원칙은 본 화에서도 등장 시간 짧음(140~158행, 약 18행)으로 준수.

### 2-4. 카이젤 귀족·로헨 침묵
- 좌중 12명 카이젤 귀족: 발화 0회 + 동기화된 비언어 (38~39행 "고개 숙이는 각도가 좌중 전부에서 거의 동일하게 한 호흡 안에 정리됐다" / 56행 "좌중이 동시에 잔을 들었다. 다마르는 한 박자 늦게" / 68행 "잔을 내려놓을 때, 잔 바닥이 식탁에 닿는 소리가 좌중의 다른 잔들과 거의 동시에 울렸다"). 12명을 12종으로 개별화하지 않고 **집단 동기화 비언어**로 처리 — 카이젤 격식의 본질에 맞는 선택. PASS.
- 로헨: 발화 0회. 비언어 3회만 등장 (10행 "뒤로 한 발, 호위 거리 안쪽" / 84행 "다섯 발쯤 뒤. 청취 거리는 아니다" / 136행 "앞으로 열 발쯤. 호위 거리 안쪽이지만, 그림자 결까지는 닿지 않는 거리"). **거리 측정 시그니처** 일관 — canon-quickref §2 로헨 "어깨 미세 굳음" 부재 / 본 화에서는 거리만으로 호위 톤. 이는 ch031 "표식 동일합니다" 발화에서 한 단계 후퇴한 침묵 호위 — **의도적 후퇴**(만찬 자리에서는 발화 부적합)로 해석 가능. PASS.

## 3. 조연 긴장 검사

### 3-1. 다마르 — 톤 한 단계 낮춤 (요구 충족)
- ch023~ch024 무기화 단계 ("약한 세계의 사람이 — 흥미로운 일이군요" 일격) → ch035 첫 영접 시 결 방향 시그니처 1회 + 가슴팍 시선 두 번째 → **ch036 만찬에서 한 박자 늦은 잔 + 정원 두 줄로 정보 흘림 모드 전환**.
- 정원 두 줄 ("이곳은 솔라리스가 아닙니다 / 한 손이 더 빠른 곳입니다") + 부연 회피 ("부연하지 않았다. 누구의 손이 빠른지, 어느 자리에서 빠른지... 한 단어도 보태지 않았다") = **방조자→경고자 미세 전환**. 캐릭터 곡선 라인 (-1.0 적대 → 미세 +0.3 비공식 경고).
- "고개 숙임을 한 번 생략" (128행) — 격식 자리에서의 한 단계 누락이 **친밀 신호 X, 경고의 진정성 신호**로 기능. 한 단계 낮춘 톤의 정확한 압축.

→ **PASS — 톤 한 단계 낮춤 명시 충족**. 적대 라인을 유지하되 만찬 자리의 발화 0회 + 정원 두 줄로 분기시켜 무기화 모드 일시 후퇴.

### 3-2. 황제 — 발화 0회의 권력 톤 (요구 충족)
- §2-1 비언어 5종으로 처리. 추가로:
  - "원경. 단상 위, 내 자리에서 다섯 발쯤. 직접 의자에 앉지 않았다" — 거리·자세 비언어. **앉지 않음**의 부재형이 카이젤 황제의 권력 톤(감시자 자세) 환기.
  - "가는 손가락" (44~46행) — 강철 = 두꺼운 손이라는 통념을 뒤집는 비언어. 지호 내면에서 "강철을 다루기보다 강철을 설계하는 손"으로 한 결 안에 인지·흩어보냄 — Part 3 천인 평의회 정화 사상 연결 가능 떡밥(canon-quickref §0 빌런 3중 라인). **신규 떡밥 비언어 — 회수 자원**.
  - 시작 신호와 종료 신호 동일 (한 손을 가볍게 들었다 내림 1회) — 황제의 모든 권력 행사가 **같은 동작 1회**로 환원되는 절제. 톤 한 단계 위 정착.

→ **PASS (강)**. 발화 0회 충족 + 향후 재등장 시 즉시 식별 가능한 시그니처 4종 정립 (단상 난간 닿음 / 가는 손가락 / 0.5초 시선 / 같은 손짓 1회).

### 3-3. 마르코 — 6회차 반회 (요구 충족)
- 회차 누적: ch020 쪽지 → ch022 정식 재회 → ch026 자객 사건 → ch027 도주자 사건 → ch028 사막 재등장(또 보네) → **ch036 가로등 그림자 6회차**.
