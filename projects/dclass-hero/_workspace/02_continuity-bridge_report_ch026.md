# ch026 연속성 보고서 — 「솔라리스 가는 길」

> 기준: ch025 「두 나라의 그림」 직후. 솔라리스 출발 당일 새벽 ~ 이튿날 저녁/밤. 1~2일.

---

## 1. 직전 화 연결점 (ch025 마지막 500자)

- **엔딩 상태**: 출발 당일 새벽. 지호가 외투·가죽 가방·함(정령석 등불 모형) 챙기고 문 앞에 섬. 짧은 "다녀올게요"/"조심해서요" 교환. 등 뒤 거울 앞 에리나가 잠옷 위로 오른손을 배에 **한 박자 가만히 얹었다 내림** — 지호 미감지. 문이 작은 소리로 닫힘.
- **직전 밤 삼킨 말**: "…아니, 다녀와서요." "길 조심하라는 말. 두 번 하면 무거우니까요." — 에리나가 임신 통보를 의도적으로 유예함.
- **열린 질문/서스펜스**:
  1) 에리나가 배에 손을 얹은 동작의 의미 (독자 감지, 지호 미감지)
  2) 모래시계에서 어제 밤 한 알 떨어진 사실 — 박자 어긋남의 다음 예약
  3) 이전 화 사절단 퇴장 직후 체포된 잠입자 계열(검은 로브 네트워크)이 끝난 게 아니라는 잔향
- **즉각 이어야 할 상황**: 문 닫힘 → 왕궁 마당 → 호송대 집결 → 성문 출발. 연속 시간대(새벽).
- **엔딩 정서**: 비장·자조·미해결 긴장. 코미디 비트 없음. ch026 오프닝은 이 톤을 받아 그대로 이어가되, 씬1 중반부터 호송대 일상 톤으로 중화 가능.

## 3. 모래시계 & 재생 상태

- **잔량**: **1/4 — 어제밤 한 알 감소** (ch025 씬4). **ch026에서 추가 감소 금지** (가드레일: 박자 어긋남 잔향만, 잔알 변동 X)
- **최근 발동**: ch021 솔라리스 태양 기둥 회상 치유가 마지막 직접 접촉. ch025는 관찰만.
- **재생 스킬 상태**: 활성. 이상 없음. 장기 누적 대가(흰머리) **아직 가시화 X** (가드레일 §12: Arc 7부터).
- **현재 부상**: 없음. (ch026 씬2 자객 공격 시 D급 재생 미세 시연 가능 — 벤 자국 즉시 회복 1회, 독백 자조만)
- **기둥 흡수 이력**: 아젤리아 빛 기둥(ch019) + 솔라리스 태양 기둥(ch021 회상). 두 기둥 안정화 완료. **세 번째 기둥은 Arc 6 카이젤 강철 — 아직 미흡수**.
- **가드레일 §11 체크**: ch026 씬1·3에서 **모래시계 꺼내기 자제**. 씬4 번개 모스 송신 후 잠깐 들여다보는 정도는 OK. **흔들기·발동·돌리기 절대 금지**. 보지 않고 셔츠 안쪽 자리를 손끝으로 한 번 누르는 신체 반응만으로도 충분.

## 8. 감정 흐름 & 톤 커브

- **ch024 엔딩 감정**: 얼음 같은 긴장 (부부 첫 어긋남)
- **ch025 엔딩 감정**: 비장·자조·미해결 긴장 (모래시계 한 알 감소 + 에리나 배 짚기 미감지)
- **코미디→고통 패턴**: ch025는 씬3 로헨 푸시업 펀치 외에 코미디 거의 없음. **ch026도 비장 우세가 자연스러움** (연속 비장 2화는 아직 경고 수준 아님).

**ch026 톤 커브 권장**:
- 씬1 (출발·야영): 평이·자조 (ch025 엔딩 여운 이어받기) — 소폭 중화
- 씬2 (자객): **액션 긴장 피크** — 짧고 건조. 신체 반응만. 로헨 시그니처 3어절로 마무리 → 미세 안도
- 씬3 (친서): **정서 피크** (임신 확정 충격 + 자조). 영웅적 결의 0회. "내가 뭐라고 아빠가" 톤
- 씬4 (번개 모스): 미세 회복 + 기능 시연의 건조함. 엔딩은 지평선·솔라리스 첫 흔적의 **부감 장면**으로 평이하게 닫기.

**Part 2 톤 공식 (코미디 2 : 비장 5 : 반전 3) 적용**:
- 코미디 2 = 씬1·씬4 호송대 일상 짧은 농담 1~2회 (지호 자조 독백 IT 비유 1회 상한 — 가드레일 §13 상기)
- 비장 5 = 씬3 친서 + 씬2 자객의 훈련된 서늘함
- 반전 3 = 씬3 임신 통보가 본 화의 진짜 반전 (사건은 자객인 척, 실제 정서 축은 친서)

- **경고 플래그**: **PERSPECTIVE_FATIGUE** (연속 5화 이상 도달) — 단, 가드레일 §1은 "1인칭 30% 이내 3인칭" 허용하는 구조. ch026은 친서 텍스트(에리나 자필)가 자연스러운 **준(準)-서브 시점** 역할을 할 수 있음 (시점 주체 전환 아님, 편지 본문 삽입).
- **권장 조치**: ch026에선 **친서 본문 인용을 한 덩어리로 배치** (150~250자) — 에리나의 목소리가 1인칭 본문 안에 삽입되는 구조. 완전 3인칭 서브 씬은 아직 유보 (chapter-architect 최종 결정).

## 11. 금지 카운트·RETCON 체크

**Part 2 RETCON 단어 — 0회 필수**:
- "회귀" / "루프" / "사이 공간" / "방출" / "다음 나라로" — 전부 0회
- 지호가 자객에 베여 재생되는 장면에서 "다시 살아났다" 대신 **"피가 멎었다" / "상처가 닫혔다" / "살이 다시 채워졌다"** 계열로. "방출" 느낌 단어 금지.

**챕터 번호 직접 언급 0회**:
- "이십육화", "26화", "지난 화" 등 전부 금지
- 시간 표현은 "지난 새벽", "사흘 전", "사절단이 떠난 이튿날" 등

**금지 표현 상한 (rule-checker 사전)**:
- "있었다" ≤ 5 / "고 있었다" ≤ 3 / "것이었다" ≤ 1 / "수 있었다" ≤ 3 / "하게 되었다" ≤ 2 / em dash(—) ≤ 5~6회

**분량 목표**: **5,000~7,000자 (공백 포함)**. ch025가 7,000자 근처였으므로 ch026도 같은 상한. 씬3 친서 인용이 길어지면 씬1 야영 묘사를 줄여 균형.

**가드레일 특별 재강조**:
- **§11 모래시계**: 씬4에서만 짧게 들여다보기 1회 (또는 생략). 흔들기·발동·돌리기 절대 금지.
- **§12 머리색 변화**: 풍 정령 부산물 미세 잔향만. 흰머리 가시화 금지.
- **§13 프로그래머 범위**: 번개 모스 통신은 정령술 + 프로그래머 사고 조합(모스 부호 = 지식 범위 내). 증기·화약·야금·전기 발전 언급 금지.
- **§14 영웅적 결의 대사 0회**: 아빠 자각도 "내가 뭐라고 아빠가" 자조 톤만. "지켜야 한다" / "반드시" / "끝까지" 류 금지.
