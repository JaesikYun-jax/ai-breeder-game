## FB-004 호감 계단 (특별 점검)

### 하은 축 (축 2) 진행 점검

**현재 3단계 심화 + 진무혈 쪽 이정표 1칸 (4단계 진입 직전)**

실현 점검:
1. ✓ "하은이다. 두 번째다." — 분노 2박자 이정표 (14화 설경 1박자 → 20화 하은 2박자)
2. ✓ 봉합 과정에서 "나는 교주가 아니라 오래된 남자에 불과하다" 인간미 접점
3. ✓ 하은 3인칭 내적 "그분의 맨발에 유리 가루 한 점이 얹혔다" — 감각적 자각 절정
4. ✓ 거실에서 "하은." 단 한 번 호명 + 체류 시간 긺
5. ✓ "너무 티 내지 마" — 하은→서연 첫 직접 견제 (1회, 경고 톤)

### 계단 순서 보호 점검

- **설경 축**: 3단계 유지 ✓ — "신교를 다시 여실 준비"는 부교주 업무 수용, 여성 축 이행 아님. 손가락 끝 한 점 접촉은 14화 연속성 내.
- **서연 축**: 3단계 유지 ✓ — SNS 미끼 철회(→방패) + 대피실 "언니" 한 번 + 찢긴 옷깃에서 눈 흔들림 — 4단계 진입 **아직 안 함** (ch022 예약).
- **하은 축**: 3단계 심화 + 진무혈 쪽 이정표 1칸 ✓ — 4단계 "질투" 본격 진입 **하지 않음** (1회 견제 대사로 씨앗만).

→ **세 축 동시 상승 금지 규칙 준수 ✓**.

### 진무혈 축 1 · 3 배경 유지 체크

- 축 1 (설경): 배경 유지 미세 신호 — "아리수 한 병 얹음"의 교환 ✓
- 축 3 (서연): 배경 유지 — SNS 철회·반 걸음·찢긴 옷깃에서 눈이 한 번 흔들림 (서연 쪽 시선, 진무혈 반응은 없음) ✓

- **MINOR-FB004-1**: 엔딩 거실에서 서연이 "하은 옆에 앉아 손등 봉합 자국을 바라봤다가 자기 손을 무릎 위로 내렸다" — 이 서연의 동작이 "하은 쪽으로 축 이동" 씨앗인지, "자기 축 다잡기"인지 모호. ch021~ch022 서연 축 4단계 진입 예약을 고려할 때 **서연 쪽 의지적 다잡음**으로 더 선명히 제시하면 FB-004 계단 순서가 더 깨끗해짐. **보강안**: "자기 손을 무릎 위로 내렸다" → "자기 손을 무릎 위로 내리고 한 번 꽉 쥐었다" (결의 신호).
