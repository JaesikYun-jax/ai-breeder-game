4. **몬스테라 잎사귀 끝 물방울 낙하** — 순찰 불빛 교차의 순간에 떨어짐 (L311)

### ch016 오프닝 회수 구도

#### 1장치 회수 (최우선)
- **김 교수 문자** → 하은이 월요일 아침 **서울대로 향하는 동선** 오프닝
- 실제 대면 씬까지 끌고 가거나, 오프닝은 이동 장면 + 중반에 연구실 대면으로 설계 가능
- 권장: 진무혈·설경에게 **문자 수신 사실을 알리는가?** — 권장 안은 **알리지 않고 혼자 감** (ch014 "이중생활 게임화" 연장선) 또는 **설경에게만 보고** (부교주 감지 루틴 작동)

#### 2장치 회수
- **고문 계약서 사진**: 두 가지 선택지
  - (A) **하은이 설경에게 먼저 공개** — 협력 관계 진전 / 적대 곡선 가속
  - (B) **교수 대면 후 돌아와 공개** — 긴장 고조 / 교수의 마지막 기만 포착 후 증거 연동
  - 권장: **(A) 새벽·아침 시간 중 설경에게 공개** → 설경이 A-01 회수 준비(17~18화 처단) 가속 / 또는 하은 단독 보유 유지 (독자 서스펜스 극대화)

#### 3장치 회수
- **순찰 불빛 교차**: 이중 순찰 체제는 박정섭이 ch015에서 확립(L31-33). ch016 새벽 보고 씬에서 **교대 결과 보고** 형태로 회수 가능 (예: "밤사이 이상 없음" / "CCTV 한 구간 추가 결손" 등 — 추가 결손은 A-11 빌드업으로 유용)

#### 4장치 회수
- **몬스테라 물방울**: 상징 이미지이므로 직접 회수보다 **반향 이미지** 권장
  - 단, ch016에서 **몬스테라 직접 묘사 금지**(비언어 Top 3 회피) — 다른 식물/물방울 치환
  - 권장: 거실 부적 종이 결에 맺힌 새벽 이슬 / 하은 핸드폰 액정 위 지문 자국 / 연구실 창가 빗물 등으로 "물방울 계열" 상징 변주
