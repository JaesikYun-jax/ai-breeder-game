## 제목 후보

1. **「심판」** — 정면. 1부 제목 스타일. 가장 안정적 1순위.
2. **「변절의 값」** — 17화 엔딩 대사("너는 살아서 네 변절의 값을 치러라") 수용. 심판 뒤에 남는 여운.
3. **「기록은 기록으로」** — 하은 기록관 축 + 여론 무기화. 3파전 축으로 기운 톤.
4. **「두 개의 자정」** — 서연 SNS 자정 비공개 전환 + 백준하 회장실 자정 대기. 병행 구조 암시.
5. **「그리 하자」** — 17화 엔딩 잔향("그리 하자") 반복. 감정 회복 축으로 기운 선택. 다만 A-12 폭발 경계.

**최선**: **「심판」** (프레임워크 §18 "처단"의 한국어 대안, 무게 유지·스포일 회피 균형).

## 2-A. 시점 구성 설계 (POV Design)

- **메인 시점**: 진무혈 1인칭 — 약 50% (심문실 일부 + 엔딩 거실 대부분)
- **서브 시점 후보 발굴**:
  1. **하은 3인칭** / 유형: 조연 — 심문 직접 수행, 연구실 이중생활 연기, 화장실 숨 한 번. 하은이 이 화의 **실질 주역 중 한 명** / 씬 1~3 배분.
  2. **백준하 3인칭** / 유형: 빌런 — 집무실 홀로. 17화 엔딩 이미지(검지 굳음) 완결 + 아베노·리하오란과의 교신 · 3일 유예 결정 / 씬 5 배분.
  3. 서연 3인칭 / 유형: 조연 — SNS 캡션 교체·팔로워 폭증 / 씬 4 · 7 배분 (단, 분량 최소).

- **최종 채택**: **하은 + 백준하 2시점** (서연은 진무혈 시점 내부의 관찰 이미지로 처리)
- **채택 사유**:
  - 하은: 김재원 정면 심문의 주체는 하은이다. 진무혈이 주도하면 카타르시스의 주체가 뒤바뀌고, B-08·A-01의 "이중 증거 시스템 = 하은의 정체성"이 약화된다. 17화까지 **구해진 자** → 18화에서 **심판을 집행하는 자**로 포지션 이동이 이 화의 핵심.
  - 백준하: 17화 엔딩 "검지 굳음"의 완결이 이 화의 빌런 축 필수 마감선. 또한 아베노·리하오란에게 **양쪽에서 밀려나는** 장면은 S-03 협정 파기의 2단계 회수로 20화 클라이맥스를 예열한다.
- **비채택 사유 (서연 단독 씬 없음)**: 서연의 이 화 기능은 "SNS 캡션 교체 1회"로 축약 가능. 진무혈 시점 태블릿 화면으로 전달되는 편이 분량 효율·3시점 상한 준수.
- **분량 배분**:
  - 진무혈 1인칭: ~50% (2,900~3,400자)
  - 하은 3인칭: ~30% (1,700~2,000자)
  - 백준하 3인칭: ~20% (1,200~1,400자)
  - 서브 시점 총합 ~50% — tone-and-style §1 "최대 3시점" + "시점당 최소 1,000자" 준수.

## 4. 보이스 퀵레프 카드

### 진무혈
- 종결어미: "~하라 / ~이다 / ~소"
- 문장 길이: 단문 + 띄엄
- 말버릇: 이름을 부르지 않는 호칭("그자") / "약속은, 약속이오" 계열 / "그리 하자"의 한 글자 변주 (17화→18화 재등장 1회 허용)
- 감정 폭발 시: 변화 없음 — 억누름도 없음. 17화 "손등 온도"의 내면 변주
- 금지 패턴: 감정 과잉 표현, "헐/대박" 계열, em dash

### 하은
- 종결어미: "~예요 / ~이에요 / ~거예요"
- 문장 길이: 관찰 기록 모드에서는 단문. 심문에서는 **평서문 + 숫자 + 시간**으로 압박
- 말버릇: "고고학도잖아요. 기록하는 게 당연하잖아요" (16화 → 18화 재호출, 공포 → 정체성으로 톤 전환)
- 감정 폭발 시: 말끝이 쉰 채 단단해짐 — **한 방울 눈물만** 흘리고 더 맺지 않음 (A-12 4단계 상한)
- 금지 패턴: 울음 폭발, "언니" 호명 (이번 화 0회 — ch015 이후 포화·극 문맥 아님)

### 김재원
- 종결어미: 가면 탈락 상태 — "~소 / 예 / 습니다"만 남음. 선배 톤·피곤 톤 제거.
- 문장 길이: 짧음. 잦은 말줄임.
- 말버릇: "어쩔 수 없었다"는 **18화에서 이미 입에 못 올림**. 대신 "......하은아"를 17화에서 1회 호명한 뒤 18화에는 호명 자체 불가능.
- 감정 폭발 시: 무릎 꿇음·고개 숙임. 17화에서 이미 완결 → 18화는 자백 소리만 남음.
- 금지 패턴: 자기 합리화 독백 재등장 금지. 합리화 언어가 **고갈됨**을 묘사.

### 백준하
- 종결어미: "~다 / ~해라 / ~이다" (부하에게) / "~소 / ~외다"는 아베노·리하오란에게
- 문장 길이: 단문
- 말버릇: 내면 독백 중 "관리인의 자식"·"약속은 지키는 사람" 두 축이 마지막으로 한 번 더 이어짐
- 감정 폭발 시: 손가락이 펴지고 다시 굳음 — 17화 검지 굳음 → 18화 검지가 풀리면서 펴짐
- 금지 패턴: 사과 대사, 손녀 이름 입에 올리는 장면은 비서 상대로만 1회

### 박정섭
- 종결어미: "~입니다 / ~습니다"
- 문장 길이: 숫자+장소+현상 3단
