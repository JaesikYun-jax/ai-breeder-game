#### 2-3. PLAUSIBILITY — 개연성

**강한 개연성 지점**:
1. 서연이 "왕자님" 포지션임을 보여주는 전사(경영서적·자회사 계획서·처방전 분석)가 나중의 L타워 제안과 "할아버지가 못 건드리는 자회사 명의"를 선행 정당화. 인과 튼튼.
2. 진무혈이 경맥에서 "봉인의 잔기"를 감지 → seal-regression §3-1 "봉인 유지 가문 후손에 축적"과 일치. 감지 메커니즘이 혈마공으로 가능(식물 속성 · 기 감지)한지 protagonist-bible §4 "기 감지" 능력으로 설명됨.
3. 서연의 "기댐 연기" → 하은의 즉각 간파(고고학 관찰 훈련) → 질투 확립. 3파전 설계 개연적.
4. 설경이 진실 공개 허락을 "진무혈의 살짝 끄덕임"으로 받는 장면 — 설경의 위계·소통 방식과 부합.

**개연성 의심 1건 (MINOR)**:

- **LOGIC MINOR**: 서연이 "봉인 진실"을 듣자마자 **그 자리에서 L타워 120층 + 백광제약 지분 투척**을 결정하는 전환이 다소 빠르다. 서연의 캐릭터(결정이 빠른 왕자님 포지션)로 일부 정당화되나, 독자 시점에서는 "충격 → 몇 초 침묵 → 즉각 재벌가 카드 꺼내기"의 심리적 쿠션이 1~2문장 더 있으면 자연스럽다. 현재 원문은 "서연이 천천히 고개를 들었다. 눈빛이 달라져 있었다" → 바로 "L타워" 제안으로 점프. **제안 직전에 "얼마 남지 않은 시간을 어디에 쓸지 방금 결정했다" 식 내면 압축 1문장** 추가가 이상적. 개연성 CRITICAL은 아니고 연출 연마 수준.

### CATHARSIS (novel-config §5 — "어그로-해소 낙차")

**1차 CATHARSIS — 업보 진실 공개**:
- 어그로 빌드업: 서연의 평생 병약 + 스탠포드 포기 + "유전이라고 들었어요"(거짓 설명) → 독자 공감 축적.
- 해소: "할아버지가 아셨으면서 안 알려주셨을 뿐" → 서연의 눈빛 전환 + 재벌 카드 투척 = **할아버지(백준하)를 향한 서연 본인의 반격 개시**.
- 빌드업 분량: 씬 1 전사(400~500자) + 씬 2~6 누적 ≈ 챕터의 약 40% → novel-config §5 "30% 이상" **충족**.
- 강도: 4.

**2차 CATHARSIS (메타) — 1화 루프 봉합**:
- 1화 "잘 왔다, 백가 / 이번엔 묻히지 않는다" → 10화 엔딩 "서울로 돌아갈 준비를 해라 / 숨어 있을 시간도 이제 끝났다" → **Arc 1 10화 누적 어그로가 1화 선언으로 회귀**하는 이중 해소.
- 강도: 5 (Arc 완결화 보너스).

**CATHARSIS MINOR — 프롬프트 지적사항 (백준하 보고 씬 부재)**:
- 프롬프트에서 명시적으로 점검 요청된 **"백준하가 '서연이 담양 이탈' 보고 받음 / 어그로 적립"** 장면이 **10화에는 직접 등장하지 않음**.
- 9화 chapter-log에 이미 "비서가 '서연 아가씨 … 간병인 동행하지 않는 날도 있습니다' 보고, 백준하 손가락 두드림, '한 주 안에 안 돌아오시면 내가 움직인다'" 씬이 **완료 배치**되어 있음 → 10화 중복 생략은 타당한 설계.
- 다만 **10화 거실 씬(씬 8~9)은 전적으로 펜션 3인(진무혈·설경·하은) 중심**이어서, **백준하 어그로의 "갱신"**이 10화 내부에서는 없다. Arc 1 완결화로서 백가의 다음 수 예고(한 컷)를 거실 씬 중간 인서트로 넣었다면 Arc 2 12화 삼파 회의까지의 어그로 곡선이 더 선명했을 것.
- 수정 제안(강제 아님, 선택적):
  - 엔딩 직전에 **백준하 컷 한 컷** 삽입 — 예: "같은 시간, 서울 16층. 비서가 조용히 고개를 숙였다. '회장님. 아가씨께서 오늘 요양 시설로 돌아오지 않으셨습니다.' 백준하는 대답하지 않았다. 야경을 보고 있었다. 강남의 한 건물. 120층." (약 4~6줄)
  - 이렇게 하면 A-02 L타워 제안 + A-06 백준하 약점 재강조 + 12화 삼파 회의 전환을 10화 내부에서 복합 회수 가능.
- 현 상태는 **완결화로서 충분히 기능**하므로 REWRITE 수준은 아님. MINOR 수준 권장 보강.
