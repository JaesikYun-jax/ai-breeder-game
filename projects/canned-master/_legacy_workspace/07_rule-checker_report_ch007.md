### BANNED

#### 1. 「있었다」 과용 — MAJOR
- **탐지 횟수**: 11회 (상한 5회/챕터)
- **초과**: +6회
- **출현 위치**:
  | 라인 | 원문 발췌 |
  |------|---------|
  | 9 | 바위에 둘러싸인 웅덩이가 **있었다** |
  | 23 | 설경이 바위 위에 앉아 **있었다**. 텀블러 하나가 옆에 놓여 **있었다** |
  | 77 | 설경이 파란 카디건을 입고 마당에 나와 **있었다**. 한쪽 팔로 빨래를 걷고 **있었다** |
  | 93 | 진무혈이 바위 위에 앉아 **있었다** |
  | 97 | 흘러내리고 **있었다** … 검은 머리가 이마에 붙어 **있었다** |
  | 121 | 키 차이가 **있었다** |
  | 129 | 천 년 동안 봉인되어 **있었던** 사람이니까 |
  | 233 | 연둣빛 눈은 계곡 방향을 보고 **있었다** |
  | 279 | 막걸리를 마시고 **있었다** |

- **수정 방향**: 명사문/능동형 전환
  - "웅덩이가 있었다" → "웅덩이가 나타났다" / "웅덩이." (묘사 분리)
  - "바위 위에 앉아 있었다" → "바위 위에 앉았다" / "바위에 앉은 채였다"
  - "빨래를 걷고 있었다" → "빨래를 걷었다" / "빨래를 걷는 중이었다"
  - "키 차이가 있었다" → "키 차이가 컸다" / 삭제 후 수치만 제시

#### 2. 「고 있었다」 과용 — MAJOR
- **탐지 횟수**: 6회 (상한 3회/챕터)
- **초과**: +3회
- **출현 위치**:
  | 라인 | 원문 |
  |------|------|
  | 71 | 하은은 버스에서 내리**고 있었다** |
  | 77 | 빨래를 걷**고 있었다** |
  | 97 | 정오에 가까워지**고 있었다** / 흘러내리**고 있었다** |
  | 233 | 계곡 방향을 보**고 있었다** |
  | 279 | 막걸리를 마시**고 있었다** |

- **수정 방향**: 현재형·단순 과거형 전환
  - "내리고 있었다" → "내렸다"
  - "가까워지고 있었다" → "가까워졌다" / "정오에 가까웠다"
  - "보고 있었다" → "보았다" / "향해 있었다"
  - "마시고 있었다" → "마셨다"

#### 3. 클리셰 — 「뇌가 작동을 멈췄다」 — MINOR
- **라인 99**: `하은의 뇌가 몇 초 동안 작동을 멈췄다.`
- **판정**: 단일 사용이지만 AI·번역소설 클리셰성 표현. 하은의 관찰자적 독백톤(독백 구어체 특징) 고려 시 구체 감각 교체 권장.
- **수정 방향**: "몇 초 동안 생각이 비었다." / "머릿속에서 말이 사라졌다." / "숨이 한 박자 늦게 돌아왔다."

#### 4. 감정 추상 — 「왜인지 슬펐다」 — MINOR
- **라인 179**: `이 사람이 밥을 먹으면서 맛을 못 느낀다는 게, 왜인지 슬펐다.`
- **판정**: 3인칭 제한시점(하은) 내부이므로 감정 직접 서술은 기술적으로 허용. 다만 "왜인지" + "슬펐다"는 AI/소설 번역투 상투어 패턴. MINOR.
- **수정 방향**: "밥을 먹으면서 맛을 못 느낀다는 게, 숟가락을 입으로 가져가다 멈칫하게 했다." / "그 한마디가 목에 걸렸다."

#### 5. 감정 추상 — 「다행이었다」 등 — MINOR
- **라인 119**: `수건을 가져와 진무혈의 어깨에 걸쳤다. **다행이었다**.`
- **라인 133**: `설경이 앞장섰다. 하은은 **다행이라는 듯** 뒤를 따랐다.`
- **판정**: "다행이었다" 근접 반복(라인 119~133, 약 350자 간격). 클리셰 인접 배치 → MINOR WARNING.
- **수정 방향**: 두 번째는 감각 전환. "설경이 앞장섰다. 하은은 그 등만 보고 뒤를 따랐다."
