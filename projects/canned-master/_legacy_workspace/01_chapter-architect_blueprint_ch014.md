**서브 비트 (밀도 보강)**

- 씬 1 서브: ① L타워 120층 비상등이 낮게 깔림(건물 자체 에너지 절감 모드). ② 설경의 태블릿 화면 네 칸 중 한 칸(옥상)이 0.5초 끊김 → 복구 → 이미 로프 세 줄 내려옴. ③ 진무혈이 맨발로 거실 카펫을 세 걸음 걸음. 창유리에 자기 실루엣이 얇게 비침.
- 씬 2 서브: ① 미끼 자객 대장(30대, 용역 출신)이 권총을 쏘는 순간 총탄이 반경 30cm 옆으로 미끄러짐 — 기로 공기 밀도를 바꾼 결과. ② 기를 쓰는 자 2인은 중국식 단봉 계열. 말 없음, 숨만 거칠음. ③ 진무혈이 한 명의 목을 오른손으로 감싸 기절시키는데 손가락 한 번 꺾는 것으로 끝. "살려보내야 할 자들이다. 백가가 듣게 할 것이 있다." 라는 원칙을 속으로 한 번 더 확인.
- 씬 3 서브: ① 장원지에의 구두는 먼지 한 점 없음(경공 사용 흔적). ② 장원지에의 호흡은 진무혈과 속도 비슷. 두 사람의 숨이 한 박자 먼저 섞임. ③ 장원지에의 왼쪽 소맷부리 안쪽에 붉은 실 매듭 하나(리하오란 가문 상징. 본 화 미공개, 26화 회수). ④ 진무혈의 왼팔 자상이 3초 만에 닫힘 — 장원지에가 그것을 봄. "......이 자를 죽이는 방법이 없다. 형님이 오셔야 합니다."
- 씬 4 서브: ① 대피실 외벽은 방탄·방폭 설계. 그런데도 뚫림(장원지에 동행 자객의 기공 파괴). ② 하은이 태블릿 통화 단축에 손가락이 미끄러짐(긴장). 두 번째 시도에서 누름. ③ 서연이 평소 장난기를 완전히 거두고 담요를 찢는 손이 빠름. "언니, 이거." 짧음. — 이 화에서 서연이 하은을 "언니"라 짧게 부르는 첫 순간(씨앗). ④ 설경의 입술이 벌어지며 "......교주님께 알려......" 말 끊김.
- 씬 6 서브: ① 진무혈이 자기 셔츠를 찢어 설경의 외상을 1차로 감쌈. 상체 일부 드러남(섹시어필 과하지 않게, 행위 묘사로만). ② 하은이 아리수 박스를 두 번 뜯음. 손톱이 부러짐(무의식). ③ 설경의 빈 왼쪽 소매가 바닥에 늘어짐. 진무혈이 한 번 그쪽을 봄. "이 사람은 천 년 전에도 한쪽으로 살림을 했다." 아주 짧은 독백. ④ 치유 마무리 즈음 진무혈 본인의 홍채 연둣빛이 1회만 선명(회복 3할→5할 경계의 실전 발현, 이후 옅어짐).
- 씬 7 서브: ① 하은이 차를 내려놓는 찻잔의 각도를 두 번 고침(의미 없음, 본인 감정 처리). ② 서연이 담요를 진무혈의 등에 걸칠 때 손끝이 어깨에 0.2초 닿음 → 바로 뗌. 진무혈 반응 없음. ③ 두 사람이 나란히 섰을 때, 서연이 하은에게 "......언니, 저도 오늘은 잠 안 올 것 같아요." 짧게. 하은 "......네." 짧게. 서로 얼굴 보지 않음.

## 2. 시점 구성 설계 (POV Design)

- 메인 시점: **진무혈 1인칭 (약 50%)** — 전투·감정 회복 중심. 첫 "분노" 감정 복귀를 독자가 내부에서 체감.
- 서브 시점 후보 (3개 검토):
  1. **장원지에 3인칭 (약 20%)** / 유형: 빌런 고수의 경외 / 용도: 자연 일체 경지의 외부 관측(A-10 회수 1단계)+리하오란 존재 인식. / 씬 위치: 씬 3-B(교전 중) + 씬 5(철수 직전 짧은 미소 각도). → **채택 확정**.
  2. **하은 3인칭 (약 15%)** / 유형: 약자·관찰자 / 용도: 대피실 설경 부상 목격, 새벽 간호의 밤 관찰. / 씬 위치: 씬 4-B, 씬 7. → **채택 확정**.
  3. **설경 3인칭 (약 15%)** / 유형: 실무 지휘자·부상자 / 용도: 씬 4 대피실 진입~관통상 직전의 냉정한 판단 + 엔딩 혼미 중 경고 발화의 외부 시점 짧게. / 씬 위치: 씬 4 도입. → **채택 확정**(1,000자 확보 가능: 씬 4 도입 500자 + 엔딩 훅 직전 500자 분산).
  4. **미끼 자객 대장 3인칭 (약 3~5%, 250~400자)** / 유형: 빌런 말단 / 용도: 씬 2-B "우리는 미끼다" 짧은 인식 → 독자 공포 빌드업. → **짧은 프리뷰 보조**(1,000자 미달. 원칙상 시점 카운트에서 제외하고 "삽입 문단"으로 처리).
- **최종 채택**: 3시점 — **진무혈 50% / 장원지에 20% / 하은 15% / 설경 15%** (4시점. 프로젝트 상한 3시점 규정과 충돌 → **아래 재조정**).
- **재조정**: novel-config.md §3 "한 챕터 최대 3시점, 각 시점 최소 1,000자" 규칙. 본 화는 **전투 회차 + 조연 부상 + 빌런 고수 경외 + 히로인 취약함 관찰**의 네 축을 모두 다뤄야 함. 해결:
  - **설경 3인칭을 "진무혈 1인칭 내부의 하은 3인칭"에 통합** → 하은 시점이 대피실 벽 뚫림~관통상~하은이 설경을 받치는 과정을 전부 담당(씬 4·4-B 통합). 설경의 혼미 중 발화는 씬 8(엔딩)에서 **진무혈 1인칭 안에서 대사·행동 관찰**로 처리.
  - **미끼 자객 대장 짧은 씬(씬 2-B)은 진무혈 1인칭 중간의 "밀착 미니 문단 250~350자"로 삽입** — `---` 구분선 대신 단락 전환만 사용(시점 변경이 아닌 **자유간접화법 허용**). 또는 `---` 한 번 쓰고 다시 복귀. **선택: `---` 구분선 2회 사용하되 분량 초단(250~350자)이므로 "3시점 규정" 카운트에서 제외**(프로젝트 전례: ch013 김재원 3인칭 1-B도 500자 이하).
  - 최종 확정 시점 배분: **진무혈 1인칭 50% / 장원지에 3인칭 20% / 하은 3인칭 25%** (3시점). **미끼 자객 대장 3~5% 삽입 문단 별도**(시점 카운트 외). **설경 3인칭 독립 씬은 생략**(설경의 내면은 진무혈의 치유 씬에서 행동·호흡으로, 엔딩 대사로 외부화).
- 채택 사유:
  - ① 진무혈 1인칭 50%는 **감정 회복 곡선 "부분 복귀" 단계의 첫 분노**를 내부에서 포착하기 위함(protagonist-bible §3). 외부 시점으로 처리하면 감정이 소실됨.
  - ② 장원지에 3인칭 20%는 **A-10 자연 일체 경지 회수 1단계**가 "적 시점의 경외"로 제시되어야 기능하기 때문(seal-regression §7-3). 진무혈 1인칭에서 자기 경지를 설명하면 자기도취로 변질됨.
  - ③ 하은 3인칭 25%는 **설경 관통상 목격자 + 새벽 간호 관찰자**가 동시에 필요하기 때문. 서연은 하은 시점 안에서 행동으로만 묘사.
- 비채택 사유(설경 독립 3인칭 없음): 설경은 의식 손상 상태가 대부분 → 시점 주체가 되면 오히려 독자 몰입이 깨짐. 진무혈의 "구해내는 자" 시점이 더 강렬.
- 분량 배분: 진무혈 1인칭 3,500~4,250자 / 장원지에 3인칭 1,400~1,700자 / 하은 3인칭 1,750~2,100자 / 미끼 자객 대장 삽입 250~350자. 합계 draft 6,900~8,400자(상한 근접).

## 4. 보이스 퀵레프 카드

### 진무혈
- 종결어미: "~하라 / ~소 / ~있었느냐 / ~다". 본 화는 전투 중심 → **~다 위주**. 간호 씬은 **~소 드물게**(설경에게 침묵 위주).
- 문장 길이: 전투 중 매우 짧음. 간호 중 1~2줄. 분노 감정 복귀 순간 **1단어 독백**("......분노.").
- 말버릇: 창 밖을 먼저 보고 말함. 손 먼저 움직이고 말함. 환자·부상자에게는 말 거의 없이 오른손 얹음.
- 감정 폭발 시: **이 화에서 첫 복귀**. 단, 폭발이 아니라 "손바닥이 뜨거움 + 단 1초". 즉시 판단 우선으로 억눌림.
- 금지 패턴: 현대 감탄사 금지. 긴 설명 금지. "기분이" "화가 났다" 같은 감정 직접어 금지.
- 샘플:
  - "설경. 대피실 잠금. 오른쪽 복도는 내가 본다."
  - "숫자가 너무 적다." (독백·자조 섞임)
  - "오늘 이 자는 오늘이 아니다." (장원지에 철수 직후 독백)
  - "......설경." (대피실 도착 후 첫 발화)
  - "......이 사람은 천 년 전에도 한쪽으로 살림을 했다." (간호 중 독백)
  - (엔딩 직전 독백) "......이 밤에 잃을 뻔한 사람이 있다."

### 장원지에 (張文杰, 신규)
- 종결어미: **"~습니다 / ~입니다 / ~겠습니다 / ~지요."** 정중한 반존대. "~합니다"조 유지.
- 문장 길이: 짧음. 한 줄 1~2 문장. 내적 독백은 3문장 이내.
- 말버릇: 상대를 "교주"라 칭함(호칭 존중). 자기 지칭 없음. "형님" 1회만 언급(가문 상징 실 매듭과 호응).
- 감정 폭발 시: **없음**. 미세한 미소 1회만. 본 화의 핵심 효과.
- 금지 패턴: 감탄사·비속어 금지. 무협지식 초식 이름 금지(맨손·경공 중심, 극양검결은 26·30화 본격). 한국어 발음 **장원지에** 엄수, 한자 **張文杰** 첫 등장 1회 병기 후 재병기 금지.
- 샘플:
  - "천마. 당신을 죽이러 온 건 아닙니다. 오늘은 당신이 어느 수준인지 보러 왔습니다."
  - "......이 자는 검을 피하지 않고 검 옆을 지나간다." (독백)
  - "......형님이 직접 오셔야 합니다." (독백·회고조)
  - "오늘은 여기까지입니다. 교주. 다음에 뵙겠습니다."
- **보이스 가이드 신규 등재 제안**: /settings-sync 때 voice-guide.md §빌런 항목에 **장원지에** 공식 등록.

### 서하은
- 종결어미: "~요 / ~거예요 / ~어요 / ~죠". 본 화는 위기 파편화 → **짧은 완결문 + 생략**.
- 문장 길이: 공포 피크에서 1~2 어절. 간호 중 3~5 어절.
- 말버릇: 본 화는 **자조 없음**. 자기 감정 분석 독백 없음(특수 상황). 대신 감각 관찰만.
- 감정 폭발 시: 눈가 붉어짐·숨 삼킴·손톱 부러짐. 말 없음.
- 금지 패턴: 본 화에서 "고고학도로서 최악의 반응" 패턴 **금지**(해당 패턴은 ch013까지 소진 + 이번 화는 감정 해소 구간).
- 샘플:
  - "설경 씨. ......설경 씨." (씬 4-B)
  - (태블릿 통화) "설경 관통상. 옆구리." (씬 4-B)
  - (서연에게) "......네." (씬 7)
  - (진무혈에게) "......차요." (씬 7, 차 내려놓으며)

### 백서연 (본 화 특수 톤 — 장난기 거둠)
- 종결어미: "~어요 / ~예요". 본 화는 **감탄사 거의 없음**.
- 문장 길이: 짧음. 평소 대담함 → **수습·짧음**.
- 말버릇: 본 화는 왕자님 포지션 일시 중단. "언니" 1회(씨앗).
- 감정 폭발 시: 담요 찢는 속도로 대체.
- 금지 패턴: "어머", "왕자님은 공짜로 안 도와주세요" 등 장난기 대사 **금지**(상황 부적합).
- 샘플:
  - (하은에게) "언니, 이거." (담요 찢어 건넴)
  - "......언니, 저도 오늘은 잠 안 올 것 같아요." (씬 7)

### 설경 (본 화 대사 극소)
- 종결어미: "~습니다 / ~겠습니까 / ~지요". 본 화는 혼미 중 **파편화 문장**.
- 문장 길이: 혼미 중 **"......"** 연속과 단어 2~3개씩 끊김.
- 말버릇: "교주님" 고수(의식 있을 때).
- 감정 폭발 시: 없음. 혼미 속에서 한 번 창 쪽으로 고개 돌리려 함.
- 금지 패턴: 긴 문장 금지. 설명체 금지. 본 화 총 대사 3~4줄 이하.
- 샘플:
  - "교주님... 그 자는... 중급 고수입니다."
  - "그보다 더 강한 자가 저쪽에 있습니다."
  - "그 자를 보낸 자는......" (의식 상실)

**메아리 대사 금지 재확인**: 설경의 "중급 고수" / "그보다 더 강한 자"를 진무혈이 반복 금지. 진무혈은 다른 각도(손을 쥐었다 놓음 + 창밖 응시 + "......이 밤에 잃을 뻔한 사람이 있다" 독백)로 응답. [source: tone-and-style.md §5]

## 5. 진무혈 상태 체크

- 혈마공 회복률: **3할 기저, 실전 5할 경계 1회 발현**. 장원지에와 교전 중 5할 경계까지 올림 → 설경 치유로 기 급격 소모 → 간호 중 3할 선으로 복귀. 숫자 직접 노출 금지, 연둣빛 홍채 1회 선명(씬 6 치유 마무리) + 옅어짐으로 간접 표현. [source: protagonist-bible §2 ch014 "5할" 이정표 / seal-regression §7-2 "형(形) 이완 14화"]
- 재생 스킬 상태: 활성. 왼팔 자상 1회(3초 내 재생, 장원지에 관측). 탈수 중간 수준 — 아리수 5L 필요.
- 모래시계: **해당 없음**(본 프로젝트 개념 없음).
- 정신 상태: 담담 유지 → 설경 부상 감지 시 **1초 분노** → 판단 우선으로 즉시 억누름 → 간호의 밤 동안 **조용한 감정 복귀** 진행. [protagonist-bible §3 "부분 복귀"]
- 기둥 흡수 이력: **본 프로젝트 해당 없음**.
- 습득 능력: 자연 일체 경지 "형(形) 이완" 실전 1차 발현(seal-regression §7-2 이정표).
- 홍채: 연둣빛 옅음 → 치유 마무리 1회 선명 → 다시 옅음. **이 화에서 체표 묘사 최소**(ch013까지 누적 사용 고려).

- **옥상(121층 상부)**: 헬리포트·안테나. 침투 경로 B(미끼 팀 8인 + 장원지에 + 숨은 자객 1).
- **계단실**: 117~120층 비상 계단. 침투 경로 C(백가 보안 특수팀 2인 보조).
- **엘리베이터**: 비상 잠금 상태(설경 사전 설정).

### 침투 경로 맵
```
옥상 (로프 3조 · 10명 진입) 
  ↓ 계단실 → 120층 거실 (진무혈 단독 응전)
  ↓ 대피실 쪽 외벽 → 장원지에 동행 자객 1인 (설경 관통상)

기계실 (2명 미끼, 거실 진입 전 진무혈에 제압)
```

### 소품 시트
- **거실**: 서쪽 창 전면 유리. 몬스테라·동백·작은 탁자. 설경의 태블릿(벽면 거치). 아리수 5L 박스 2개(냉장고 옆, 평시 3할 수련용).
- **대피실**: 방탄문·방폭 외벽. 방공시트·담요 4장·생수 박스 1개·구급상자. 벽 뚫림(가로 60cm 구멍, 파괴 공구 + 기공 복합).
- **장원지에**: 검은 정장(단추 2개, 프리미엄 원단), 무기 없음, 오른손 약지에 얇은 은반지 1개(26화 극양검결 회수 전조). 왼쪽 소맷부리 안쪽 붉은 실 매듭(리하오란 가문 상징).
- **미끼 자객 대장**: 블랙 전술복, 권총 1(탄창 2), 통신 이어피스(본사로부터 실시간 지시).
- **설경**: 파란 카디건 위에 검은 양복 조끼. 왼쪽 빈 소매 핀 고정. 태블릿 손에 쥔 채로 관통상.
- **진무혈**: 회색 니트·맨발(평시). 셔츠 찢어 설경 감쌈(씬 6). 상체 일부 드러남.
- **하은**: 검은 트레이닝복(대피실 도착 시). 손톱 부러짐·손바닥 붉게 젖음.
- **서연**: 실크 잠옷 위에 카디건. 담요 찢는 손 빠름.

### 엔딩 훅 공간
- 거실. 동쪽 창. 동이 트는 새벽 하늘. 설경의 입이 움직이고 의식 잃음. 진무혈의 오른손이 설경의 손 위.

### 소품 체인
창가 도시 등불 → 옥상 로프 세 줄 → 설경 태블릿 점 깜박임 → 미끼 자객의 권총 탄피 3개 → 장원지에의 단추 → 대피실 벽 구멍 60cm → 설경의 옆구리 붉은 젖음 → 하은의 손바닥 붉음 → 서연의 찢긴 담요 → 진무혈의 찢긴 셔츠 자락 → 아리수 박스 두 번 뜯김 → 몬스테라 잎 끝 물방울 → 동쪽 창 새벽빛.

소품이 감정 곡선을 받침: **빛(평온) → 금속(침투) → 붉음(부상) → 물/초록(회복) → 빛(재각성)**.
