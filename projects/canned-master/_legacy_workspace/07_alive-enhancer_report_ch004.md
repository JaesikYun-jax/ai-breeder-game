#### MINOR #1 — "......이 도시가 여기냐." / "네. 서울이에요."

- **판정**: MINOR (경계선). "네." + 고유명사 응답만으로 구성되어 정보 추가량이 낮음. 다만 직후 서술("사내는 말없이 화면을 보고 있었다. 천 년 전에는 벌판이었을 곳에 천만 명이 사는 도시가 서 있다...")로 **침묵+내적 서술** 보강이 이뤄져 꼭두각시 느낌은 없음. 기능적으로는 PASS에 근접하나, 대사 단위만 보면 MINOR.
- **수정 후보** (원본 수정 금지, 참고용):
  - 대체 A: "네. 서울이에요. 천 년 전에는 이 자리에 아무것도 없었을 거예요." (하은이 시대 대조 정보 추가)
  - 대체 B: 대사 삭제. "하은은 고개를 끄덕였다. 대답할 필요가 없었다. 화면 속의 한강이 모든 걸 말하고 있었다." 비언어로 전환.
  - 대체 C: "서울이에요. 지금 우리가 있는 곳도 여기 어딘가예요." (위치감각 부가)

#### MINOR #2 — "한글이요?" + "좋은 왕이었나." / "네. 좋은 왕이었어요."

특히 15번 "좋은 왕이었나." / "네. 좋은 왕이었어요." 는 **진무혈 대사의 핵심어("좋은 왕")를 하은이 그대로 반복**한 구조로, 조연이 아닌 주인공(진무혈)의 대사를 조연 포지션(POV는 하은이지만 대화 구조상 질문자=진무혈, 응답자=하은)이 복창하는 역방향 메아리다.

- **판정**: MINOR. voice-guide의 하은 "존댓말 기본, 당황 시 말끝 올림" 설정과 부합하긴 하나, 정보 추가량이 0이고 직전 "세종이라는 왕이 만들었어요" 대사 직후라 반복 누적이 눈에 띈다.
- **12번 "한글이요?"** 는 진무혈이 먼저 "이 시대의 글자도" 언급 → 하은이 "한글"을 받아 되묻기. 단독으로는 MINOR지만, 직후 "한글이에요. 약 600년 전에..." 로 주체적 설명을 이어가므로 꼭두각시 느낌 경감. 15번과 합쳐 **MINOR 1건으로 합산**.
- **수정 후보** (원본 수정 금지, 참고용):
  - 15번 대체 A: "글자 반포가 훈민정음 머리말 그대로라면요. 백성을 위해 만든 왕이에요." (학부·대학원생 어휘로 새 정보 부가)
  - 15번 대체 B: "네. 이 시대 사람은 다 그분 글자로 이름을 써요." (시대 설명)
  - 15번 대체 C: 무대사 처리. "하은이 말없이 고개를 끄덕였다. 1443년은 천 년 기준으로는 가까운 과거다." (비언어+내적 서술)
  - 12번 대체: "한글이요? 한자 말고 우리 글자요." (반문+자체 정의)

#### 총계

- 메아리형 2건 → **MINOR 2건, MAJOR 0건**. 판정 기준(3건+ → MAJOR) 미달.
- 이 챕터는 "현대문물 학습" 장면이 집중되어 **구조적으로 질문-응답 대사가 많다**. 이 자체는 정당한 설계(chapter-architect의 "간극 체험" 비트)이며, 대부분의 응답이 새로운 사실/맥락 정보를 담고 있어 메아리 빈도는 억제되어 있다.

#### 개별 점검

1. **"사내는 말없이 화면을 보고 있었다. 천 년 전에는 벌판이었을 곳에 천만 명이 사는 도시가 서 있다. 그 사실을 소화하는 데 시간이 걸리는 모양이었다."** (101행)
   - "말없이" 직접 서술이지만 **직후 해석적 서술(천 년 전→천만 명)**이 침묵의 무게를 의미화함. voice-guide의 진무혈 "감정 무뎌짐, 담담" 설정과 일치.
   - **판정**: 경계선. 단독 MINOR.
   - **수정 후보**: "사내가 화면에 바싹 다가섰다. 손가락이 화면 위 한강 다리를 따라 움직였다. 천 년 전에는 벌판이었을 곳..." (비언어 부착)

2. **"사내가 입을 다물었다. 이 시대의 여자는 말이 많다는 표정이었다. 아니면 천 년 만에 누군가와 대화를 이어가는 것 자체에 익숙하지 않은 걸지도."** (241행)
   - "입을 다물었다" 직접 서술 + **"표정이었다"** 비언어 해석 즉시 부착. 구조적으로 침묵→비언어 전환이 내장되어 있음. voice-guide의 "감탄·감격 표현 적음" 설정과 일치.
   - **판정**: 기능적 PASS이나 직접 서술 카운트는 1회로 기재.

3. **"사내는 이해했는지 못했는지, 화면 앞에 앉아 채널을 돌리기 시작했다. 뉴스, 드라마, 예능, 다큐멘터리. 한글 자막이 나오는데 읽지 못하니 영상만 보고 있었다."** (95행)
   - 침묵 직접 서술 없음. 행동 연쇄로만 처리. **PASS**.

4. **"사내가 잠시 하은을 봤다. 고르는 것이 아니라 가늠하는 눈이었다. 이 사람에게 어디까지 말해야 하는가를 재는 눈."** (261행)
   - 침묵 직접 서술 없음. **"가늠하는 눈"** 비언어 처리. voice-guide의 진무혈 "짧고 담담" 그대로. **PASS**.

5. **"사내가 고개를 돌려 하은을 봤다. 뭔가 말하려다가, 고개만 끄덕였다."** (35행)
   - "뭔가 말하려다가" = 망설임 비언어(voice-guide 표의 "입을 열었다 닫음"에 근접) + "고개만 끄덕였다" = 결의/수용 비언어. **모범 사례**. PASS.

6. **"목소리가 살짝 갈라졌다. 사내가 이 감정에 익숙하지 않다는 것을 하은은 알 수 있었다."** (31행)

   - 정지-비언어. "표정 변하지 않음" + "눈 달라짐"의 대비로 감정 전달. 직접 침묵 아님. PASS.

8. **"하은은 숨을 삼켰다."** (215행)

#### 침묵 직접 서술 총계

- "말없이" 1회 + "입을 다물었다" 1회 = **2회**.
- 판정 기준(4회+ → MAJOR, 2~3회 → MINOR, 1회 이하 → PASS) 적용 → **MINOR 1건**.
- 2건 모두 비언어 해석 서술이 즉시 부착되어 있어 MAJOR로 악화될 위험은 낮음. 다만 다음 화에서 같은 패턴(말없이 + 직후 해석)이 반복되면 MINOR 누적.
