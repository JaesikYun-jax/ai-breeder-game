## 2. 공간 상태

### A. 강화 별장 (메인 무대)
- **정문**: 검은 차 3대가 자갈 마당 안쪽으로 이미 진입 완료 (ch016 L327). 앞차 조수석 창 내려감. 운전자 쪽 차량 번호판에 가로 먼지 한 줄(특징).
- **마당**: 자갈. 앞차 바퀴 앞으로 자갈 한 개가 튀었다가 굴러감 (5장치 중 '자갈' + '바퀴 소리' 근원).
- **지하 창고**: 콘크리트 바닥(하은의 왼쪽 뺨이 닿아 혈흔 자국 남음). 손목·발목 결박. 콘크리트 냉기 + 창 천장에서 빗소리(이미 잦아듦). 창밖 나뭇잎에 빗방울 맺혀 미끄러짐. **창은 있으나 고정창 추정**(하은이 빗속도를 신호처럼 읽는 묘사).
- **진입 경로 추정**: 정문 → 자갈 마당 → 별장 본관 → 지하 창고 (별장 지하 1층). ch015 C-10 단계에서 이서명이 좌표 제공 완료, 설경이 화면에 찍음. 박정섭은 "지하 창고 위치를 기억합니다" 확인 완료.
- **예상 배치 인원**: 김재원(확정, 창고 안), 장원지에 또는 부하(미확정, 검은 차 3대 탑승자), 별장 관리인 소수.
- **특징적 톤 장치**: 빗소리가 오프닝 직전 잦아듦 → 천장 조명 소리·바람·자갈 밟는 소리만 남는 **정적 긴장** 상태.

### B. 강화대교 (진무혈 이동 루트)
- 진무혈 세단: 조수석 **아리수 4병 (낡은 라벨 1병은 반쯤 빈 상태)** + 뒷좌석 **설경 접은 부적 3장**.
- 박정섭 운전, 진무혈 단독 탑승 (추가 경호 없음).
- 기감 상태: "강의 기운"이 닿자 **기감이 조금 돌아옴**—물의 경로 원격 기감(A-13) 1차 발현 증거. 철골이 저녁 빛에 반쯤 붉음 = **혈마공+저녁 빛 시각 연동**.
- 진무혈 내적: "한 시간이면 닿는다" 결의 (출발 직후) → ch017 오프닝은 **이미 다리 통과 후 강화 북부 진입 중** 또는 **별장 진입 직전 5분 구간**.

### C. L타워 120층 (후방 상황실)
- **설경**: 거실 소파에 앉아 태블릿 정좌 지휘. 왼팔 없는 몸이 소파 깊이에 붙잡혀 일어서지 못함. 관통상 60% 회복 (14화 관통 → 15~16화 2.5일 경과). 오른손 하나로 지휘 + 서연 경맥 어긋남 직접 짚을 수준.
- **서연**: 노트북에서 **백광제약 IT팀 로그 + SNS 전선** 이중 가동. "저녁에 L타워 야경 + '저는 여기 있어요' 글" 게시 예정 = ch017 시점에 **실제 게시되었을 타이밍**.
- **박정섭 부재**: 진무혈 동행으로 L타워 경비 공백 → 담양 교우회 보안업체 12명 중 일부가 120층 커버 (15화 체제).
- **박지원**: 엘리베이터 복도에서 뒷걸음(ch016 엔딩). ch017 시점 **L타워 내부 어딘가에 잔류** (아직 퇴장 안 했음) → A-18 씨앗 유지.
- **위협 예측**: 진무혈 지시 "이게 함정이라면, L타워가 다음이다" → **2차 습격 가능성**을 설경이 이미 계산 중.

## 3. 인물 상태

### 진무혈 (1인칭 복귀 예정)
- **물리적**: 아리수 4병 + 부적 3장 적재. 혈마공 **3할 유지** (Arc 2 12~16화 동안 명시적 상승 없음, 14화 "연둣빛이 얇게 짙어졌다. 손끝까지만 끌어올리고 멈췄다" 절제 운용).
- **정서적**: **천 년 만에 도착한 "분노" 이후 추가 도약 중**. 16화에서 "누군가의 외출이 몸에 걸리는구나"+"내 기감이, 여기서 저기까지 가 닿지 않는구나"라는 **두 번째·세 번째 "천 년 전엔 떠올린 적 없는 문장"** 축적. 14화 "오래 살아야겠다"+15화 "이 나라가 바뀐다는 말을 두 번째로 듣는다"+16화 "거리가 걸리는 자 = 집착 대상이 생긴 자" 3연속 감정 복귀 계단.
- **출정 모드**: 단독 출정 결단 ("아니다. 내가 간다"). 박정섭 동행은 **운전만**—전투는 혼자. 관찰자 모드(ch011 L121 "구분할 필요도 없었다") 완전 해제, **행동자 모드 복귀**.
- **전투 자원 한계**: 장원지에(중급) 상대 시 "쫓을 수 없었다"(14화). 즉 ch017에서 **장원지에 급 인물 조우 시 여전히 정면 대결은 미완**. 아베노 계열 검사는 대피실 벽을 뚫는 수준으로 더 위험.
- **현대 감각 적응**: 휴대폰 서툰 조작(1화 이래), 자동차 후좌석 경험 축적(11화 KTX·13화 세단·16화 세단 3회차).

### 서하은 (3인칭)
- **물리적**: 결박(손목·발목). 콘크리트 냉기 + 왼쪽 뺨 얕은 혈흔. **혀 탐지 반격** 성공(약물 감지) 후 뒷목 손날 기절. 현재 의식 복귀 + 반쯤 앉은 자세(벽에 등 기댐).
- **무기**:
  - (a) 외투 안주머니 **예비 녹음기 + GPS 패치** 작동 중 (B-08 배치 완료). 붉은 점 깜빡.
  - (b) **녹음 내용**: ①김재원 교수실 전화 "목격자 확보/강화 별장/협상 카드/회장님/서연 아가씨 건" 전 발언 ②교수가 하은 앞에서 무너지는 세 박자 ③자기 합리화 독백("어쩔 수 없었다") ④침실 이송 과정 잡음까지.
  - (c) **심리적**: 이미 교수를 언어로 꺾은 상태 ("가방 가져가셨잖아요. 그거 한 대뿐인 줄 아시고"). 교수가 "벽에 부딪혀 뒤로 물러섬" → **이후 상태는 '공황 → 재결심' 구간**일 가능성.
- **정체성**: 고고학도 기록자로서 자기 역할 완전 수용. "고고학도로서 최악이다" 3회 내적 반복(1회·2회·3회가 점층적).
- **위험**: 교수가 공황 후 **재무장하면 2차 폭력 가능**(약물 재투입, 결박 조임, 물리적 입막음). ch017 오프닝에서 이 틈을 누가 먼저 메우는가.

### 김재원 (3인칭 또는 관찰 대상)
- **심리 상태**: 하은의 "이 모든 걸 기록했다" 발언 직후 **가면 완전 탈락**. 의자에서 벽까지 물러섬. 손가락 펴지지 않음. 이마 땀. **자기 합리화 시스템 (아내 수술비·아들 사립초) 외벽이 뚫린 상태**.
- **재기 가능성**: 높음—외벽 뚫려도 "어쩔 수 없었다"로 돌아가는 반복 패턴(ch016 내 2회 확인). 공황 지나면 **더 잔혹한 결정** 가능: (a) 하은 입막음 (b) 녹음기 회수 2차 시도 (c) 별장 도착한 동료(검은 차 3대)에게 처리 떠넘김.
- **물리력**: 낮음(서울대 교수, 비무장 일반인). 단, 단독 대치에서 결박된 하은은 제압 가능.
- **운명 축**: A-01 최종 회수 = 17~18화 "처단 씬". 녹음이 결정타.

### 장원지에 (3인칭, 미확정 현장 여부)
- **14화 이후 상태**: 진무혈과 비무 3회 교차 후 "오늘은 여기까지" 철수. 내적 "형님이 직접 오셔야 합니다" 판정. 현재 **리하오란의 지시 대기 중**.
- **ch017 현장 개입 여부**: 가능성 3단계
  - (a) **부하만 파견**: 검은 차 3대 탑승자 = 일반 용역 수준(김재원이 백준하에게 이송 보고하고, 백가 측이 용역 파견). 장원지에 본인 부재.
  - (b) **원격 감시**: 별장 밖 차량 또는 인근 지점에서 관찰. 진무혈 도착 시 개입 여부 결정.
  - (c) **현장 지휘**: 14화 실패 복구용 직접 투입. 단 12화 "2주 내 재소집·서로 치지 않기로 금기" 합의 파기 비용이 큼—**가능성 낮음**.
- **권고**: (a) 부하 파견이 김재원의 "회장님께서 직접 움직이시기 어려우시다" 발언과 정합적. 장원지에는 지문 남기지 않음. ch017 구출전은 **김재원+백가 용역**을 주 상대로, 장원지에 급은 엔딩 훅이나 18화 개입으로 보류 권고.

### 백준하 (3인칭, 본사 예상)
- **위치**: 백광 본사 회장실. 서연 SNS 미끼에 **시선이 묶인 상태** 예정 (서연 전술 작동).
- **심리**: A-06(서연 약점) + C-13(자기규율 "혈통은 건드리지 않는다") 내적 충돌. 김재원이 "회장님께서 직접 움직이시기 어려우시다는 것, 알고 있습니다" 발언으로 **백준하의 행동 제약을 외부자가 이미 인지하고 활용 중**임이 본인에게도 전달되었을 가능성—**수치감 + 초조**.
- **ch017 등장 확률**: 중간. 별장 현장엔 없을 가능성 높음 (아래 "협상 카드" 지시자 논리).
- **역할**: 배경 압력. 19화 대면 씬 예정이므로 ch017에서 무대 밖 유지 권고.

### 설경 (3인칭, L타워)
- **물리적**: 관통상 **60% 회복** (14화 관통 → 15화 오른손 태블릿 → 16화 소파 지휘). 여전히 일어서지 못함. 왼팔 부재 → 오른손 하나로 지휘.
- **기능**: 태블릿 지휘 + 서연 경맥 보호 + 박정섭 정보 관제 + 담양 교우회 순찰 조율. **4중 관제 중**.
- **ch017 역할**: 후방 지휘. 진무혈 현장 정보 전달 + L타워 2차 습격 방어 준비. **직접 전투 불가** (추가 부상 = A-07 회수 곡선 최종 단계, 20화 예정).

## 6. 직전 엔딩 훅 5장치 연결점

ch016 마지막 문단 (L327~331)의 5장치:
1. **자갈** ("자갈 하나가 앞차의 바퀴 앞으로 튀었다가 옆으로 굴렀다")
2. **먼지 번호판** ("운전자 쪽 차량 번호판에 먼지 한 줄이 가로로 그어졌다")
3. **붉은 점** ("예비 녹음기의 작은 붉은 점이, 외투 옷감 아래에서 옅게 깜빡였다")
4. **바퀴 소리** ("자갈 위를 천천히 밟는 바퀴 소리가 창고 벽 너머에서 들어왔다")
5. **검은 세단 3대** ("검은 차 세 대가 나란히 멈춰 섰다 → 정문을 통과해 마당 안으로")

### ch017 오프닝에서의 수신 방식 (권고)

**A안. 동일 자극의 다른 시점 수신** (추천)
- ch016 엔딩: 하은 관점 — 창고 안에서 **벽 너머**로 들리는 바퀴 소리.
- ch017 오프닝: 진무혈 관점 — 세단에서 내려 **마당 밖에서** 자갈 바퀴 소리를 듣는 본인 위치의 소리. 또는 하은이 **두 번째** 자갈 바퀴 소리(진무혈 도착)를 구별하는 장면.
- 5장치 대응:
  - 자갈 → 진무혈의 구두가 자갈 위에 내려앉는 순간 (A-10 자연 일체 경지 연출: 자갈이 먼저 길을 내줌)
  - 먼지 번호판 → 진무혈 세단의 번호판은 **깨끗함**—대비로 백가 차량의 먼지 강조
  - 붉은 점 → 창고 안에서 하은이 눈을 뜨고 붉은 점을 재확인하는 한 컷
  - 바퀴 소리 → **두 번째** 바퀴 소리 (하은이 박정섭 세단 특유의 엔진 톤을 "맞다, 설경 씨가 말한 그 차" 식으로 기감 없이도 식별)
  - 검은 세단 3대 → 진무혈이 별장 진입로에서 **3대의 차량 배치를 한 눈에 읽음** (자연 일체 경지 감각)

**B안. 교차 컷 병렬 유지** (대안)
- ch016처럼 강화 별장 / L타워 / 세단 3분할 교차를 1~2분 더 유지.
- 진무혈 도착 직전까지 긴장 누적 → 중반에 폭발.
- 단점: 16화 톤 반복 위험. **권고하지 않음.**

**C안. 하은 3인칭 단독 선행** (권고)
- 오프닝 2~3분간 **하은 단독 3인칭**. 창고 내부의 소리·빛·습기 감각을 통해 외부 접근을 먼저 읽게 함.
- 창고 벽 너머 두 번째 바퀴 소리가 첫 번째와 다르다는 것을 하은이 알아챔 ("이 차는 느리지 않다. 이 차는 먼저 멈추지 않는다").
- **진무혈의 도착을 하은이 먼저 감지**하는 구조 = A-13 원격 기감의 역방향 (하은→진무혈)이라는 **관계 곡선 신규 이벤트**.

### 권고 결론
**A안 + C안 하이브리드**: 오프닝 2/3를 하은 3인칭 단독(C안), 이어 진무혈 1인칭 진입(A안). 5장치 전원 다른 각도로 재사용. 교차 컷은 중반 이후로 유보.

## 7. 톤·리듬

### ch016 톤 프로필
- 감정 아크: 코미디 0 / 긴장 70 / 공포 25 / 감동 5
- 구조: **교차 컷 4시점** (하은 / 진무혈 / 김재원 / 서연) 병렬 긴박
- 페이싱: 아침(평온) → 오전(습격 시작) → 오후(수색 가속) → 저녁(양방 진입)—**4단 가속**
- 특징: 비언어 중심. 대사보다 **행동·이미지·시간 표지** 밀도 높음.

### ch017 오프닝 선택지 평가

**A. 즉시 돌입 전투**
- 장점: 16화 긴박 유지. 엔딩 5장치 즉각 회수.
- 단점: 하은의 1시간 구금 누적 긴장이 해소되지 않음. 진무혈 감정 복귀 단계가 "전투 흥분"으로 덮임—**A-12 곡선 낭비**.
- **권고 등급**: 하

**B. 잠시 숨고르기** (진무혈 세단 안 독백 2~3분, 또는 창고 내 시간 멈춤)
- 장점: 14~16화 누적된 감정 문장 3종("이 밤에 잃을 뻔한 사람" / "이 나라가 바뀐다는 말을 두 번째로" / "거리가 걸리는")의 **계승 문장** 1종 추가 여지. 혈마공 3할의 절제 운용 톤 유지.
- 단점: 긴박 끊어질 위험.
- **권고 등급**: 중상

**C. 하은 3인칭 선행** (권고)
- 장점:
  - (1) 하은 내면의 1시간 구금 감각(혈흔·냉기·녹음기 붉은 점·빗소리 잔향)을 **독자 감정과 동기화**.
  - (2) 진무혈 도착을 **하은이 먼저 감지**하는 구조로 A-12·A-13 양방향 정서 곡선 성립.
  - (3) 교차 컷 피로 방지 — ch016과 톤 차별화.
  - (4) **비언어 TOP 3 전환**이 자연스러움 (아날로그 텍스처·청각·촉각 중심의 하은 시점은 "손끝/태블릿/박자" 과포화를 자동 리셋).
- 단점: 진무혈 1인칭 복귀 지연 (5~7천자 중 후반 60~70%에서 1인칭 전환).
- **권고 등급**: 상

### 권고 오프닝 구조
```
씬 1: 하은 3인칭 — 창고 내부 (현재 자세·녹음기 붉은 점·빗소리 잦아듦·두 번째 바퀴 소리 감지)  [약 15%]
씬 2: 진무혈 1인칭 — 별장 마당 진입 (자갈·검은 차 3대·부적 잔기 감지·A-10 2단계 연출)  [약 20%]
씬 3: 하은·진무혈 재회 (창고 문 열림·A-12 "천 년 만에 도착한 문장" 4번째·결박 풀기)  [약 25%]
씬 4: 김재원 심판 1차 (녹음기 발동·진무혈의 물리적 처단은 아직 보류·하은 주도)  [약 20%]
씬 5: 엔딩 훅 (L타워 2차 습격 조짐 or 장원지에 엔딩 카메오 or 강화대교 퇴각 중 추격)  [약 20%]
```

### 리듬 가이드
- **문장 길이**: ch016의 짧고 탁탁 끊는 리듬 기조 유지. 단, **씬 3 재회**에서 2~3문장 긴 호흡 1회 삽입 허용 (A-12 계단 노출용).
- **대사 밀도**: 낮게. 하은과 진무혈의 첫 교환은 **3~4 줄** 이내. 김재원 대사는 이미 16화에 다 나왔으므로 ch017에선 **무너진 자의 잔여 발화**만.
- **비언어**: 위 TOP 3 대체 후보 각각 3회 이상 분산 배치.
- **시간 표지**: "한 박자"는 **ch017 전체에 2회 이하**로 제한. 대체 후보(숨 단위·물방울 단위·공간 단위) 우선.

### 금지 / 경계
- 진무혈의 감정 언어화 **폭발 금지** (28화 첫 눈물까지 계단 유지).
- 진무혈이 김재원을 **직접 죽이지 말 것** (18화 처단 씬 유보). 본 화는 **무력화·굴복까지만**.
- 장원지에 직접 전투 투입 금지 권고 (14화에서 이미 등장, 20화 습격 재회까지 유보).
- "고개를 끄덕였다" 재사용 금지 (ch003~ch007 과포화 흔적 회복 중).
- 태블릿 디지털 매개 비언어 **설경 후방 씬 1회만** 허용.
