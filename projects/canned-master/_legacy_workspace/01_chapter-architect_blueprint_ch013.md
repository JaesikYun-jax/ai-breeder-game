**서브 비트 (밀도 보강)**

- 씬 1 서브: ① 김재원이 하은의 커피잔 쪽으로 티백 하나를 더 권함(“피곤해 보여. 한 잔 더 할래?”) → 하은이 거절 불가능한 분위기 → 그냥 잔만 받고 안 마심. ② 책장 한쪽에 새로 들어온 자문 명함이 반쯤 빼꼼. ③ 벽의 시계가 두 번 째깍.
- 씬 2 서브: ① 카페 바리스타가 라테 아트를 너무 열심히 그려줘서 시간이 지체됨(일상감 유지). ② 옆자리 대학생이 노트북 스티커 얘기로 떠듦(소음 속에 시선 감지). ③ 하은이 주문대에서 결제하며 입구 유리에 비친 SUV 넘버를 조각 눈으로 담음(발굴 훈련 활용).
- 씬 5 서브: ① 박정섭의 구두는 낡았지만 깨끗함. ② 상대 앞사내의 손가락 마디 굳은살이 권투장이 아니라 핸들 잡는 자의 굳은살. 용역 출신임을 관찰자 정보로 흘림(박정섭의 속 한마디: “경찰 출신은 아니군.”). ③ 박정섭이 “제가 공권력은 아닙니다만, 공부는 오래 했습니다.” 짧은 자기 소개.
- 씬 7 서브: ① 하은이 카페에서 챙긴 레몬 캔디 하나를 주머니에서 꺼내 진무혈 손바닥에 얹음. 진무혈이 손바닥을 두 번 보다가 주머니에 넣음. ② 설경이 눈꼬리 살짝 내리는 **정도만**의 웃음(대사 없이 비언어). ③ 거실 화분의 몬스테라 잎 끝이 물방울 한 방울로 반짝.

## 2. 시점 구성 설계 (POV Design)

- 메인 시점: **서하은 3인칭 제한시점 (약 60%)** — 주역 회차. 독자에게 공포·안도·연기력·증거 수집을 밀착 체험.
- 서브 시점 후보:
  1. **김재원 3인칭 (약 20%)** / 유형: 빌런 행동자 / 용도: 삼파 결정의 말단 실행이 ‘평범한 학교 교수의 통화 한 통’으로 개시됨을 독자에게만 공개. 어그로 빌드업 토대. / 씬 위치: 도입 직후(씬 1-B), 필요 시 엔딩 부근에 2~3줄 미러 장면 가능하나 축소 권장.
  2. **진무혈 1인칭 (약 10~15%)** / 유형: 관찰자 + 주역의 균형추 / 용도: 자연 일체 경지 연출 지속(참새·동백·바람). 톤 낙차 담당(정적). / 씬 위치: 씬 6 · 씬 9.
  3. **설경 3인칭 (약 10~15%)** / 유형: 조연 실무 지휘자 / 용도: 원격 대응의 냉정 · 신교 외곽 첫 가동. 왼쪽 빈 소매 반복 묘사 자제하되 1회 삽입. / 씬 위치: 씬 4. 엔딩 한 줄 대사.
- 최종 채택: **하은 60% / 김재원 20% / 진무혈 15% / 설경 10%** (네 시점. 프로젝트 상한 3시점 규칙과 충돌 가능성 있음 → **아래 재조정** 참조).
- **재조정**: novel-config.md §3 “한 챕터 최대 3시점, 각 시점 최소 1,000자” 규칙. 진무혈 1인칭은 씬 6·9 합산 1,000자 확보(창가·참새·짧은 대사·내면), 설경은 씬 4 단독으로 1,000자 확보 어려움 → **설경 씬은 하은 3인칭 속의 외부 카메라로 녹임**(하은의 폰 화면으로 설경의 지시가 전달되는 구조 + 박정섭 도착 장면 뒤 한 번 ‘설경 3인칭 500자 미니’). 또는 **설경 씬을 삭제하고 박정섭의 행동으로 설경의 지시만 드러냄**. 최종 선택: **3시점 — 하은 60% / 김재원 20% / 진무혈 20% (설경은 하은 시점 안에서 대사·문자로만 나타남)**. 설경 3인칭 독립 씬은 이번 화에 생략. 이유: 12화에서 설경 3인칭 독립 씬이 이미 강하게 있었고, 13화는 ‘하은 주역 회차 + 빌런 실행 첫 단계’ 대비를 선명히 하기 위함. [source: novel-config.md §3 / 프롬프트 시점 배분 60/20/20 수렴]
- 채택 사유: ① 약자 시점의 공포 빌드업이 어그로 핵심이라 하은 60%는 톤 공식상 당연. ② 빌런 행동 노출 20%는 12화 삼파 결정의 말단 실행이 이번 화 중심 고리임을 못박는 역할. ③ 진무혈 20%는 자연 일체 경지 지속 연출·톤 숨쉬기·엔딩 훅 전조 담당.
- 비채택 사유 (설경 독립 시점 없음): 시점 수를 3개로 제한하고, 대신 설경의 실무 지휘는 하은의 폰과 박정섭의 행동으로 외부화하여 **‘그림자처럼 조율하는 부교주’** 이미지를 강화. 12화 단독 시점과 중복 회피.
- 분량 배분: 하은 3,900~4,500자 / 김재원 1,300~1,500자 / 진무혈 1,300~1,500자. 서브 시점 합 약 40% (가이드 20%를 상회하나 본 화는 ‘주역 회차 + 빌런 첫 실행 노출’ 특수 상황이라 정당화 가능).

## 4. 보이스 퀵레프 카드

### 서하은
- 종결어미: “~요 / ~거예요 / ~어요 / ~죠”. 내면 독백은 구어체(“아, 이거 어떻게 하지”).
- 문장 길이: 평소 중간. 공포 상승 시 파편화. 안도 시 숨 삼킴.
- 말버릇: “......저거 일부러다.” 류의 자기 관찰 독백. 위기엔 말이 짧아짐.
- 감정 폭발 시: 말 끝이 올라가거나(“...진짜예요?”), 즉시 자조(“고고학도로서 최악의 반응이다”).
- 금지 패턴: 본인 감정을 직접 선언하지 않는다(“무서웠다” 같은 상태어 금지). 감각 묘사로 드러낸다.
- 샘플: “교수님, 그럼 저 나가 볼게요.” / “......지금 저거, 따라오는 거지?” / (설경 문자 보고) “......다행이다.” / (어깨 접촉 후 독백) “고고학도로서 최악의 반응이다.”

### 김재원
- 종결어미: “~지 / ~해요 / ~지 않나 / ~할게.” 학생 상대로 친근한 선배 톤. 내면은 정중함이 희석됨.
- 문장 길이: 대화는 중간. 통화 장면에선 더 짧고 공손(백가 대리 경찰 지인에게는 예의).
- 말버릇: “하은아,” 로 시작. 자기 방어적 문장(“나도 어쩔 수 없지.”).
- 감정 폭발 시: 본 화에서는 폭발 없음. 손끝으로 안경테만 잠깐.
- 금지 패턴: 직설적 협박 금지. 분위기와 권유형으로만.
- 샘플: “하은아, 요즘 잘 지내?” / “...수상한 사람 만나는 것 같더라. 선배 말이니까 그냥 듣고 넘겨.” / (통화) “......번거로우시겠지만, 우리 아이 하나만 봐 주시죠. 며칠만.” / (내면) “어쩔 수 없다. 어쩔 수 없다.”

### 진무혈
- 종결어미: “~하라 / ~소 / ~있었느냐 / ~다”. 이 화는 **~다**(담담) 위주.
- 문장 길이: 매우 짧음. 한 씬당 대사 3~5줄 내.
- 말버릇: 창 밖을 먼저 보고 말함. 동물을 보는 동안 보고를 들음.
- 감정 폭발 시: 이 화엔 없음. 대신 어깨에 손 한 번, 3초.
- 금지 패턴: 현대 감탄사 금지. 긴 설명 금지. “기분이” 같은 감정 직접어 금지.
- 샘플: “그자가 시작했군.” / “고생했다.” / “그럼 오늘은 하은이 여기서 자야겠군.” / (내면) “치지 않아도 되는 손으로 이 사람을 지키는 법을 다시 배워야겠다.”

### 설경
- 종결어미: “~습니다 / ~겠습니까 / ~지요.” 짧고 명료.
- 문장 길이: 짧음. 한 문장 1~2줄.
- 말버릇: 교주에게 보고할 때 사실만. 하은에게는 “아가씨” 호칭 준수.
- 감정 폭발 시: 없음. 한 번 눈을 감고 뜨는 것으로 대체.
- 금지 패턴: 감정어 금지. 냉소는 아주 가끔.
- 샘플: (폰 문자) “도서관 3층 비상구. 10분.” / (통화) “박 사범님. 남쪽 정문 쪽, 관악 본관 동. 바로 가 주시지요.” / (진무혈에게) “상황 정리했습니다. 서 하은 아가씨 안전.” / “다음은 그들이 직접 옵니다. 오늘 밤이 아니라도, 이틀 안에.”

### 박정섭(신규, voice-guide.md 추가 대상)
- 종결어미: “~습니다 / ~지요 / ~하십시오.” 전직 경찰 억양. 군더더기 없음.
- 문장 길이: 매우 짧음.
- 말버릇: 목적 + 지시의 2단 문장. 존칭 과하지 않되 정중.
- 감정 폭발 시: 없음. 상황 종료 직후 “수고하셨습니다” 정도.
- 금지 패턴: 감탄사 · 자기 자랑 금지. 무공 용어 금지(현대 격투·관절기만).
- 샘플: “아가씨, 이쪽으로.” / “저희가 모시겠습니다.” / “차에 앉으시고 고개만 숙이세요.” / “경찰 출신은 아니군.” / “L타워까지 가십니다. 안심하세요.”

**메아리 대사 금지 재확인**: 설경의 “다음은 그들이 직접 옵니다.”를 진무혈이 반복해선 안 됨. 진무혈은 다른 각도(“그럼 오늘은 하은이 여기서 자야겠군.”)로 응답. [source: tone-and-style.md §5]

## 5. 진무혈 상태 체크

- 혈마공 회복률: **3할 유지** (12화에서 3할, 본 화에서 전투 없음. 자연 일체 경지 심화만). [source: protagonist-bible.md §2]
- 재생 스킬 상태: 안정. 이번 화 자기 부상 없음.
- 모래시계(시간 감각): 본 프로젝트에서는 봉인 해제 후 봉인진 상시 계수 없음(대신 혈마공 회복률·단전 기 지표). “모래시계 잔량” 개념 자체는 본 프로젝트에 맞지 않으므로 **해당 없음**으로 기록. [source: novel-config.md §1, §4 — 1093년 고정 수치 외 시간 카운터 없음]
- 정신 상태: 담담 유지 + 감정 미세 복귀(부분 복귀 단계 초중반). [protagonist-bible §3 10~14화 구간]
- 기둥 흡수 이력: **본 프로젝트에 ‘기둥’ 개념 없음 → 해당 없음**. (해당 개념은 dclass-hero 프로젝트용. canned-master는 봉인/업보/혈마공 프레임만.) [source: novel-config.md, worldbuilding.md]
- 습득 능력: 자연 일체 경지 연출(참새·동백·바람). 공격성 없이 동물 접근. [foreshadowing.md A-10]
- 홍채: 연둣빛 짙음. 햇빛 드는 각도에 짙게 반짝. 본인의 자각 한 번만. [protagonist-bible §2 체표 + §3]

## 7. 톤 설계

- 주요 톤 구간: **일상 외피 속 공포 빌드업 → 현대 격투 해소 → L타워 정적·온기**.
- 감정 전환점 1: 카페에서 SUV·뒷자리 남자 감지 시(씬 2 후반).
- 감정 전환점 2: 박정섭 등장·손목 제압 3초(씬 5).
- 감정 전환점 3: 엘리베이터 문이 열리고 진무혈이 시야에 들어올 때(씬 7).
- 감정 전환점 4: 설경의 “오늘 밤이 아니라도, 이틀 안에” + 진무혈의 “그럼 오늘은 하은이 여기서 자야겠군.”(씬 9).
- 낙차 공식: **권력의 부드러운 칼(김 교수의 정중함) vs 현대 격투의 사무적 제압 vs 진무혈의 잔잔한 자연 일체**. 세 겹의 낙차가 한 화 안에 맞물리게. [tone-and-style.md §3, §7]
- 금지 패턴: 스프레드시트식 나열 · 영어 혼용 · em dash · 감정 직접어(“무서웠다”, “기분이 좋았다”) · 목적절 강제(“~하기 위해 ~했다” 과용) · 내면독백의 긴 분석 문단(2문장 이내로 끊기) · 메타 서술(“다음 화”, “이 순간이 전설이 된다” 등) 0회. [tone-and-style.md §6, §6-A / novel-config.md §7]
- 내면 독백 샘플 (하은):
  - “아, 커피잔 온도가 미지근해질 때까지 교수 얼굴을 봐야 한다.”
  - “......지금 저거, 따라오는 거지?”
  - “고고학도로서 최악의 반응이다.”
- 내면 독백 샘플 (진무혈):
  - ‘이 도시에는 약속을 거래로 바꾸는 자들이 많다.’
  - ‘치지 않아도 되는 손으로 이 사람을 지키는 법을 다시 배워야겠다.’
- 내면 독백 샘플 (김재원):
  - ‘어쩔 수 없다. 하은이도 다치진 않을 것이다. 그저 겁을 주는 것뿐이다.’

## 8. 훅 전략

- 오프닝 훅: **미스터리 + 정중한 압박** / 강도: **3/5**. 김재원이 하은에게 “요즘 수상한 사람 만나는 것 같더라”를 웃으며 꺼내는 순간. 폭력 없음이 오히려 긴장감.
- 엔딩 훅: **전조 + 미공개 인물 실루엣** / 강도: **4/5**. 창 너머 멀리 다른 빌딩 옥상에서 누군가가 로프를 챙기는 손. 얼굴·이름 미공개(14화 자객 전조). 설경의 “이틀 안에” 대사로 밑에 깔린 시한.
- 포인트 장면:
  1) 카페 유리 너머 SUV 번호판을 하은이 조각 눈으로 담는 장면(씬 2) — 어그로 빌드업 최고조 직전.
  2) 박정섭의 3초 관절기 + “경찰 출신은 아니군.” 짧은 진단(씬 5) — 카타르시스 피크.
  3) 어깨에 3초 손 + “고생했다.”(씬 7) — 섹시어필 피크 + 톤 숨쉬기.

## 9. 현대 지식 활용 후보 (MODERN_REF)

- 설경의 태블릿 **원격 지휘**(화면 세 개, 위치 점, 통화 단축) — 관찰자 서술로 간결히.
- 하은의 **SUV 번호판 각인** 관찰력 — 발굴 훈련을 현대 감시 상황에 전용(캐릭 개성).
- **용역 vs 전직 경찰** 손 굳은살 차이 — 박정섭의 현장 진단.
- 김 교수의 **카카오톡이 아닌 음성 통화** 사용 — 기록 흔적 최소화(교활의 소품).
- 주의: IT 비유 챕터당 3회 이내. 본 프로젝트는 dclass-hero의 ‘프로그래머 비유’가 아니므로 **영어·IT 용어 직접 노출 금지**. [novel-config.md §7, tone-and-style.md §6] “태블릿”, “SUV”, “카카오톡” 등 일반 고유명 수준은 허용.

## 12. 공간 / 소품 디테일 시트

- **서울대 본관 김재원 연구실**: 책상 오른쪽 서랍, 각진 서류철 모서리. 자문 명함이 책장 한 칸에 반쯤 삐죽. 창은 운동장 쪽. 벽 시계 초침 소리 또렷.

## 14. 집필 시 Do / Don’t 요약

### DO
- 하은 시점에서 ‘발굴 훈련 관찰력’을 번호판·손 굳은살·서류철 모서리로 실행.

- 진무혈 씬은 **참새·동백·바람** 중 2개 이상. 대사 3~5줄.
- 하은-진무혈 어깨 3초 접촉 + “고생했다.” + 하은 자조 독백. 한 씬.
- 엔딩 훅: 창 너머 멀리 로프 든 손 실루엣, 이름 미공개.

### DON’T
- em dash 사용(0회 엄수).
- 메타 서술(“N화”, “떡밥”, “복선”, “다음 화”) 0회.
- “무서웠다 / 기분이 나빴다 / 사랑스러웠다” 등 감정 직접어.
- 혈마공 회복률 숫자 노출(“3할이다” 등).
- 섹시어필을 직접어로(“매력적이었다”).
- 설경 독립 3인칭 1,000자 블록 추가(3시점 규정).
- 서연·박지원·박순영·환자 등장(본 화 외 인물 비중 방해 금지).
- 중국·일본 이름 임의 한국어 변형(본 화 직접 등장 없음, 엔딩 훅의 손은 무명 유지).
