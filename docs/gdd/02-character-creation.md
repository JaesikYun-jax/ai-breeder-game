# 02. 캐릭터 생성 — 성격 퀴즈 시스템

## 개요

플레이어는 게임 시작 시 5개의 3지선다 질문에 답한다.  
각 답변은 **5가지 능력치(STR, INT, CHA, PER, FAT)에 가중치**를 부여하며, 최종 능력치 분포에 따라 **9개 진입점 중 하나에 배정**된다.

---

## 질문 디자인 원칙

1. 질문은 "당신이라면?" 상황 제시 (직접적인 능력치/지역 언급 금지)
2. 어떤 답이 "정답"인지 알 수 없도록 설계
3. 모든 선택지가 매력적으로 느껴져야 함
4. 질문 순서: 가치관 → 행동 양식 → 관계 → 위기 대응 → 운명관

---

## 5개 질문

---

### Q1. 가치관 — "눈앞에 두 갈래 길이 있다"

> 낯선 세계에 떨어졌다. 눈앞에 세 방향의 길이 보인다.  
> 하나는 밝고 넓은 대로, 하나는 울창한 숲속 좁은 길, 하나는 아래로 향하는 어두운 계단.

| 선택지 | 텍스트 | 능력치 보너스 |
|--------|--------|--------------|
| A | 밝은 대로로 간다. 사람이 있을 것이다. | CHA+3, FAT+1 |
| B | 숲속 길로 간다. 조용히 관찰하고 싶다. | PER+3, INT+1 |
| C | 계단을 내려간다. 뭔가 끌린다. | STR+2, FAT+2 |

**설계 의도**: 세계와의 첫 접촉 방식 — 사교적/관찰적/본능적

---

### Q2. 행동 양식 — "전투 상황"

> 거대한 야수가 길을 막는다. 도망치기엔 이미 눈이 마주쳤다.

| 선택지 | 텍스트 | 능력치 보너스 |
|--------|--------|--------------|
| A | 주변을 살핀다. 약점이 있을 것이다. | PER+2, INT+2 |
| B | 돌을 집어 던진다. 일단 맞서야 한다. | STR+3, PER+1 |
| C | 눈을 마주보며 천천히 다가간다. 대화가 통할 수도. | CHA+2, FAT+2 |

**설계 의도**: 위험에 대한 기본 반응 — 분석/전투/교감

---

### Q3. 관계 — "동료의 배신"

> 함께 여행하던 동료가 뒤에서 칼을 뽑는다.  
> "미안하지만 여기까지야" 라고 말한다.

| 선택지 | 텍스트 | 능력치 보너스 |
|--------|--------|--------------|
| A | "왜?" 라고 묻는다. 이유를 들어야 한다. | INT+3, CHA+1 |
| B | 슬프지만 칼을 뽑는다. 살아남아야 한다. | STR+2, PER+2 |
| C | "그래도 널 원망하지 않아." 진심이다. | CHA+2, FAT+2 |

**설계 의도**: 인간관계에서의 핵심 가치 — 이해/생존/용서

---

### Q4. 위기 대응 — "마을의 재앙"

> 작은 마을에 도착하니 역병이 퍼져 있다.  
> 주민들이 도움을 청한다. 하지만 당신도 감염될 수 있다.

| 선택지 | 텍스트 | 능력치 보너스 |
|--------|--------|--------------|
| A | 원인을 조사한다. 치료법을 찾아야 한다. | INT+3, PER+1 |
| B | 일단 환자들을 돌본다. 내 몸은 나중 문제. | CHA+2, STR+2 |
| C | 위험을 무릅쓰고 역병의 근원지로 직접 간다. | FAT+3, STR+1 |

**설계 의도**: 타인의 고통에 대한 대응 — 지적 해결/헌신/운명적 돌진

---

### Q5. 운명관 — "신의 목소리"

> 잠에서 깨어났더니 어디선가 목소리가 들린다.  
> "네 운명은 이미 정해져 있다." 라고 말한다.

| 선택지 | 텍스트 | 능력치 보너스 |
|--------|--------|--------------|
| A | "그건 네 생각이고." 운명 따윈 없다. | STR+2, PER+2 |
| B | "정해진 운명이라면 이용해주지." 역이용한다. | INT+2, FAT+2 |
| C | "...어떤 운명인데?" 일단 들어본다. | CHA+2, FAT+2 |

**설계 의도**: 초월적 존재/운명에 대한 태도 — 거부/이용/수용

---

## 능력치 → 진입점 매핑

5개 질문 응답 후 총점 (각 능력치 0~15 범위, 총합 고정 20점)

| 진입점 | 1순위 능력치 | 2순위 능력치 | 배정 조건 |
|--------|-------------|-------------|----------|
| 아젤리아 왕국 | CHA | FAT | CHA 최고 & FAT ≥ 4 |
| 카이젤 제국 | STR | INT | STR 최고 & INT ≥ 4 |
| 솔라리스 (남방) | PER | STR | PER 최고 & STR ≥ 3 |
| 프로스트헬 (북방) | STR | PER | STR 최고 & PER ≥ 4 (INT < 4) |
| 용화국 (동방) | INT | STR | INT 최고 & STR ≥ 3 |
| 리베르타 (군도) | CHA | PER | CHA 최고 & PER ≥ 4 (FAT < 4) |
| 셀레스티아 (천공) | INT | FAT | INT 최고 & FAT ≥ 4 |
| 카즈모르 (지하) | STR | INT | STR 최고 & INT ≥ 4 & PER < 4 |
| 아비살 (심해) | FAT | CHA | FAT 최고 or (CHA ≥ 4 & FAT ≥ 4) |

### 동점 처리
1. 1순위 능력치가 동일한 경우 → 2순위 능력치 비교
2. 그래도 동점 → FAT가 높은 쪽 (운명이 개입)
3. 완전 동점 → 아비살(심해) 배정 (심연은 모든 것을 받아들인다)

### 회귀 이후 (2회차~)
- 퀴즈를 다시 할 수 있음 (새로운 진입점 탐색)
- 또는 퀴즈 건너뛰고 **직접 진입점 선택** 가능 (이전에 방문한 적 있는 지역만)
- 미방문 지역은 퀴즈로만 접근 가능 → 리플레이 동기 부여

## 예시 플레이

### 플레이어 선택: A, C, A, C, B
| 질문 | 선택 | STR | INT | CHA | PER | FAT |
|------|------|-----|-----|-----|-----|-----|
| Q1 | A (대로) | 0 | 0 | 3 | 0 | 1 |
| Q2 | C (다가감) | 0 | 0 | 2 | 0 | 2 |
| Q3 | A (이유물음) | 0 | 3 | 1 | 0 | 0 |
| Q4 | C (근원지) | 1 | 0 | 0 | 0 | 3 |
| Q5 | B (역이용) | 0 | 2 | 0 | 0 | 2 |
| **총합** | | **1** | **5** | **6** | **0** | **8** |

→ FAT 최고(8) → **아비살(심해)** 배정  
→ 초기 능력치: STR 1 / INT 5 / CHA 6 / PER 0 / FAT 8

### 플레이어 선택: C, B, B, B, A
| 질문 | 선택 | STR | INT | CHA | PER | FAT |
|------|------|-----|-----|-----|-----|-----|
| Q1 | C (계단) | 2 | 0 | 0 | 0 | 2 |
| Q2 | B (돌던짐) | 3 | 0 | 0 | 1 | 0 |
| Q3 | B (칼뽑음) | 2 | 0 | 0 | 2 | 0 |
| Q4 | B (환자돌봄) | 2 | 0 | 2 | 0 | 0 |
| Q5 | A (거부) | 2 | 0 | 0 | 2 | 0 |
| **총합** | | **11** | **0** | **2** | **5** | **2** |

→ STR 최고(11), PER ≥ 4 → **프로스트헬(북방)** 배정  
→ 초기 능력치: STR 11 / INT 0 / CHA 2 / PER 5 / FAT 2
