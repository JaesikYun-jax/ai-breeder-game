# 01. 게임 개요

## 장르
이세계 회귀 루프물 선택지 스토리게임 (비주얼노벨 스타일)

## 핵심 컨셉
> "죽을 때마다 처음으로 돌아간다. 하지만 기억은 남는다."

현대 한국에서 죽은 주인공이 판타지 세계에 전이. 이 세계에서도 죽으면 **최초 전이 시점으로 회귀**한다. 매 회귀마다 이전 루프의 기억이 축적되며, 선택에 따라 세계의 진실에 점점 가까워진다.

## 핵심 루프

```
[현대 사망] → [이세계 전이] → [성격 퀴즈로 진입점 결정]
    ↓
[스토리 진행: 매 분기 2~3개 선택지]
    ↓
[사망 또는 엔딩 도달]
    ↓ (사망 시)
[회귀 → 기억 축적 + 일부 능력 유지]
    ↓
[같은 진입점 or 새로운 진입점 선택 가능]
```

## 회귀 메커닉

| 요소 | 회귀 시 리셋 | 회귀 시 유지 |
|------|-------------|-------------|
| 스토리 진행 | O | X (기억만 남음) |
| NPC 관계 | O (NPC는 기억 못함) | X |
| 플래그/선택 기록 | O | △ (메타 플래그) |
| 주인공 기억 | - | O (축적) |
| 특수 능력 | △ (일부만) | △ (일부만) |
| 아이템 | O | X |
| 회귀 횟수 | - | O (카운트 증가) |

### 메타 플래그
회귀해도 유지되는 특수 플래그. 특정 진실을 알게 되면 해금되며, 이후 루프에서 새로운 선택지가 열림.
- 예: "제국의 비밀을 알게 됨" → 다음 루프에서 제국 관련 대화에 새 선택지 등장
- 예: "인어 여왕의 진짜 이름을 암" → 수중 왕국 진입 시 특수 루트 개방

## 게임 플로우

### 1단계: 성격 퀴즈 (캐릭터 생성)
- 5~7개 질문에 답하면 초기 성향/능력치 결정
- 답변 조합에 따라 **9개 진입점 중 하나에 배정**
- 회귀 이후에는 퀴즈를 다시 하거나 직접 진입점 선택 가능

### 2단계: 스토리 진행
- 매 분기 2~3개 선택지 제시
- 선택에 따라 스토리 분기, 능력치 변동, NPC 호감도 변화
- 각 진입점마다 고유 1장(챕터 1)이 있고, 이후 세계 전체로 확장

### 3단계: 사망/엔딩
- 사망 → 회귀 (메타 플래그 축적)
- 일반 엔딩 → 해당 루트 클리어 (리플레이 가능)
- 트루 엔딩 → 다수 메타 플래그 필요, 세계의 진실 도달

## 초기 능력치 체계

| 능력치 | 설명 | 영향 |
|--------|------|------|
| **힘 (STR)** | 물리적 전투력 | 전투 선택지 성공률 |
| **지능 (INT)** | 마법·지식 | 마법/지식 선택지, 복선 조기 파악 |
| **매력 (CHA)** | 대인관계 | NPC 설득, 호감도 보너스 |
| **감각 (PER)** | 관찰·직감 | 함정 감지, 숨겨진 선택지 발견 |
| **운명 (FAT)** | 운·회귀 친화도 | 치명적 상황 생존률, 메타 플래그 획득 보너스 |

성격 퀴즈 답변에 따라 5개 능력치에 초기값 배분.
총합은 고정(예: 25점), 배분 비율만 다름.
