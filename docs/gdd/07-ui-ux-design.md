## 텍스트 연출 시스템

### 타이핑 효과
- **기본 속도**: 30ms/글자
- **slow**: 60ms/글자 (독백, 충격적 내용)
- **fast**: 15ms/글자 (급박한 상황)
- **클릭/탭**: 즉시 전체 표시
- **자동 모드**: 텍스트 완료 후 2초 대기 → 다음 진행

### 화자 표시
- NPC 대사: 화자명 표시 + 캐릭터 하이라이트
- 나레이션: 화자명 없음, 이탤릭체
- 주인공 독백: "(괄호 안 텍스트)", 주인공 색상
- 시스템 메시지: `[대괄호]`, 글리치 폰트

### 선택지 UI
- 일반 선택지: 반투명 박스 + 흰 텍스트
- 호버/터치: 박스 밝아짐 + 미세한 확대
- 능력치 조건 선택지: 능력치 아이콘 표시 (충족 시 밝게, 미충족 시 어둡게)
- **메타 선택지** (회귀 기억):
  - 박스 테두리 글리치 이펙트
  - 텍스트 색상: 시간의 기둥 색 (보라/금색)
  - 라벨: `[이전 기억]` 또는 `[기시감]`
  - 등장 애니메이션: 화면이 찢어지듯 나타남

## 화면 이펙트

### 전환 효과
| 상황 | 전환 |
|------|------|
| 일반 씬 전환 | 페이드 인/아웃 (0.5초) |
| 같은 장소 시간 경과 | 디졸브 (1초) |
| 충격적 장면 | 화이트/블랙 플래시 (0.3초) |
| 회상 진입 | 세피아 필터 + 슬로우 페이드 |
| 회상 복귀 | 세피아 제거 + 빠른 페이드 |
| 사망 | 붉은 비네팅 → 글리치 → 되감기 |
| 회귀 | 화면 깨짐 → 재조립 |

### 감정 이펙트
| 감정 | 이펙트 |
|------|--------|
| 긴장 | 화면 미세 진동, 심장 박동 SFX |
| 공포 | 비네팅 강화, 채도 저하, 앰비언트 사운드 |
| 슬픔 | 블루 틴트, 배경 흐려짐 |
| 분노 | 레드 틴트, 화면 경계 진동 |
| 충격 | 화면 정지 0.3초 + 줌 인 |
| 기시감 | 짧은 글리치 + 이전 씬 플래시백 (0.1초) |

## 메뉴 구조

### 메인 메뉴 (게임 시작 화면)
```
[게임 제목 로고]

  새 게임
  이어하기
  기억의 서재 (메타 플래그 열람)
  설정
```

```

│ [진실의 서가]  해금: 3/9                │
│  ✦ 아젤리아의 진실                      │
│  ✦ 카이젤의 진실                        │
│  ✦ 솔라리스의 진실                      │

│                                         │

│  ✦ 에리나                              │
│  ✦ 벨타                                │

│                                         │
│ [멜로디 컬렉션]  수집: 4/9              │
```

### 인게임 메뉴 (ESC / 햄버거)
```
  이어하기
  세이브 / 로드
  대화 로그
  기억의 서재
  설정
  메인 메뉴로
```

### 설정
```
  텍스트 속도: [느림 — 보통 — 빠름]
  자동 진행: [ON / OFF]
  자동 진행 대기: [1초 — 2초 — 3초]
  BGM 볼륨: [━━━━━━━○━━]
  SFX 볼륨: [━━━━━━━━○━]
  화면 이펙트: [ON / OFF] (접근성)
  텍스트 크기: [작게 — 보통 — 크게]
  언어: [한국어]
```

## 세이브/로드 시스템

### 슬롯 구성
- **자동 저장**: 3슬롯 (선택지 직전, 챕터 시작, 10분 간격)
- **수동 저장**: 20슬롯
- **메타 저장**: 1슬롯 (자동, 회귀 시 갱신) — 메타 플래그 전용

### 슬롯 표시 정보
```
[슬롯 12]
Ch.1 아젤리아 — "에리나와의 첫 만남"
회귀 3회차 | 2024.03.15 14:23
[스크린샷 썸네일]
```

### 회귀와 세이브의 관계
- 회귀 시 **일반 세이브는 유지** (해당 루프의 세이브)
- 메타 저장은 항상 최신 상태
- 로드 시 "이 세이브는 현재 루프와 다른 루프입니다" 경고

## 반응형 디자인

### 해상도 대응
- 기본: 1280×720 (16:9)
- 스케일 모드: FIT + 중앙 정렬
- 텍스트 박스: 화면 하단 30% 고정
- 선택지: 텍스트 박스 위에 오버레이

### 모바일 대응
- 터치 탭 = 클릭 (텍스트 진행)
- 선택지: 큰 터치 영역 (44px 최소)
- 스와이프 좌: 대화 로그
- 스와이프 우: 메뉴
- 길게 누르기: 자동 모드 토글

## 접근성

| 기능 | 설명 |
|------|------|
| 텍스트 크기 조절 | 3단계 |
| 화면 이펙트 OFF | 깜빡임/흔들림 비활성화 |
| 고대비 모드 | 텍스트 배경 불투명도 100% |
| 키보드 조작 | Enter=진행, 1/2/3=선택, ESC=메뉴 |
| 스크린 리더 | 대사/선택지 텍스트 접근 가능 |
