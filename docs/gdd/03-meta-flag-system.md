# 03. 메타 플래그 시스템

## 개요

**메타 플래그**는 회귀해도 유지되는 특수 상태 변수.  
일반 플래그는 루프 시 초기화되지만, 메타 플래그는 영구 저장되어 다회차 플레이에서 새로운 선택지/이벤트/엔딩을 해금한다.

---

## 메타 플래그 카테고리

### 1. 진실 플래그 (Truth Flags)
세계의 비밀을 발견하면 해금. 트루 엔딩 조건.

| 플래그 ID | 해금 조건 | 효과 |
|-----------|----------|------|
| `truth_azelia` | 대신관의 음모 폭로 | 2회차+ 아젤리아 시작 시 새 루트 |
| `truth_kaizer` | 황제의 수명 착취 발견 | 제국 반란 루트 해금 |
| `truth_solaris` | 정령=기둥 파편 인지 | 정령 대화에 새 선택지 |
| `truth_frosthel` | 봉인 존재의 정체 확인 | 봉인지 직접 접근 가능 |
| `truth_yonghwa` | 천명 시스템의 인위성 발견 | 천명 거부 루트 해금 |
| `truth_liberta` | 해적왕=전이자 확인 | 레온과 현대 대화 해금 |
| `truth_celestia` | 대몰락=천인의 죄 발견 | 천인 평의회 대면 이벤트 |
| `truth_kazmor` | 광맥=기둥의 뿌리 발견 | 광맥 심층 루트 해금 |
| `truth_abyssal` | 여왕=전이자 환생 인지 | 여왕 전생 기억 각성 이벤트 |

**트루 엔딩 조건**: 9개 진실 플래그 중 최소 6개 해금

---

### 2. 관계 플래그 (Bond Flags)
NPC와의 깊은 유대를 형성하면 해금.

| 플래그 ID | 해금 조건 | 효과 |
|-----------|----------|------|
| `bond_erina` | 에리나 신뢰도 MAX | 에리나 개인 엔딩 해금 |
| `bond_velta` | 벨타와 반란 공모 | 제국 반란 합류 가능 |
| `bond_naira` | 나이라의 비밀 수용 | 정령 교감 공유 이벤트 |
| `bond_kai` | 카이와 생사고락 | 봉인지 동행 가능 |
| `bond_hyeonmu` | 사부의 가르침 완수 | 천명 바깥의 힘 해금 |
| `bond_leon` | 레온과 동맹 | 다지역 항해 루트 해금 |
| `bond_ariel` | 아리엘과 진실 추적 | 대몰락 기록 열람 |
| `bond_thorg` | 토르그 대장간 수료 | 특수 무기 제작 가능 |
| `bond_abigail` | 여왕의 전생 기억 회복 | 시간의 기둥 정보 해금 |
| `bond_marco` | 마르코의 진심 대화 3회 | 세계수에 대한 직접 언급 |
| `bond_blackrobe` | 검은 로브 여인 3지역 대화 | 시간의 기둥 관리자 인지 |

---

### 3. 멜로디 플래그 (Melody Flags)
각 지역에서 발견하는 멜로디 파편.

| 플래그 ID | 지역 | 발견 장소/이벤트 |
|-----------|------|----------------|
| `melody_azelia` | 아젤리아 | 궁정 연회 참석 시 간주에서 발견 |
| `melody_kaizer` | 카이젤 | 야간 행군 중 병사가 부르는 노래 |
| `melody_solaris` | 솔라리스 | 정령 소환 의식 참관 |
| `melody_frosthel` | 프로스트헬 | 대빙하 위에서 바람 소리 경청 |
| `melody_yonghwa` | 용화국 | 내공 수련 중 내면에서 들림 |
| `melody_liberta` | 리베르타 | 레온이 술 마시며 부르는 뱃노래 |
| `melody_celestia` | 셀레스티아 | 에테르노의 기록실에서 고대 악보 발견 |
| `melody_kazmor` | 카즈모르 | 최심층 광맥에서 울려오는 진동 |
| `melody_abyssal` | 아비살 | 여왕의 자장가 |

**완성 효과**: 9개 모두 수집 → `melody_complete` 해금 → 세계수 부활/파괴 선택지

---

### 4. 회귀 플래그 (Loop Flags)
회귀 횟수에 따라 자동 해금.

| 플래그 ID | 조건 | 효과 |
|-----------|------|------|
| `loop_3` | 3회 회귀 | 선택지 텍스트에 (기시감) 표시 |
| `loop_5` | 5회 회귀 | NPC 대사 중 "전에도..." 류 선택지 등장 |
| `loop_10` | 10회 회귀 | 주인공 독백 변화, 냉소적 선택지 추가 |
| `loop_20` | 20회 회귀 | "모든 것이 익숙하다" — 스킵 가능 |
| `pillar_awakening_1` | 5회차 | 손끝 발광 이펙트 시작 |
| `pillar_awakening_2` | 10회차 | NPC 반응 "빛이 보인다" |
| `pillar_awakening_3` | 20회차 | 시간 정지 능력 (짧은) 해금 |
| `pillar_awakening_4` | 30회차 | 주인공 변질 경고 — 기둥화 진행 |

## 플래그 저장 구조

```typescript
interface MetaFlags {
  // 진실 플래그
  truths: Record<string, boolean>;
  
  // 관계 플래그
  bonds: Record<string, boolean>;
  
  // 멜로디 플래그
  melodies: Record<string, boolean>;
  
  // 루프 관련
  loopCount: number;
  pillarAwakeningLevel: number;
  
  // 방문 지역 기록 (회귀 후 직접 선택 가능)
  visitedRegions: string[];
  
  // 엔딩 해금 기록
  endingsUnlocked: string[];
}
```

## 엔딩 해금 조건

### 일반 엔딩 (지역별)
- 해당 지역 챕터 끝까지 도달 → 자동 해금
- 9개 지역 × 2~3개 엔딩 = 약 20~27개

### 히든 엔딩
| 엔딩 | 조건 |
|------|------|
| **복원 엔딩** | `melody_complete` + `truth` 6개+ + `bond_blackrobe` |
| **파괴 엔딩** | `melody_complete` + `truth` 6개+ + `loop_20` |
| **공존 엔딩** | `melody_complete` + `truth` 9개 + `bond_abigail` + `bond_marco` |
| **고독 엔딩** | `loop_30` + 관계 플래그 0개 (아무와도 유대하지 않음) |
| **관측자 엔딩** | `bond_marco` + 마르코 최종 선택에서 "함께 관찰하겠다" |

## UI 표시

- 메타 플래그 해금 시 화면 상단에 짧은 알림: `[기억의 파편을 얻었다]`
- 회귀 시 메타 플래그 목록 열람 가능 (메뉴 → 기억의 서재)
- 해금된 멜로디는 음악 재생 가능
- 진실 플래그는 실루엣 → 해금 시 텍스트 공개
