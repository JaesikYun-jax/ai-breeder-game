# 11. 크로스 지역 디펜던시 맵

**최상위 설계**: 9개 지역은 독립적으로 플레이 가능하되, 깊은 분기와 트루 엔딩에 도달하려면 **다른 지역에서 획득한 것**이 필요하다. 이것이 리플레이와 지역 탐험의 동기.

---

## 핵심 원칙

1. **각 지역 1회차는 독립 클리어 가능** — 다른 지역 경험 없이 일반 엔딩까지 도달
2. **깊은 분기(히든 루트, 관계 엔딩)는 타 지역 의존** — 최소 1~2개 다른 지역 경험 필요
3. **트루 엔딩은 광범위한 지역 탐험 필수** — 9개 중 6개+ 지역 진실 + 멜로디 9개
4. **의존은 양방향** — A가 B를 필요로 하면, B도 A에서 뭔가를 얻어야 깊어짐
5. **사이클 없는 진입 장벽** — 첫 지역 어디서든 시작 가능, 갈수록 의존이 심화

---

## 지역 간 의존 매트릭스

### 범례
- 🔑 = 영혼 귀속 아이템이 필요
- 📜 = 진실 플래그가 필요
- 💎 = 멜로디 파편이 필요
- 👤 = 특정 NPC 관계(bond)가 필요
- 🗺️ = 해당 지역 방문 기록(visited)만 있으면 됨

```
              TO (혜택을 받는 지역)
              아젤  카이젤  솔라  프로스  용화  리베  셀레  카즈  아비

아젤리아      ──    📜     🗺️    🗺️     🗺️    🗺️    📜     🗺️    📜
```

## 트루 엔딩 크리티컬 패스

### 공통 전제
- `melody_complete` (9개 멜로디 수집) → 최소 9개 지역 모두 방문
- 진실 플래그 6개+ (트루 엔딩 3종 공통)
- 진실 플래그 9개 전부 (공존 엔딩)

### 복원 엔딩 크리티컬 패스

```
[아무 지역] → truth 1개 획득
    ↓
[셀레스티아] → truth_celestia (대몰락=천인의 죄, 세계수 분할)
    ↓ + soul_celestia_archive
[프로스트헬] → truth_frosthel (봉인=관리자) + soul_guardian_fragment
    ↓
[검은로브 3지역 대화] → bond_blackrobe (관리자 인식)
    ↓
[아비살] → truth_abyssal (여왕=시간 기억자) → bond_abigail 불필요
    ↓
[나머지 지역] → 총 truth 6개 이상 + melody 9개
    ↓
→ 복원 엔딩 해금 (melody_complete + truth 6+ + bond_blackrobe)
```

**최소 루프 수 추정**: 6~8루프 (지역당 1~2루프 × 6+지역)

### 파괴 엔딩 크리티컬 패스

```
[아무 지역] → 반복 회귀 축적
    ↓ (총 20루프+)
[loop_20 자동 해금]
    ↓
[6개+ 지역] → truth 6개 이상
    ↓
[melody 9개 수집]
    ↓
→ 파괴 엔딩 해금 (melody_complete + truth 6+ + loop_20)
```

**최소 루프 수 추정**: 20루프+ (조건에 loop_20 포함)

### 공존 엔딩 크리티컬 패스 (가장 어려움)

```
[전 지역] → truth 9개 전부
    ↓
[아비살 심층] → bond_abigail (여왕 전생 기억 완전 회복)
    ↓ 필요: truth_frosthel + truth_azelia + truth_celestia
[마르코 3회 대화] → bond_marco
    ↓ 필요: 3개+ 지역 방문 + 각 지역에서 마르코 심층 대화
[melody 9개]
    ↓
→ 공존 엔딩 해금 (melody_complete + truth 9 + bond_abigail + bond_marco)
```

**최소 루프 수 추정**: 12~15루프 (모든 지역 심층 탐험 필요)

## 권장 탐험 순서 (가이드, 강제 아님)

### 초보 플레이어 자연스러운 흐름

```
1주차: [퀴즈로 배정된 지역] → 일반 엔딩 or 사망
    ↓ "다른 곳은 어떨까?"
2주차: [다른 지역] → 일반 엔딩 or 사망
    ↓ "전에 알게 된 것이 여기서 통하네?"
3~4주차: [크로스 정보로 이전 지역 재방문] → 히든 루트
    ↓ "멜로디를 모아야 하는 것 같다"
5~8주차: [남은 지역 탐험] → truth/melody 수집
    ↓ "검은 로브 여인... 모든 곳에 있었어"
9~12주차: [트루 엔딩 조건 달성 시도]
```

### 핵심 허브 지역 (다른 지역에 가장 많이 영향)

1. **셀레스티아** — 세계수 분할 정보의 원천. 가장 많은 지역에 📜/🔑 제공
2. **리베르타** — 교역 허브. 레온의 👤bond가 전 지역에 유용
3. **프로스트헬** — 관리자(검은 로브 여인) 접점. 시간의 기둥 핵심 단서

### 아젤리아 관점에서의 최소 의존 경로

```
아젤리아 일반 엔딩: 의존 없음 (독립 클리어)
아젤리아 히든 루트: 카이젤 truth + 1회 사망
아젤리아 관계 엔딩: 아비살 bond_abigail (루프 종결 약속)
아젤리아 → 트루 엔딩: 셀레스티아 archive + 프로스트헬 guardian + 6 truth
```
