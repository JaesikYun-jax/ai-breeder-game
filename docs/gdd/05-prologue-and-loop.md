# 05. 프롤로그 & 회귀 연출

---

## 프롤로그 — 현대 파트

### 설계 의도
- 1회차: 몰입감 있는 사망 → 전이 연출로 강렬한 첫인상
- 2회차+: 빠르게 스킵 가능하되, 루프 횟수에 따라 미묘한 변화를 심어둠

---

### 씬 구성

#### SCENE 01: 일상
- **배경**: 현대 한국, 대학교 캠퍼스 근처 횡단보도
- **연출**: 따뜻한 색감, 일상적인 BGM
- **텍스트**:
  > 평범한 화요일이었다.  
  > 수업이 끝나고 횡단보도 앞에 서 있었다.  
  > 신호가 바뀌었고, 나는 발을 내딛었다.

- **선택지 없음** (일상의 자동적 흐름을 표현)

#### SCENE 02: 사고
- **연출**: 갑작스러운 경적 소리(SFX), 화면 백색 플래시
- **텍스트**:
  > 옆에서 비명이 들렸다.
  > 고개를 돌렸을 때, 이미 늦었다.

- **화면**: 1초간 완전 백색 → 천천히 암전
- **SFX**: 충돌음 → 이명 → 정적

#### SCENE 03: 사이 공간 (Between)
- **배경**: 완전한 암흑. 글자만 표시
- **연출**: 타이핑 속도 매우 느림, 한 글자씩
- **텍스트**:
  > ...
  > 아프지는 않았다.
  > 다만 차가웠다.
  > 무언가가 나를 끌어당기고 있었다.

- **이펙트**: 화면 구석에서 미세한 빛 입자가 모여듦
- **텍스트 (계속)**:
  > 목소리가 들렸다. 아니, 목소리라기보다는 — 울림.
  > "네가 필요하다."
  > 누구의 목소리인지 모르겠지만, 거부할 수 없었다.

- **복선**: 이 목소리 = 세계수의 의지. 트루 엔딩에서 같은 목소리가 재등장

#### SCENE 04: 전이
- **연출**: 화면 중앙에서 빛이 폭발 → 다양한 색의 파편이 흩어짐
- **텍스트**: 없음. 순수 비주얼 연출 3~5초
- **음악**: 아르카디아 메인 테마 첫 소절
- **전환**: 성격 퀴즈로 이동

---

### 2회차+ 프롤로그 변화

#### 3회차
- 사이 공간 텍스트 추가:
  > ...이 감각은 처음이 아닌 것 같다.

#### 5회차
- 사고 씬 텍스트 변화:
  > 고개를 돌렸을 때, 이미 늦었다.  
  > (...이번에도.)

#### 10회차
- 사이 공간에서 선택지 등장:
  > A: "또야..." (회귀에 지침)
  > B: "...이번에는 다르게." (결의)
  > C: [침묵] (감정 없음)
- 각 선택에 따라 초기 FAT 미세 보정

#### 20회차+
- 프롤로그 완전 스킵 옵션 제공
- 스킵 시 특수 텍스트:
  > [기억이 되감긴다. 이제는 익숙하다.]
- 스킵하지 않으면 새로운 대사:
  > "네가 필요하다" 대신 "...미안하다."
  - **복선**: 세계수가 주인공을 이용하고 있다는 죄책감

## 성격 퀴즈 연출

### 배경
- 사이 공간이 서서히 밝아지며, 추상적인 풍경이 형성
- 각 질문마다 배경이 미세하게 변화 (선택에 반응하는 느낌)

### 퀴즈 UI
- 화면 중앙: 질문 텍스트 (나레이션 형식)
- 하단: 3개 선택지 (카드 형태로 나열)
- 선택 시: 카드가 빛나며 흡수되는 이펙트

### 퀴즈 종료 후 전이 연출
- 5개 답변 종료 → 능력치 계산 → 진입점 결정
- 결정된 지역에 따라 전이 연출 분기:
  - **아젤리아**: 빛 기둥 속으로 떨어짐 → 왕궁 홀에 착지
  - **카이젤**: 어둠 속 쇳소리 → 빈민가 골목에서 깨어남
  - **솔라리스**: 뜨거운 바람 → 모래 위에 쓰러져 있음
  - **프로스트헬**: 극심한 한기 → 설원에서 동사 직전
  - **용화국**: 안개 속 산 → 산길에서 기억 없이 방황
  - **리베르타**: 파도 소리 → 난파선 잔해 위에서 깨어남
  - **셀레스티아**: 상승 기류 → 구름 위 섬에 떨어짐
  - **카즈모르**: 지진 → 갱도 속에 매몰
  - **아비살**: 물속으로 가라앉음 → 빛나는 손에 이끌려 깨어남

## 사망 → 회귀 연출

### 사망 시퀀스 (공통)

#### STAGE 1: 사망 직전 (1~2초)
- 화면 가장자리부터 붉은 비네팅
- BGM 왜곡 (피치 다운)
- 사망 원인에 따른 효과음

#### STAGE 2: 정지 (0.5초)
- 화면 완전 정지 (프리즈 프레임)
- 모든 소리 멈춤
- 짧은 정적

#### STAGE 3: 깨짐 (2~3초)
- 화면이 유리처럼 금이 감
- 금 사이로 빛이 새어나옴
- 글리치 이펙트 (화면 찢어짐)

#### STAGE 4: 되감기 (3~5초)
- 빠르게 역재생되는 느낌으로 장면들이 스쳐감
- 플레이했던 장면들의 스냅샷이 거꾸로 흘러감
- 점점 빨라지다가 —

#### STAGE 5: 사이 공간 (Between) 재방문
- 다시 암흑
- 텍스트:
  > 다시, 여기.
  > [회귀 횟수]번째.
- 메타 플래그 획득 알림 (있을 경우):
  > [기억의 파편을 얻었다]
  > "대신관의 진실" — 빛의 이면에 숨겨진 것

#### STAGE 6: 선택
- 선택지:
  > A: 같은 세계로 돌아간다 (같은 진입점)
  > B: 다른 길을 찾는다 (퀴즈 재시작)
  > C: [해금된 지역 직접 선택] (방문한 적 있는 지역만)

---

### 사망 유형별 특수 연출

| 사망 유형 | 추가 연출 |
|----------|----------|
| **전투 사망** | 무기 소리 반향 → 붉은 플래시 |
| **함정 사망** | 바닥이 무너지는 느낌 → 낙하 |
| **배신 사망** | 뒤에서 칼날 소리 → 배신자의 얼굴 잔상 |
| **희생 사망** | 따뜻한 빛 → 서서히 소멸 (아름답지만 슬픈) |
| **사고 사망** | 현대 교통사고와 겹치는 데자뷰 연출 |

---

### 회귀 횟수별 사이 공간 변화

| 회차 | 사이 공간 변화 |
|------|---------------|
| 1~3회 | 완전한 암흑, 목소리 들림 |
| 5회 | 암흑 속에 미세한 빛 입자가 떠다님 |
| 10회 | 빛 입자가 나무 형태로 모이기 시작 (세계수 실루엣) |
| 15회 | 세계수 실루엣이 점점 선명, 뿌리에서 빛이 흐름 |
| 20회 | 세계수가 완전히 보임, 주인공의 손에서 같은 빛 |
| 25회 | 세계수와 주인공이 실로 연결된 것이 보임 |
| 30회 | 세계수가 갈라지며 주인공을 흡수하려 함 → 위기감 |

## 프롤로그 스킵 정책

| 상황 | 스킵 가능 여부 |
|------|---------------|
| 1회차 프롤로그 | **불가** (첫 경험 보장) |
| 2~4회차 프롤로그 | 사고 씬까지만 스킵 가능 |
| 5~19회차 | 사이 공간까지 스킵 가능 |
| 20회차+ | 완전 스킵 가능 (퀴즈 선택 화면 직행) |
| 퀴즈 (2회차+) | 스킵 가능 → 직접 지역 선택으로 |
