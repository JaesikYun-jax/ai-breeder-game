# 08. 사운드 & 음악 기획

---

## 음악 설계 원칙

1. **지역 = 악기**: 각 지역마다 고유 악기/음색 할당
2. **멜로디 복선**: 10번째 기둥의 멜로디가 각 지역 BGM에 변형되어 숨겨져 있음
3. **회귀 반영**: 같은 BGM이 루프 횟수에 따라 미세하게 변조
4. **감정 레이어**: 상황에 따라 BGM에 레이어 추가/제거

## 10번째 기둥의 멜로디 (숨겨진 테마)

### 원곡 구조
- 9개 음으로 구성 (각 지역 기둥의 음)
- 간단하지만 강렬한 선율
- 이름: "세계수의 자장가"

### 지역별 변형 (숨겨진 삽입)
| 지역 | 삽입 위치 | 변형 방식 |
|------|----------|----------|
| 아젤리아 | 궁정 연회 BGM 간주 (3:20~) | 하프 아르페지오로 변형 |
| 카이젤 | 야간 행군 BGM 중반 | 트럼펫 솔로의 일부 |
| 솔라리스 | 정령 소환 의식 BGM | 정령 보이스의 허밍 |
| 프로스트헬 | 대빙하 탐색 앰비언트 | 바람 소리에 실린 자연 음 |
| 용화국 | 내공 수련 BGM 명상 파트 | 소(簫) 솔로 |
| 리베르타 | 레온의 뱃노래 후렴 | 아코디언+보컬 |
| 셀레스티아 | 에테르노 기록실 BGM | 합창의 언더톤 |
| 카즈모르 | 최심층 광맥 앰비언트 | 금속 진동음의 배음 |
| 아비살 | 여왕의 자장가 | 직접적 보컬 (가장 원곡에 가까움) |

### 완성 시 (9개 파편 수집 후)
- 메뉴에서 "세계수의 자장가" 풀버전 재생 가능
- 트루 엔딩 시퀀스에서 BGM으로 사용

## 상황별 BGM 운용

### 레이어 시스템
기본 BGM에 상황별 레이어를 얹어 동적 사운드스케이프 구현

| 레이어 | 트리거 | 효과 |
|--------|--------|------|
| Base | 지역 진입 | 지역 메인 테마 |
| Tension | 긴장 씬 | 저음 드론 추가, 하이햇 |
| Combat | 전투 진입 | 퍼커션 레이어, BPM 상승 |
| Emotional | 감정 씬 | 피아노/스트링 레이어, 볼륨 다운 |
| Discovery | 복선 발견 | 글로켄슈필 + 10번째 멜로디 한 소절 |
| Death | 사망 직전 | 역재생 + 피치 다운 + 페이드 |
| Meta | 메타 선택지 | 글리치 사운드 + 사이 공간 BGM 잔향 |

### 전투 음악
- 일반 전투: 지역 BGM의 고속/공격적 변형
- 보스전: 고유 보스 테마
- 사망 직전: 역재생 감속 → 정지

## 회귀 횟수별 음악 변조

| 루프 | 변화 |
|------|------|
| 1회차 | 원본 BGM |
| 3회차 | 미세한 리버브 추가 (몽환적) |
| 5회차 | 일부 악기 빠짐 (빈 공간 생김) |
| 10회차 | 반음 디튜닝 (미세한 불협화음) |
| 15회차 | 사이 공간 BGM의 요소가 모든 지역에 섞임 |
| 20회차+ | 모든 BGM에 시계 소리가 미세하게 깔림 |
| 30회차 | BGM 자체가 느려지고 왜곡됨 (주인공 변질 반영) |

## 음성

### 1단계 (MVP)
- 음성 없음, 텍스트 전용
- SFX와 BGM으로 분위기 전달

### 2단계 (확장)
- 핵심 대사만 음성 (감정적 씬, 사망 씬)
- NPC별 보이스 톤 설정 (짧은 감탄사 정도: "흠", "아!", "...")
- 10번째 기둥 멜로디의 보컬 버전
