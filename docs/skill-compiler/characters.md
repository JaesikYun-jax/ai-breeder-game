## 주인공: 안강해 (安剛海)

| 항목 | 내용 |
|---|---|
| 의미 | "강한 바다처럼 견뎌라" — 부모가 아픈 아이에게 지어준 이름 |
| 나이 | 28세 |
| 직업 | 백엔드 개발자 (4년차, 중소 SI 업체) |
| 외형 | 마른 체형, 다크서클, 후드 즐겨 입음, 사람과 눈 안 마주침 |
| 트라우마 | 어린 시절 정신병원 입원 이력. 가족과 거리. 친구 없음. 타인 부정 감정 누수로 사회 회피형 |
| 표면 능력 | F급 각성자. 공식 스킬 3개 모두 D급 이하 |
| 진짜 능력 | 오리지널 상태창 v0.1 (베타). 코드 편집 가능. 본인은 모르는 백도어 권한 |
| 욕망 | 초기 "그냥 평범하게 살고 싶다" → 후기 "이 시스템의 진실을 알고 싶다" |
| 약점 | 타인 감정 누수. 군중 속 두통. 강한 감정엔 일시 의식 끊김 |
| 시그니처 톤 | 자조 + 냉소 + 우월감 대비 (브리프 §10) |

### 안강해 — 1부 성장 아크 (시드)

- EP001~005: 무능력자 → "수정 가능" 자각 → 첫 사이다
- EP006~020: 솔플 먼치킨 + 길드 영입 거절 + 메인 히로인 첫 만남
- EP021~050: 사기 콤보 + 길드 전쟁 + 협회 최상위 인증
- EP051~080: 이상 던전 → 시스템 정체 자각 → 1차 결심

## 적대자

### 1층 — 인간 (1부 ~ EP080)

- 5대 길드 마스터 중 일부 (1~2명 얼굴 빌런 — 큰 설계에서 명명)
- 약자 착취·던전 독점·신규 각성자 갈취 → 사이다 동력원

### 2층 — 시스템 (EP050~)

- 상태창 = 의념 탈수기. 게이트·던전·레이드 = 농장 사이클
- 강한 감정일수록 양분 큼 → 사이다와 고난 정교 배치
- 강해는 베타 테스터로 코드 직접 본 유일한 존재

### 0층 — 시스템 설계자 (라스트, 떡밥만)

- 신? AI? 더 상위 차원? — 후반 공개

## Dialogue DNA (스켈레톤)

> `/design-big` 단계에서 캐릭터별 말투·시그니처 어휘·금지 표현 정식 정의.

### 안강해 (시드)
- **시그니처 어휘**: "...뭐야 이거", "...별 거 아니네", "...아 진짜"
- **자주 쓰는 IT 용어**: 함수, 변수, 컴파일, 디버그, 런타임, 주석
- **금지 어휘**: 영웅적 결의 / 사명감 / "지켜내겠다" / "구하겠다" 류
- **속마음과 발화의 괴리**: 속으로는 다 보이지만 입으로는 무뚝뚝/회피
