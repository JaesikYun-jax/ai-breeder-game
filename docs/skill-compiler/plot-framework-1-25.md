# 플롯 프레임워크 — EP001~EP025 (1막: 도입 + 1차 갈등)

> 큰 설계(`/design-big`) 단계에서 본격 작성. 현재는 브리프 §8 비트 시드 보존.
> 작은 설계(`/design-small`) 단계에서 EP별 4씬 구조로 분해 → 절단신공 위치까지 명시.

---

## 매크로 골격 (브리프 §8)

```
EP001~005 ──── 도입부 (후크 + 각성 + 첫 사이다)
EP006~020 ──── 1차 갈등 (무시받던 자의 두각)
EP021~025 ──── 1차 절정 진입 (길드 전쟁 시작 + 히로인 합류)
EP026~050 ──── 1차 절정 본격 (다음 framework 작성 예정)
EP051~080 ──── 1차 클라이맥스 (시스템 정체 자각 + 1부 결심)
```

## 1. EP001~005 — 후크 + 각성 + 첫 사이다 (도입부)

> **의무**: 1화 안에 후크 박힘 / 5화 안에 각성 + 첫 사이다.
> **분량**: EP당 5,000~7,000자 / 절단신공 우선.

### EP001 — 모두에게 상태창이 열린 날
- 대각성 1년 후. 강해만 각성 못한 일상
- 회사·사회에서 무능력자 표면
- 후크: "모두에게 상태창이 열린 날, 나는 그게 새삼스럽지 않았다." (브리프 부록)
- 절단 위치: TBD (큰 설계 후 비트 시트로 분해)

### EP002 — F급 인증 (또)
- 헌터 협회 등급 측정 또 F급
- 상태창은 보이지만 능력 안 깸
- 측정관·주변 반응 + 강해 무덤덤
- 절단 위치: TBD

### EP003 — "수정 가능하잖아?"
- 길드 갈취 사건 목격
- 본인 스킬 코드 보다가 깨달음
- 손이 떨림
- 절단 위치: 깨달음 직후 (강력 후크)

### EP004 — 첫 수정
- 조심스레 D급 스킬 한 줄 수정 → 위력 2배
- 시뮬레이션·테스트
- 절단 위치: 첫 적용 직전

### EP005 — 첫 사이다
- 첫 던전 솔플
- 약한 스킬 조합으로 자기 등급 위 클리어
- **첫 사이다** — 사이다 사이클 정착
- 절단 위치: 클리어 직후 협회 보고

## 2. EP006~020 — 1차 갈등 (무시받던 자의 두각)

### EP006~010 — 솔플 반복 + 등급 갱신
- 솔플로 던전 클리어 반복
- 등급 인증 갱신 (F→E→D 단계적)
- 주변 충격, SNS·기사 첫 노출
- 헌터 협회 분석관 이도현 등장 (FS-010)

### EP011~014 — 5대 길드 영입 시도 → 거절 → 적대
- 길드 마스터의 거만한 영입 → 강해의 거절
- 1~2개 길드 명확한 적대 노선 (FS-007 활성)
- 갈취 사건의 배후가 그 중 1곳임이 드러남

### EP015~020 — 메인 히로인 첫 등장
- 던전 사고 / 협회 의뢰 / 임시 파티 등 트리거 신
- 그녀의 단일 강력 스킬 코드를 강해가 봄 → 응용법 알려줌
- 충격 → 따라옴 (FS-003 미세 이질감 배치)
- EP020 마무리: 첫 콤보 합 성공

---

## 3. EP021~025 — 1차 절정 진입

### EP021~023 — 길드 전쟁 시작
- 적대 길드의 직접 공격
- 강해 + 히로인 첫 합동 작전
- 사기 콤보 본격 가동

### EP024~025 — 협회 최상위 인증 직전
- 등급 인증 갱신 — A급 또는 B급 (큰 설계에서 확정)
- 협회 분석관 이도현이 강해 능력의 학문적 이질감 감지 (FS-008·FS-009 활성)
- EP025 클로징: 협회 내부 음모 떡밥 (FS-007 후속)
