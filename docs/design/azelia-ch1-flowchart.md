# 아젤리아 왕국 — 챕터 1 분기 플로우차트

## 타이틀: "소환된 빛"

---

## 분기 구조도

```
[001: 소환의 홀] ← 프롤로그에서 전이
    │
    ├─ A: 에리나의 손을 잡는다 (CHA+1, 에리나 호감+5)
    │   └→ [002a: 왕좌의 알현]
    │
    ├─ B: 주변을 먼저 살핀다 (PER+1)
    │   └→ [002b: 관찰자의 시선]
    │
    └─ C: "여기가 어디지?" (INT+1)
        └→ [002c: 질문하는 자]
        
    ★ 메타 선택지 [truth_azelia]: "에리나, 대신관을 조심해."
        └→ [002d: 예언자의 경고] (에리나 호감+15, 특수 분기)

─── 002a/b/c 합류 ───

[003: 용사의 방]
    │ 에리나가 상황 설명, 마르코 첫 등장
    │
    ├─ A: 에리나에게 더 물어본다 (CHA+1, 에리나 호감+3)
    │   └→ [004a: 공주의 속마음]
    │
    ├─ B: 마르코에게 말을 건다 (PER+1)
    │   └→ [004b: 상인의 힌트] (마르코 "또 만났군" 복선)
    │
    └─ C: 혼자 방을 탐색한다 (INT+1)
        └→ [004c: 왕궁의 비밀] (도서관 단서 발견)

─── 004a/b/c 합류 ───

[005: 대신관의 축복]
    │ 카르디날 모르데우스 등장, "축복 의식" 제안
    │
    ├─ A: 축복을 받는다 (FAT+2, 성광 감응)
    │   └→ [006a: 빛의 세례] (능력 해금, 하지만 기둥 연결 강화)
    │
    ├─ B: 정중히 거절한다 (INT+1, CHA+1)
    │   └→ [006b: 불신의 씨앗] (모르데우스 경계 시작)
    │
    └─ C: "그 전에 몇 가지 물어볼 게 있습니다" (INT+2)
        └→ [006c: 의심의 시선] (모르데우스의 미묘한 불쾌감)
    
    ★ 메타 선택지 [bond_blackrobe]: 검은 로브 여인(사서)의 시선을 느낌
        └→ [006d: 도서관의 그림자] (목걸이 복선 심기)

─── 006 합류 ───

[007: 첫 밤 — 왕궁 탐색]
    │ 자유 탐색 파트
    │
    ├─ [007-A: 정원] → 에리나와 대화 (관계 심화)
    │   └ 에리나가 아버지(왕)의 병에 대해 암시
    │
    ├─ [007-B: 도서관] → 검은 로브 여인(사서) 만남
    │   └ "아르카디아의 역사"에서 10번째 기둥 암시 발견
    │
    ├─ [007-C: 병사 막사] → 왕국 기사들과 대화
    │   └ 제국과의 긴장 관계 정보 입수
    │
    └─ [007-D: 왕의 침실 근처] (PER 4 이상만)
        └ 대신관이 왕에게 "약"을 주는 장면 목격
        → 플래그: king_medicine_seen

─── 탐색 후 ───

[010: 마왕 토벌 선언]
    │ 다음 날, 대신관이 마왕 토벌 명령을 공표
    │
    ├─ A: 영웅적으로 수락 (CHA+2, 민중 지지↑)
    │   └→ [011a: 출정 준비] (정통 용사 루트)
    │
    ├─ B: 더 많은 정보를 요구 (INT+2)
    │   └→ [011b: 의심의 용사] (조사 루트)
    │
    └─ C: "혼자서는 무리인데" (PER+1, CHA+1)
        └→ [011c: 동료 모집] (파티 구성 루트)
    
    ★ 메타 선택지 [king_medicine_seen]: "대신관, 왕의 약은 뭡니까?"
        └→ [011d: 정면 대결] (모르데우스 적대 루트 조기 돌입)

═══ 중반부 분기 (동시 진행 가능 이벤트) ═══

[020: 근교 마을 사건]
    │ 마을에서 괴물 출현 → 첫 전투
    │
    ├─ 전투 승리 (STR or INT 5+)
    │   └→ [021: 승리의 용사] → 민중 신뢰↑
    │
    ├─ 고전 끝 승리 (STR/INT 낮음)
    │   └→ [022: 상처투성이] → 에리나 간호 이벤트
    │
    └─ 패배 → [사망 이벤트 #1]
        └→ 첫 회귀 트리거
        → 사망 원인: 전투 미숙
        → 획득 메타 플래그: 없음 (첫 사망)

[030: 에리나의 비밀]
    │ 조건: 에리나 호감 15+
    │
    ├─ A: "왕의 병, 알고 있어?" (INT+1)
    │   └→ 에리나가 대신관의 독살 의심 고백
    │   → 플래그: erina_confession
    │
    ├─ B: "힘들면 말해" (CHA+2, 에리나 호감+10)
    │   └→ 감정적 교류, 비밀은 아직 안 나옴
    │
    └─ C: 그냥 같이 있어준다 (FAT+1, 에리나 호감+5)
        └→ 침묵 속의 위로

═══ 후반부 (챕터 1 클라이맥스) ═══

[040: 제국 침공 소식]
    │ 카이젤 제국이 변경을 침공했다는 급보
    │
    ├─ A: 전장에 나간다 (STR+2)
    │   └→ [041: 전장의 용사]
    │       ├─ 승리 (STR 8+) → 영웅으로 귀환
    │       └─ 패배 → [사망 이벤트 #2: 전사]
    │           → 메타 플래그: imperial_encounter
    │
    ├─ B: 대신관의 반응을 관찰한다 (PER+2, INT+1)
    │   └→ [042: 그림자의 움직임]
    │       └ 모르데우스가 제국에 밀서를 보내는 장면 목격 가능
    │       → 플래그: mordeus_spy_seen (PER 7+ 필요)
    │
    └─ C: 에리나를 보호한다 (CHA+1, 에리나 호감+10)
        └→ [043: 궁전 방어전]
            ├─ 성공 → 에리나 신뢰 최대
            └─ 실패 → [사망 이벤트 #3: 궁전 함락]
                → 메타 플래그: palace_fall

[050: 챕터 1 결말 분기점]
    │ 축적된 플래그에 따라 분기
    │
    ├─ [루트 A: 빛의 용사] (erina_confession 없음, mordeus_spy 없음)
    │   → 대신관의 축복 아래 마왕 토벌 출정
    │   → 일반 엔딩(NE-01)으로 향하는 정통 루트
    │   → 챕터 2: 마왕 토벌 여정
    │
    ├─ [루트 B: 진실의 추적자] (erina_confession 있음 or mordeus_spy 있음)
    │   → 마왕 토벌을 핑계로 왕국 밖 조사
    │   → 대신관의 진실에 접근하는 루트
    │   → 챕터 2: 왕국의 그림자
    │
    ├─ [루트 C: 반역의 용사] (메타: truth_azelia)
    │   → 이미 진실을 아는 2회차+. 대신관을 직접 대면
    │   → 가장 빠르지만 위험한 루트
    │   → 챕터 2: 왕좌의 전쟁
    │
    └─ [사망 → 회귀]
        │ 어떤 사망이든 여기로
        └→ 프롤로그 → 퀴즈(또는 스킵) → 재시작

═══ 사망 이벤트 상세 ═══

### 사망 #1: 첫 전투 패배
- **상황**: 근교 마을 괴물에게 패배
- **연출**: 괴물의 일격 → 붉은 화면 → "아직 약하다"는 깨달음
- **감정**: 충격과 당혹 (첫 사망은 항상 충격적으로)
- **대사**: "이것으로... 끝인 건가..."
- **회귀 후 효과**: 전투 관련 선택지에 [기시감] 표시

### 사망 #2: 전장에서 전사
- **상황**: 제국군과의 전투에서 압도당함
- **연출**: 혼전 속 쓰러짐 → 카이젤 병사의 얼굴 → 암전
- **감정**: 무력감, 제국의 강대함 인식
- **대사**: "다음에는... 더 강해져서..."
- **회귀 후 효과**: 제국 관련 대화에서 새 선택지 (imperial_encounter)

### 사망 #3: 궁전 함락
- **상황**: 궁전 방어 실패, 에리나 눈앞에서 쓰러짐
- **연출**: 에리나의 비명 → 주인공의 손에서 피 → 에리나 얼굴 잔상
- **감정**: 분노와 자책, 에리나를 지키지 못한 것에 대한 후회
- **대사 (에리나)**: "안 돼...! 용사님...!"
- **회귀 후 효과**: 에리나 관련 선택지에 강한 감정적 반응 추가
