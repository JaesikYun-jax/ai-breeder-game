# 카이젤 제국 — 챕터 1 분기 플로우차트

## 타이틀: "철의 그림자"

---

## 분기 구조도

```
[001: 빈민가의 아침] ← 프롤로그에서 전이
    │ 제국 변경 빈민가에서 깨어남. 기억은 있으나 아무것도 없음.
    │
    ├─ A: 주변 사람에게 도움을 청한다 (CHA+1)
    │   └→ [002a: 빈민가의 인정] — 노인이 빵 나눠줌
    │
    ├─ B: 일자리를 찾아 거리를 돌아다닌다 (STR+1)
    │   └→ [002b: 노동자의 하루] — 공장 일용직
    │
    └─ C: 빈민가 구석에서 상황을 파악한다 (INT+1, PER+1)
        └→ [002c: 관찰과 적응] — 징병 포스터 발견

─── 002 합류 ───

[003: 징병]
    │ 제국군 징집관이 빈민가 소탕. 도망치든 잡히든 결국 징집.
    │
    ├─ A: 순순히 따라간다 (FAT+1) → 벨타 소대에 즉시 배속
    │
    ├─ B: 도망치려 한다 (STR+1, PER+1) → 추격전 후 체포, 벌칙부대 배속
    │   └→ [004b: 벌칙부대] (하드 모드, 하지만 숨겨진 동료 만남)
    │
    └─ C: "자원입대 하겠습니다" (CHA+2) → 벨타 소대 배속 + 소대장 첫인상 좋음
    
    ★ 메타 선택지 [truth_kaizer]: "황제의 수명 연장에 대해 알고 있다"
        └→ [004d: 위험한 선언] (즉시 감시 대상, 특수 루트)

─── 병영 합류 ───

[005: 벨타 소대]
    │ 소대장 벨타와의 첫 만남
    │ "이름?" — "... (주인공 이름)" — "쓸모를 증명해."
    │
    │ 마르코 등장: 군 물자 납품상으로 위장
    │ "또 만났군." (복선)
    │
    ├─ A: 훈련에 전념한다 (STR+2)
    │   └→ [006a: 유망한 신병]
    │
    ├─ B: 동료 병사들과 친해진다 (CHA+2)
    │   └→ [006b: 소대의 유대]
    │       └ 동료 NPC 소개: 갈드(덩치큰 낙천가), 민(소년병, 겁쟁이)
    │
    └─ C: 군 시스템과 마기공학을 연구한다 (INT+2)
        └→ [006c: 기술자의 자질]
            └ 마기공학 기초 습득, 벨타의 관심 획득

[008: 첫 실전 — 변경 순찰]
    │ 소대 단위 변경 순찰. 프로스트헬 유민 무장단과 조우.
    │
    ├─ A: 교전 명령에 따른다 (STR+1)
    │   └→ [009a: 명령에 따른 전투]
    │       ├─ 승리 → 승진 기회
    │       └─ 패배 → [사망 이벤트 #1: 첫 전투]
    │           → 메타 플래그: first_battle_kaizer
    │
    ├─ B: 유민들과 대화를 시도한다 (CHA+2)
    │   └→ [009b: 대화의 시도]
    │       ├─ 성공 (CHA 5+) → 유민 정보 획득: "제국이 마을을 비웠다"
    │       │   → 플래그: refugee_testimony
    │       └─ 실패 → 전투 돌입 (불리한 조건)
    │
    └─ C: 상황을 관찰하며 벨타에게 보고한다 (PER+1, INT+1)
        └→ [009c: 전술적 판단]
            └ 벨타의 신임 획득

    ★ 메타 선택지 [imperial_encounter]: "이 유민들, 제국에 의해 쫓겨난 거야"
        └→ [009d: 진실의 증인] (유민과 연대 루트)

─── 순찰 후 ───

[012: 마기공학 도시 견학]
    │ 소대가 제국 수도 인근 마기공학 도시로 이동
    │ 거대한 공장, 마석 정제소, 군수 시설
    │
    ├─ [012-A: 마석 정제소 견학]
    │   └ 마석이 어디서 오는지 (카즈모르 채굴) 정보
    │   → PER 6+: "이 마석... 뭔가 살아있는 느낌이"
    │
    ├─ [012-B: 군사 연구소]
    │   └ 마기공학 무기 시연, 생체 인터페이스 기술
    │   → INT 6+: "이 에너지 패턴, 아젤리아의 성광과 비슷한데?"
    │
    ├─ [012-C: 빈민가 통행]
    │   └ 도시 빈민가에서 "징발된 가족"의 흔적 발견
    │   → 플래그: city_poor_evidence
    │
    └─ [012-D: 벨타의 사적 시간] (벨타 호감 10+)
        └ 벨타가 동생 초상화를 꺼내보며 독백하는 장면 목격
        → 플래그: velta_sister_portrait

[015: 수상한 의식]
    │ 조건: 특정 플래그 2개 이상 보유
    │
    │ 밤중에 제국 수도 지하에서 이상한 의식 목격
    │ 수십 명의 사람이 기계에 연결되어 있고, 에너지가 추출되고 있음
    │
    ├─ A: 깊이 잠입 조사한다 (PER+2, 위험)
    │   └→ [016a: 수명 변환로]
    │       ├─ 성공 (PER 7+) → 황제의 수명 연장 메커니즘 발견
    │       │   → 메타 플래그: truth_kaizer (!!!)
    │       └─ 발각 → [사망 이벤트 #2: 처형]
    │           → 메타 플래그: truth_kaizer (사망해도 진실은 기억)
    │
    ├─ B: 벨타에게 보고한다 (CHA+1)
    │   └→ [016b: 벨타의 결심]
    │       └ 벨타가 동생의 죽음과 연결 → 반란 의지
    │       → 플래그: velta_rebellion_seed
    │
    └─ C: 아무 것도 안 본 척한다 (FAT+1)
        └→ [016c: 외면의 대가]
            └ 이후 민(소년병)이 "징발" 대상으로 선정되는 이벤트

[020: 사라지는 마을]
    │ 소대에 명령: 변경 마을 "이주 지원"
    │ 실제로는 마을 주민 전원을 수도로 "호송" (수명 변환로 재료)
    │
    ├─ A: 명령에 따른다 (무거운 마음으로)
    │   └→ [021a: 군인의 의무]
    │       └ 마을 주민들의 절규 → 동료 갈드가 "이거 아닌 거 같은데..."
    │       → 플래그: duty_followed
    │
    ├─ B: 마을 사람들을 도망치게 한다 (CHA+2, STR+1)
    │   └→ [021b: 반역의 시작]
    │       ├─ 성공 (CHA+STR 10+) → 마을 사람 일부 탈출
    │       │   → 메타 플래그: village_saved
    │       └─ 실패 → [사망 이벤트 #3: 군법 처형]
    │           → 메타 플래그: village_saved (실패해도 시도의 기억)
    │
    └─ C: 증거를 수집한다 (INT+2, PER+1)
        └→ [021c: 기록자]
            └ 명령서, 호송 루트, 수명 변환로 관련 문서 확보
            → 플래그: evidence_collected

    ★ 메타 선택지 [truth_kaizer]: "이 마을 사람들은 황제의 먹이가 된다"
        └→ [021d: 폭로] (소대원 전원에게 진실 고백 → 벨타 합류 or 적대)

[025: 챕터 1 결말 분기점]
    │
    ├─ [루트 A: 충성의 병사] (duty_followed, 진실 모름)
    │   → 승진하지만 비밀을 모른 채 더 깊이 관여
    │   → 챕터 2: 제국의 톱니바퀴
    │
    ├─ [루트 B: 의심하는 병사] (evidence 일부 보유)
    │   → 벨타와 함께 제국 내부 조사
    │   → 챕터 2: 철의 이면
    │
    ├─ [루트 C: 반역자] (village_saved or velta_rebellion_seed)
    │   → 제국 탈출, 반란 세력 합류
    │   → 챕터 2: 제국의 반역자
    │
    └─ [루트 D: 진실의 전파자] (메타: truth_kaizer + 2회차+)
        → 이미 모든 것을 아는 상태에서 최적 루트
        → 챕터 2: 황제에 대항하여

═══ 사망 이벤트 상세 ═══

### 사망 #1: 첫 전투 패배
- **상황**: 프로스트헬 유민과의 교전에서 패배
- **연출**: 적의 화살/검에 맞음 → 눈(雪) 위에 쓰러짐
- **감정**: "이런 곳에서 이렇게 끝나는 건가"
- **대사**: "...춥다."
- **회귀 후 효과**: 전투 선택지에 [기시감], 유민 관련 정보 파편

### 사망 #2: 수명 변환로 발각
- **상황**: 잠입 발각 → 즉결 처형
- **연출**: 기계에서 나오는 에너지가 주인공을 관통
- **감정**: 진실을 알았으나 아무것도 못 한 분노
- **대사**: "...다음엔 반드시..."
- **회귀 후 효과**: truth_kaizer 해금 + 수명 변환로 위치 기억

### 사망 #3: 군법 처형
- **상황**: 마을 사람 탈출 실패 후 반역죄로 처형
- **연출**: 총살형 → 벨타의 얼굴 잔상 (그녀가 당신을 살리지 못한 표정)
- **감정**: 자책하지만 후회는 없음
- **대사 (벨타)**: "...바보 같은."
- **회귀 후 효과**: village_saved 플래그 + 벨타 관련 강화 선택지

## 카이젤 챕터 1 핵심 감정선

```
무기력 (빈민) → 소속감 (군대) → 동료애 (소대)
    → 의문 (시스템의 균열) → 분노 (진실 발견) → 결의 (행동 선택)
```

챕터 1 플레이타임 예상: 25~35분 (1회 통과)
