# 사망 & 회귀 서사 규칙

> 이 소설의 핵심 메커닉은 "죽고 다시 태어난다"가 아니라
> **"죽는 것이 얼마나 끔찍한지, 그럼에도 다시 깨어나야 한다"**이다.
> 사망은 파워업 이벤트가 아니다. 트라우마다.

---

## 1. 사망 유형 카탈로그

### 확정된 사망 (1-5화 기준)

| # | 장소 | 방식 | 속도 | 고통 수준 | 서사적 의미 |
|---|------|------|------|----------|-----------|
| 0 | 서울 횡단보도 | 트럭 충돌 | 즉사급 (~30초) | ★★★★☆ | 이세계 진입 — "소설 같은 일이 실제로" |
| 1 | 아젤리아 대성당 | 축복 의식 (영혼 추출) | 느림 (수분) | ★★★★★ | 신뢰의 배신 — "보호자가 살인자" |

### 예정된 사망 (지역별)

| 지역 | 예상 방식 (설정 기반) | 서사적 의미 |
|------|---------------------|-----------|
| 솔라리스 | 사막 환경/태양폭풍/정령 교감 대가 | 자연의 무관심 — "세계는 너를 신경 쓰지 않는다" |
| 카이젤 | 전장/처형/생명력 수확 | 시스템의 폭력 — "개인은 국가에 소모품" |
| 프로스트헬 | 동사/봉인 역류/극한 환경 | 고독과 인내의 한계 |
| 용화국 | 수련 실패/천명 역류/무투 | 운명에 대한 저항의 대가 |
| 리베르타 | 해적 전투/난파/배신 | 자유의 대가 |
| 셀레스티아 | 추락/이단 심판/금기 마법 역류 | 진실을 아는 대가 |
| 카즈모르 | 낙반/살아있는 금속 폭주/갱도 붕괴 | 탐욕의 결과 |
| 아비살 | 익사/심해 압력/심연의 존재 | 가장 깊은 곳에서의 죽음 |

### 사망 방식 규칙
- **같은 방식으로 두 번 죽지 않는다**: 각 사망은 고유한 감각 경험
- **지역의 본질이 사망 방식에 반영**: 빛의 왕국에서는 빛에 의해, 얼음의 땅에서는 추위에 의해
- **사망이 단서를 남긴다**: 죽는 순간 보이는 것(기둥 내부, 균열 등)이 다음 아크의 열쇠

## 2. 사망 묘사 프로토콜

### 6단계 구조 (5화에서 확립)

```
1단계: 전조
   - 환경이 변한다 (공기 밀도, 온도, 빛)
   - 지호의 분석 뇌가 마지막으로 작동
   - 예: "발이 문양 위를 밟는 순간, 발바닥에서 진동이 올라왔다."

2단계: 발동
   - 고통의 시작 — 처음에는 약하게
   - "따뜻했다"에서 시작하는 패턴 (기만적 안도)
   - 예: "처음에는 따뜻했다. 목욕탕에 들어가는 것 같은."

3단계: 상승
   - 감각 과부하 — 구체적 신체 묘사
   - 분석이 깨지기 시작 ("이걸 알아서 뭐하냐")
   - 예: "안에서부터 탔다. 골수가 불붙는 것 같은."

4단계: 임계점
   - 뇌가 사고를 포기 — 본능/공포만 남음
   - 문장이 파편화됨
   - 예: "울었다. 아이처럼. 엄마를 부르고 싶었는데."

5단계: 의식 소멸
   - 감각이 하나씩 꺼짐 (시각 → 청각 → 촉각)
   - 마지막 생각 하나만 남음
   - 예: "죽고 싶지 않아." (반복)

6단계: 암전
   - "어둠" — 하지만 매번 다른 종류의 어둠
   - 1차: 완전한 무 (감각 부재)
   - 2차: 무게가 있는 어둠 (존재의 밀도)
```

### 감각 묘사 우선순위
| 감각 | 빈도 | 용도 |
|------|------|------|
| 촉감/온도 | ★★★★★ | 고통의 핵심 전달 채널 |
| 청각 | ★★★★☆ | 자기 비명 인식, 소리 왜곡으로 의식 상태 표현 |
| 시각 | ★★★☆☆ | 시야 축소/확대로 의식 변화 표현 |
| 후각 | ★★☆☆☆ | 피 냄새, 타는 냄새 — 짧게 |
| 미각 | ★☆☆☆☆ | 입 안의 피 맛 정도 |

## 3. 사이 공간 (회귀 중간지대)

### 공간 묘사
- 완전한 감각 부재에서 시작
- 어둠의 질감이 매번 다름:
  - 1차: 텅 빈 무 — "존재 자체가 사라진 것 같은"
  - 2차: 밀도 있는 어둠 — "어둠에 무게가 있었다"
  - 이후: 점점 더 "무언가가 있는" 느낌으로 변화

### 존재의 목소리
| 항목 | 규칙 |
|------|------|
| 발화 방식 | 귀가 아닌 머리 안에서 직접 울림 |
| 성별 | 없음 — 중성적 |
| 감정 | 미안함 + 무력감 + 약간의 절박함 |
| 정보량 | 매번 약간씩 더 줌 — 하지만 완전한 설명은 절대 안 함 |
| 지호와의 관계 | 고용주와 피고용인? 보호자와 피보호자? — 아직 불명확 |

### 사이 공간에서의 지호 반응 변화
| 회차 | 반응 |
|------|------|
| 1차 (0→1) | 혼란 + 수동적 수용 ("돌아가는 것보다 낫지") |
| 2차 (1→2) | 분노 + 추궁 ("이 정도? 죽었거든? 두 번째야?") |
| 이후 (예정) | 거래/협상 시도, 정보 요구, 점차 대등한 관계로 |

### 재배치 규칙
- 같은 지역으로 돌아갈 수 없음: "결계가 너를 기억하고 있어" (5화)
- 다른 지역으로 보내짐 — 선택권은 없음 (초반), 이후 선택 가능해질 가능성
- 도착 시 각 지역 특유의 감각으로 깨어남 (아젤리아: 금빛 소환진, 솔라리스: 모래+이중태양)

## 4. 회귀 시 유지/리셋 규칙

### 핵심 원칙: "영혼의 깨달음"만 남는다

회귀를 관통하는 것은 **지식이나 능력이 아니라 깨달음**이다.
"아, 그때 그걸 이해했구나" 하는 영혼 수준의 인식 변화만 다음 생으로 이어진다.
체력이나 기술은 리셋되고, 성격의 찌질함도 그대로지만,
**"세계를 보는 눈"**만 조금씩 넓어진다.

### 유지/리셋 테이블
| 유형 | 유지 여부 | 세부 |
|------|----------|------|
| **영혼의 깨달음** | ✅ 핵심 유지 | 세계의 구조, 기둥의 진실, 사람의 본질에 대한 이해. "아는 것"이 아니라 "깨달은 것" |
| 사실 기억 | ✅ 유지 | 누가 뭘 했는지, 어디서 뭘 봤는지 — 정보로서 남음 |
| 감각 기억 | ✅ 강하게 유지 | 고통, 온도, 냄새 — 트라우마로 각인 |
| 감정 기억 | ⚠️ 점차 마모 | 첫 사망의 공포감이 루프 반복 시 옅어짐 |
| **멘탈 내성** | ⚠️ 아주 서서히 강화 | 공포에 대한 내성이 미세하게 쌓임. 하지만 "강해진다"기보다 "버티는 법을 조금 안다" 수준 |
| 신체 능력 | ❌ 완전 리셋 | 체력, 근력, 반사신경 — 매번 28세 마른 몸으로 리셋 |
| 마법/전투 기술 | ❌ 리셋 | 몸이 기억하는 것은 전부 사라짐. 검 잡는 법을 "안다"고 해도 몸이 안 따라줌 |
| **성격/찌질함** | ❌ 리셋 안 됨 (=유지) | 돈미새, 여미새, 겁쟁이, 분석 도피 — 기본 장착. 죽어도 안 고쳐짐 |
| 아이템 | ⚠️ 모래시계만 | 깨진 모래시계 목걸이는 유지 (영혼 귀속) |

### "깨달음"과 "지식"의 차이
- **지식**: "모르데우스는 추기경이고 축복 의식을 주관한다" → 정보. 유지됨.
- **깨달음**: "선의를 가장한 시스템적 악이 가장 위험하다" → 영혼에 새겨짐. 다음 지역에서 비슷한 구조를 직감적으로 감지.
- 깨달음은 다음 지역에서 "감"으로 발현: "이 사람 좋은 말 하는데 뭔가 이상하다" — 이유를 설명 못 하지만 몸(영혼)이 반응

### 성격이 안 나아지는 이유
- 지호의 찌질함은 **결함이 아니라 본성**
- 죽음을 겪어도 돈 계산하고, 예쁜 여자 보면 여미새 뇌 가동하고, 위험하면 분석으로 도피
- 이것이 오히려 지호를 인간답게 유지하는 앵커
- 영웅적으로 각성하면 캐릭터 붕괴 — "찌질한데 버틴다"가 이 소설의 공식
- 변하는 건 **깨달음의 깊이**뿐: 같은 겁쟁이인데, 겁먹으면서도 한 발짝 더 나가게 됨

### 서사적 활용
- **데자뷔**: 이전 루프 기억이 현재 경험과 겹칠 때 발생
  - 예: 같은 음식 냄새 → "이 냄새를 맡은 적 있다... 아, 아젤리아 왕궁 식당."
  - 초반: 가벼운 기시감
  - 후반: 날카로운 기억 소환
- **트라우마 플래시백**: 사망 관련 자극에 의해 촉발
  - 예: 금빛 빛 → 축복 의식의 고통 기억
  - 의도적 회피 행동으로 연결
- **깨달음 발현**: 이전 루프의 핵심 교훈이 "직감"으로 튀어나옴
  - 예: 새 지역의 권력자가 친절하게 다가옴 → "...이거 아젤리아랑 같은 패턴 아닌가?" (사실 기억이 아니라 뼈에 새겨진 경계심)

## 5. 감정 마모 — 반복 사망의 대가

### 마모 곡선
```
1~2회차: 공포 100% — 죽음이 절대적 위협
3~4회차: 공포 70% + 분석 30% — "어떻게 죽느냐"에 집중
5~7회차: 공포 40% + 분석 40% + 무감각 20% — 사망이 "과정"이 됨
8~10회차: 공포 20% + 분석 30% + 무감각 50% — 위험 신호
10회차+: 인간성 유지가 의식적 노력이 됨
```

### 무감각화의 징후 (집필 시 삽입)
- "또인가" 류의 건조한 반응
- 타인의 고통에 대한 공감 능력 저하
- 유머가 더 어두워짐 — 자조에서 냉소로
- 사망 직전의 마지막 생각이 점점 짧아짐 ("죽고 싶지 않아" → "아." → "...")

### 인간성 유지 갈등 (핵심 테마)
- 지호의 최대 적은 외부 위협이 아니라 **반복된 죽음에 의한 감정 마모**
- "죽음이 무서워야 정상이다. 무섭지 않으면 나는 이미 사람이 아니다."
- 각 아크의 NPC 관계가 인간성 앵커 역할 (에리나, 나이라, 카이 등)
- 궁극적 질문: "무한히 죽고 깨어나는 존재는 여전히 인간인가?"

## 6. 회귀 후 첫 씬 규칙

### 각성 순서
```
1. 감각 하나가 먼저 돌아옴 (지역 특성 반영)
   - 솔라리스: 열기 → "입술이 갈라져 있었다"
   - 프로스트헬: 추위 → "손가락 끝이 없는 것 같았다"
   
2. 환경 인식 (어디인지 파악)
   - 지호의 이세계 지식 DB가 가동
   - "모래? 솔라리스. 정령과 모래폭풍의 땅."

3. 신체 상태 확인
   - 매번 건강하지만 "속이 비어 있는" 느낌
   - 추출당한 무언가가 돌아오지 않은 감각

4. 이전 루프 기억 정리
   - "나는 살아 있었다. 죽었는데 또 살아 있었다."
   - "두 번째 생이다."

5. 다음 행동 결정
   - 1~2회차: 수동적 ("이번에는 얼마나 버틸 수 있을까")
   - 이후: 능동적 ("이번에는 _를 알아내야 한다")
```

### 5화 마지막 — 솔라리스 각성 (확립된 패턴)
```
눈을 떴다.
모래.
하늘에 태양이 두 개.
입술이 이미 갈라져 있었다. 온몸이 열병처럼 뜨거웠다.
"모래? 솔라리스. 정령과 모래폭풍의 땅."
팔이 말을 듣지 않았다. 머리가 둔했다.
"나는 살아 있었다. 죽었는데 또 살아 있었다."
"두 번째 생이다."
"이번에는 얼마나 버틸 수 있을까."
```

## 7. 모래시계의 역할

### 확인된 능력
| 항목 | 내용 |
|------|------|
| 발동 조건 | 극한 상황에서 자동 (의지와 무관) |
| 효과 | 0.3초간 시간 완전 정지 |
| 빛 색상 | 은색 (기둥의 금색과 대비) |
| 발동 시 현상 | 주변 모든 것 정지 — 물방울, 음파, 빛의 맥동까지 |
| 사용자 상태 | 의식 유지, 이동 불가, 관찰만 가능 |

### 서사적 기능
- 죽음 직전 0.3초 = "진실을 볼 수 있는 창"
- 아젤리아: 기둥 내부의 갇힌 영혼들 + 금빛 균열 목격
- 이후 아크: 각 기둥의 핵심 비밀을 죽는 순간 목격하는 패턴?
- 마르코가 의도적으로 준 것 → 마르코는 이 효과를 알고 있었다

### 미확정 사항 (향후 결정 필요)
- 의도적 발동 가능 여부
- 0.3초 이상 확장 가능 여부 (루프 축적에 따라?)
- 다른 시간 관련 효과 존재 여부
- 모래시계의 정체 (시간 기둥의 파편?)
