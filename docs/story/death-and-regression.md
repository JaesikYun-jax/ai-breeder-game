# 기둥 흡수 & 전이 서사 규칙

> 이 소설의 핵심 메커닉은 "죽고 다시 태어난다"가 아니라
> **"기둥에 빨려 들어가 재생과 분해를 반복하며, 끝없는 고통 끝에 다음 나라로 방출된다"**이다.
> 주인공은 죽지 않는다. 죽는 것보다 나쁜 일을 겪는다.

---

## 1. 기둥 흡수 유형 카탈로그

### 확정된 기둥 체험 (1-5화 기준)

| # | 장소 | 방식 | 체험 시간 | 고통 수준 | 서사적 의미 |
|---|------|------|----------|----------|-----------|
| 0 | 서울 횡단보도 | 트럭 충돌 → 이세계 전이 | 즉사급 (~30초) | ★★★★☆ | 이세계 진입 — "소설 같은 일이 실제로" |
| 1 | 아젤리아 대성당 | 축복 의식 → 기둥 흡수. 재생 루프(분해→재생→분해)로 장시간 고통 | 체감 수십 분~수 시간 | ★★★★★+ | 신뢰의 배신 — "보호자가 살인자". 재생이 고통을 끝내주지 않음 |

### 예정된 기둥 흡수 (지역별)

| 지역 | 기둥 흡수 양상 (설정 기반) | 서사적 의미 |
|------|--------------------------|-----------|
| 솔라리스 | 태양 기둥 — 열과 빛에 의한 소각→재생 루프. 불의 정령 계약 흔적이 보호막 역할? | 자연의 무관심 — "세계는 너를 신경 쓰지 않는다" |
| 카이젤 | 전쟁 기둥 — 생명력 수확 시스템에 재생이 무한 공급원으로 작용 | 시스템의 폭력 — "개인은 국가에 소모품" |
| 프로스트헬 | 얼음 기둥 — 동결→재생→동결 루프. 감각이 돌아올 때마다 극한의 추위 | 고독과 인내의 한계 |
| 용화국 | 천명 기둥 — 기공 역류와 함께 내장이 분해→재생 반복 | 운명에 대한 저항의 대가 |
| 리베르타 | 바다 기둥 — 익사→재생→익사. 폐에 물이 차고 비우기를 반복 | 자유의 대가 |
| 셀레스티아 | 하늘 기둥 — 추락 충격→재생 루프. 높은 곳에서의 끝없는 낙하감 | 진실을 아는 대가 |
| 카즈모르 | 대지 기둥 — 살아있는 금속이 신체를 관통→재생→관통 반복 | 탐욕의 결과 |
| 아비살 | 심해 기둥 — 수압에 의한 압사→재생. 가장 깊은 곳에서 가장 오래 | 가장 깊은 곳에서의 고통 |

### 기둥 흡수 규칙
- **같은 기둥에 두 번 흡수되지 않는다**: 각 기둥의 결계가 지호를 기억함 ("결계가 너를 기억하고 있어")
- **지역의 본질이 흡수 양상에 반영**: 빛의 왕국에서는 빛으로, 얼음의 땅에서는 냉기로 분해
- **흡수 중 관찰이 가능하다**: 재생 루프 사이사이에 기둥 내부를 관찰. 모래시계의 0.3초 정지가 관찰 창을 열어줌
- **재생 스킬이 체험을 장기화한다**: 다른 사람은 즉사(의식 소멸)하지만, 지호는 재생 덕분에 의식이 유지된 채 고통을 겪음. 이것이 축복이자 저주

## 2. 기둥 흡수 묘사 프로토콜

### 6단계 구조 (5화에서 확립)

```
1단계: 전조
   - 기둥이 반응한다 (공기 밀도, 온도, 빛의 변화)
   - 지호의 분석 뇌가 마지막으로 작동
   - 예: "발이 문양 위를 밟는 순간, 발바닥에서 진동이 올라왔다."

2단계: 흡수 시작
   - 기둥의 힘이 신체를 잡아당김 — 처음에는 약하게
   - "따뜻했다"에서 시작하는 패턴 (기만적 안도)
   - 예: "처음에는 따뜻했다. 목욕탕에 들어가는 것 같은."

3단계: 분해-재생 루프 시작
   - 기둥의 힘이 신체를 분해하기 시작
   - 재생 스킬이 즉시 반응 — 분해된 부위가 재생됨
   - 기둥이 다시 분해 → 재생 → 분해 루프 돌입
   - 예: "안에서부터 탔다. 골수가 불붙는 것 같은. 
     꺼졌다. 아니, 돋아났다. 새 살이 차오르는 그 몇 초가 — 
     다시 탔다."

4단계: 임계점 — 의식이 버티는 한계
   - 뇌가 사고를 포기 — 본능/공포만 남음
   - 하지만 재생 때문에 의식이 꺼지지 않음
   - 문장이 파편화됨
   - 예: "울었다. 아이처럼. 멈춰. 재생하지 마. 제발 그냥 꺼져."

5단계: 모래시계 발동 (0.3초 정지)
   - 모래시계가 자동 발동 — 분해-재생 루프가 일시 정지
   - 이 찰나에 기둥 내부를 관찰
   - 갇힌 영혼들, 기둥 깊은 곳의 금빛 균열 등
   - 예: "세계가 멈췄다. 은빛이 번졌다. 
     금빛 덩어리 안에 — 사람들이 있었다. 입을 벌리고 있었다."
   - 모래시계의 모래가 약간 줄어듦 (매 발동마다 감소)

6단계: 의식 소실 → 방출
   - 모래시계 효과가 끝나고 분해-재생이 재개
   - 마침내 의식이 소실 (재생이 뇌까지 완전 재건하기 전 잠깐의 공백)
   - 어둠 — 하지만 매번 다른 종류의 어둠
     - 1차: 완전한 무 (감각 부재)
     - 2차: 무게가 있는 어둠 (존재의 밀도)
   - 다음 나라의 기둥 근처에서 방출. 신체 온전. 옷, 소지품 유지.
```

### 핵심 차이: "사망"이 아닌 "통과"
- 지호는 죽지 않는다. 기둥을 통과한다.
- 재생 스킬 덕분에 다른 사람이라면 즉사할 과정을 살아서 겪는다
- 의식이 유지되는 시간이 길수록 기둥 내부를 더 많이 관찰 가능
- 이것이 지호만의 자격: 아무도 볼 수 없는 기둥의 내부를 본 유일한 존재

### 감각 묘사 우선순위
| 감각 | 빈도 | 용도 |
|------|------|------|
| 촉감/온도 | ★★★★★ | 분해-재생 고통의 핵심 전달 채널 |
| 청각 | ★★★★☆ | 자기 비명 인식, 기둥 내부의 소리(영혼들의 울음) |
| 시각 | ★★★☆☆ | 기둥 내부 관찰 — 특히 모래시계 발동 시 |
| 후각 | ★★☆☆☆ | 타는 냄새, 자신의 살이 타는 냄새 — 재생 후 다시 |
| 미각 | ★☆☆☆☆ | 입 안의 피 맛 — 재생되어도 맛은 남아있음 |

## 3. 사이 공간 (기둥 흡수 → 방출 사이)

### 공간 묘사
- 기둥 내부에서 의식 소실 후, 방출 전 잠깐 거치는 중간지대
- 어둠의 질감이 매번 다름:
  - 1차: 텅 빈 무 — "존재 자체가 사라진 것 같은"
  - 2차: 밀도 있는 어둠 — "어둠에 무게가 있었다"
  - 이후: 점점 더 "무언가가 있는" 느낌으로 변화

### 존재의 목소리
| 항목 | 규칙 |
|------|------|
| 발화 방식 | 귀가 아닌 머리 안에서 직접 울림 |
| 성별 | 없음 — 중성적 |
| 감정 | 미안함 + 무력감 + 약간의 절박함 |
| 정보량 | 매번 약간씩 더 줌 — 하지만 완전한 설명은 절대 안 함 |
| 지호와의 관계 | 고용주와 피고용인? 보호자와 피보호자? — 아직 불명확 |

### 사이 공간에서의 지호 반응 변화
| 회차 | 반응 |
|------|------|
| 1차 (0→1) | 혼란 + 수동적 수용 ("돌아가는 것보다 낫지"). 재생 스킬 선택 |
| 2차 (1→2) | 분노 + 추궁 ("이 정도? 살아서 통과했거든? 두 번째야?") |
| 이후 (예정) | 거래/협상 시도, 기둥 내부에서 본 것에 대한 정보 요구, 점차 대등한 관계로 |

### 전이 규칙
- 같은 지역의 기둥으로 돌아갈 수 없음: "결계가 너를 기억하고 있어" (5화)
- 다른 지역의 기둥 근처에서 방출됨 — 선택권은 없음 (초반), 이후 선택 가능해질 가능성
- 도착 시 각 지역 특유의 감각으로 깨어남 (아젤리아: 금빛 소환진, 솔라리스: 모래+이중태양)
- **신체 유지**: 이전 지역의 몸 그대로. 옷, 소지품, 상처 흔적(재생 완료 상태) 유지
- **불의 정령 계약 흔적**: 솔라리스에서 계약한 정령 마크가 다음 지역에서도 유지됨
- **모래시계**: 항상 유지 (영혼 귀속). 단, 모래가 매번 약간씩 줄어있음

## 4. 기둥 통과 시 유지/변화 규칙

### 핵심 원칙: "몸도 기억도 남는다 — 잃는 것은 모래시계의 모래뿐"

기둥 흡수와 전이의 핵심 차별점: **신체가 리셋되지 않는다.**
지호는 같은 몸을 가지고 다음 나라로 이동한다.
이전 지역에서 익힌 기술, 만든 상처, 얻은 물건이 모두 남는다.
대신 모래시계의 모래가 줄어든다 — 이것이 유한성의 원천.

### 유지/변화 테이블
| 유형 | 유지 여부 | 세부 |
|------|----------|------|
| **신체** | ✅ 유지 | 몸이 리셋되지 않음. 재생으로 상처는 회복된 상태이지만, 같은 체격·같은 몸 |
| **체력/근력** | ✅ 유지 (누적) | 이전 지역에서 훈련한 만큼 남아있음. 리셋 없음 |
| **기술** | ✅ 유지 (누적) | 정령술, 검술 등 몸이 기억한 것 그대로 유지 |
| **소지품** | ✅ 유지 | 옷, 무기, 도구 — 기둥 통과 시 함께 방출 |
| **정령 계약** | ✅ 유지 | 불의 정령 계약 마크 지속. 다른 지역에서도 소환 가능 |
| **모래시계** | ✅ 유지 (감소) | 영혼 귀속. 단, 매 기둥 통과마다 모래가 줄어듦 |
| **기둥 내부 기억** | ✅ 강하게 유지 | 모래시계 정지 중 본 것들 — 영혼, 균열, 구조. 트라우마로 각인 |
| **감각 기억** | ✅ 강하게 유지 | 분해-재생 루프의 고통. 트라우마로 각인. 재생 때문에 매번 풀파워 |
| **사실 기억** | ✅ 유지 | 누가 뭘 했는지, 어디서 뭘 봤는지 — 정보로서 남음 |
| **감정 기억** | ⚠️ 점차 마모 | 첫 기둥 체험의 공포감이 반복 시 옅어짐. 하지만 고통 자체는 매번 동일 (신경 재생) |
| **멘탈 내성** | ⚠️ 아주 서서히 강화 | 공포에 대한 내성이 미세하게 쌓임. 하지만 고통 내성은 불가 (신경 재생) |
| **성격/찌질함** | ❌ 안 바뀜 (=유지) | 돈미새, 여미새, 겁쟁이, 분석 도피 — 기본 장착. 기둥을 통과해도 안 고쳐짐 |
| **모래시계 모래** | ⚠️ 감소 | 매 기둥 통과마다 줄어듦. 모래가 다 떨어지면? — 미확정 |

### 모래시계 정지 중 보이는 것
- 기둥 흡수 과정 중 모래시계가 자동 발동 (0.3초 시간 정지)
- 이 순간 분해-재생 루프가 멈추고, 기둥 내부를 관찰할 수 있음
- **아젤리아 기둥**: 갇힌 영혼들, 금빛 균열
- **이후 기둥**: 각 기둥의 핵심 비밀을 이 순간에 목격하는 패턴
- 모래시계의 은빛 ↔ 기둥의 금빛 대비
- 매 발동마다 모래가 줄어듦 — 유한한 자원

### 성격이 안 나아지는 이유
- 지호의 찌질함은 **결함이 아니라 본성**
- 기둥을 통과해도 돈 계산하고, 예쁜 여자 보면 여미새 뇌 가동하고, 위험하면 분석으로 도피
- 이것이 오히려 지호를 인간답게 유지하는 앵커
- 영웅적으로 각성하면 캐릭터 붕괴 — "찌질한데 버틴다"가 이 소설의 공식
- 변하는 건 **세계 이해의 깊이**뿐: 같은 겁쟁이인데, 겁먹으면서도 한 발짝 더 나가게 됨

### 서사적 활용
- **데자뷔**: 이전 기둥 체험 기억이 현재 경험과 겹칠 때 발생
  - 예: 같은 음식 냄새 → "이 냄새를 맡은 적 있다... 아, 아젤리아 왕궁 식당."
  - 초반: 가벼운 기시감
  - 후반: 날카로운 기억 소환
- **트라우마 플래시백**: 기둥 흡수 관련 자극에 의해 촉발
  - 예: 금빛 빛 → 축복 의식의 분해-재생 고통 기억
  - 의도적 회피 행동으로 연결
- **기둥 관찰 활용**: 모래시계 정지 중 본 것들이 다음 지역의 열쇠
  - 예: 아젤리아 기둥 내부의 균열 패턴이 솔라리스의 기둥 위치 단서

## 5. 감정 마모 — 반복 기둥 체험의 대가

### 마모 곡선
```
1~2회차: 공포 100% — 기둥 흡수가 절대적 위협
3~4회차: 공포 70% + 분석 30% — "기둥 안에서 뭘 볼 수 있나"에 집중
5~7회차: 공포 40% + 분석 40% + 무감각 20% — 흡수가 "과정"이 됨
8~10회차: 공포 20% + 분석 30% + 무감각 50% — 위험 신호
10회차+: 인간성 유지가 의식적 노력이 됨
```

**단, 고통은 마모되지 않는다.** 감정적 공포는 줄어들어도, 신경 재생으로 인해 분해-재생 루프의 물리적 고통은 매번 첫 번째와 동일한 강도로 느껴진다. 이것이 핵심 모순: 마음은 무뎌지는데 몸은 매번 비명을 지른다.

### 무감각화의 징후 (집필 시 삽입)
- "또인가" 류의 건조한 반응 — 하지만 기둥에 닿는 순간 비명
- 타인의 고통에 대한 공감 능력 저하
- 유머가 더 어두워짐 — 자조에서 냉소로
- 기둥 흡수 직전의 마지막 생각이 점점 짧아짐 ("제발 멈춰" → "아." → "...")

### 인간성 유지 갈등 (핵심 테마)
- 지호의 최대 적은 외부 위협이 아니라 **반복된 기둥 체험에 의한 감정 마모**
- "기둥이 무서워야 정상이다. 무섭지 않으면 나는 이미 사람이 아니다."
- 하지만 재생이 고통을 유지시키므로: "마음이 무뎌져도 몸이 기억한다"
- 각 아크의 NPC 관계가 인간성 앵커 역할 (에리나, 나이라, 카이 등)
- 궁극적 질문: "기둥을 통과할 때마다 모래가 줄어드는데, 모래가 다 떨어지면 나는 어떻게 되는가?"

## 6. 기둥 방출 후 첫 씬 규칙

### 각성 순서
```
1. 감각 하나가 먼저 돌아옴 (도착 지역 특성 반영)
   - 솔라리스: 열기 → "입술이 갈라져 있었다"
   - 프로스트헬: 추위 → "손가락 끝이 없는 것 같았다"
   
2. 신체 확인 — 자기 몸이 온전한지
   - 재생 완료 상태. 옷도 그대로. 소지품도 있음.
   - 하지만 "속이 비어 있는" 느낌 — 기둥이 빼간 무언가?
   - 불의 정령 계약 마크 확인 (솔라리스 이후)

3. 환경 인식 (어디인지 파악)
   - 지호의 이세계 지식 DB가 가동
   - "모래? 솔라리스. 정령과 모래폭풍의 땅."
   - 근처에 기둥이 보임 — 이 나라의 기둥 근처에 방출되었음

4. 이전 기둥 체험 정리
   - "나는 살아 있었다. 기둥을 통과했는데 살아 있었다."
   - 분해-재생 루프의 감각 기억이 플래시백
   - 모래시계 확인 — 모래가 줄어있음을 발견

5. 다음 행동 결정
   - 1~2회차: 수동적 ("이번에는 얼마나 버틸 수 있을까")
   - 이후: 능동적 ("이번에는 기둥 안에서 _를 더 자세히 봐야 한다")
```

### 5화 마지막 — 솔라리스 방출 (확립된 패턴)
```
눈을 떴다.
모래.
하늘에 태양이 두 개.
입술이 이미 갈라져 있었다. 온몸이 열병처럼 뜨거웠다.
손을 봤다. 내 손이다. 아젤리아에서 입었던 옷 그대로.
목에 모래시계가 있었다. 모래가 — 줄었나?
"모래? 솔라리스. 정령과 모래폭풍의 땅."
팔이 말을 듣지 않았다. 머리가 둔했다.
기둥 안에서의 — 생각하지 마. 아직 떨리고 있잖아.
"나는 살아 있었다. 기둥을 통과했는데 살아 있었다."
"두 번째 나라다."
"이번에는 얼마나 버틸 수 있을까."
```

## 7. 모래시계의 역할

### 확인된 능력
| 항목 | 내용 |
|------|------|
| 발동 조건 | 극한 상황에서 자동 (의지와 무관). 특히 기둥 흡수 중 발동 |
| 효과 | 0.3초간 시간 완전 정지 — 분해-재생 루프도 정지 |
| 빛 색상 | 은색 (기둥의 금색과 대비) |
| 발동 시 현상 | 주변 모든 것 정지 — 물방울, 음파, 빛의 맥동까지 |
| 사용자 상태 | 의식 유지, 이동 불가, 관찰만 가능 |
| 모래 감소 | 매 발동마다 모래가 약간 줄어듦 — 유한한 자원 |

### 서사적 기능
- 기둥 흡수 중 0.3초 = "진실을 볼 수 있는 창"
- 분해-재생의 고통이 멈추는 유일한 순간 = 관찰이 가능한 순간
- 아젤리아: 기둥 내부의 갇힌 영혼들 + 금빛 균열 목격
- 이후 아크: 각 기둥의 핵심 비밀을 흡수 중 목격하는 패턴
- 마르코가 의도적으로 준 것 → 마르코는 이 효과를 알고 있었다
- 모래가 줄어든다 = 영원히 기둥을 통과할 수 있는 게 아니다

### 미확정 사항 (향후 결정 필요)
- 의도적 발동 가능 여부
- 0.3초 이상 확장 가능 여부 (경험 축적에 따라?)
- 다른 시간 관련 효과 존재 여부
- 모래시계의 정체 (시간 기둥의 파편?)
- 모래가 전부 떨어지면 어떻게 되는가?
