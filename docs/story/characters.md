### 기본 정보

| 시점 | 1인칭 |
| 성격 | 자조적, 분석적, 코미디 회피 본능 (위기에서 IT 비유로 도피) |

### 핵심 능력
- **D급 재생** (시작 스킬, 제3자만 등급 언급) — 손가락 베인 상처 15분, 장기 손상 5분, 미세 신경 10분
- **화염 정령 계약** (8화, 솔라리스 열기둥) — 손바닥 동전 크기 화상 자국. 첫 정령
- **번개 정령 계약** (15화, 벼락 마당) — 왼쪽 손목 흰색 나뭇가지(리히텐베르크) 문양. 21회 감전 끝 계약
- **체내 순환** (12~13화) — 정령 에너지를 혈관 따라 흘려 효율 3배. 처음 한계 15초
- **신경 가속/근육 강화** (16화) — 번개로 뇌·근섬유에 과부하 걸어 반사속도와 근밀도 증폭
- **문신 코딩** (17화) — 등·양팔·목·가슴·양손목에 새긴 5개 함수. 솔라리스 곡선 + 직선 분기:
  - 방어: 외부 충격 감지 시 등 번개 방벽 자동 전개
  - 경로: 번개 흐름 자동 제한 (성공률 60% → 90%+)
  - 감지: 척추 라인이 위험 신호로 미세 전류
  - 보온: 가슴 화염으로 체온 유지 (사막 횡단·아젤리아 추위)
  - 차단: 양 손목 임계치 초과 시 전류 자동 차단

### 한계와 대가
- 신경/근육 파괴 속도가 재생 속도를 초과하면 영구 소실 가능 (16화에서 손가락 감각 10분 소실로 발견)
- 폭식 체질 (밥 6인분 기본). 과부하 가동 후 더 심함
- 모래시계의 모래가 대가 단위로 보임. 13화 체내 순환 후 첫 감소 자각, 20화 시점 1/4만 남음
- 재생은 피로·수명·고통 기억(공포 누적)을 복구하지 못함

### 캐릭터 호 (Arc 1~3)
- Arc 1 (아젤리아): 트럭→소환→축복 의식에서 분해-재생 6회 루프 후 솔라리스로 전송
- Arc 2 (솔라리스): 캐러밴 합류, 화염 계약, 야간 훈련 390회로 횃불 도달, 체내 순환 자득
- Arc 3 (각성): 행상인의 아젤리아 소식 → 번개 계약 → 문신 코딩 → 사막 횡단 → 빛의 기둥 리팩토링 → 모르데우스 제압 → 에리나와 결혼 → 떠남
- Part 2: 부왕(사실상 새 왕) 포지션으로 솔라리스 방향, 나머지 8개 기둥 순례

### 호칭 변화
| 시기 | 지호 → 에리나 | 에리나 → 지호 |
|------|----------------|----------------|
| 1~5화 | 호칭 회피, 내면 "에리나"/"공주" | "용사님" 일관 |
| 18화 (재회) | "에리나" / "공주" | "용사님" — 다만 신원 확인 후 곧바로 신뢰 톤 |
| 19~20화 (결혼 후) | "에리나" | "지호" (이름 전환, 결혼식 이후 정착) |

### IT 비유 사용 (가드레일: 챕터당 최대 3회)
컴파일/런타임 에러, 무한 루프, try-catch, CI/CD, 그리드 서치, 베이지안 최적화, DDoS, SQL 인젝션, 레거시 코드 리팩토링, return 문 등.
