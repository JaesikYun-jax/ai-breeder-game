# 아젤리아 왕국 — 지역 상세 설정

> 빛의 기둥이 지탱하는 중앙 평원의 왕국.
> "빛이 밝을수록 그림자는 짙다."

---

## 1. 첫인상 — 지호가 처음 도착했을 때

### 시각
소환의 홀: 거대한 돌 기둥들, 희미하게 빛나는 금빛 문양, 스테인드글라스로 쏟아지는 석양빛. 천장은 "압도적으로" 높다. 수십 명의 기사, 사제, 왕이 내려다보고 있다.

### 후각
알 수 없는 달콤하면서도 금속적인 냄새 — 소환진에서 나는 마법의 잔향. 서울에서 한 번도 맡아본 적 없는 냄새.

### 촉각
소환진 위의 진동 — 발바닥에서 시작해 뼈까지 울리는 공명. 금빛 빛의 따뜻함.

### 청각
웅웅거리는 마법 잔향, 갑옷 부딪히는 금속음, 군중의 숨소리.

---

## 2. 지리 & 기후

| 항목 | 내용 |
|------|------|
| 위치 | 대륙 중앙 평원 |
| 지형 | 광활한 평야, 왕도 주변 경작지, 성벽으로 둘러싸인 도시 |
| 기후 | 서울보다 따뜻한 온대 기후 ("바람은 서울보다 따뜻했다" — 3화) |
| 계절 | 사계절 있으나 본편에서는 주로 늦봄~초여름 묘사 |
| 밤하늘 | 서울과 비교 불가하게 많은 별 ("평생 본 별보다 이 밤 하나에 보이는 별이 더 많다" — 2화) |
| 특이사항 | 빛의 기둥 근처는 밤에도 희미한 금빛 잔광, 첨탑 꼭대기의 맥동하는 빛 |

---

## 3. 왕도 — 도시의 모습

### 거리 풍경 (4화 기반)
- 돌 건물이 줄지어 선 포장 도로
- 마법 위생 시스템 (악취 없음 — 중세치고 깨끗)
- 마차가 다님
- 상인들의 호객 소리
- 빵 굽는 냄새가 거리에 퍼짐
- 중세 유럽풍이되 현실 중세보다 훨씬 청결

### 시장
- 다양한 지역 상품 거래
  - 솔라리스산 보라색/붉은색 과일
  - 카이젤산 정밀 기계
  - 보석, 건조 약초, 고대 지도
- 소란스럽고 활기참
- 마르코의 텐트 노점이 시장 중앙에 위치

## 4. 핵심 장소

### 왕궁 (소환의 홀, 객실, 식당)
- **소환의 홀**: 거대한 아치형 천장, 금빛 문양 기둥, 스테인드글라스
- **객실**: 서울 원룸 다섯 개 크기, 샹들리에, 태피스트리, 거대한 침대
- **목욕탕**: 대리석, 거대 석조 수반, 5성급 스파 수준
- **식당**: 갓 구운 빵, 버터, 치즈, 훈제 고기, 과일, 허브차
- **분위기**: 호화롭지만 어딘가 감시당하는 느낌

### 훈련장
- 모래 깔린 왕궁 안마당
- 한쪽 무기 거치대, 반대쪽 과녁
- 기사 십여 명이 이미 훈련 중
- 아침 햇살, 금속 부딪히는 소리
- 2층 발코니에서 관찰 가능 (에리나의 위치)

### 대성당 (루미엘 칸타레)
- 아치형 천장, 스테인드글라스, 아침 빛
- 바닥에 복잡한 기하학 문양 (원 안의 원 안의 별 안의 고대 문자)
- 중앙에 **수정 기둥** (빛의 기둥) — 사람 키 2배, 반투명, 내부 금빛 맥동
- 의식 시 사제 12명이 원을 이루며 기도
- **이곳의 진짜 정체**: 300년간 32명의 영혼을 먹어온 제단

### 첨탑
- 대성당 서쪽 끝의 뾰족한 탑
- 꼭대기에 불규칙한 금빛 — 밝아졌다 어두워졌다 심장박동처럼
- 맥동마다 지호 가슴에 공명
- 밤에만 보임

### 에리나의 골목
- 성문 근처, 건물 사이 좁은 틈
- 두 사람이 겨우 들어갈 정도
- 에리나가 지호를 끌어들여 진실을 폭로한 곳

## 5. 일상 & 문화

### 인사
- "루미엘라" (빛이 함께하기를) — "에트 루미엘라" (당신에게도)

### 식사
| 음식 | 묘사 |
|------|------|
| 갓 구운 빵 | 따뜻하고 바삭, 버터와 함께 — 지호 曰 "편의점 김밥의 대척점" |
| 훈제 고기 | 아침 식사에 포함, 서울에서 먹던 것과 차원이 다른 풍미 |
| 허브차 | 왕궁 표준 음료, 에리나가 대접 |
| 황금 스튜 (아우룸 스튜) | 뜨거울 때 먹어야 맛이 변하지 않음 (Blue 스크립트) |
| 광명빵 (판 데 룩스) | 금빛 빵, 딱딱해서 꿀에 찍어 먹음 (Blue 스크립트) |

### 화폐
- **쿠론** — 아젤리아 기본 화폐
- 빵 1개 = 1 쿠론, 과일 = 3 쿠론, 고기 꼬치 = 5 쿠론, 검 = 200 쿠론
- 지호 환산: 1 쿠론 ≒ 500원

### 장례 문화
- 등불을 빛의 기둥 쪽으로 띄워 보냄 ("빛으로 돌아간다")

### 축제
- 광명절 — 연 3일, 군사 대회 + 마지막 밤 빛의 눈(빛 입자가 하늘에서 내리는 현상)

## 6. 정치 & 권력 구조

| 직위 | 인물 | 실권 |
|------|------|------|
| 국왕 | 세드릭 | 명목상 최고 권력, 실제로는 병으로 약화 |
| 추기경 | 모르데우스 | **실질적 지배자** — 기둥 접근권, 의식 주관, 왕에게 "치료제" 제공 |
| 제2왕녀 | 에리나 | 정치적 영향력 제한, 개인적으로 용사 구출 시도 |
| 근위대 부단장 | 로헨 | 군사력 수장, 모르데우스 지시 따름 |

### 권력 갈등
- 모르데우스 vs 에리나: 에리나는 의식의 진실을 알지만 막을 힘이 없음
- 왕의 병: 모르데우스의 "치료"가 오히려 악화시키는 의심 (Blue 스크립트 — 약 먹은 후 검은 혈관)
- 기사단: 왕에게 충성하지만 실제 명령은 모르데우스 경유

## 7. 위험 요소 — 지호를 죽이는 것들

| 위험 | 성격 | 결과 |
|------|------|------|
| 축복 의식 | 구조적/정치적 | 영혼 추출 → **사망 (1회차)** |
| 모르데우스 | 인적 위협 | 모든 것을 통제하는 실권자 |
| 마법 결계 | 물리적 장벽 | 의식 전날 가동, 탈출 봉인 |
| 정보 부재 | 시스템적 | 세계의 규칙을 모르는 상태로 던져짐 |
| 신체 무능 | 개인적 한계 | 전투/마법 모두 불가 |

## 8. NPC 관계도 (아젤리아 내)

```
        [세드릭/왕]
         ↑ 아버지    ← 치료? → [모르데우스]
         |                      ↑ 지배
      [에리나]  ←── 적대/두려움 ──→  |
         ↑ 경고/보호              ↓ 명령
      [지호] ←── 훈련 ── [로헨]
         ↑ 모래시계
      [마르코] (외부인, 독자적 행동)
```

### 관계 요약
- **에리나 → 지호**: 죄책감에 기반한 보호 본능 (하루카의 기억)
- **모르데우스 → 지호**: 소모품 (33번째 제물)
- **마르코 → 지호**: 의도적 투자 (모래시계 제공, 미래를 아는 듯)
- **로헨 → 지호**: 직업적 의무, 개인적 감정 최소화
- **세드릭 → 지호**: 미안함이 있지만 왕국을 위해 묵인

## 9. 마법 체계 — 성광술 (聖光術)

| 항목 | 내용 |
|------|------|
| 원천 | 빛의 기둥에서 흘러나오는 에너지를 신체로 채널링 |
| 기초 | 치유의 빛 (초보) |
| 중급 | 빛의 방패 |
| 상급 | 성스러운 심판 (광역 정화) |
| 금기 | 영혼 소환 (타 차원 존재 추출) — 축복 의식의 원리 |
| 대가 | 피로, 집중력 소모, 일시적 시력 상실, 금기 시 수명 |
| 지호와의 관계 | 마나 회로 미형성으로 사용 불가, 다만 시간 기둥 파편으로 인해 치유의 빛이 자동 반응할 가능성 |

## 10. 분위기 키워드

이 지역 씬을 쓸 때 참조할 이미지/형용사:

1. **금빛**: 소환진, 기둥, 스테인드글라스, 석양 — 아젤리아의 색은 금
2. **무거운 호화**: 화려하지만 압도적. 왕궁의 모든 것이 "너무 크다"
3. **감시**: 항상 누군가 보고 있는 느낌 (에리나의 발코니, 기사의 시선, 모르데우스의 존재감)
4. **석양/석조**: 돌과 빛의 조합. 차가운 돌에 따뜻한 빛이 스미는 이미지
5. **심장박동**: 첨탑의 맥동, 소환진의 진동, 기둥의 빛 — 모든 것이 "살아 있다"
