# 스토리 피드백 누적 로그

> 스토리에 대한 피드백과 그로 인한 변경사항을 시간순으로 기록한다.
> 각 피드백이 영향을 미친 챕터, 설정 문서, 파급 효과를 추적한다.

---

## 사용법

새 피드백 발생 시 아래 템플릿을 복사하여 **최상단에** 추가한다 (최신순 정렬).

```
### FB-XXX — YYYY-MM-DD

**피드백 원문**: (사용자가 준 피드백 원문 그대로)

**요약**: 한 줄 요약

**영향 받은 챕터**:
- N화 「제목」 — 변경 내용 요약
- M화 「제목」 — 변경 내용 요약

**설정 변경**:
- [문서명] — 변경 내용

**파급 효과**:
- 향후 N화~ 에 영향: (구체적 내용)
- 캐릭터 X의 행동 동기 변경 등

**상태**: 반영 완료 / 부분 반영 / 미반영
```
