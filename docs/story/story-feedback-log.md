# 스토리 피드백 누적 로그

> 스토리에 대한 피드백과 그로 인한 변경사항을 시간순으로 기록한다.
> 각 피드백이 영향을 미친 챕터, 설정 문서, 파급 효과를 추적한다.

---

## 사용법

새 피드백 발생 시 아래 템플릿을 복사하여 **최상단에** 추가한다 (최신순 정렬).

```
### FB-XXX — YYYY-MM-DD

**피드백 원문**: (사용자가 준 피드백 원문 그대로)

**요약**: 한 줄 요약

**영향 받은 챕터**:
- N화 「제목」 — 변경 내용 요약
- M화 「제목」 — 변경 내용 요약

**설정 변경**:
- [문서명] — 변경 내용

**파급 효과**:
- 향후 N화~ 에 영향: (구체적 내용)
- 캐릭터 X의 행동 동기 변경 등

**상태**: 반영 완료 / 부분 반영 / 미반영
```

### FB-003 — 2026-04-13

**피드백 원문**: 코믹한 요소가 너무 없어져버렸고 주인공이 안쎄서 초반 하차할게 걱정됨. '재생'이라는 스킬을 받지만 초반에 빛기둥까지 가는데 별 도움이 안되는 것으로 해서 넘길 것. 이틀차 아침에 몰래 일어나서 화장실에서 몸의 재생능력을 간단하게 테스트하다가 조식먹으러 끌려나감. 8화와 함께 자신의 스킬에 대해서 속속들이 알게되는 계기. 재생의 속도가 매우 빠르지는 않지만 분명하게 재생한다. 신체 절단의 재생도 잘 붙여놓으면 서서히 수복됨. 5화때 오랜시간 고통받은것도 재생때문이었고, 빛의 기둥에서 축복 의식 도중 분명 죽었다 생각했는데, 신체적 죽음 직후에 재생해버린 것. 이정도면 불사 아닌가? 하지만 목이 잘리고 통조림에 들어가면 복구 불가다... 정말 쓰레기같다. 신경세포도 무조건 재생하기때문에 고통에서 익숙해질 수 없다. 또한 '죽음'이 아니라 각 지역의 기둥에 스스로 들어가거나 들어가짐 당하는 방식으로 진행시킬 것. 5화 단위는 너무 짧다. 6화부터 25~30화까지 솔라리스 에피소드들로. 9개의 나라를 거치며 기둥을 온몸으로 받아내고 성장하여 나중에는 이 세계를 구하는 영웅이 되어가는 서사시.

**요약**: 재생(D급) 스킬 추가, 기둥 흡수/전이 메커닉(죽음이 아닌 통과), 아크 확장(솔라리스 6~30화), 코미디 밸런스 복원, 서사시적 스케일 확립

**영향 받은 챕터**:
- 1화 — 재생 스킬 선택 씬 추가 (사이 공간에서 D급 스킬 목록 중 재생 선택)
- 2화 — 화장실 재생 테스트 씬 추가 (이틀차 아침, 15분 만에 베인 상처 회복)
- 3화 — 검술 훈련 중 재생 정신승리 추가 ("긁지 않은 복권", 무한 훈련 루프 구상)
- 5화 — 기둥 의식 장면 대폭 수정 (분해→재생→분해 루프, 연장된 고통, 사망이 아닌 전이)
- 6~10화 — 구조 변경 필요 (솔라리스 아크 확장에 따른 재배치)
- 11~15화 — 카이젤 아크 연기 (솔라리스 아크 확장으로 번호 밀림)

**설정 변경**:
- `protagonist-bible.md` — 재생 스킬 섹션 신설, 성장 공식에 재생 통합, 기둥 체험 반응 변화 곡선으로 변경
- `death-and-regression.md` — 전면 재작성: "사망 & 회귀" → "기둥 흡수 & 전이". 몸/옷/아이템 유지, 재생이 기둥 통과를 가능하게 함
- `story-framework-6-30.md` — 전면 재작성: 5화 단위 아크 → 솔라리스 20+화 장편 아크, 기둥 흡수 메커닉 반영

**파급 효과**:
- 전체 서사 구조: 5화 단위 속행 → 지역별 장편 아크 (20~30화씩). 총 분량 대폭 확대
- 기둥 메커닉: "죽음"이 아닌 "기둥 흡수→재생 루프→방출→다음 나라" 패턴 확립
- 재생과 고통참기의 필연적 연결: 신경 재생으로 고통 면역 불가 → 고통참기 비술이 필수
- 솔라리스 6~30화: 노예→비밀 정령술 훈련→상행 조력자→기둥 흡수까지 3개월간의 성장
- 코미디 밸런스: 재생 스킬이 지속적 코미디 소스 (D급의 현실, 화장실 테스트, 정신승리)

**상태**: 반영 완료 (1~5화 수정, 설정집 3종 업데이트, 프레임워크 재작성)

### FB-002 — 2026-04-13

**피드백 원문**: 마르코가 자주 등장하는 것은 좋지 않음. 극의 몰입을 방해하는 것 같으며 모래시계를 알아보는것도 문제가 있음. 알아보는 것 처럼 하면서 오히려 처음 보는 물건인 것 처럼 큰돈을 주고 매입하려는 듯하게 보이는게 낫지. 지호가 마케터 뇌라고 딱히 뛰어날 필요는 없음. 오히려 삼류 프로그래머 정도로 설정을 바꾸자. 기둥 떡밥이 이렇게 빨리 회수될 필요가 없음. 멀리 모래폭풍 기둥이 보이는 정도로 하자. 6화에서는 특이하게 일단 나이라의 시점으로 3인칭 묘사를 자세히 하고 사막 중심의 서술을 하자. 솔라리스에서는 사람들이 정말 많이 태어난다. 다들 생명을 연료삼아서 살아가는데 익숙하기 때문에 정령술사 평균 수명이 짧다는 설정. 간혹 정령의 사랑을 받으면 큰 대가 없이도 쓰게 해준다. 나이라가 마지막에 겨우 참여하게 된 상행에서 살짝 뒤쳐졌는데 그늘에서 모래에 반쯤 파묻힌 주인공 찾아냄. 사막에서는 정령술에 눈뜨게 되는 계기. 장생에 대한 열망이 강한 주인공을 쓰는법을 알아만두고 잘 쓰지 않는다. 계약 안한 정령은 갑자기 나타나지 않음. 주위 원소가 강한곳에서 계약을 맺어두어야 함. 솔라리스에서 불의 정령과 맺은 계약이 나중에 북쪽 나라에서 요긴하게 쓰이게 됨. 전체적으로 너무 핵심만 뱉으면서 진행되는데, 각 화에서 무조건 1인칭 주인공 시점만 유지하지 말고 적절한 씬전환을 통해서 설정도 보여주고 주인공이 찌질하지만 조금씩 강해질 수 있는 떡밥을 주기. 근데 바로 강해지는게 아니라, 모든게 참기힘든 고통을 수반하는데 나중에 고통참기의 비술을 익히면서 모든 강함의 방식이 하나로 연결되게 하자. 그리고 너무 무재능이라서 이것들을 합치는게 가능하다는 것으로.

**요약**: 마르코 등장 최소화, 주인공 직업 삼류 프로그래머로 변경, 기둥 떡밥 느린 회수, 3인칭 씬 전환 활용, 정령 계약 시스템 확립, 고통참기+무재능 통합 성장 메커닉

**영향 받은 챕터**:
- 6~30화 전체 — `story-framework-6-30.md` 새로 작성. 모든 피드백 반영
- 6화 — 나이라 3인칭 시점 도입, 사막 캐러밴 중심 서술, 모래에 파묻힌 지호 발견
- 8화 — 정령 계약 장면 (원소 강한 열기둥에서, 고통 수반)
- 16화~ — 솔라리스 불의 정령이 프로스트헬에서 체온 유지에 활용
- 22화 — 고통참기의 비술 체득 (현무 사부)
- 28화 — 마르코 스침 (모래시계 매입 시도)

**설정 변경**:
- `protagonist-bible.md` — 직업: 마케터 → 삼류 프로그래머. 비유/유머 패턴 전면 수정
- `voice-guide.md` — 마르코 등장 규칙 추가. 모래시계 반응 방식 변경 (알아보는 척 X → 매입 시도)
- `tone-and-style.md` — 3인칭 씬 전환 규칙 추가 (§1 시점 규칙)
- `project_novel_structure.md` (메모리) — 주인공 직업 변경, 성장 아크 수정

**파급 효과**:
- 마르코: 6~30화에서 2회만 등장 (10화 부근 스침, 28화 매입 시도). 정체 공개는 31화 이후로 이연
- 성장 메커닉: 고통참기가 모든 힘의 통합 키. 용화국(Arc 5)에서 비술 체득 후 리베르타(Arc 6)에서 실전 적용
- 정령 계약: 원소 강한 곳에서만 계약 가능. 솔라리스 불 → 프로스트헬 활용 → 용화국 통합 → 리베르타 실전
- "정령의 사랑" 복선: 솔라리스에서 설정만 배치. 회수는 훨씬 후반 클라이맥스

**상태**: 반영 완료

### FB-001 — 2026-04-13

**피드백 원문**: 죽음을 거쳐 다시 태어날 때 유지되는 것은 영혼의 깨달음 중심. 육체는 자꾸 리셋되고 멘탈이 조금씩 강해져가지만 찌질함이 기본 장착이라서 성격은 잘 나아지지 않는편으로

**요약**: 회귀 시 유지되는 것 = 영혼의 깨달음(직감/이해). 육체 리셋, 멘탈 미세 강화, 찌질한 성격은 불변.

**영향 받은 챕터**:

**설정 변경**:
- `death-and-regression.md` §4 — "기억 규칙" → "유지/리셋 규칙"으로 개편. 영혼의 깨달음 중심 재구성, 찌질함 불변 명시
