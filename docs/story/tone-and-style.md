# 문체 & 톤 가이드라인

> 이 소설의 톤 공식: **코미디로 시작해서 고통으로 착지한다.**
> 독자는 웃다가 얼어붙는 경험을 반복한다. 이 낙차가 핵심이다.

---

## 1. 시점 규칙

### 1인칭 내면 독백 중심
- 서술자 = 강지호. 그의 눈에 보이지 않는 것은 쓰지 않는다
- 타인의 심리 직접 서술 **금지**: "에리나는 슬펐다" (X) → "에리나의 입술이 떨렸다" (O)
- 지호가 모르면 독자도 모른다. 정보 비대칭 유지
- 단, 지호의 **해석**은 적극 서술: "저 표정은... 공포다. 축복이 아니라 뭔가 다른 것에 대한."

### 시점 예외
- 씬 첫 문장에서 공간 묘사는 객관적 허용 ("성당의 천장은 높았다")
- 다만 2~3문장 안에 반드시 지호 주관으로 복귀 ("높다기보다 압도적이었다. 무릎이 꺾일 것 같은.")

### 3인칭 씬 전환 (적극 활용)
- 챕터 도입이나 중간에 **타 캐릭터 시점의 3인칭 씬**을 삽입할 수 있다
- 용도: 세계관 설정 전달, 지호가 모르는 상황 보여주기, 서브 캐릭터의 심리/배경 묘사
- 예: 6화 — 나이라 시점으로 사막과 상행을 먼저 보여주고, 나이라가 지호를 발견하는 씬으로 시작
- 3인칭 씬은 `---` 구분선으로 확실히 분리하고, 지호 시점 복귀 시에도 `---`로 전환
- **주의**: 3인칭 씬에서도 톤(코미디→고통)과 감각 구체화 규칙은 동일 적용
- 3인칭 씬의 분량은 챕터의 30% 이내가 적정. 메인 서술은 여전히 지호 1인칭

## 2. 문장 스타일

### 기본: 짧은 문장
- 평서문 위주, 복문보다 단문
- 호흡이 빠른 웹소설 리듬
- **좋은 예**: "걷고 있다. 제 발로 죽으러 걸어가고 있다."
- **나쁜 예**: "나는 지금 제 발로 내 죽음을 향해 걸어가고 있다는 사실을 인식하면서도 멈출 수 없었다."

### 길어지는 조건
| 상황 | 문장 길이 | 이유 |
|------|----------|------|
| 전투/사망 | **매우 짧음** — 파편화 | 의식이 분해되는 감각 재현 |
| 분석/추론 | **길어짐** | 프로그래머 뇌 가동, 논리 전개 |
| 회상 (현대) | **2~3문장 이내** | 길어지면 현재 긴장감 상실 |
| 감각 묘사 | **중간** | 구체적 감각 나열 (소리, 온도, 촉감) |
| 유머 | **짧음** | 펀치라인은 짧아야 터짐 |

### 문장 리듬 예시 (5화 사망 씬)
```
빛이 쏟아졌다.
처음에는 따뜻했다. 목욕탕에 들어가는 것 같은. 부드러운.
0.5초.
그리고 안에서부터 탔다.
```
→ 짧음 → 약간 김 → 초짧음 → 짧음. 이 리듬이 긴장과 이완을 만든다.

## 3. 유머 규칙

### 허용되는 유머
| 유형 | 예시 | 효과 |
|------|------|------|
| **자조** | "스쿼트 3개가 한계인 28세 직장인의 위엄" | 주인공 친밀감 |
| **메타 코멘트** | "금빛 로브? 악역 확정." | 장르 인식, 독자 공감 |
| **상황 아이러니** | 소환 직후 "화장실 어디예요?" | 기대 전복 |
| **돈미새/여미새** | "목숨 위험한데 물가 분석 중" | 캐릭터 일관성 |
| **직업병** | 이세계 상황을 삼류 개발자 관점으로 분석 ("이건 스펙 누락이야") | 웃기면서 세계관 설명 |
| **부적절한 타이밍** | 사망 직전 분석: "이 사람도 서른두 번 봤구나" | 갤로우즈 유머 |

### 금지되는 유머
- **타인 비하**: 이세계인을 미개하다고 깔보거나, NPC 외모 비하 금지
- **과도한 언어유희**: 말장난 중심 유머는 이 소설의 톤이 아님
- **개그 과잉**: 한 씬에 유머가 3개 이상 연속되면 긴장감 소멸
- **비극 중 개그**: 사망 씬 한복판에서 유머 금지 (사망 직전/직후의 갤로우즈 유머는 가능)

### 유머 → 고통 전환 패턴 (본편 대표 예시)
```
[유머] "여미새 뇌, 지금 이 상황을 로맨스로 해석하지 마라."
[전환] 하지만 에리나의 표정은 심각했다. 핏기 없는 얼굴.
[고통] "서른두 명이 죽었어요. 그중 살아 돌아온 사람은—없어요."
```
→ 유머가 독자의 가드를 내린 순간, 진실이 꽂힌다.

## 4. 고통 묘사 규칙

### 원칙: 감각 구체화
- 추상적 고통 금지 ("극심한 고통을 느꼈다" → X)
- 구체적 감각으로 분해:
  - **소리**: "뼈가 '득' 하는 둔탁한 소리" (1화)
  - **온도**: "안에서부터 탔다" / "비정상적으로 차가운 손" (5화/3화)
  - **촉감**: "세포 하나하나가 분해되는 느낌" (5화)
  - **호흡**: "숨을 들이마시는데 폐에 뭔가가 차 있었다" (1화)
  - **시각**: "시야가 좁아진다. 터널처럼." (1화)

### 미화 금지
- 영웅적 사망 묘사 금지 ("장렬하게 쓰러졌다" → X)
- 실제 죽음은 추하고 무기력하다:
  - "비명도 아니고, 외침도 아니고, 그냥 울었다. 아이처럼." (5화)
  - "엄마를 부르고 싶었는데 목에서 소리가 나지 않았다." (5화)
  - "0.01%의 생존 확률을 위해 뇌가 순응을 선택한다." (5화)

### 고어 과잉 금지
- 장기 묘사, 혈액량 수치, 해부학적 디테일 과잉은 금지
- 감각 전달에 필요한 만큼만. "뼈가 부러졌다"는 되지만 "경골이 45도 각도로 꺾이며..."는 과잉
- **기준**: 독자가 "으" 하고 움찔하는 수준 = 적절. "읽기 싫다" = 과잉

### 사망 씬 구조 (확립된 패턴)
```
1. 전조 (환경 변화, 공기가 무거워짐)
2. 발동 (고통 시작 — 처음엔 약하게)
3. 상승 (감각 과부하 — 분석이 깨지기 시작)
4. 임계점 (뇌가 사고를 포기 — 본능/공포만 남음)
5. 의식 소멸 (감각 하나씩 꺼짐 — 마지막 생각 하나)
6. 암전
```

## 5. 회상 규칙

### 현대 회상
- **길이**: 2~3문장 이내. 절대 플래시백 씬 안 만듦
- **용도**: 비유 소재로만. "서울 지하철 퇴근길처럼 사람에 치이는 느낌이었다"
- **트리거**: 이세계 상황이 현대 경험을 환기할 때만
- **금지**: 현대 에피소드를 길게 풀기 (독자가 이세계에서 이탈)

### 이전 루프 회상 (2회차 이후 해당)
- 플래시백보다는 **데자뷔** 형태로
- "이 복도를 걸어본 적이 있다. 언제? 아, 그때." — 짧은 인식
- 감각 기억 우선 (논리적 기억보다 "이 냄새를 맡은 적 있다"가 먼저 옴)
- 루프가 쌓일수록 데자뷔가 날카로워짐

## 6. 씬 전환 규칙

### `---` (씬 구분선)
- 시간 경과 또는 장소 전환 시 사용
- 렌더링 시 `· · ·`로 표시됨
- 한 챕터 내 3~5개가 적정 (너무 많으면 씬이 얕아짐)

### 전환 후 첫 문장 규칙
| 전환 유형 | 첫 문장 패턴 |
|----------|-------------|
| 장소 이동 | 감각 묘사로 시작 ("모래가 입에 들어왔다") |
| 시간 경과 | 상태 묘사 ("잠을 못 잤다") |
| 각성/부활 | 감각 복귀 ("눈을 떴다. 빛이 아팠다") |
| 긴장 전환 | 행동 시작 ("문이 열렸다") |

### 챕터 시작/끝 패턴
- **시작**: 훅 — 짧고 강한 문장. 1화: "트럭이 오는 건 알고 있었다."
- **끝**: 여운 또는 클리프행어. 5화: "이번에는 얼마나 버틸 수 있을까."
- 끝에 이탤릭 마무리: "*N화 끝. 다음 화: 제목*"

## 7. 금지 목록 (절대 하지 않는 것)

| 금지 항목 | 이유 |
|----------|------|
| 만능 주인공 묘사 | 지호는 무능하다. 그게 전제 |
| 설명충 나레이션 | "이 세계의 마법 체계는..." 식의 백과사전 서술 금지. 지호가 경험으로 발견 |
| 감정 직접 명명 | "슬펐다" → 보여주기 ("시야가 흐려졌다") |
| 히로인 외모 장황 묘사 | "은발, 보라색 눈" 정도. "달빛을 머금은 실크같은 은발이..." 금지 |
| 시스템 용어 | 레벨, 스킬, 스탯 창, 경험치 — 이 세계에 없음 |
| 영웅적 독백 | "나는 이 세계를 구하겠다!" — 지호는 이런 말 안 함 |
| 타인의 내면 서술 | 1인칭 시점 엄수. 지호가 추측할 수는 있지만 확정 불가 |
| 이세계 미화 | "아름다운 중세 풍경"보다 "화장실이 없다"가 이 소설의 톤 |

## 8. 대사 포맷 규칙

### 기본
- 큰따옴표 `""`로 감싼다: `"용사님, 의식장으로 이동하겠습니다."`
- 내면 독백은 작은따옴표 `''`로 감싼다: `'여미새 뇌, 지금 이걸 로맨스로 해석하지 마라.'`
- 나레이션/독백은 따옴표 없이 평서문
- 괄호 `()` 사용 금지 — Blue 스크립트 포맷이지 소설 포맷 아님

### 독백 구분
| 표현 | 용도 |
|------|------|
| 따옴표 없음 | 서술/나레이션 |
| `''` 작은따옴표 | 내면 독백 (파란색 렌더링) |
| `""` 큰따옴표 | 실제 대사 (볼드 렌더링) |
| `*텍스트*` | 강조/이탤릭 |

## 9. 톤 밸런스 가이드 (챕터별)

### 이상적 배합
```
일상/탐색 씬: 유머 70% / 긴장 20% / 고통 10%
수사/발견 씬: 유머 30% / 긴장 60% / 고통 10%
전투/사망 씬: 유머 5%  / 긴장 30% / 고통 65%
회귀 직후 씬: 유머 20% / 긴장 30% / 고통 50%
```

### 챕터 단위 톤 곡선 (1-5화 실제 패턴)
| 화 | 시작 톤 | 중간 톤 | 끝 톤 |
|----|--------|--------|-------|
| 1화 | 일상+자조 유머 | 공포(사고) → 초현실(사이공간) | 어색한 안도+수치 |
| 2화 | 물질적 안도(목욕+방) | 불안(에리나 방문) | 오싹한 미스터리(첨탑 빛) |
| 3화 | 기대(훈련) | 굴욕(완전 무능) | 으스스한 불안(모르데우스) |
| 4화 | 호기심(도시 탐방) | 흥미(마르코) → 공포(에리나 폭로) | 절망(탈출 불가) |
| 5화 | 체념(가택연금) | 극한 고통(의식) | 공허한 각성(솔라리스 도착) |

→ **패턴**: 매화 시작은 비교적 가볍게, 끝으로 갈수록 무거워진다. 이 곡선을 유지할 것.
