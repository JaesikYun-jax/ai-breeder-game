## Part 1 — 연명 (Survival) — ch001~ch025

**부 테마**: 죽기 싫어서 발버둥 + 초반 자원 확보로 도약
**부 목표**: 각성 → 이리스 조우 → 탈출 액션 → 변두리 정착 → 미다스 씨 신분 → 1차 신체 개조
**시간**: D+0 ~ D+120 (약 4개월)
**끝 훅**: 마도 공학자 길드의 존재를 알게 된다.

### Part 1 재설계 핵심 변화 (vs 기존)
- 탈출을 **마차 짐칸 수동 은신 → 옌스 처치·자원 탈취 액션**으로 변경
- 탈출 시점 **ch010 → ch007~008로 앞당김**
- 초반 자원 확보로 "찢어지게 가난" 구간 축소 (ch011~017 단축)
- 호문클루스 신체 스펙 **ch007에서 시현** (강신우가 옌스 처치로 입증)

### ch001. 어디지, 이건
- **시점**: D+0
- **플롯 비트**:
  - **오프닝**: 강신우가 지독한 두통 속에서 눈을 뜸. 낯선 복장의 노인들. 알 수 없는 대화.
  - 노인들의 진단 마법: 강신우 몸 주변에 섬광 문양. "다이그노시스"
  - 소울 색 변화·오염 판정 대화
  - 텔레포트 실험 실패 언급 ("차원의 틈 이동 중 외부 물질에 오염된 것")
  - 양피지에 기록 → 노인들 퇴장 → 강신우 다시 정신 잃음
  - **엔딩**: 어두운 방에서 깨어남. 자기가 누군지 정리하려 애씀. "나는 호문클루스 603호... 아니, 강신우다. 죽었을텐데?"
- **감정 아크**: 혼란 → 고통 → 관찰 → 공포(기억 회복) → 결단 씨앗
- **내면 톤**: 사용자 원본 샘플 표준 준수. 더듬거림·말줄임표·실시간 반응
- **배치 복선**: FS-003 (소울 색 오염 판정 — 연금술사들 대화에서 자연 배치)
- **핵심 장면**: 진단 마법 시 고통. 그리고 후반 어두운 방에서 자기 정체성 확인하는 내면 독백 (사용자 원본 샘플과 동일 결)
- **수치 박스 없음** (이 화는 상황 파악이 우선)

### ch002. 위장
- **시점**: D+1 ~ D+10
- **플롯 비트**:
  - 실험실 생활 시작. 연금술사들의 반복 검진
  - 강신우가 "실패한 호문클루스" 연기 시작. 감정·반응 제거
  - 603호의 단편 기억들이 조금씩 정리됨 (호문클루스 제조 공정·산업 구조)
  - 신체 감각 놀라움: 생각보다 몸이 가볍고 단단하다는 걸 깨닫는 순간 → 호문클루스 신체 스펙 복선 씨앗
  - 연금술사들이 결국 "실험 실패" 결론 → 폐기동 이송 결정
- **감정 아크**: 긴장 → 관찰·위장 → 신체 자각 → 결단(일단 살아남기)
- **핵심 내면**: "일단 실험이 실패한 것으로 계속 위장해야 된다" (사용자 원본 샘플 직접 계승)
- **수치 박스**: 없음

### ch003. 폐기동
- **시점**: D+11 ~ D+30
- **플롯 비트**:
  - 폐기동 이송. 울타리 안에서 다른 호문클루스들과 섞임
  - 식물 같은 홈들의 상태 관찰 → 강신우의 복잡한 심경 (연민·공포·계산)
  - 엘릭서 정기 주입 체험. 배터리 같은 감각 (직접 프로그래머 용어 아닌 **기질**로 묘사)
  - 폐기동 관찰: 마차 주기, 경비, 시설 배치
  - 밤마다 고독한 관찰자 (잠 필요 없음)
- **감정 아크**: 절망 → 적응 → 관찰 → 체념과 탐색 혼재
- **배치 복선**: FS-022 (폐기동 동료들의 운명 — 식물 상태 + 자연사)
- **수치 박스**: 필요 시 간결하게 1회 (수명·잔고·주입 주기)

### ch004. 591호
- **시점**: D+30 ~ D+35
- **플롯 비트**:
  - 폐기동 배회 중 갑자기 어깨에 손이 올라옴 — 591호(이리스)
  - 그녀의 속삭임: "너, 소울 잃어버린 거 아니지?"
  - 이리스의 수다. 강신우는 여전히 무반응 연기
  - 이리스의 핵심 폭로: "엘릭서를 더 주입하면 좀 더 오래 살 수 있을 텐데..."
  - 강신우 내면에서 "돈만 있으면 안 죽는다"는 방정식이 조용히 완성됨
  - 강신우 연기 살짝 풀림 → 이리스 감지 → 공조 암묵적 성립
  - 첫 외부 대사: "…고마워"
  - 번호 아닌 이름 부여: "이리스"
- **감정 아크**: 경계 → 충격 → 깨달음 → 연대
- **배치 복선**: FS-001 (이리스 자아 근원 씨앗), FS-032 (멜로디)
- **감정 톤**: 이 화는 유머 거의 없음. **비밀을 공유한 두 존재**의 조용한 연대
- **핵심 대사**:
  - 이리스: "폐기동 홈들은 다 이렇게 조용해. 안 그래도 짧은 생인데, 재미가 하나도 없어."
  - 이리스: "엘릭서를 더 주입하면 좀 더 오래 살 수 있을 텐데..."

### ch005. 어떤 남자 (옌스 등장)
- **시점**: D+35 ~ D+50
- **플롯 비트**:
  - 이리스와의 공조 심화. 낮엔 연기, 밤엔 속삭임
  - 시설 정보 교환 (마차 주기·경비·구조)
  - **옌스 첫 등장**: 야간 당직 연구원. 폐기동 순찰 중 다른 호문클루스를 발로 차고 조롱
  - 강신우 최초의 분노. 하지만 일단 관찰만
  - 이리스도 옌스의 괴롭힘 경험 암시 (여성형이라 노골적 대상)
  - 옌스의 패턴 분석: 순찰 시간, 휴대품, 동선, 권한 (시체 처리조·엘릭서 창고 열쇠)
  - **엔딩**: 강신우 결심. "탈출, 동시에 저 놈을 없애자."
- **감정 아크**: 관찰 → 분노 → 냉정 → 결단
- **감정 톤**: 도덕적 긴장과 실용적 계산의 공존
- **엔딩 훅**: 옌스 처치 계획의 그림자

### ch006. 준비
- **시점**: D+50 ~ D+55
- **플롯 비트**:
  - 옌스 처치 + 자원 탈취 + 위장 도주의 시나리오 수립
  - 옌스의 순찰 시간·습관·약점 정밀 관찰
  - 시체 처리 용해조의 작동 원리 이리스가 설명 (그녀가 기억 파편으로 안다)
  - 이리스는 엘릭서 창고·자금고 위치 파악
  - 옌스가 특정 호문클루스를 또 괴롭히려는 낌새
  - 강신우·이리스의 역할 분담 확정
- **감정 아크**: 냉정한 기획 → 긴장 고조 → 결의
- **엔딩 훅**: 내일 밤이 실행일

### ch007. 처치와 탈취
- **시점**: D+55 저녁 ~ 밤
- **플롯 비트**:
  - **옌스가 이리스를 괴롭히려 접근하는 순간** → 강신우 매복 → **격투 → 처치**
  - 이 씬에서 **호문클루스 신체 스펙 처음 시현** (맨몸으로 성인 남자 2~3배 근력)
  - 고어 절제하되 타격감은 명확
  - 시체를 용해조에 투입 (증거 소멸)
  - 옌스의 신분증·방 열쇠로 엘릭서 창고 진입 → 엘릭서 상당량 탈취
  - 연구 자금·귀금속 탈취
  - 옌스가 "자원 훔쳐 도주"한 것처럼 증거 조작
- **감정 아크**: 각오 → 폭발 → 냉정한 처리 → 피로 + 자각("나는 이제 사람을 죽인 몸이다")
- **신체 시현**: 강신우가 자기 손을 내려다보며 "이 몸이 이 정도였구나" 하는 짧은 내면
- **엔딩 훅**: 새벽. 정문을 향해 걷는 두 사람

### ch008. 바깥
- **시점**: D+56 새벽 ~ D+65
- **플롯 비트**:
  - 정문 통과 (옌스의 신분증 + 야간 경비 허술)
  - 숲으로 피신 → 로브·옷 갈아입기
  - 추격 없음 확인 (길드는 옌스 추적에 집중)
  - 처음 보는 바깥 세상 — 강신우의 짧은 감회 (전생 회상 2~3문장)
  - 이리스와 도피 경로 의논
  - 탈취한 자산 점검: 엘릭서 수 병 + 금화 수백 닢 + 귀금속
  - **자원 확보 = 초반 생존 보장**
- **감정 아크**: 긴장·도피 → 안도 → 다음 계획 (아르테미온 진입)
- **수치 박스**: 확보 자산 간결 정리 (1회)
- **엔딩 훅**: 수도 아르테미온 방향으로

### ch009. 도시의 냄새
- **시점**: D+65 ~ D+70
- **플롯 비트**:
  - 아르테미온 외곽 진입
  - 이세계 도시 첫 관찰 (냄새·소음·사람)
  - 빈민굴·변두리 탐색
  - 호문클루스 신분 은폐 (로브·후드)
  - 낯선 곳에서의 밤
- **감정 아크**: 긴장·관찰 → 피로 → 조심스러운 희망
- **엔딩 훅**: 변두리 지역에서 적당한 거처를 찾는다

### ch010. 오두막
- **시점**: D+70 ~ D+75
- **플롯 비트**:
  - 변두리 외곽 낡은 석조 오두막 발견. 임대 협상 (옌스 자금 일부 사용, 금화 30닢 선납)
  - 지하실 발견 (추후 작업실 가능성)
  - 오두막 내부 정리
  - 이리스·강신우 첫 "집"의 조용한 안도
- **감정 아크**: 지친 끝의 안도 → 시스템 가동 ("이제부터 뭘 할까")
- **엔딩 훅**: 강신우가 이세계에서 할 일을 진지하게 고민하기 시작

## Part 2 — 개발 (Invention) — ch026~ch050

**부 테마**: 프로그래머 스킬 = 돈
**부 목표**: 마도 공학자 길드 입문 → 첫 히트 → 월 흑자 → 2차 신체 개조
**시간**: D+120 ~ D+300
**끝 훅**: 연금술사 길드가 "이상한 호문클루스" 소문 추적 시작.

### ch026~030. 입문기
- 26. 기어 거리 정찰
- 27. 견습 시험 (그레고르와 첫 조우)
- 28. "자네, 어디서 이런 걸 배웠나?" (그레고르의 감탄)
- 29. 견습 등록 완료. 강신우의 마도 공학자 길드 공식 진입.
- 30. 길드 내 동료·경쟁자 관찰. 수련 시작.

### ch031~035. 첫 히트
- 31. 일상 마도구 관찰 (낡은 설계). 강신우의 리팩토링 아이디어.
- 32. "자동 세탁 마도진" 프로토타입. 이리스가 테스트.
- 33. 길드 내 발표. 조롱 받음. 강신우 분노 억제.
- 34. 중산층 상인의 관심 → 첫 판매.
- 35. 히트작 확정. 주문 쇄도. 첫 금화 100닢 수익.

### ch036~040. 월 흑자
- 36. 이리스도 미다스 씨 밑에서 "조수 호문클루스"로 공식 등록.
- 37. 생산 라인 고민 (혼자로는 한계). 그레고르와 협업 제안.
- 38. 그레고르의 개인 공방 방문. 고대 설계도의 존재 힌트.
- 39. 강신우의 마도구 성능 압도적 → 경쟁사 견제 시작.
- 40. 월 수입 금화 100닢 돌파. 월 엘릭서비 금화 50닢. **처음으로 흑자**.

### ch041~045. 귀족 고객
- 41. 귀족 시장 진출 결심. 고급 마도구 라인 기획.
- 42. 세라핀 드 아르곤 첫 방문 — 오두막에서 603호와 첫 조우 (연민 단계).
- 43. 세라의 재방문. 603호 옆에서 혼잣말하기 시작.
- 44. 귀족용 "온도 조절 마도구 침대" 설계. 아르곤 후작 대량 주문 임박.
- 45. 세라의 아버지 아르곤 후작과 첫 대면. 은근한 정치적 시험.

### ch046~050. 의심의 그림자
- 46. 아르곤 후작 대량 주문 성사. 금화 2,000닢 계약.
- 47. 그레고르가 고대 설계도를 정식 공개. 2차 개조 연구 시작.
- 48. 에녹 칼러스가 "이상한 마도구 소문"을 듣고 조사 시작.
- 49. 강신우가 표식 감청을 재점검 → 길드 연락 증가 확인. 위험 신호.
- 50. **2차 신체 개조 (Stage 2)**. 연비 1년/병. 마법진 슬롯 4개 구축.
  - **Part 2 엔딩 훅**: 에녹이 폐기동 기록을 다시 뒤지기 시작했다는 소식.

## Part 3 — 정치공작 (Market Manipulation) — ch051~ch075

**부 테마**: 엘릭서 값 좀 내려달라고
**부 목표**: 호문클루스 산업 규제법 통과 → 시장 폭락 → 3차 개조 → 세라 정체 발각
**시간**: D+300 ~ D+540
**끝 훅**: 연금술사 길드의 전면 공격 개시.

### ch051~055. 공방 독립
- 51. 공방 독립 선언. 장인구에 정식 점포. 이리스 서브 운영.
- 52. 그레고르 가문 복원 약속 (세라 후작가와 정치적 동맹 밑그림).
- 53. 세라의 자각·갈등 단계 진입. 방문 중단.
- 54. 강신우의 계산: "세라 = 연간 금화 500닢 가치, 그런데 왜 공허한가?"
- 55. 엘릭서 시장 구조 분석 완료. 규제법 아이디어 착상.

### ch056~060. 정치공작 설계
- 56. 3파 동원 전략 수립: 종교계 / 장인 길드 / 반길드 귀족.
- 57. 은빛별 교단 카타리나 벨제에게 익명 자료 제공.
- 58. 장인 길드 연합에 "호문클루스 노예가 일자리 뺏는다" 통계 유출.
- 59. 할베르크 공작가 정적 귀족에게 접근. 리노의 레버리지 활용.
- 60. 정치공작 본격 점화. 귀족원에 청원 올라옴.

### ch061~065. 법안 통과
- 61. 귀족원 청원 심의. 강신우는 미다스 씨로서 "합리적 제안" 제출.
- 62. 연금술사 길드의 반격. 에녹 직접 나섬.
- 63. 여론전. 교단 설교, 장인 시위. 강신우의 정보 조율.
- 64. 세라핀이 아버지 아르곤 후작 마음 돌리기 시도.
  - **회수 복선**: FS-013 (후작의 이중성) 초기 힌트
- 65. **호문클루스 산업 규제법 통과**. 강신우 승리.

### ch066~070. 시장 폭락 & 3차 개조
- 66. 엘릭서 수요 급감 시작. 시세 20% 하락.
- 67. 1급 엘릭서 대량 매집. 강신우의 장기 비축.
- 68. 이리스 자아 근원 부분 공개 (FS-001 회수 1단계).
- 69. 시세 40% 폭락 달성. 강신우의 월 비용 절반.
- 70. **3차 신체 개조 (Stage 3)**. 연비 3~5년/병. 갑주용 슬롯 확장. 소울 색 위장 완성.

### ch071~075. 발각과 폭로
- 71. 에녹의 조사가 "미다스 씨 = 603호의 주인 = 정치공작 배후"로 좁혀짐.
- 72. **암살단 습격**. 미다스 씨 오두막이 타깃. 강신우 부재, 세라와 603호(강신우)만 있음.
- 73. 강신우의 반격. 마도 갑주 즉석 작동. 암살단 제압.
- 74. **세라핀 정체 발각**. 충격·사랑·분노 폭발. "당신이... 당신이었어?"
- 75. 세라의 각오 — "함께 하겠다". 공범 선언.
  - **Part 3 엔딩 훅**: 에녹 "이제 확신했다. 전면전이다."

## Part 4 — 합성 (Synthesis) — ch076~ch100

**부 테마**: 진짜 영생 = 독점 해체 + 사랑
**부 목표**: 인공 합성 성공 → 시장 붕괴 → 에녹 타도 → 은둔
**시간**: D+540 ~ D+780
**끝 훅**: 엔딩

### ch076~080. 합성 연구
- 76. 비밀 연구소 설립 (세라·그레고르 공동 투자).
- 77. 세 단서(그레고르 설계도 + 리노 정보 + 세라 어머니 노트) 존재 인식.
- 78. 세라 어머니 사건 전모 (FS-012 회수). 세라의 각오 재확인.
- 79. 그레고르가 길드 보복으로 중상 (FS-010 확장).
- 80. 첫 합성 시도. 실패. 강신우의 암흑기.

### ch081~085. 돌파
- 81. 이리스의 각성 (FS-015) — 마법사 잠재력으로 실험 돌파 기여.
- 82. FS-001 완전 회수 (이리스 자아 근원). 고대 자장가 연결.
- 83. FS-024 회수 (은빛별 교단 = 마도 공학 창립자 후손). 동맹 강화.
- 84. 세 단서 조합 완성. 이론적 돌파구.
- 85. **인공 엘릭서 첫 합성 성공**. 강신우·이리스·세라의 감정 폭발.

### ch086~090. 안정화
- 86. 합성 공정 안정화. 대량 생산 준비.
- 87. 강신우 Stage 4 달성 (자가 순환). **실질 영생**.
- 88. 이리스도 동일 수준 개조 완료.
- 89. 합성법을 어느 수준까지 공개할지 결정. 강신우의 "시장 붕괴" 계획.
- 90. 합성법 암시장 유출 (리노 경유).

### ch091~095. 최종전 서막
- 91. 엘릭서 시세 90% 폭락. 연금술사 길드 재정 파탄.
- 92. 에녹의 진실 추적 (FS-011 활용). 소울 색 해석 재검.
- 93. 에녹의 공식 선전포고. 왕실 인증 추적 영장 발부.
- 94. 세라의 마지막 가족 설득 — FS-013 최종 회수, 아르곤 후작 중립 선언.
- 95. 강신우의 사망 위장 계획 수립.

### ch096~100. 엔딩
- 96. **에녹과 최종 대결**. 연금술사 대주문 vs 마도 갑주. 1:1 대결.
- 97. 에녹 사망. 마지막 대사: "당신이 연금술사였다면..." (FS-025 회수).
- 98. 연금술사 길드 붕괴. 사회적 파급.
- 99. 미다스 씨·603호·591호 공식 사망 처리. 대륙 북부로 이동.
- 100. **엔딩 장면** — 별장, 세라·이리스와 함께. 강신우가 통장 대신 일기를 쓴다.
  - FS-030 회수 (편의점 커피) — 세라가 건넨 커피가 "그 맛 비슷해."
  - FS-002 회수 (미다스 이름) — 세라: "당신은 미다스가 아니야."
  - 마지막 대사: "드디어, 일 안 해도 안 죽는다."
  - *100화 끝. 完.*
