# 캐논 퀵레퍼런스 (Compact Canon for Pipeline)

> **목적**: /create, /polish 에이전트가 정본 12개를 매번 읽지 않고 이 한 문서로 충분하게 만든다.
> **사용 규칙**: 에이전트는 우선 이 문서를 읽고, 챕터별 특수 디테일이 필요할 때만 원본 정본 1~2개 추가 참조.
> **갱신 주기**: 신규 캐논 결정 시 즉시 반영. 본문 진전(아크 완료)마다 활성 복선/관계 곡선만 갱신.

## 0. 작품 핵심 (1줄씩) — 갱신 2026-04-20 (Arc 4 종료 시점)
- **장르**: 이세계 선형 멸망 저지물 (회귀·루프 X)
- **주인공**: 강지호 (28세 삼류 프로그래머 → 아젤리아 부왕). D급 재생 + 무재능 = 다중 체계 흡수
- **위협**: 9기둥 = 이계 차단 배리어. 소멸 시 → 배리어 off → 이계 괴물 침략 → 세계 병합
- **빌런 (3중 라인, 카이젤 ≠ 검은 로브)**:
  - **천인 평의회** (셀레스티아) — Part 3 최종 적
  - **카이젤 황실** — Arc 6 직접 적. 모르데우스와 300년 동맹
  - **검은 로브 네트워크** — 이름 없음, 표식 = 깨진 모래시계 모양. ch023·ch027 잠입자/사보타주 도주자
- **엔딩 (Part 3)**: 모든 기둥 치유 → 본인 수명 전부 소진 → 죽음 (해피&새드)
- **현재 진행**: Arc 4 (ch021~028) 완료. Arc 5 진입 직전.

### 메인 서사 방향 (2026-04-25 강화)
- **주인공 무쌍**: 정령술 + 문신 코딩 + D급 재생 + 기후 프로그래밍(FS-A11) 통합 운용으로 압도적 전투력 점진 발현
- **나라 부강**: 아젤리아·솔라리스 → 카이젤·프로스트헬·용화국·리베르타 외교·기술·정령석 양산 확장
- **전쟁 승리**: Arc 6 카이젤 내전 / Arc 7 천인 평의회 첫 충돌 / Arc 8 용화국 균열 / Arc 9 이계 침공 첫 격퇴 / Part 3 클라이맥스 — 결정적 승리 시연

- **마르코는 부차 안내자**: 메인이 아닌 곁가지. 빈도 아크당 0~1회로 제한.

- 비언어: 입꼬리 0.5초, 어깨 미세 굳음, 문틀 어깨 부딪침
- "푸시업 200" 펀치 — 코미디 비트 자원

### 모르데우스 (수감, ch019 이후)

## 4. 마법·메커닉 (RETCON 정본)

### 재생 D급
- 신경 세포까지 완벽 복구. 100번째 화상 = 1번째 동일 강도 (고통 면역 불가)
- 장점: 무한 훈련 가능
- 약점: 머리 분리 등 극단 상황 사망

### 정령술 (솔라리스 한정)
- 9화 솔라리스 화염 기둥 파편 = 정령
- 사용 = 기둥 약화 (장기적)
- **계약 = 솔라리스 사막 한정** (외지인 거의 불가, 지호는 외부인 최초)
- **수명 비대칭**: 일반 술사 머리 탈색·수명 단축 큼 / **지호는 D급 재생 덕에 미미** → 다중 정령 가능
- 단, 기둥 직접 치유에는 본인 수명 직접 지불 (Part 3 회수)
- 정령 다른 지역 가져가도 페널티 (지호는 미미)
- **카이젤이 못 가지는 이유**: 솔라리스 사막 한정 + 신민 수명 강요 불가

### 강지호 정령 보유 (Arc 4 ch028 종료 시점)
- 화염 정령 (ch008 계약) — 손바닥 인장

- 풍 정령 (ch021 회상에서 추가) — 발목 안쪽 + 왼쪽 관자놀이 머리카락 끝 빛바램
- **3정령 동시 운용 사례**: ch027 모래바람 진정 시연 (풍+화염+문신 코딩 5함수), ch028 정령석 사치품 봉인 (화염+번개+회로 사고)

   - 그 외 단순 언급·관조·풍경 묘사는 **불가**.

3. **자동 발동 메커닉 (FS-A12 신규):**
   - **지호 의지 X** — 부르려 해도 안 됨. 통제 불가.
   - **트리거**: 생명 위협 / 중대 결정 / 치명적 위기 (셋 중 하나 충족 시 무작위 발동).
   - **효과**: 재생 가속 OR 판단 보조 (또는 둘 다 동시).
   - **대가**: 잔량 미세 감소 + 본인 수명 (Arc 7~ 흰머리 누적과 직접 연결).
   - **묘사 톤**: "세상이 한 박자 늘어졌다" / "초가 길어졌다" 같은 짧은 1~2문장 신호. **박자 단위 디테일 금지** — 모호하게.

4. **잔량 한 알 감소 묘사 절제**: 큰 플롯 전환점에만 (Arc당 0~1회). 매 화 알 떨어짐 X.
