# tone-and-style.md — 톤 / 문체 / 구조 규칙

## 1. 시점

### 기본
- **주서술자**: 진무혈 1인칭 내면 독백.
- **보조 서술자**: 3인칭 제한시점 (하은, 서연, 빌런, 환자 가족).

### 다시점 전환 규칙
- `---` 구분선으로 명시.
- 한 챕터에 최대 3시점. 시점당 최소 1,000자.
- **5화 연속 주인공 단독 시점 금지**. 빌런/관찰자/조연 시점 의무 삽입.

### 시점 배분 표준 (챕터당)
| 구성 | 주인공 | 조연/히로인 | 빌런 | 관찰자/환자 |
|------|--------|-----------|------|-----------|
| 일상/수련 | 60~80% | 20~30% | - | - |
| 카타르시스 이벤트 | 20~30% | 10~20% | - | 40~50% |
| 전투 | 60~70% | 10% | 20~30% | - |
| 빌런 모의 | 30~40% | - | 40~50% | - |

## 2. 문장 스타일

### 기본
- 단문 위주. 호흡 빠른 웹소설 리듬.
- 길이 있는 문장은 분석·추론 장면에만.

### 상황별
- **전투/사망/각성**: 매우 짧음. 파편화.
- **감각 재생(수분·기)**: 구체적 신체 감각. 추상 감정 금지.
- **회상 (천 년 전)**: 2~3문장으로 압축. 길어지면 현재 긴장감 상실.
- **유머**: 짧은 자조·메타 코멘트. 펀치라인은 짧을수록 터짐.

## 3. 톤 공식 — 카타르시스 구조

### 원칙
매 챕터 최소 1회, **어그로 빌드업 → 해소** 낙차 배치.

### 어그로 빌드업
- 피해자 시점 상세 묘사 (환자 가족, 서민, 약자).
- 권력·시스템의 횡포 디테일 (응급실 방치, 재벌 협박, 공무원 단속).
- 호소 묵살 장면 → 독자 분노 유발.
- **빌드업 분량이 챕터의 30% 이상** 차지해야 해소가 짜릿함.

### 해소
- 주인공 직접 개입. 시스템/경찰에 맡기지 않음.
- 악역 반응 3단계: 무시 → 현실 직시 → 공포/굴복.
- 증인 반응 삽입 (피해자 감사, 주변 보호자 야유).
- **너무 빠르지 않게**. 어그로가 충분히 쌓인 뒤 해소.

### 챕터당 빈도
- 최소 1회, 최대 2회. 남발 시 긴장감 상실.

## 3-B. 위기 층위 규칙 (적의 강함 연출)

**원칙**: 주인공 일방 승리 금지. 매 전투 회차에 **"진무혈이 쉽게 못 뚫는 벽"** 하나 이상 배치.

### 적 강자 3등급

| 등급 | 기준 | 대표 인물 |
|------|------|----------|
| **상급** (Tier S) | 진무혈 전성기(9할)가 아니면 불리. 1부에선 대등 불가. | 리하오란 (당 가주), 아베노 교이치(결계 조건부) |
| **중급** (Tier A) | 진무혈 5~7할과 호각 ~ 소폭 열세. 단독 상대 시 시간 끌림. | 장원지에, 아베노 고위 음양사, 백가 보안 총책 |
| **하급** (Tier B) | 진무혈 3할+ 상태에서 다수라도 제압 가능. 하지만 수분·기 소모 유발. | 일반 자객, 기를 쓰는 관리인 |

### 위기 연출 공식 (전투 회차 필수)

**어느 하나 이상 포함**:
1. **숨은 고수**: 표면 적 + 배후 고수 구조. 표면 적을 제압하는 동안 배후가 조연 공격.
2. **수분 고갈**: 장기 전투로 재생력 한계. 물병을 찾거나 비까지 기다리는 장면.
3. **조연 부상**: 설경·하은·서연 중 1인이 의미 있는 부상(단순 찰과상 아님).
4. **환경 압박**: 결계·정전·방화·민간인 인질 등 전투 외 변수로 판단 제약.
5. **다중 전선**: 같은 시간 두 장소 이상에서 동시 위기 발생.

### "뚫어내지 못한 첫 상대" 필수

1부 Arc 3에 **진무혈조차 승리하지 못한 상대**를 반드시 1명 배치 (현재 설계: 리하오란, 28~30화).
- 무승부·잠정 철수·보류 중 택일.
- "다음에 만나면 결판 내자" 식 여운.
- 2부 메인 적으로 승계.

### 스케일 인플레이션 규칙

| 회차 | 적 규모 | 공격 형태 |
|------|--------|----------|
| 14화 | 소규모 팀 (미끼 10 + 고수 1) | 침투 |
| 16~17화 | 인질 작전 (김재원 + 별장 경비 10) | 인질극 |
| 20화 | **삼파 연합 총력전** | 드론+결계+저격+방화+정전 동시 |
| 26화 | 중규모 정예 (자객 5~8) | 도시 내 거점 공격 |
| 29~30화 | **가주급 단독 + 소수 정예** | 일대일 클라이맥스 |

**핵심**: 규모가 아니라 **"상대가 얼마나 강한가"**를 보여주는 것이 몰입의 열쇠.

## 4. 섹시어필 가이드

### 원칙
진무혈의 신체적 매력·카리스마를 **간접적으로**, **조연 시점으로** 보여준다.

### 금지
- "그의 근육이 섹시했다" 같은 직접 서술.
- "여자들이 그를 욕망했다" 같은 욕망 서술.
- 성적 노골 묘사.

### 권장
- 조연 시점의 시선 끌림, 귀 붉어짐, 말 더듬음.
- 체형/의복 디테일을 행동 묘사로 암시 (젖은 상체, 땀 한 방울, 삭은 옷 사이로 드러난 선).
- 여성 캐릭터의 사고가 잠깐 흐트러지는 순간 독백.

### 히로인 적극 어필 (서연)
- 신체 접촉 시도 (허리·가슴에 기대기).
- 대담한 발언 ("제가 책임지겠습니다", "왕자님은 공짜로 안 도와주세요").
- 다른 여성 등장 시 견제 대사·행동.

### 질투 구도 (하은 vs 서연)
- 작은 견제. 대놓고 싸우지 않음.
- 각자 어필 방식 차이로 캐릭 구분.
  - 하은: 실무 기여·지적 교류.
  - 서연: 재력·신체 접촉·대담함.
- 진무혈은 둔감/의도적 무시로 반응.

### 빈도
- 챕터당 2~4회. 본 플롯 방해 금지. 양념 수준.

## 5. 내면 독백 / 대사 규약

### 내면 독백 (`'작은따옴표'`)
- 진무혈의 사고·감정.
- 분석적 (기 흐름 체크, 상대 평가).
- 냉소적 (자기 비하 유머).
- 메타적 (천 년 전과의 대비).

### 대사 (`"큰따옴표"`)
- 각 캐릭터 보이스표 준수 (voice-guide.md).
- 한 문장에 두 캐릭터 대사 금지.

### 메아리 대사 금지
조연이 주인공 말을 단순 반복 금지. 각자 맥락에서 반응.

## 6. 금지 표현

| 패턴 | 제한 | 대체 |
|------|------|------|
| `—` em dash | **0회** | 쉼표, 마침표, `...` |
| `이었다` | 5회 이하 | `~였다`, `~이다` |
| `고 있었다` | 3회 이하 | `~했다` |
| `것이었다` | 1회 이하 | `~였다`, `~셈이었다` |
| `미라` 반복 | 3회 이하 | 감각 묘사로 우회 |
| `섹시` 직접어 | 금지 | 조연 반응 묘사 |

### 6-A. 메타 서술 절대 금지 (제4의 벽)

본 세계관은 **작중 인물에게는 완결된 현실**. 독자·작가의 시점에서 서술하는 메타 문구는 즉시 삭제 대상.

| ❌ 금지 예시 | ✓ 대체 |
|------------|--------|
| "그리고 1화의 펜트하우스가 시작된다" | 작중 시간의 자연스러운 장면 묘사로 끝맺음 |
| "다음 화에서..." | 훅 대사·행동으로 암시 |
| "이 순간이 훗날 전설이 된다" | 작중 인물의 예감·직관으로 표현 |
| "N화에서 회수될 복선" | 서사 안에서 자연스럽게 배치 |
| "독자가 보기에..." / "작가의 의도..." | 어떤 형태로도 금지 |

**검출 패턴** (rule-checker·polish 체크 항목):
- `N화` · `다음 화` · `이전 화` 원고 본문 내 사용
- `에필로그` · `프롤로그` · `서장` 본문 서술어로 사용
- `이제 ~가 시작된다` 류 연출적 미래 선언
- "~할 것이다"가 작중 인물이 아닌 서술자 전지 시점 예고인 경우

**예외**:
- 회상 장면에서 "그때" / "천 년 전" 등 시간 지시어는 허용 (작중 인물 기억)
- 챕터 끝 `*N화 끝. 다음 화: 제목*` 이탤릭 예고문은 **메타 영역(파라텍스트)**이라 허용 — 본문과 분리

이 규칙을 어긴 서술이 발견되면 **polish 자동 감점** + 즉시 삭제·대체 대상.

## 7. 아이러니 / 낙차 규칙

### 코미디-고통 낙차 (dclass-hero 계승)
본 프로젝트에서는 "현대-고전 낙차"로 응용:
- 현대 일상 묘사 → 갑자기 천 년 전 무공/봉인 감각 전환
- 진무혈의 정중한 옛 말투 × 현대인의 단순 반응

### 낙차 예시
- "교주라기보다는 문 앞에 서는 개에 가까웠다" (1화)
- "대학원 생활이 이렇게 막장이 될 줄은 몰랐다" (3화)
- "천마에게 한글을 가르치는 중입니다" (4화)

## 8. 회차별 톤 목표

|----------|-----|----------|---------|
| 각성·전환 | 서늘함 + 경이 | 보조 | 0~1회 |
| 적응·일상 | 코미디 + 호기심 | 보조 | 2~3회 |

| 전투 | 파편화 + 공포 | 주인공 개입 형태 | 0~1회 |
| 빌런 회차 | 서스펜스 | - | - |
| 로맨스 견제 | 은근함 | - | 3~4회 |
