# chapter-log.md — 화별 요약

## Arc 1 — 개봉 (ch001~ch010)

**Arc 1 한 줄 요약**: 천 년 봉인에서 풀려난 교주·부교주가 현대 서울의 무력 체제(백광) 앞에 맞서며 기록자(하은)+후원자(박순영/서연)+대척자(백준하/김재원) 13개 관계축을 확립, 1화 펜트하우스 루프 봉합 완료.

### 1화. 손님
- **시점**: 진무혈 1인칭 (현재 D+? — Arc 1 개봉편의 미래 시점, 2019년 4월)
- **공간**: 서울 120층 펜트하우스 (후일 L타워로 10화 회수)
- **등장**: 진무혈(가운/맨발), 설경(검은 양복·왼쪽 소매 핀 고정), 1093년 전 회상 인물 — 신교 장로 호연(역병 토로), 당 무림맹주 리커톈(봉인 제안), 왜국 음양사 아베노 교겐(봉인진 설계), 백가 부하(전화)
- **주요 사건**:
  - 펜트하우스 유리 너머 서울 야경. "천만 명이 산다". 남산 항공장애등 붉은 빛 → 천 년 전 불 회상 트리거
  - 1093년 전 회상: 발해 상경용천부 신교 본산 → 역병 → 본산 지하 장서각 금서 → 혈마공 수련 → 주화입마 3달 → 불사 획득 + 감정 무뎌짐 / 우물 독 발각(당 독문 출처) → 삼면 협공 → 교도 3천 명 인질 → 봉인 수락 → 석관 밀봉
  - 현재 귀환: 전화기 서툰 조작 후 "오게 놔둬라" 지시. 백가 무장 6명(기 사용 2 + 총 4) 접근 중
- **훅**: "잘 왔다, 백가." / "이번엔 묻히지 않는다."
- **설정 언급**: 전사 전체, 혈마공 원리·대가(감정 무뎌짐), 봉인진 구조(체내 수분 추출), 봉인 기간 1093년, 삼파 구성(당 무림/왜국 음양사/발해 백씨), 현재 삼파 후예 좌표(중국 뒷골목/일본 정재계 그림자/한국 재벌)

### 2화. 석관
- **시점**: 서하은 3인칭 (3주 전 = D-20)
- **공간**: 강남 영동대로 지하 30m 굴착 현장
- **등장**: 서하은, 김재원 교수, 현장 인부, 석관 속 진무혈
- **주요 사건**:
  - 석관 발견, 이두 경고문 해독 ("水를 近이 하지 말 것")
  - 미라 발견. 김 교수의 수상한 반응과 철수 지시
  - 하은이 실수로 물통을 쏟음 → 진무혈 각성
  - "나는 진무혈이다"
- **훅**: "1093년 동안 어둠 속에 있었던 눈"
- **설정 언급**: 발해 이두문, 물=재생 트리거, 봉인 경고 이유

### 3화. 조수석
- **시점**: 서하은 3인칭 (D-20, 밤)
- **공간**: 굴착 현장 → 하은의 아반떼 → 시내
- **등장**: 진무혈, 하은, (멀리서) 김 교수 + 백가 무인들
- **주요 사건**:
  - 사다리로 지상 탈출
  - 공사장 물탱크로 수분 보충 (10L)
  - 하은이 진무혈의 눈에서 연둣빛 발견
  - 1093년을 셀 수 있는 원리(혈마공) 설명
  - 백가 무인 접근 감지, 차로 도주
  - 하은 원룸으로 향함
- **훅**: "하은의 원룸에 천 년 된 천마가 굴러들어왔다"
- **설정 언급**: 연둣빛 홍채 첫 언급, 혈마공의 시간 인식 원리, 적 기 감지 능력

### 4화. 아리수
- **시점**: 서하은 3인칭 + 진무혈 대화
- **공간**: 하은의 관악구 원룸 → 서울대 → 다시 원룸
- **등장**: 진무혈, 하은, 김재원 (간접)
- **주요 사건**:
  - 원룸 적응: 수돗물, 샤워, 전등, TV, 트레이닝복
  - 진무혈의 샤워 3시간 후 대화 ("아리수가 최적이다")
  - 하은 등교. 김 교수 부재. 백씨 가계도 조사
  - 백광그룹 = 발해 백씨 후손 확인
  - 한글 수업 개시 (ㄱ부터)
- **훅**: "한글 수업 첫째 날"
- **설정 언급**: 아리수 설정, 혈마공 회복과 물의 관계, 백씨 가문 현대 위치, 한글 신조어

### 5화. 부교주
- **시점**: 서하은 3인칭
- **공간**: 하은 원룸 → 북한산 지하 → 강남 호텔 최상층
- **등장**: 진무혈, 하은, 설경 (첫 등장), 백가 관리인 4명 (간접)
- **주요 사건**:
  - 10일 경과. 진무혈 상태 회복, 하은 매력 인지 독백
  - 설경의 봉인 감지
  - 진무혈 단독 구출, 관리인 4명 도주
  - 설경 등장 (왼쪽 소매 빈 상태, 봉인 당시 상실 설명)
  - 백광 법인카드로 호텔 체크인
  - 하은이 이 순간을 기록 시작
- **훅**: "시작하자" / "백씨 본가. 어디에 있는지 알아내라" / "이미 알아냈습니다"
- **설정 언급**: 설경 공식 등장, 왼팔 상실 경위, 혈마공 vs 일반 무공 차이

### 6화. 남행
- **시점**: 진무혈 1인칭 + 백가 (백준하) 시점 + 하은 3인칭
- **공간**: 서울역 → KTX → 담양 펜션 / 백광 본사 지하 / 서울대 연구실
- **등장**: 진무혈, 설경, 서울역 노숙자, 백준하, 백씨 관리인들, 하은, 김 교수
- **주요 사건**:
  - 서울역 노숙자 구조. "천마님?" → 신교 후예 존재 암시
  - KTX 담양행
  - 백준하가 관리인 4명을 가족 호적으로 협박
  - 하은의 이중생활 개시 (김 교수 복귀에 순응 연기)
  - 담양 펜션 도착. 설경이 응급실 의사 부족 기사 제시
- **훅**: "두고 보자"
- **설정 언급**: 신교 후예 존재, 백씨 수뇌부 악랄함, 하은의 정보책 역할 확립

### 7화. 계곡
- **시점**: 진무혈 1인칭 + 하은 3인칭 + 마을 이장 시점
- **공간**: 담양 계곡 / 펜션 마당 / 마을 회관
- **등장**: 진무혈, 설경, 하은, 마을 이장, 약국집 박 약사
- **주요 사건**:
  - 계곡 수련 (3일차, 무공 2할 회복)
  - 감정 무뎌짐 자각 대화
  - 하은 주말 방문. 계곡에서 상의 탈의 상태의 진무혈 목격
  - 점심 식사. 응급실 기사 재상기. 진무혈 결심
- **훅**: "이 평온이 다음 주에 박살난다"
- **설정 언급**: 주화입마 후유증 설정, 하은의 점진적 매력 인식

### 8화. 응급실
- **시점**: 박순영 3인칭 + 진무혈 1인칭 + 하은 3인칭 + 엔딩 간호사 관찰
- **공간**: 박순영 국밥집(면 소재지) → 구급차 → ○○군립병원 응급실 → 주차장 → (엔딩) 응급실 당직 간호사실
- **등장**: 진무혈, 설경, 하은, 박순영(50대 국밥집 주인), 박철수(남편, 심근경색), 박지원(딸, 대학생 — 광주에서 상경), 응급실 간호사 1인(유일 근무), 행정실장(50대 남), 복도 보호자들, 당직 간호사(CCTV 녹화자)
- **주요 사건**:
  - 국밥집 주방에서 박철수 쓰러짐. 구급차 도착 15분 → 군립병원까지 40분
  - 군립병원 응급실: 유일 의사 타 병원 이송 중 + 길 막힘으로 복귀 불가. 외래 의사는 응급 자격 없음 → 사실상 무의사
  - 행정실장 "저희도 살아야죠" → 박순영 바닥에 주저앉아 애원. 박철수 거품 토함, 경고음
  - 진무혈 3인 도착. 설경이 접수대에 "보호자 아님. 지인일 뿐. 이의는 법적으로" 논리 설정
  - 진무혈 30초 치유 (심장 위 손). "이 자는 죽지 않소. 몇 달은 기름진 음식 금하고 걷기를 많이 해야 하오" — 사극 말투 1차 대중 노출
  - 행정실장 경찰 신고 시도 → 박순영 "40분 방치는 법 아니에요?" 반격 + 박지원 "저희 과 기자 지망 선배" 카드 + 복도 보호자 야유 → 굴복(전화기 내려놓음)
  - 박지원이 진무혈 손을 두 손으로 잡으며 이름 요청 → 진무혈 회피. 하은이 "또 시작이네" 예감
  - 주차장 이동 중 진무혈이 하은에게 먼저 "네 말이 맞았다. 고맙다" — 감정 복귀 징후 + 하은 호감 곡선 1차 상승
  - (엔딩) 당직 간호사가 CCTV 30초 회복 영상 혼자 3번 시청, 핸드폰으로 녹화. 다음 날 아침 박지원이 담양 맘카페에 "기적" 글 작성
- **훅**: "○○군립병원 응급실에서 기적을 봤어요" (글 제목 노출)
- **카타르시스 구조**: 시스템 부조리(방치→ 책임 회피→ 신고 위협) 3중 쌓기 → 민중의 역공 3중(피해자·가족·복도 군중) → 공권력 대표자 무너뜨림 — Arc 1 최초 완성형 사이다 패턴
- **섹시어필**: 박지원(21~22세, 여대생) 호감 표현 — 3파전 경쟁 구도(하은·서연·지원)의 첫 시드
- **설정 언급**: 진무혈의 치유 공식 데뷔, "의원이 아닌 자가 의원 일을 했다" 원칙, CCTV 영상 B-06 배치

### 9화. 기적
- **시점**: 진무혈 1인칭 + 환자/마을 관찰자 + 백준하 회장실 시점 + 진무혈 독백
- **공간**: 담양 펜션 마당 / 마을 어귀(정찰 차) / 백광 본사 16층 회장실 / 엔딩 마당
- **등장**: 진무혈, 설경, 마당 환자 다수(당뇨 합병증 남자·난소암 4기 여성·근육병 7세 남아와 모·휠체어 아이 엄마), 마을 이장 김 노인, 보건소 공무원 2명, 백준하, 비서 상무, 엔딩 — 백서연(휠체어, 간병인 동반)
- **주요 사건**:
  - 맘카페 글 4만 조회. 댓글 "그 분 어디서 뵐 수 있나요" 수백 건. 지원은 주소 비공개
  - 펜션 마당에 환자 30명 줄. 설경 4대 규칙 확립 (급한 사람 먼저 / 무료, 자발적 기부만 / 완치 아닌 진행 차단 / 미디어 비공개)
  - 진무혈 하루 15명. 치유할수록 혈마공 회복 가속 — 설경 해설 "기꺼이 받는 자의 기는 주는 자의 기에 더 잘 얹힙니다"
  - 근육병 7세 남아에게 즉흥 시(詩): "다리도 마음이 있다" → 진무혈 자기 자각 "내가 약해진 거 같다" → 설경 "회복되고 계신 겁니다" (감정 복귀 징후 + 복선 C-09 배치)
  - 마을 어귀 검은 승용차 정찰. 설경 판단 "건드리지 말고 평범한 치유사로 보이게 두십시오"
  - 보건소 공무원 2명 단속 시도. 당뇨 남자 발·난소암 여성·휠체어 아이 모 연쇄 증언 + 이장 "월급 깎일 일 없다" 일갈 → 공무원 퇴각
  - 백광 본사 회장실: 백준하가 정찰 영상 실시간 수신. 아베노 내일 서울 도착 / 리씨 가문 3명 모레 도착 / 삼파 회의 모레 저녁 7시 확정. 그리고 서연 이탈 사실 보고 — 손가락 떨림 + "내 손녀가 그 자를 봤다면" 약점 + 업보 교차 노출 (S-01, A-06, C-06, C-07 다층 배치)
  - 엔딩: 백서연 휠체어로 마당 진입. "안녕하세요, 선생님"
- **훅**: "안녕하세요, 선생님" (서연 첫 대사)
- **카타르시스 구조**: 치유 받은 환자들의 연쇄 증언으로 공권력 격퇴 — 8화 응급실 반격의 확장형 집단 버전
- **설정 언급**: 치유 = 수련 대체 패턴, 자의(自意) 수용 시 기 배가 원리, 업보의 기 치유 능력 예고, 삼파 회의 일정 확정, 백준하 약점 공식화

### 10화. 서연
- **시점**: 백서연 3인칭 + 진무혈 1인칭 + 설경/하은 관찰 + 엔딩 펜션 거실 3인 대화
- **공간**: 담양 요양 시설(전사) → 담양 펜션 마당 → 엔딩 펜션 거실
- **등장**: 진무혈, 설경, 하은(나무 그늘에서 원거리 관찰), 백서연(23세, 선천성 면역결핍), 간병인
- **주요 사건**:
  - 서연 전사: 태어날 때부터 병약, 스탠포드 치료 불가 판정, "왕자님" 자기 포지션(공주 아님), 할아버지 서재 침투 + 중학생 때 자회사 설립 계획서 사건
  - 서연 펜션 도착. 줄 시스템 보고 "야전 병원에 가깝다" 인식
  - 1시간 반 대기 → 서연 차례. 진무혈이 손목 맥진 중 "봉인의 잔기 = 업보의 기" 감지. "한 번으로 완치되지 않는다. 일주일에 한 번씩 와라" (A-05 배치)
  - 설경이 옆에서 "교주님"이라 부르게 유도 → 서연 "교주님!" 호칭 채택
