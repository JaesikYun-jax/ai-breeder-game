# 지역 간 연결 관계

아르카디아 9개 지역은 고립되어 있지 않다. 교역, 전쟁, 외교, 밀수 등으로 복잡하게 얽혀 있으며, 주인공은 회귀를 거듭하며 이 관계망을 파악하게 된다.

---

## 지정학적 관계도

```
                 [셀레스티아] (천공)
                 고립주의, 지상 경멸
                 ↓ (추락 잔해 교역)
                 
[프로스트헬]  ←긴장→  [카이젤 제국]  ←전쟁→  [아젤리아 왕국]
 북방 수렵          서부 군국주의          중앙 봉건제
  ↓ (모피 교역)       ↓ (마기공학 수출)       ↓ (성광 순례)
  |                   |                      |
  └───── [카즈모르] ──┘                      |
         지하, 광물 공급                      |
         (제국에 마석 원석 공급)               |
                                             |
[용화국] ←고립→ (대산맥 장벽)              [솔라리스]
 동방 선협                                  남방 부족
 자급자족                                   ↓ (향신료 교역)
                                            |
                                     [리베르타 군도]
                                     해상 교역 허브
                                     (모든 지역 연결)
                                            |
                                     [아비살] (심해)
                                     거의 단절, 군도만 접촉
```

## 양자 관계 상세

### 아젤리아 ↔ 카이젤 (대전쟁)
- **상태**: 현재 휴전 중이나 긴장 고조
- **원인**: 카이젤의 팽창 정책 vs 아젤리아의 대륙 중심부 사수
- **숨겨진 이유**: 양쪽 모두 상대 기둥의 힘을 노리고 있음
- **스토리 연결**: 아젤리아 루트에서 제국 침공 이벤트 / 카이젤 루트에서 왕국 원정 이벤트
- **회귀 활용**: 양쪽 루트 모두 경험하면 전쟁의 전체 그림이 보임

### 카이젤 ↔ 카즈모르 (공급 관계)
- **상태**: 표면적 동맹. 실질적으로 착취
- **내용**: 카즈모르가 마석 원석 공급 → 카이젤이 마기공학 제품 제공
- **갈등**: 카이젤이 채굴 할당량을 계속 올림 → 드워프들의 불만 축적
- **숨겨진 위험**: 과다 채굴 = 기둥 뿌리 약화 = 세계 불안정
- **스토리 연결**: 카즈모르 루트에서 제국의 무리한 요구 거부 이벤트

### 아젤리아 ↔ 솔라리스 (신앙 연결)
- **상태**: 우호적이나 문화 차이
- **내용**: 아젤리아 대신관이 "빛의 포교"를 남방에 시도 → 부족들은 정령 신앙 고수
- **갈등**: 표면적 선교 활동 뒤에 기둥 간 영향력 확장 의도
- **스토리 연결**: 솔라리스 루트에서 왕국 선교사와의 충돌

### 프로스트헬 ↔ 카이젤 (긴장)
- **상태**: 비공식 적대. 프림족은 제국 징집을 거부
- **내용**: 제국이 북방 자원(빙하석, 봉인지 에너지)을 노림
- **갈등**: 프림족의 독립 vs 제국의 팽창
- **스토리 연결**: 프로스트헬 루트에서 제국 정찰대와의 조우

### 리베르타 ↔ 모든 지역 (교역 허브)
- **상태**: 중립. 모든 세력과 교역
- **내용**: 유일하게 모든 지역의 물자가 오가는 곳
- **특이점**: 해적왕이 중립을 유지하는 이유 = 정보 수집 (전이자로서 세계를 이해하려는 욕구)
- **스토리 연결**: 리베르타는 다른 지역으로의 "경유지" 역할. 타 지역 NPC 만남 가능

### 셀레스티아 ↔ 지상 전체 (단절)
- **상태**: 거의 완전 단절. 천인은 지상을 경멸
- **내용**: 대몰락 이후 하늘에 틀어박힘. 죄책감을 우월감으로 위장
- **유일한 접점**: 추락한 섬 잔해에서 나오는 고대 유물 → 리베르타 상인이 거래
- **스토리 연결**: 셀레스티아 루트에서 지상과의 재접촉 이벤트

### 아비살 ↔ 리베르타 (유일한 접점)
- **상태**: 비공식 접촉. 세이렌 혼혈 항해사(루나)가 중개
- **내용**: 심해 산물(진주, 산호, 심해 광물)과 지상 물건 교환
- **갈등**: 아비살 왕실은 지상과의 접촉을 금지 → 루나처럼 추방되기도
- **스토리 연결**: 리베르타 루트에서 아비살 단서 발견 / 아비살 루트에서 지상 탐험 이벤트

### 용화국 ↔ 나머지 (대산맥 장벽)
- **상태**: 물리적 고립. 대산맥이 동방과 서방을 분리
- **내용**: 극소수 상인만 산맥 넘어 교역. 문화 교류 거의 없음
- **천명의 벽**: 기의 기둥이 만든 천명 시스템이 동방인의 "외부 유출"을 억제
- **스토리 연결**: 용화국 루트에서 산맥 너머 세계의 존재를 알게 되는 것이 각성 계기

## 크로스 지역 이벤트 (다회차 해금)

### 이벤트 해금 조건
최소 2개 지역의 진실 플래그를 보유한 상태에서 특정 지점 방문 시 발동

| 이벤트 | 필요 플래그 | 내용 |
|--------|-----------|------|
| 용사와 황제 | `truth_azelia` + `truth_kaizer` | 두 기둥의 진실을 아는 주인공이 전쟁의 진짜 이유를 양측에 폭로 가능 |
| 정령의 뿌리 | `truth_solaris` + `truth_kazmor` | 정령(기둥 파편)과 광맥(기둥 뿌리)의 연결 인식 → 세계 구조 이해 |
| 해적왕의 고백 | `truth_liberta` + 아무 진실 1개 | 레온이 주인공에게 현대 세계 이야기를 상세히 공유 → 공통점 발견 |
| 봉인과 관리자 | `truth_frosthel` + `bond_blackrobe` | 봉인된 존재 = 10번째 기둥 관리자 = 검은 로브 여인의 본체 |
| 천인의 고백 | `truth_celestia` + `truth_yonghwa` | 천명 시스템과 대몰락이 같은 뿌리(세계수 분할)임을 이해 |
| 심해의 기억 | `truth_abyssal` + `loop_10` | 여왕의 전생 기억과 주인공의 회귀 기억이 공명 → 시간의 기둥 정보 |

## 마르코의 교역 네트워크

떠돌이 상인 마르코는 모든 지역에 출현하며, 지역 간 아이템 교류의 핵심.

### 마르코에게 살 수 있는 것
- **일반 물품**: 각 지역 특산품 (어디서든 구매 가능)
- **정보**: 다른 지역에서 일어나고 있는 사건 힌트
- **특수 아이템**: 회귀해도 유지되는 아이템 (마르코만 판매)
  - "기억의 결정" — 메모를 저장하여 다음 루프에 전달
  - "세계수의 잎사귀" — 기둥 에너지 감지 범위 확장
  - "시간의 모래" — 짧은 시간 되감기 1회 (선택지 재선택)

### 마르코의 가격
- 돈이 아닌 **"이야기"** 로 거래. 주인공이 경험한 이야기를 들려주면 물건을 줌
- 같은 이야기는 한 번만 통함 → 다양한 경험을 해야 더 많은 것을 살 수 있음
- **메타적 의미**: 플레이어가 다양한 루트를 경험할 동기 부여
