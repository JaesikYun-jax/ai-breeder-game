```
[배경: bg_azelia_summoning_hall / 대낮]

귀족들이 고개를 숙인다.

[배경 전환: bg_azelia_throne_room / 대낮]
[이펙트: 화면 슬라이드 전환]
[SFX: sfx_door_heavy — 거대한 문이 열리는 소리]

[나레이션]
소환의 홀보다 더 큰 공간이 펼쳐졌다.
왕좌의 간.
붉은 카펫이 저 끝까지 이어지고, 그 끝에 왕좌가 있었다.

[캐릭터 등장: 세드릭 / 위치: 중앙 / 표정: 온화하나 창백 / 애니메이션: 없음(이미 착석)]

[나레이션]
왕좌에 앉은 남자는 — 왕이라 불리기엔 너무 마른 사람이었다.
은빛 머리카락에 백발이 섞여 있고, 왕관이 무거워 보인다.
그러나 눈빛만은 온화했다. 피로하지만 따뜻한 눈.

[SFX: sfx_cough_light — 가벼운 기침]

세드릭: "오셨... 크흠. 오셨군요, 용사여."

[나레이션]
왕이 기침을 삼키며 말을 이었다.
옆의 시의(侍醫)가 걱정스러운 눈으로 왕을 바라본다.

세드릭: "이 나라를 대신하여... 감사를 전하오. 먼 곳에서, 이 세계의 부름에 응해주셨으니."

(감사라니. 내가 원해서 온 것도 아닌데.)

세드릭: "자세한 이야기는... 쉬신 뒤에 하겠소. 지금은 다만... 당신이 무사히 도착했다는 것이 기쁘오."

[SFX: sfx_cough_deep — 더 깊은 기침]

[나레이션]
왕이 심하게 기침했다. 시의가 급히 다가가고, 에리나의 표정이 순간 일그러졌다.
하지만 그것도 잠시, 에리나는 다시 미소를 만들었다.

[캐릭터 등장: 모르데우스 / 위치: 왼쪽 / 표정: 자비로운 미소 / 애니메이션: slideIn]
[SFX: sfx_robe_silk — 비단 법의 스치는 소리]

[나레이션]
그때, 왕좌 옆에서 한 사람이 앞으로 나섰다.
흰 법의에 금실 자수. 마른 체형에 부드러운 미소.
60대쯤 되어 보이는 남자였다.

모르데우스: "폐하, 무리하시면 안 됩니다."

[나레이션]
목소리가 부드러웠다. 너무 부드러웠다.
꿀처럼 달콤한데, 무언가 씁쓸한 뒷맛이 남는 목소리.

모르데우스: "용사님, 저는 아젤리아 왕국의 대신관 모르데우스라고 합니다."

[캐릭터: 모르데우스 / 표정: 자비로운 미소 — 눈은 차분히 관찰하는 중]

모르데우스: "빛의 기둥이 당신을 이 세계로 이끌었습니다. 그대의 빛은... 이 세계를 구할 것입니다."

(빛이라는 말을 유독 강조하는 것 같은데.)

모르데우스: "오늘은 편히 쉬시죠. 내일, 빛의 축복 의식을 통해 그대의 힘을 일깨워 드리겠습니다."

[나레이션]
모르데우스가 고개를 숙이며 물러났다.
자비로운 미소가 한 번도 흐트러지지 않았다.
왜인지 모르겠지만, 등골이 서늘했다.

에리나: "용사님, 방으로 안내해 드리겠습니다."

[이펙트: 화면 페이드 아웃]

→ AZL_003으로 이동
```

## ===== AZL_002b: 관찰자의 시선 (PER 루트) =====

```
[씬 ID: AZL_002b]
[배경: bg_azelia_summoning_hall / 대낮]
[BGM: bgm_azelia_summoning → 볼륨 낮게 유지, 긴장감]

[나레이션]
손을 잡지 않았다.
아니, 잡을 수 없었다.
먼저 해야 할 일이 있었다 — 여기가 어딘지 파악하는 것.

에리나: "용사님...?"

[캐릭터: 에리나 / 표정 변경: 안도 → 약간의 걱정]

[나레이션]
에리나의 손이 허공에 멈췄다.
약간의 당혹감이 스쳤지만, 그녀는 곧 손을 내렸다.

(미안하지만, 지금은 상황 파악이 먼저야.)

[나레이션]
바닥의 마법진을 살폈다.
원형 구조 안에 복잡한 기하학 문양이 겹겹이 새겨져 있다.
중심부에 가까울수록 문양이 촘촘해지고, 가장 안쪽에는—

[이펙트: 화면 줌인 — 바닥 마법진 클로즈업]
[SFX: sfx_discovery — 작은 발견음]

[나레이션]
이상한 표식이 있었다.
마법진의 나머지 문양과는 다른, 어딘가 이질적인 기호.
원형 안에 모래시계 같은 형상 — 그런데 깨져 있다.
주변의 사제들은 이 표식을 의식하지 않는 것 같았다.

(복선: 10번째 기둥 — 시간의 기둥 표식)

(뭐지, 이건. 다른 문양이랑 느낌이 다른데...)

[나레이션]
시선을 들어 홀 전체를 둘러보았다.
스테인드글라스에 그려진 것은 — 빛의 기둥을 중심으로 모여든 사람들의 그림.
그런데 유리의 한 부분이 다른 색으로 수리된 흔적이 있다.
마치 원래 그림의 일부를 의도적으로 바꾼 것처럼.

[나레이션]
기사들의 표정도 살폈다.
도열한 기사 중 대부분은 경건한 표정이었지만, 일부는 —
아니, 한 명은 분명히 불안한 눈으로 출구를 흘겨보고 있었다.

(이 소환이라는 게, 모두가 원한 건 아닌 모양이야.)

[캐릭터: 에리나 / 위치: 오른쪽 / 표정: 걱정]

에리나: "용사님, 괜찮으신... 겁니까?"

[나레이션]
에리나가 한 발 다가왔다.
가까이서 보니 눈 밑에 옅은 다크서클이 있었다.
이 소환 의식 준비로 잠을 못 잤거나, 아니면 다른 이유로 잠을 못 잤거나.

에리나: "처음에는 혼란스러우실 겁니다. 천천히... 안내해 드리겠습니다."

(이 사람도 뭔가 숨기고 있다. 하지만 악의는 아닌 것 같아.)

[이펙트: 화면 페이드 아웃]

→ AZL_003으로 이동
```

```
[나레이션]
모르데우스가 차분히 설명했다.
마치 아이에게 이야기를 들려주듯, 천천히, 다정하게.
```
