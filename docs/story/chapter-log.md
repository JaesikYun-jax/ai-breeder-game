### ch001. 트럭이 오는 건 알고 있었다 (Arc 1 — 아젤리아)
- **시놉시스**: 트럭 사망 → 사이 공간 → D급 재생 선택 → 아젤리아 소환

- **공개 설정**: 세계가 10조각→9조각으로 깨짐 / "아무것도 아닌 자의 시선" 필요 / 스킬 등급제(S~D) 존재 / 강지호가 D급 재생 선택
- **감정 아크**: 권태 → 사고 공포 → 사이 공간 초현실 → 소환 후 "화장실 어디예요?" 수치

### ch002. 아젤리아 왕궁의 밤은 길다 (Arc 1 — 아젤리아)
- **시놉시스**: 왕궁 적응 + 첨탑의 맥동하는 빛 발견, 페이퍼나이프로 재생 D급 검증

### ch003. 용사라는 직업의 현실 (Arc 1 — 아젤리아)

### ch005. 축복이라 쓰고 제물이라 읽는다 (Arc 1 — 아젤리아)

- **감정 아크**: 체념적 분석 → 극한 고통(분해-재생 무한 루프) → 아이처럼 울음 → 모래시계 발동 경이 → 거부당한 채 사막 각성
- **다음 화 훅**: 솔라리스 사막에서 깨어남, "두 번째 생", Arc 2 시작
- **배치된 복선**: 기둥 균열, 모래시계 25%로 줄어듦(반→1/4), 갇힌 영혼들, 결계가 "기억"하는 가능성

- **감정 아크**: 두 번째 죽음에서 깨어난 탈진 → 사막 적응 → "정령술사가 될 수 있나" 막연한 호기심

### ch007. 불을 빌리는 자들 (Arc 2 — 솔라리스)
- **시놉시스**: 오아시스 도착, "태양에 닿은 자들"의 벽 목격, 장로 할리마와의 첫 대면
- **등장**: 강지호, 나이라, 할리마(70대, 정령술 안 쓰고 살아남은 전설), 사막 아이들
- **공개 설정**: 정령은 원소 강한 곳에서만 감지(열기둥/수맥/산위 기류) / 정령술 = "대출 시스템"(수명을 이자로 지불) / 할리마는 비계약자 70년 생존 / 할리마의 딸은 16세 계약 → 31세 사망 / 다섯 살부터 계약 시작 / 부고 게시판이 "태양에 닿은 자들"이라는 명예 형식
- **감정 아크**: 부고 벽의 잔인한 아름다움 충격 → 할리마와의 대화 → "오래 살기 vs 안전하게 살기" 모순 자각
- **다음 화 훅**: 정령 계약 결심 직전, 열기둥 앞에서 망설임
- **배치된 복선**: 지평선 기둥의 맥동(심장처럼), 모래시계 미세 흔들림, 할리마의 "어떻게 죽을지 선택하는 거야"

### ch008. 계약 (Arc 2 — 솔라리스)
- **시놉시스**: 사흘 횡단을 위해 술사 한 명이 더 필요해 정령 계약 결심, 10초간 정령에게 "읽힘" 후 계약 성립
- **등장**: 강지호, 나이라, 하산, 변경 초소 술사 쿠르드·파리(인서트)
- **공개 설정**: 계약은 "정령이 사람의 안을 디컴파일하듯 읽는" 과정 / 손바닥 동전 크기 화상 자국이 계약 인장 / 정령의 반응: "너, 재밌다" — 모래시계와 0.3초 시간 정지를 호기심 있게 응시 / 평균 5초 만에 끝나는데 지호는 10초 걸림 / 라이터 불꽃이 첫 출력
- **감정 아크**: 사흘 횡단 압박 → 두려움 극복 → 계약 후 "삼류는 세계가 바뀌어도 삼류" 자조
- **다음 화 훅**: 다음 구간에서 모래폭풍 예고
- **배치된 복선**: 정령이 모래시계를 인식, "흥미로운 버그", 지호의 불이 "보통 정령과 다른 느낌"

### ch009. 모래폭풍 (Arc 2 — 솔라리스)
- **시놉시스**: 대폭풍 중 나이라가 동·북 방벽 동시 유지로 한계 돌파, 지호의 "투명한 불"이 나이라 방벽 강화·대가 감소시킴
- **등장**: 강지호, 나이라, 하산, 캐러밴 술사들
- **공개 설정**: 지호의 불은 색이 없고 뜨겁지 않음 / 다른 술사 방벽에 닿으면 균열을 메우고 대가를 줄임 / 나이라의 화상이 어깨→쇄골까지 번짐, 머리카락 앞쪽 1/3이 실시간 백화 / 하산은 60년간 술사 11명을 잃음(15명 중 4명 생존)

- **다음 화 훅**: 나이라를 살릴 방법 모색
- **배치된 복선**: 지호 불의 특수성(투명·대가 감소), 나이라 수명 단축 가속, FS-B03 화상 누적

### ch010. 꺼지지 않는 불 (Arc 2 — 솔라리스)
- **시놉시스**: 재생을 이용한 비밀 야간 훈련 시작 — 화상-재생 무한 루프로 일반 술사의 일주일치를 하룻밤에
- **등장**: 강지호, 나이라(부분)
- **공개 설정**: 재생이 화상을 15분 만에 원상복구 → 술사 훈련 횟수 제약 우회 / 통증 내성 형성 불가(매번 첫 번째와 동일) / 정신적 공포는 누적(파블로프 효과) / 호흡 동기화로 효율 15% 향상 / 출력 키우기보다 유지 시간 늘리는 게 대가가 적음
- **감정 아크**: 가능성의 발견 → 반복적 자가 고문 → 새벽에 나이라가 "산책?"이라며 깨끗한 손 발견(반쯤 노출)
- **다음 화 훅**: 비밀이 반쯤 노출됨, 매일 밤 훈련 지속
- **배치된 복선**: 재생+훈련 무한 루프, 도구화 우려("들키면 소모품")

### ch011. 명예로운 노예들 (Arc 2 — 솔라리스)
- **시놉시스**: 다른 캐러밴 술사 아브라(34세)의 장례 참관, 솔라리스의 "명예로운 소모" 구조 인식
- **등장**: 강지호, 나이라, 하산, 장로(70대), 마을 사람들
- **공개 설정**: 장례 = 모래 구덩이에 안치 → 술사들이 손바닥 불로 모래를 녹여 유리관 봉인 / "태어나면 불이고, 살면 불이고, 정령의 곁으로 돌아가면 불로 돌아간다" 의식 / 솔라리스의 정령술은 희생이 아니라 노동으로 인식("수고했어"라고 인사) / 하산: "미안하니까" 예쁘게 보내준다고 고백
- **감정 아크**: 장례의 잔인한 아름다움 → "이 세계는 사람을 먹는 구조 위에 세워져 있다"는 인식 → 명예로운 노예가 되지 않으려는 결의
- **다음 화 훅**: 매일 밤 훈련 지속, 횃불 수준 도달 임박
- **배치된 복선**: 아젤리아(속여서 먹음) vs 솔라리스(자발적으로 바치게 함) 구조 동일성, "이 세계는 사람을 먹는 구조 위에 세워져 있다"

### ch012. 최적화 (Arc 2 — 솔라리스)

## Part 1 종합

강지호는 야근하던 28세 삼류 프로그래머로 트럭 사고로 죽고 사이 공간의 존재에게 D급 재생만 받은 채 빛의 왕국 아젤리아에 용사로 소환된다. 무능 판정을 받은 그는 모르데우스의 "축복 의식"이 사실 영혼을 빛의 기둥에 바치는 제의임을 에리나에게 듣지만 결계로 인해 도망치지 못하고 의식대에서 분해-재생 6회 루프를 견딘 끝에 기둥에 거부당한 채 솔라리스 사막으로 토해진다.

솔라리스에서 정령술사 나이라의 캐러밴에 합류한 그는 불의 정령과 계약하고 재생을 이용해 일반 술사가 2년 걸리는 라이터→횃불 성장을 3주에 압축하지만, 매번 첫 번째와 동일한 화상 고통을 견뎌야 했다. 솔라리스가 "명예로운 노예"의 구조로 정령술사를 소모하는 사회임을 인식하던 중 행상인을 통해 아젤리아에서 모르데우스가 섭정·축복 의식 재개·에리나 행방불명 소식을 듣는다.

70년 동안 4명이 모두 감전사한 번개 정령에 21번 심장 정지·재생을 반복해 계약을 맺고, 신경 가속·근섬유 파괴-재생으로 신체를 강화한 뒤, 할리마와 나이라의 도움으로 등·양팔·목·가슴·손목에 5개 함수의 문신 코딩(방어·경로·감지·보온·차단)을 새긴다. 사막 12일 횡단으로 아젤리아에 돌아와 에리나의 저항 세력 50명과 합류, 대성당 지하 수로로 의식장에 침입해 빛의 기둥 내부의 "300년 전 누군가가 심은 기생 코드"를 우회하는 새 경로를 deploy함으로써 모르데우스를 무력화하고 축복 의식을 영구 폐지시킨다.

기둥 단절 후 노인이 된 모르데우스를 본 지호는 그가 "버그를 정상이라 믿은 시스템 관리자"였음을 이해하지만 용서와 방면은 다르다. 왕국 재건은 에리나에게 맡기고, 청혼·결혼식·첫날밤을 거친 뒤 나머지 8개 기둥을 고치기 위해 솔라리스 방향으로 단신 출발한다. 성문 밖에 마르코가 두고 간 보따리와 "또 만났군"의 회수 쪽지가 다음 여정의 시작점을 가리킨다. 모래시계의 모래는 1/4 남았다.
