## 전체 흐름 요약

```
1화: 현대 사망 → 소환 (도입)
2화: 왕궁 적응 + 불안 조성 (세팅)
3화: 무능 확인 + 모르데우스 접근 (위기 구축)
4화: 진실 폭로 + 탈출 불가 (전환점)
5화: 사망 + 회귀 (클라이맥스 + Arc 전환)
```
