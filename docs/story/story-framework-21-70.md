## 2. 핵심 리트콘 (정본 우선순위)

이 4개는 기존 설정 문서를 덮어씀.

### RETCON-1. 모래시계 재정의
- **이전**: 마르코가 준 깨진 모래시계 = 0.3초 시간정지 도구
- **신규**: 세상 모든 기둥의 잔여 시간을 보여주는 카운트다운 지표
- ch005에서 발동했던 0.3초 정지는 "기둥이 자기 카운트다운에 닿은 순간 동기화" 일회성 현상으로 재해석
- 이후 도구로 사용 안 함. ch022에서 마르코 재회 시 진실 공개
- 모래 잔량 감소 = 세계의 시간이 줄어드는 것이지, 지호의 수명이 줄어드는 게 아님

### RETCON-2. 정령 수명 대가의 비대칭
- **이전**: 정령 계약 → 균일 수명 단축 (나이라 흰머리)
- **신규**: 비대칭. 일반인은 머리색 탈색·수명 단축 큼 / 지호는 D급 재생 + 무재능 체질 덕에 미미
- 근거: 재생이 신경·세포 단위까지 복구하므로 정령력 침식이 즉시 회복됨
- 결과: 지호는 정령을 다중 계약 가능. 다른 누구도 못 하는 빌드.
- **단** 기둥 치유에는 본인 수명을 직접 지불 → 엔딩 회수 (Part 3)

### RETCON-3. 진짜 위협
- **이전**: 막연한 "세상 구원"
- **신규**: 기둥 = 이계 차단 배리어. 기둥 소멸 시 → 배리어 off → 이계 괴물 침략 → 세계 병합
- "용사" 호칭은 사실상 배리어 유지용 소모품 시스템이었음
- 빌런 진영(천인 평의회 등)은 의도적으로 기둥 약화·파괴를 가속화

### RETCON-4. 기둥 흡수 → 다음 나라 방출 메커닉 폐기
- **이전**: death-and-regression.md "기둥 흡수 후 다른 나라 기둥 근처에서 방출"
- **신규**: 본문에서 이미 폐기됨. 지호는 캐러밴/도보로 직접 이동
- 기둥 자체는 "치유 대상"이지 "통과 게이트" 아님

## 5. 부강 에피소드 핵심 아이디어

### 5-1. 아젤리아 혁신 (Arc 4 분산)

삼류 프로그래머의 제한된 지식이 오히려 설득력 — 만능 치트 X.

| 분야 | 지호가 이식 가능 | 지호가 못 하는 것 |
|---|---|---|
| 행정 | 복식부기, 카드 색인, 세금 DB, 법령 분류체계 | 정밀 통계 |
| 공정 | 업무 흐름도, 분업 표준화, 도량형 통일 | 기계화 |
| 공중보건 | 손씻기·끓인 물 가이드라인, 격리 개념 | 백신·약물 |
| 교육 | 표음문자화 보급, 초급 산수, 사고 훈련 | 고등 과학 |
| **약점** | — | 증기기관, 화약 개량, 금속 야금 (전공 밖) |

에리나가 과도정부 실행을 맡고, 지호는 "설계도 그리는 컨설턴트"에 가까움. 여전히 노트 펴놓고 도식 그리는 그림.

### 5-2. 솔라리스 천막 기술 (Arc 5 핵심)

지호 = 캠핑 브랜드 리뷰 영상 500개 본 28세 직장인.

- **이중 천막** (공기층 단열) + **은박 반사 코팅** + **통풍 굴뚝 구조** + **경량 텐트폴 규격화**
- 기존 사막민 가죽 천막 대비 체감 실내온도 -15℃
- 용도 확장: 유목 이동식 상점 → 카이젤 야전 막사 역수출 → 프로스트헬 혹한용 역설계
- 나이라의 정령 가호로 정령력 보존 처리까지 추가
- "사막 피가 없는 외지인이 이런 걸 볼 줄 안다" — 지호의 **설계 + 상인 직감**을 부족장들이 정식 인정
- Arc 5 정서적 피크: 모멸당하던 외지인이 부족에게 받아들여지는 순간
