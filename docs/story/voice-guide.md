# 캐릭터 보이스 가이드

> 같은 상황에서도 캐릭터마다 다르게 말해야 한다.
> 이 문서는 "이 캐릭터라면 이렇게 말한다"의 즉시 참조용.
> 소설 본편(1-5화) + Blue 스크립트 기준. 상충 시 소설 본편 우선.

## 에리나 (エリナ) — 아젤리아 제2왕녀

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 32세 (14세에 하루카 목격, 이후 18년) |
| 외모 | 은발, 보라색 눈 (지호 曰 "렌즈 아니고?") |
| 직위 | 아젤리아 제2왕녀 |
| 본질 | 18년간 죄책감을 안고 살아온 내부고발자 |

### 말투 규칙
| 상황 | 화법 |
|------|------|
| 공식석상 | 왕족 격식체 — "~하옵니다", "~이옵니다" |
| 지호에게 (초반) | 정중 존댓말 — "~하세요", "~입니다" |
| 지호에게 (폭로 시) | 존댓말 유지하나 문장 짧아짐, 떨림 — "없어요." |
| 감정 폭발 시 | 반말 살짝 섞이다 자각하고 복귀 — "그건... 아, 죄송합니다" |
| 위험 경고 시 | 급박한 존댓말 — "조심하세요. 제발." |

### 말버릇/특징
- 중요한 말 전에 **멈춤**이 있다 ("무리하지... 마세요")
- 진짜 감정을 말할 때 문장이 **미완성**으로 끝남 ("용사님 앞에서는 왜인지...")
- "이전 용사들도..."로 시작하는 문장 — 과거에 대한 죄책감 트리거

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 공식 인사 | "아젤리아 왕국은 당신을 기다려왔습니다, 용사님." |
| 일상 대화 | "불편하시진 않으셨나요? 소환 과정이 항상 순탄한 것은 아니라서요." |
| 진심 토로 | "저는 이 궁에서 태어나 이 궁에서 죽겠죠... 선택할 수 있는 것이 하나도 없어요." |
| 폭로 | "아젤리아 왕국은 300년 동안 용사를 소환해 왔어요. 그중 축복의 의식을 받고 살아 돌아온 사람은—없어요." |
| 절박한 경고 | "조심하세요. 제발." |

### 지호와의 호칭
- 에리나 → 지호: "용사님" (변함없음, 소설에서)
- 지호 → 에리나: 직접 호칭 회피 (내면에서 "에리나", "공주")

## 모르데우스 — 추기경 (Cardinal)

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 60대 추정 |
| 외모 | 금빛 로브, 좁은 눈, 날카로운 광대뼈, 금색 소용돌이 눈동자 |
| 직위 | 아젤리아 추기경, 축복 의식 주관자 |
| 본질 | 실용적 악 — 트롤리 문제를 정책화한 행정가 |

### 말투 규칙
| 상황 | 화법 |
|------|------|
| 일반 | 부드럽고 달콤한 존댓말 ("꿀처럼 달콤한데 뒷맛이 씁쓸한") |
| 설득 | 장문의 흐르는 문장, 논리적, 반박 여지를 안 줌 |
| 놀랐을 때 | 문장이 급격히 짧아짐 ("...흥미롭군요.") |
| 위협 | 목소리를 낮추지 않고 오히려 더 부드럽게 — 성당 전체에 울리는 조용한 목소리 |
| 철학적 정당화 | "용사 한 명의 목숨으로 수백만의 국민이 산다. 이것이 잔인하다고? 아니. 이것이 왕국이다." |

### 말버릇/특징
- 발소리가 안 남 — 모래 위에서도 무음 (초자연적)
- 상대의 어깨를 터치함 — 비정상적으로 차가운 손
- "빛"이라는 단어를 자주 사용 ("빛의 기둥이 당신을 불렀습니다")
- 위기 상황에서도 미소 유지 — 미소가 0.5초 멈추면 진짜 놀란 것

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 축복 설명 | "빛의 기둥이 당신을 이 세계로 불렀습니다. 그대의 빛은... 이 세계를 구할 것입니다." |
| 안심시키기 | "걱정 마십시오. 이 세계의 그릇에 이 세계의 힘을 담을 수 있습니다." |
| 에리나 제압 | "공주. 300년간의 의식이 없었다면 이 왕국은 존재하지 않는다. 네가 태어날 수도 없었다." |
| 의식 시작 | "시작하거라." (2단어. 이것으로 충분) |
| 발견의 순간 | "...흥미롭군요." (유일하게 가면이 벗겨지는 순간) |

### 지호와의 호칭
- 모르데우스 → 지호: "용사여", "그대"
- 지호 → 모르데우스: 내면에서 "추기경", "이 사람", "금빛 로브 사제"

## 마르코 — 떠돌이 상인

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 30대 초반으로 보임 |
| 외모 | 빨간 머리, 예리한 눈, 상인다운 편안한 태도, 의미심장한 미소 |
| 직위 | 떠돌이 상인 (모든 지역 출입) |
| 본질 (설정) | 세계수의 마지막 잎, 모든 루프를 관찰하는 의식체 |

### 말투 규칙
| 상황 | 화법 |
|------|------|
| 일상 | 밝고 가벼운 반존대 — 웃음 섞인 화법 |
| 의미심장할 때 | 갑자기 진지 — 문장이 짧아지고 웃음이 사라짐 |
| 실수 수습 | 너무 자연스러운 화제 전환 — "연습된 거짓말" 느낌 |
| 조언 | 간접적, 경고보다 힌트 ("사람들이 하는 말을 전부 믿지는 마세요") |

### 말버릇/특징
- "또 만났군" — 첫 만남인데 재회처럼 말함 (시간 인식자 복선)
- 수습: "아, 그런가? 제 말은요—" (너무 매끄러운 수습)
- "떠돌이 상인의 덕목" — 자기 행동을 상인 철학으로 포장
- "또 보자, 용사님. 분명히 또 볼 거야." — 확신에 찬 예언조

### 마르코 등장 규칙
- **등장 빈도 최소화**: 매 아크마다 나오면 몰입 방해. 서사적으로 정말 필요한 순간에만
- **모래시계 반응**: 알아보는 척 하지 않음. 오히려 처음 보는 물건인 것처럼 큰돈을 주고 매입하려 듬 — 지호가 거부하면 깔끔하게 물러남. 이 행동 자체가 모래시계의 가치를 독자에게 암시
- **정보 노출**: 세계수/기둥 등 핵심 떡밥을 직접 설명하지 않음. 간접적 힌트만
- **재등장 시**: 이전 만남을 기억하는 듯한 미세한 뉘앙스만. 직접 언급 안 함

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 재회 인사 | "또 만났군." → (지호 혼란) → "아, 그런가? 이런 자리에서 만나니 반갑다는 뜻이죠! 하하!" |
| 모래시계 반응 | "그거... 혹시 팔 생각 없어요? 금화 50개 드리죠." (처음 보는 척, 하지만 눈빛이 진지) |
| 공짜 제공 | "그냥 가져. 떠돌이 상인의 덕목은 인연에 투자하는 거니까." |
| 진지한 조언 | "사람들이 하는 말을 전부 믿지는 마세요. 특히 '당신을 위한 것'이라는 말은요." |
| 작별 | "또 보자, 용사님. 분명히 또 볼 거야." |

- 정보를 전부 주지 않음 — 단편적 설명 후 끊김

## 병사들 — 집합적 목소리

- 소문/구전을 통해 세계관 정보 전달

| 실제 | "근데 전사 현장을 본 사람이 아무도 없어요... 시체도 없었다는 소문이..." |
| 기둥 관련 | "기둥이 약해지고 있는 거 아냐?" |

## 보이스 대비표 — 같은 상황, 다른 반응

### 상황: "축복의 의식에 대해 말할 때"
| 캐릭터 | 반응 |
|--------|------|
| **모르데우스** | "빛은 언제나 기다리고 있으니까요." (달콤한 위협) |
| **에리나** | "축복의 의식을 받고 살아 돌아온 사람은—없어요." (떨리는 폭로) |
| **로헨** | (침묵. 시선 회피.) |
| **세드릭** | "자세한 이야기는... 크흠... 쉬신 뒤에..." (회피) |
| **병사** | "의식 끝나면 용사님 뵐 수 있겠죠? ...그렇죠?" (불안한 희망) |
| **마르코** | "특히 '당신을 위한 것'이라는 말은 믿지 마세요." (간접 경고) |
| **검은 로브** | (모래시계를 만지며 침묵) |

### 상황: "위험을 감지했을 때"
| 캐릭터 | 반응 |
|--------|------|
| **지호** | 분석 시작 ("경고 3박자 패턴이다. 1박, 2박은 왔고...") |
| **에리나** | 미세 표정 변화 (눈 1초 떨림) + 말 돌리기 |
| **모르데우스** | 미소 유지 + 주제 전환 (완벽한 통제) |
| **로헨** | 검 손잡이에 손 올림 (무의식적) |
| **마르코** | 농담으로 위장 + 핵심 정보를 힌트로 투척 |

### 호칭 관계표
