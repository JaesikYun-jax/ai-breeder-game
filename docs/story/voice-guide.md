## 에리나 (Erina) — 아젤리아 제2왕녀

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 32세 (14세에 하루카 목격, 이후 18년) |
| 외모 | 은발, 보라색 눈 (지호 曰 "렌즈 아니고?") |
| 직위 | 아젤리아 제2왕녀 → 저항 세력 지도자 → 과도 정부 핵심 → 지호의 배우자 |
| 본질 | 18년간 죄책감을 안고 살아온 내부고발자 → 행동하는 지도자 |

### 말투 규칙 (시기별 변화)
| 시기 | 화법 |
|------|------|
| 1~5화 공식석상 | 왕족 격식체 — "~하옵니다", "~이옵니다" |
| 1~5화 지호에게 (초반) | 정중 존댓말 — "~하세요", "~입니다" |
| 4화 폭로 시 | 존댓말 유지하나 문장 짧아짐, 떨림 — "없어요." |
| 5화 의식장 | 떨리는 외침, 문장 끊김 — "이건... 아닙니다, 이건 축복이 아닙니다!" |
| 18화 재회 | 억누른 목소리, 소리를 죽이는 톤 — 지하 거점에서의 화법. 떨리지 않으려고 누르는 중. 살짝 농담 가능 — "용사님이 아니라 용사님을 먹은 괴물 같은데요." |
| 18~19화 작전 | 브리핑 모드. 감정을 상자에 넣고 뚜껑을 닫은 목소리. 지도자의 얼굴 |
| 19화 의식 폐지 선언 | 떨리지 않는 한 문장 — "축복의 의식을 영구 폐지합니다." |
| 20화 프러포즈/결혼식 | 갈라진 목소리, 떨림이 다시 돌아옴 — 18년 닫혀 있던 감정이 풀려나는 톤 — "저도요." / "지금은 기다림이 두렵지 않아요." |

### 말버릇/특징
- 중요한 말 전에 **멈춤**이 있다 ("무리하지... 마세요")
- 진심을 말할 때 문장이 **미완성**으로 끝남 ("용사님 앞에서는 왜인지...")
- "이전 용사들도..."로 시작하는 문장 — 죄책감 트리거 (Arc 1)
- 18화 이후: 18년 닫혀 있던 감정이 종종 0.5초씩 새어 나옴. 미소가 0.5초 넓어지고 돌아오는 패턴
- 결혼 후: 농담을 받아치는 여유 — "용사님, 지금이 식사할 때인가요?" + 입꼬리 미세 상승

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 공식 인사 (1화) | "아젤리아 왕국은 당신을 기다려왔습니다, 용사님." |
| 일상 대화 (2화) | "불편하시진 않으셨나요? 소환 과정이 항상 순탄한 것은 아니라서요." |
| 진심 토로 (4화) | "하루카 님이 끌려가는 걸 아무것도 못 하고 지켜본 사람은, 다음에도 아무것도 못 하면 안 되니까요." |
| 폭로 (4화) | "아젤리아 왕국은 300년 동안 용사를 소환해 왔어요. 그중 축복의 의식을 받고 살아 돌아온 사람은—없어요." |
| 절박한 경고 (4화) | "조심하세요. 제발." |
| 18화 재회 신원 확인 | "당신... 정말 용사님?" |
| 19화 작전 브리핑 | "C조. 용사님과 저, 그리고 다섯 명. 대성당 지하 수로로 기둥에 접근합니다." |
| 19화 폐지 선언 | "축복의 의식을 영구 폐지합니다." |
| 20화 프러포즈 응답 | "저도요. 18년을 기다렸어요. 아버지가 나을 거라고. 왕국이 바뀔 거라고." |
| 20화 서약 | "기다릴게요. 이번에는... 기다리는 게 두렵지 않아요." |
| 20화 성문 작별 | "조심하세요. 제발." (1~5화와 같은 단어, 다른 의미 — 이번엔 두려움이 아니라 믿음 섞인 기다림) |

### 지호와의 호칭 변화
| 시기 | 에리나 → 지호 | 지호 → 에리나 |
|------|----------------|----------------|
| 1~17화 | "용사님" 일관 | 직접 호칭 회피 (내면 "에리나", "공주") |
| 18~19화 | "용사님" (재회 직후 신뢰 톤) | "에리나", "공주" (저항 세력 회의에서) |
| 결혼식 후 (20화 후반) | **"지호"** (이름 전환) | "에리나" (이름 정착) |

## 모르데우스 (Mordeus) — 추기경, 후일 섭정

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 60대 (기둥 연결 단절 전엔 외모상 수십 년 더 젊어 보임) |
| 외모 | 금빛 로브, 좁은 눈, 날카로운 광대뼈, 금색 소용돌이 눈동자, 비정상적으로 차가운 손 |
| 직위 | 아젤리아 추기경, 축복 의식 주관자 → 섭정 → 폐위·왕궁 지하 수감 |
| 본질 | 실용적 악 — 트롤리 문제를 300년간 정책화한 행정가. 버그를 정상이라 믿은 시스템 관리자 |

### 말투 규칙 (시기별)
| 상황 | 화법 |
|------|------|
| 일반 (1~5화) | 부드럽고 달콤한 존댓말 ("꿀처럼 달콤한데 뒷맛이 씁쓸한") |
| 설득 | 장문의 흐르는 문장, 논리적, 반박 여지를 안 줌 |
| 놀랐을 때 | 문장이 급격히 짧아짐 ("...흥미롭군요.") |
| 위협 | 목소리를 낮추지 않고 오히려 더 부드럽게 — 성당 전체에 울리는 조용한 목소리 |
| 19화 기둥 단절 후 | 처음으로 미소 사라짐. 문장이 깨짐 — "기둥이... 무너지면... 이 왕국은..." |

### 말버릇/특징
- 발소리가 안 남 — 모래 위에서도 무음 (초자연적, 기둥 연결 효과). 19화 무릎 꿇을 때 처음으로 마찰음 발생
- 상대의 어깨를 터치함 — 비정상적으로 차가운 손
- "빛"이라는 단어를 자주 사용
- 위기 상황에서도 미소 유지 — 미소가 0.5초 멈추면 진짜 놀란 것
- 19화 정체 노출 직후 — 문장 끝나기 전 단어 단위로 끊김

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 축복 설명 (3화) | "빛의 기둥이 당신을 이 세계로 불렀습니다. 그대의 빛은... 이 세계를 구할 것입니다." |
| 안심시키기 | "이 세계의 그릇에 이 세계의 힘을 담을 수 있습니다." |
| 의식 시작 (5화) | "시작하거라." (2단어) |
| 발견의 순간 (5화/19화) | "...흥미롭군요." (가면이 벗겨지는 순간) |
| 정당화 (19화) | "용사 한 명의 목숨으로 수백만의 국민이 산다. 이것이 잔인하다고? 아니. 이것이 왕국이다." |
| 충격 (19화) | "문양이... 생각하고 있다?" (지호의 문신 코딩이 빛 패턴을 읽었을 때) |
| 패배 (19화) | "기둥이... 무너지면... 이 왕국은..." (장문의 논리 사라지고 단어가 조각남) |
| 무너짐 (19화) | "원래... 코드?" (60년 믿음이 무너지는 순간) |

### 지호와의 호칭
- 모르데우스 → 지호: "용사여", "그대"
- 지호 → 모르데우스: 내면에서 "추기경", "이 사람", "금빛 로브 사제" → 19화 전투 중에는 반말 ("...이 개새끼가 또 사람을 먹이겠다고?" — 14화 내면 / 19화 직접 발화 "그 논리를 300년간 돌렸으니까 기둥이 이 꼴인 거야")

## 로헨 (Rohen) — 근위대 부단장 → 근위대장

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 40대 초반 |
| 외모 | 거대한 체격 (이두근이 지호 허벅지만 함), 턱에 흉터 |
| 직위 | 왕실 근위대 부단장 → 모르데우스에 맞서 투옥 → 19화 구출 → 20화 근위대장 복귀 |
| 본질 | 직업군인. 원칙주의자. 감정 노출은 0.5초 단위 |

### 말투 규칙
| 상황 | 화법 |
|------|------|
| 명령 | 군대식 간결체 + **존댓말 기본** — "검을 잡아 본 적 있습니까?" |
| 실망 | 말이 줄어듦 — "...그렇겠지." |
| 포기 | 주제 전환 — "마법 적성을 보겠습니다." (검술 포기 선언) |
| 가르침 | 행동으로. 말보다 시범이 먼저 |
| 19~20화 재회 | 짧고 단단한 톤. 입꼬리 0.5초 단위로만 감정 노출 — "...살아 있었나." / "뭐, 그런 거지." |

> **호칭/말투 정본 정리**: 본편 일부 챕터에서 로헨이 반말로 표기된 대사가 있으나, 정본은 **공식적 존댓말 + 짧은 종결 + 흐리는 어미**로 통일한다 (사용자 피드백 ch001~005 리뷰 시 확정). 향후 윤문 시 반말 표기는 존댓말로 교정 대상.

### 말버릇/특징
- 지호의 무능을 보고 "아무 표정도 짓지 않는다" — 경악할 수 있었는데 참는 것
- 문장 끝을 흐리는 경향 — "뭐, 그런 거지"
- 입꼬리가 정확히 0.3~0.5초 단위로 올라갔다 내려옴 — 이 사람의 축하/인정 표현
- 침묵이 인정이다 (20화 모의 대련에서 검 내리고 침묵)

### 대사 샘플
| 상황 | 대사 |
|------|------|
| 첫 만남 (3화) | "검을 잡아 본 적 있습니까?" |
| 실력 확인 후 | "...마법 적성을 보겠습니다." |
| 격려 (드묾) | "검고 뭐고, 일단 살아 있으면 그걸로 된 겁니다." |

| 20화 재대련 | "검을 잡아 본 적 있습니까?" (1화와 동일한 첫 마디 반복, 새로운 답을 기대) |
| 20화 푸시업 인정 | "한 100개." (목표치 제시) |

## 마르코 (Marco) — 떠돌이 상인

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 30대 초반으로 보임 (실제 정체·나이 불명) |
| 외모 | 빨간 머리, 예리한 눈, 상인다운 편안한 태도, 의미심장한 미소 |
| 직위 | 떠돌이 상인 (모든 지역 출입) |
| 본질 (설정) | 세계수의 마지막 잎, 모든 루프를 관찰하는 의식체. 본편엔 직접 노출 안 됨 |

### 말투 규칙
| 상황 | 화법 |
|------|------|
| 일상 (4화 시장) | 밝고 가벼운 반존대 — 웃음 섞인 화법, 영업 모드 |
| 의미심장할 때 | 갑자기 진지 — 문장이 짧아지고 웃음이 사라짐 |
| 실수 수습 | 너무 자연스러운 화제 전환 — "연습된 거짓말" 느낌 |
| 쪽지 (20화) | 손글씨 — 빠르고 자신감 있는 필체. 한 문장 — "또 보자, 용사님. 분명히 또 볼 거야." |

### 말버릇/특징
- "또 만났군" — 첫 만남인데 재회처럼 말함 (시간 인식자 복선)
- 영업 → 비고객 판정까지 0.3초 (4화 "돈 없는데요" 대답에 표정 즉변)
- "떠돌이 상인의 덕목" — 자기 행동을 상인 철학으로 포장
- "또 보자, 용사님. 분명히 또 볼 거야." — 확신에 찬 예언조

### 마르코 등장 규칙
- **등장 빈도 최소화**: 4화 시장 단 한 번 직접 대화. 이후는 흔적/물건/쪽지로만 등장
- **모래시계 반응**: 알아보는 척 하지 않음. 처음 보는 물건처럼 큰돈을 주고 매입하려 듦 — 지호가 거부하면 깔끔하게 물러남
- **20화 출구**: 직접 만나지 않고 성문 밖 3미터에 보따리 + 쪽지 + 솔라리스 약도 남김

### 대사/쪽지 샘플
| 상황 | 대사 |
|------|------|
| 영업 (4화) | "이봐, 이봐! 거기 용사님! 우와, VIP 고객이다!" |
| 모래시계 거래 회피 | "이거 가져. 재고 정리야, 재고 정리." |
| 상인 철학 | "떠돌이 상인의 덕목은 인연에 투자하는 거니까." |
