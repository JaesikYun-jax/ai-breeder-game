## 에리나 (Erina) — 아젤리아 제2왕녀

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 32세 (14세에 하루카 목격, 이후 18년) |
| 외모 | 은발, 보라색 눈 (지호 曰 "렌즈 아니고?") |
| 직위 | 아젤리아 제2왕녀 → 저항 세력 지도자 → 과도 정부 핵심 → 지호의 배우자 |

### 말투 규칙 (시기별 변화)
| 시기 | 화법 |
|------|------|

| 18화 재회 | 억누른 목소리, 소리를 죽이는 톤 — 지하 거점에서의 화법. 떨리지 않으려고 누르는 중. 살짝 농담 가능 — "용사님이 아니라 용사님을 먹은 괴물 같은데요." |

| ch024 침소 (모르데우스 직후) | 정중 존댓말 복귀, 호칭 회피. "다녀오셨어요." / "많이... 걸리셨네요." / "저는 먼저 자야겠어요." 평이한 박자에 한 박자 멈춤이 끼어듦. 침대 등잔을 끄지 않고 반대편 끝에 누움 (한 사람 분 자리 비움) |
| ch025 다실 화해 | 손등 위에 손 얹기. "미안하다는 말은 안 받을게요. 받으면 당신 잘못으로 두는 게 되니까. 다음에 그런 자리 가야 할 일이 또 생기면, 가기 전에 한 번, 말해줘요." |
| ch025 출발 전날 | 미완성 발화 — "…아니, 다녀와서요." / "별 거 아니에요. 길 조심하라는 말. 두 번 하면 무거우니까요." 새벽 작별 "조심해서요." (3음절) + 거울 앞 잠옷 위 배에 오른손 한 박자 (지호는 등 돌려 못 봄) |
| ch026 친서 (사적) | 네 줄 — "지호에게. / 사흘간 고민하다 부친 씁니다. 아이가 생겼어요. 놀라지 말고, 돌아오세요. / 에리나." 마지막 한 획만 살짝 흔들림 |
| ch028 친서 (사적, 두 번째) | "지호에게" + 출산 한 달 전·반지 헐거움·새벽 배의 발길질·사관 한 줄 읽음·"돌아오는 길은 급하지 않게 / 에리나로부터" |

### 말버릇/특징
- 중요한 말 전에 **멈춤**이 있다 ("무리하지... 마세요")
- 진심을 말할 때 문장이 **미완성**으로 끝남 ("용사님 앞에서는 왜인지...")
- "이전 용사들도..."로 시작하는 문장 — 죄책감 트리거 (Arc 1)
- 18화 이후: 18년 닫혀 있던 감정이 종종 0.5초씩 새어 나옴. 미소가 0.5초 넓어지고 돌아오는 패턴
- 결혼 후: 농담을 받아치는 여유 — "용사님, 지금이 식사할 때인가요?" + 입꼬리 미세 상승

## 모르데우스 (Mordeus) — 추기경, 후일 섭정

### 기본 정보
| 항목 | 내용 |
|------|------|
| 나이 | 60대 (기둥 연결 단절 전엔 외모상 수십 년 더 젊어 보임) |
| 외모 | 금빛 로브, 좁은 눈, 날카로운 광대뼈, 금색 소용돌이 눈동자, 비정상적으로 차가운 손 |
| 직위 | 아젤리아 추기경, 축복 의식 주관자 → 섭정 → 폐위·왕궁 지하 수감 |
| 본질 | 실용적 악 — 트롤리 문제를 300년간 정책화한 행정가. 버그를 정상이라 믿은 시스템 관리자 |

### 말투 규칙 (시기별)
| 상황 | 화법 |
|------|------|
| 일반 (1~5화) | 부드럽고 달콤한 존댓말 ("꿀처럼 달콤한데 뒷맛이 씁쓸한") |
| 설득 | 장문의 흐르는 문장, 논리적, 반박 여지를 안 줌 |
| 놀랐을 때 | 문장이 급격히 짧아짐 ("...흥미롭군요.") |
| 위협 | 목소리를 낮추지 않고 오히려 더 부드럽게 — 성당 전체에 울리는 조용한 목소리 |
| 19화 기둥 단절 후 | 처음으로 미소 사라짐. 문장이 깨짐 — "기둥이... 무너지면... 이 왕국은..." |
| ch024 지하 감옥 재면담 | 평이한 톤 (회한도 비웃음도 아님). **한 문장 끝의 마지막 단어가 한 박자 늦게 나옴** — "오셨군요." / "…부왕 전하." / "감옥에 누가 직접 내려오는 일은, 보통, 없거든요." 손이 가만히 있다가 한 번씩 떨림 (자기 신호인 줄 알면서 내버려두는 노쇠). 미소 부재. 5단으로 정보 인계 — 검은 로브·300년 거래·다른 기둥 보호자·솔라리스 인지·"자네는 나와 다르지 않게 될 거예요" |

### 말버릇/특징
- 발소리가 안 남 — 모래 위에서도 무음 (초자연적, 기둥 연결 효과). 19화 무릎 꿇을 때 처음으로 마찰음 발생
- 상대의 어깨를 터치함 — 비정상적으로 차가운 손
- "빛"이라는 단어를 자주 사용
- 위기 상황에서도 미소 유지 — 미소가 0.5초 멈추면 진짜 놀란 것
- 19화 정체 노출 직후 — 문장 끝나기 전 단어 단위로 끊김

## 로헨 (Rohen) — 근위대 부단장 → 근위대장

- 비서가 한 박자 늦게 웃으면 시선 한 번으로 거두게 함 (수행원 군기)

## 검은 로브 단역들 — 잠입자 / 자객 / 도주자 — Arc 4

### 공통 정본
- 모든 검은 로브 단역은 **이름 없음** (모르데우스 ch024 인계 — "이름이 있으면 거래가 추적되니까")
- 표식: 두 줄 호선이 위아래로 겹친, 한쪽이 깨진 듯 끊긴 무늬 (위치는 잠입자=목 뒤, 자객=목덜미~어깨, 도주자=장신구). 모르데우스 장신구는 정본, 단역들의 것은 같은 계열·같지 않음

### 잠입자 (ch023~24 심문)
| 상황 | 대사 |
|------|------|
| 이름 | "…없습니다." |
| 소속 | "…모릅니다." |
| 무늬 출처 | "…모릅니다." |
| 누가 보냈는지 | "…**검은 로브**입니다." (첫 자발 단어 — 이 단어 자체가 매뉴얼이 아닌 듯) |
| 검은 로브 이름 | "…이름은 없습니다." |

**말투 규칙**:
- 같은 톤·같은 박자. 매뉴얼대로 답하는 자의 평이한 체념
- 협박조도 저항조도 아님. **도구로 쓰인 자의 톤**
- 표정 매뉴얼처럼 평평. 한쪽 무릎만 자꾸 한 번씩 작게 떨림

### 자객 (ch026 사막 야영) — 발화 없음
- 짧게 한 번 비명 삼키고 쓰러짐
- 훈련된 호흡, 발화 0
- 캐릭터화 금지: 이름·말 모두 없는 도구

### 사보타주 도주자 (ch027 협곡) — 발화 없음
- 비명 한 번 삼키고 그늘로 도주
- 손에서 깨진 모래시계 장신구 떨굼
- 캐릭터화 금지: 이름·말 모두 없는 도구
