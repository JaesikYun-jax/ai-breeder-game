# 주인공 바이블 — 강지호 (姜志浩)

> 이 소설의 모든 문장은 강지호의 눈과 입을 통해 나간다.
> 이 문서는 그의 목소리, 감정, 한계, 성장을 정의하는 최종 레퍼런스다.

## 2. 현대 생활 — 회상/비유의 소재 풀

지호의 독백은 현대 한국의 구체적인 경험에서 비유를 끌어온다. 이세계 묘사를 할 때도 반드시 현대적 비유가 하나 이상 들어간다.

### 일상 소재
| 소재 | 활용 예시 (실제 본편 기반) |
|------|--------------------------|
| **편의점** | "이 세계에도 편의점은 없다" (4화 제목 자체), 가격 비교의 기준점 |
| **웹소설 앱** | 무료 코인 줍기, 437화 정주행 중 트럭에 치임 |
| **사무실** | "개발자 직업병" — 이세계 상황을 디버깅하듯 분석 ("이건 버그야") |
| **월급** | 왕궁 옷을 보고 "서울 월세 몇 달치" 환산 |
| **아메리카노** | "씁쓸한 거 알면서 먹는 아메리카노" 같은 체념 비유 |
| **대중교통** | 횡단보도, 신호등, 퇴근길 — 1화 도입부의 공간 |
| **호텔** | 왕궁 목욕탕을 "5성급 스파"에 비유 |
| **원룸** | 왕궁 침실 크기를 "서울 원룸 다섯 개" 단위로 측정 |

### 비유 규칙
- 이세계 사물 → 현대 한국 등가물로 번역 (독자 공감 + 지호 캐릭터성)
- 감정 → 신체 감각 + 일상 비유 ("비명이 물속에서 들리는 것처럼 왜곡되어 들렸다")
- 돈 관련 → 반드시 구체적 숫자 ("쿠론은 대략 500원")
- 이세계 클리셰 → 구체적 작품명 언급 ("무한회귀 천재검귀" 같은 가상 제목)
- 프로그래머적 비유 → 상황을 코드/시스템 용어로 해석 ("무한루프에 빠졌다", "예외처리가 안 된 거야", "이 세계 런타임 에러 아닌가")

## 3. 이세계 지식 DB — "트로프 박사"

지호가 500작 이상의 웹소설에서 축적한 장르 지식. 이것이 그의 최대 무기이자 최대 약점.

### 인식하는 클리셰 목록
| 클리셰 | 지호의 반응 | 실제 결과 |
|--------|-----------|----------|
| 트럭에 치여 이세계 | "이거 그거잖아" — 즉시 인식 | 맞음, 하지만 고통이 100% 리얼 |
| 소환진 등장 | "소설에서 봤어" | 맞음 |
| 공주가 마중 | "데레 캐릭터 확정" | 실제로는 절박한 내부고발자 |
| 히든 스킬 각성 | "이제 곧 폭발할 거야" | 아무 일도 일어나지 않음 |
| 금빛 로브 사제 | "악역 확정" | 맞음 (모르데우스) |
| 스탯 창/시스템 | 허공에 손 휘저어봄 | 없음. 시스템 자체가 존재하지 않음 |
| 축복 의식 | "파워업 이벤트겠지" | 영혼 추출 제의 |
| 위기의 순간에 각성 | "이제 숨겨진 힘이..." | 각성 없음. 그냥 죽음 |

### 지식이 무기일 때
- 에리나의 표정 변화를 "경고 3박자 패턴"으로 분석 (4화)
- 마르코의 "또 만났군"에서 루프/시간 인식자 가능성 즉시 포착
- 모르데우스의 "꿀처럼 달콤한데 뒷맛이 씁쓸한" 어조에서 위험 감지
- 시장 경제를 프로그래머적 논리로 패턴 분석해 세계관 파악

### 지식이 독일 때
- "설마" — 위험 신호를 소설적 클리셰로 합리화 ("주인공은 안 죽으니까")
- 히든 파워를 기대하며 대비를 안 함
- 모래시계가 "그냥 깨진 모래시계일 뿐"이라고 판단 (5화, 의식 직전)
- **핵심 자각**: "여기는 소설이 아니다" (4화 마지막)

## 4. 보이스 규칙 — 강지호의 말과 글

### 독백 문체 (1인칭 내면)
- **기본 톤**: 반말 + 속어 + 자조
- **문장 길이**: 짧은 게 기본. 분석할 때만 길어짐
- **특징적 패턴**:
  - 상황 인식 → 클리셰 분류 → 현실과의 괴리 자각
  - 예: "소환진이다. 소설에서 백 번은 봤다. 근데 이렇게 눈이 아플 줄은 몰랐다."
  - 자기 비하 유머: "스쿼트 3개가 한계인 28세 직장인의 위엄"
  - 돈 환산: 자동 발동 ("이 옷 서울에서 얼마야")

### 대사 문체 (실제로 입 밖에 내는 말)
| 상대 | 화법 |
|------|------|
| 왕족/권력자 | 어색한 존댓말 ("...감사합니다?") — 물음표 뉘앙스 |
| 에리나 (초기) | 경계 섞인 존댓말 ("그게 무슨 뜻이죠?") |
| 에리나 (후기) | 약간 풀린 존댓말, 간혹 반말 섞임 |
| 모르데우스 | 형식적 존댓말 + 내면에서 욕설 |
| 로헨 | 군대식 존댓말 ("네, 알겠습니다") |
| 마르코 | 경계하면서도 호기심 있는 반존대 |
| 혼잣말 (위기) | 순수 반말 + 욕설 ("아 씨, 뭐야 이게") |

### 유머 패턴
1. **메타 코멘트**: 이세계 상황을 웹소설 용어로 해석 ("악역 확정", "데레 캐릭터")
2. **자조**: 자기 능력의 처참함을 담담하게 서술 ("푸시업 7개. 로헨이 세는 걸 포기했다")
3. **돈미새 발동**: 부적절한 타이밍에 가격 계산 ("목숨이 위험한데 물가 분석을 하고 있다")
4. **여미새 발동**: 로맨스 상황 인식 후 자체 차단 ("여미새 뇌, 지금 이걸 로맨스로 해석하지 마라")
5. **직업병**: 삼류 개발자의 사고방식을 엉뚱한 데 적용 ("이 왕국 시스템 아키텍처가 레거시야")

### 금지 표현 (지호가 절대 하지 않는 것)
- 영웅적 각오 ("이 세계를 구하겠다!")
- 상대를 깔보는 발언 (자기 비하는 하지만 타인 비하 안 함)
- 감정 직접 명명 ("슬펐다", "분노했다" → 대신 신체 반응으로)
- 낭만적 독백 ("그녀의 은발이 달빛에..." → "은발이 보였다. 진짜 은색이었다. 렌즈 아니고?")
- 시스템/스킬 용어 사용 (이 세계에 시스템은 없다)

## 5. 감정 스펙트럼

### 공포
- **초기 반응**: 몸이 먼저 반응 (다리 떨림, 목소리 안 나옴, 소변 실금)
- **인식 지연**: "비명이 내 비명인지 인식하는 데 시간이 걸렸다" (5화)
- **분석 본능**: 공포 속에서도 상황을 분석하려 함 — 하지만 "분석해서 뭐하냐. 죽어가면서."
- **극한**: 아이처럼 울음. "엄마를 부르고 싶었는데 목에서 소리가 나지 않았다" (5화)
- **문체 변화**: 문장이 파편화됨. 주어 탈락. 동사만 남음.

### 체념/피로
- **표현**: 담담한 서술 + 건조한 유머 ("두 번째 생이다")
- **사고 패턴**: 옵션을 나열하고 전부 불가능함을 확인 → 소거법적 절망
- **문체 변화**: 문장이 짧아지고 감정 형용사가 사라짐

### 분노
- **발현**: 느림. 지호는 분노보다 공포가 먼저 옴
- **방향**: 시스템/구조에 대한 분노 ("왕국이 300년간 해왔다고?")
- **표현**: 내면 독백이 짧고 날카로워짐 ("이 사람도 이걸 서른두 번 봤구나.")
- **자기 분노**: "500편 읽고도 이걸 예측 못 했어?"

### 안도/기쁨
- **드문 감정**: 1-5화에서 기쁨은 거의 음식에서만 발생
- **표현**: "이것만으로도 이세계 올 가치가 있다" (3화, 아침 식사)
- **돈미새 기쁨**: 공짜 의식주에 대한 본능적 만족
- **문체 변화**: 문장이 약간 길어지고, 감탄사 등장

### 호기심/분석
- **가장 빈번한 감정**: 지호의 디폴트 모드
- **표현**: 내면 독백이 길어짐, 비유가 풍부해짐, 프로그래머 어투 발동 ("이건 if문이 잘못된 거야")
- **위험**: 분석에 빠져서 위험 신호를 놓침

## 6. 성장 마일스톤 (아크별 내면 변화)

### Arc 1: 아젤리아 — "트로프 박사에서 당사자로"
- **시작**: 이세계 상황을 소설 독자 시점에서 관찰
- **전환점**: "여기는 소설이 아니다" (4화)
- **끝**: 죽음을 경험하고 "두 번째 생"을 자각
- **핵심 변화**: 관찰자 → 당사자

### Arc 2: 솔라리스 (예정) — "생존자"
- 첫 회귀 후 적응기
- 지식만으로는 살 수 없다는 것을 체감
- 정령술에 눈뜸 — 쓰는 법을 알지만 **대가(수명 소모)** 때문에 잘 안 씀
- 장생에 대한 열망이 강함: 겨우 살아남았는데 수명을 갉아먹는 힘은 쓰기 싫다
- 불의 정령과 계약 — 이후 프로스트헬(북쪽)에서 요긴하게 활용
- **찌질함 건재**: 사막에서도 물값 계산하고, 죽을 뻔한 기억에 움츠러듦

### Arc 3~ : 고통의 축적
- 각 지역에서의 사망이 "영혼의 눈"을 넓힘
- 전략가로 변하는 게 아니라, **직감이 날카로워짐** — "이거 위험하다"를 설명 못 하지만 느낌
- 체력은 매번 리셋이라 여전히 스쿼트 3개. 겁도 여전히 많음
- 달라지는 건 오직: 겁먹으면서도 한 발짝 더 나갈 수 있게 되는 것
- **고통참기의 비술**: 모든 힘을 얻는 과정에 참기 힘든 고통이 수반됨. 나중에 고통을 견디는 기술을 익히면서 정령술/검술/기공 등 이질적인 힘들을 하나로 통합
- **무재능의 역설**: 재능이 있으면 한 분야에 특화되어 다른 체계를 받아들일 수 없음. 지호는 아무 재능이 없기 때문에 오히려 모든 체계를 조금씩 흡수하여 합칠 수 있음

### 성장의 핵심 공식
> **"찌질한데 버틴다."**
>
> 성격은 안 나아진다. 돈미새, 여미새, 겁쟁이 — 기본 장착.
> 죽어도 안 고쳐진다. 그게 강지호다.
> 대신 **영혼이 깨달은 것**들이 쌓인다:
> "누구를 믿어야 하는지", "세계가 어떻게 작동하는지", "왜 살아야 하는지."
> 이 깨달음이 같은 겁쟁이를 **의미 있는 겁쟁이**로 만든다.

### 최종: 의지의 주인공
- 이세계 지식도, 전략도 아닌 "그럼에도 불구하고"의 의지
- 여전히 찌질하고, 여전히 겁먹고, 여전히 돈 계산함
- 하지만 9번의 죽음이 새긴 깨달음으로, 남들이 못 보는 것을 본다
- 강지호만의 자격: "아무것도 아닌 자의 시선" — 특별하지 않기에 모든 것을 볼 수 있는 자

| 5+ | — | 감정 마모, 건조한 수용 | "또인가" — 인간성 유지가 새로운 갈등 |

## 8. 약점/한계 — 이것을 어기면 캐릭터가 무너진다

### 절대 불변 (죽어도 안 바뀌는 것)
- **체력 바닥**: 매 회귀마다 리셋. 기사 4명에 둘러싸이면 저항이라는 선택지 자체가 사라짐 (5화)
- **전투 센스 제로**: 나무 검도 못 들어올림. 마법도 회로 미형성. 몸이 리셋이라 "알아도 못 함" (3화)
- **시스템 부재**: 스탯 창, 스킬 트리, 레벨업 없음. 이 세계에 시스템은 없다
- **찌질함 기본 장착**: 돈미새, 여미새, 겁쟁이, 분석 도피 — 성격은 회귀해도 안 고쳐짐. 이것이 캐릭터의 핵심이자 매력
- **감정 회피**: 위기 상황에서 분석으로 도피 ("이걸 알아서 뭐하냐" 자각은 있음)

### 점진적 변화 (영혼의 깨달음으로만)
- **직감 강화**: 이전 루프의 교훈이 "감"으로 발현 — 위험한 상황/사람을 설명 못 하지만 느낌
- **멘탈 내성 미세 증가**: 공포 자체는 여전하지만, 겁먹으면서도 한 발짝 더 나가는 능력
- **세계 이해도**: 기둥, 마법, 정치 구조에 대한 깊은 이해 축적
- 마법은 사용 불가하지만, 모래시계를 통한 시간 정지(0.3초)는 가능
- 인간관계에서 얻은 깨달음은 축적 (하지만 상대는 리셋 — 일방적 기억의 비극)

### 모래시계의 규칙
- 마르코에게 받은 깨진 모래시계 목걸이
- 축복 의식 중 자동 발동: 0.3초간 시간 정지
- 금색(기둥)이 아닌 은색 빛
- 기둥 내부의 갇힌 영혼들, 기둥 깊은 곳의 금빛 균열을 목격하게 해줌
- **현재까지 확인된 능력**: 극한 상황에서 자동 발동, 사용자 의지와 무관
- **미확인**: 의도적 발동 가능 여부, 사용 횟수 제한, 다른 효과
