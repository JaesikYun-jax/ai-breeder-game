## 0. 요약 — 역방향 체크 결론

ch002, ch003는 **이미 확정된 기설정**으로 굳어 있다. ch001 재작성은 이 두 화의 전제를 **깨뜨리지 않는** 방식으로 작성되어야 한다. 상세 시놉시스(00_detailed_synopsis)가 사용자 승인을 받았으므로 시놉시스를 정본으로 하되, 아래 정합 지점을 반드시 맞춘다.

## 1. 후속 2화 정합 체크리스트 (필수 준수)

### 1-1. 엘릭서 주입 시점 — "사흘 뒤 주입" / "8일 후 주입"

- ch002 본문에서 흰 수염이 D+5에 "엘릭서는 사흘 뒤 주입하네"라고 통보 → **실제 첫 주입은 D+8 저녁**.
- ch001 엔딩(D+0 현재)에서의 수치 박스 "다음 주입까지 8일"은 **D+0에서 D+8 첫 주입까지의 총 대기 기간**을 가리킨다.
- ch001에서 "사흘 뒤"라고 쓰지 말 것. ch002 D+5의 "사흘 뒤"는 ch002 흰 수염의 재통보 시점 기준이므로 ch001 엔딩과 단위가 다르다.
- **정합 확인**: ch001 엔딩(D+0) 시점에 "다음 주입까지 8일" → ch002 D+1 아침 시작 → ch002 D+5 재통보 "사흘 뒤" → ch002 D+8 첫 주입. 모순 없음.

### 1-2. 연금술사 3인 실루엣 — 이름 부여 절대 금지

| 식별자 | 표식 | ch001 | ch002 | ch003 |
|--------|------|-------|-------|-------|
| A | 흰 수염 (권위자, 장부 기록) | 검진 주재 | 매일 검진 주재, 이송 판정 | — |
| B | 확대경 (수정 사용, 실무) | 수정으로 소울 색 측정 | 검진 실무, 주입 집행 | 마지막 주입 집행, 이송 동행 |
| C | 둥근 안경 (젊은 학자, 노트) | "연구 가치" 묵살 | 무언으로 관찰·기록 | 동행 없음 (표식 관심만 잔재) |

- **금지**: 이름, 직함, 소속 상세 (그냥 로브 입은 노인 3인).
- ch001에서 3인을 처음 소개 → ch002에서는 "어제와 같은 순서" "같은 수정" "같은 장부"로 반복.
- ch001에서 설정한 특징(흰 수염의 권태, 확대경의 수정, 둥근 안경의 노트)은 ch002에서 그대로 이어지므로 **절대 변경 금지**.

### 1-3. 소울 색 키워드 — "검보라" "오염"

- ch001 씬 3에서 정면 배치: `푸른→갈색→회색→검보라` 변색 시퀀스.
- 대사: "소울 색이 검보랍니다." / "오염됐네."
- ch002에서도 동일 키워드 재언급 ("소울 색, 여전히 검보랍니다. 이틀 연속 동일.").
- ch002 본문: "거리도 어제와 같았다. 2cm"라고 나옴 → ch001에서 수정 거리를 설정하면 ch002가 이를 참조. **ch001에서 "눈앞 2cm" 정도로 명시하거나 단순히 "눈앞에 들어 올렸다"까지만 써도 무방** (ch002 "2cm"는 ch001과 독립적으로 성립).

### 1-4. D+10 엔딩 = ch002의 엔딩점, ch003의 진입점

**시간선 스택**:
- ch001 = D-10 플래시백 → D+0 현재 (씬 4 엔딩이 D+0, 첫 주입까지 8일)
- ch002 = D+1 아침부터 시작 → D+5 (흰 수염의 "일주일 더 본다, 엘릭서는 사흘 뒤") → D+8 (첫 주입) → D+10 밤 (폐기동 이송 통보)
- ch003 = D+16 새벽 (이송 실행)

**중요**: **ch001에서 "폐기동 이송"을 기정사실로 선언하지 말 것**. ch001의 판정은 "검진 후 비공식 판정 = 실패체로 판정됐구나"까지만이고, **정식 폐기동 이송 통보는 ch002 엔딩(D+10)**에 흰 수염이 직접 전달하는 장면으로 이미 집필되어 있다. ch001이 이를 선점하면 ch002 엔딩 훅이 무너진다.

### 1-5. 수치 박스 초기값 (ch001 씬 4)

시놉시스의 박스와 일치시킬 것:
```
[호문클루스 603호 · 현재 상태]
남은 수명: 11개월 (다음 주입까지 8일)
월 엘릭서 비용: 금화 150닢
현재 잔고: 금화 0닢
→ 결제 실패 시 사망
```

- ch002 주입 전 박스: "남은 엘릭서: 1일분 / 잔고: 금화 0 / 지불 주체: 연금술사 길드".
- ch002 주입 후 박스: "남은 엘릭서: 30일분 / 유효기간 D+38".
- **ch001의 박스 수치(11개월, 8일, 150닢, 0닢)는 ch002와 모순되지 않는다** — ch001은 "제조 직후 남은 총 수명"을 11개월로, 주입 사이클은 "8일 후"로 명시하는 것. ch002 박스는 "현재 엘릭서 잔량"을 일 단위로 표시하므로 축이 다르다.

### 1-6. 강신우 연기 패턴 — ch002에서 확립된 스펙

ch001에서 씨앗을 뿌리되 **과잉 명시 금지**:
- **입력 지연 3초** (ch002에서 "어젯밤 결정"으로 나옴 → ch001 엔딩에서 "느리게 움직여야 한다는 감"까지만 암시 권장, 정확한 수치 3초는 ch002에서 확정)
- **초점 풀린 눈** (ch001 씬 3 검진 중 이미 연기 시작)
- **손가락 미구부림** (ch002 D+10에 "오늘 처음 구부렸다"로 회수 → ch001에서 **손가락 구부리는 묘사 금지**)
- **음성 0회** (시놉시스에 명시. 강신우 외부 대사 없음)

### 2-1. ch002/ch003에 이미 쓴 표현과의 정합

### 2-2. "실패체 판정"의 수위

- ch001 씬 3: 강신우 내면 독백으로 **"실패체로 판정됐구나"**까지만 (본인의 해석).
- 외부 대사로 흰 수염이 "폐기한다", "이송한다"라고 말하는 순간이 오면 ch002 엔딩과 중복된다 → **금지**.
- ch001 씬 3의 외부 대사는 시놉시스대로 "다음 실험대로" 정도의 무관심한 종결이 최선.

### 2-3. 표식/마법진 노출 수위

- ch002 본문: "왼쪽 쇄골 아래 봉인 표식"이 둥근 안경 학자의 시선을 끈다 (FS-023 씨앗).
- ch003: "왼쪽 쇄골의 봉인 표식"이 재언급되며 "다른 회로와 다른 박자" 감지.
- **ch001의 표식 묘사**: 발목의 **숫자 603 각인**이 메인 (시놉시스 씬 1). 왼쪽 쇄골 봉인 표식은 **ch001에서 언급하지 않아도 무방**, 혹은 "몸 여기저기 마법진 각인이 있다"는 수준의 전체 스케치까지만. FS-023 본격 씨앗은 ch002에서 뿌려진 상태.
