## 1. 직전 2화 연결점

### 1-1. ch001 결말 상태 (D+0 밤)
- **장소**: 아르테미온 연금술사 길드 실험동 3구역, 603호 대기실
- **강신우 상태**: 실험대 옆 바닥에 앉아 벽에 기댄 자세. 눈 초점 풀린 연기 유지. 첫 상태창 세움. 병원비 면제의 기묘한 안도 + 월 금화 150닢 구독료 인식.
- **독자가 아는 정보**:
  - 강신우는 전생에서 췌장암 + 빚 8,200만 원으로 판교 옥상 투신 자살 → 호문클루스 603호에 빙의
  - 이 세계는 연금술사 길드 중심, 호문클루스 = 엘릭서 구독제 생명 (월 150닢)
  - 연금술사 3인(흰 수염 / 확대경 / 둥근 안경)에게 검진 받음 → "소울 색 검보라, 오염체"로 판정
  - 폐기 대기 판정. 단 둥근 안경 학자가 "관찰 기록 남기겠다"고 자원함
  - 마법진이 회로 구조라는 직관 (프로그래머적 시선)
  - 603호 기억 파편 폴더 `/603` 존재

### 1-2. ch002 결말 상태 (D+10 밤)
- **장소**: 같은 검진실 침상. 복도 마력등 빛이 손등 마법진을 비추는 장면.
- **강신우 상태**:
  - **신체**: 주입 직후 반응 속도 0.3초 → 0.25초로 미세 향상. 손가락 처음으로 구부려봄 (의도적 일탈 허가).
  - **심리**: 폐기동 이송 확정 통보를 받고 0.5초간 내면이 뒤틀렸으나 표정 사수. 20초 공포 → 계산 → 결심으로 변환 완료.
  - **결정**: "실험동은 나를 본다. 폐기동은 나를 잊으려 할 것이다." — 이송을 **기회**로 재정의. 보내지는 방향이 처음으로 반가웠음.
- **독자가 아는 정보**:
  - 다섯 번의 검진 통과. 소울 색 검보라 일관. 확대경의 "여전히 이상하지만" 한마디.
  - D+8에 **첫 엘릭서 주입** 완료. 영문 독백 "Your subscription has been renewed. 유효기간 30일."
  - D+10 밤 흰 수염이 "관찰 가치 낮네. 폐기동으로 이송하게. 내일 새벽." 단독 통보.
  - "엘릭서는 표준대로 한 병 주입해뒀으니 이송 중 문제 없을 걸세" — 이송 시점 재고 30일.
  - 호문클루스 산업 = SaaS 구조 해석 완성.
  - 왼쪽 쇄골 봉인 표식의 마력 박자가 손등 나머지 마법진과 다르다는 감지 (해석 보류).
  - 둥근 안경 학자의 시선이 여전히 표식에 머무름. 노트는 계속 쓰임.

### 1-3. 즉각 이어야 할 상황
- **시작 시점**: D+16 새벽 (이송 실행). ch002 엔딩 D+10 밤 → ch003 도입 **6일 뒤**.
- **공백 6일 처리**: ch003 도입부에서 간략히 요약 허용. "이송 통보 후 엿새 동안 같은 방, 같은 자세, 같은 침묵"식의 건너뛰기. 또는 새벽 이송 직전부터 바로 도입 가능.
- **강신우의 심리 궤도**: 경계 유지 → 새 환경 관찰 → 30일 시한 재계산 → 결심 (5장의 감정 아크에 선행하는 씨앗).

## 6. 설정 공개 범위 (ch003 전용)

### 6-1. 이 화에서 공개해도 되는 것
1. **폐기동의 공식 운영 구조**
   - 자연사 대기 시설. 폐기 비용보다 유지 비용이 쌀 때까지만 최소 엘릭서 주입.
   - 회수 요원이 **정기 방문** (주기는 ch003에서 확정 — "7~10일마다 한 번" 정도 자연스러운 수치 권장).
   - 수명 만료 후 시신은 "재료 재활용" 공정으로 회수 [직접 묘사는 톤다운, 암시 선].
2. **폐기동 내부 풍경**
   - 변두리, 울타리 너머 하늘 보이는 반(半) 개방 건물.
   - 석조 건물 1~2동 + 공동실 1~2개. 창문 또는 채광구 있음.
   - 호문클루스 10~20체 수용 [수치 가변 — 작가 재량].
3. **동료 호문클루스들의 상태**
   - 대부분 **식물 같은 반응** (눈 초점 없음, 하루 종일 같은 자세, 호명 없이는 움직이지 않음, 대화 없음).
   - 번호만 있고 이름 없음.
   - 강신우 자신이 연기하는 상태가 "정상 샘플"임을 재확인하는 계기.
4. **회수 요원**
   - 연금술사 길드 보조급(수련생·보조). 권태의 결이 실험동 노인들보다도 짙음.
   - 호문클루스를 가축·재고처럼 대함. 이름 없이 번호 호명.
   - 대화는 극소. 검진 없음 (주입만).

### 6-2. 감춰야 할 것 (ch004~ 이후 공개)
- **이리스의 존재** — ch004 첫 조우까지 단 하나의 힌트도 금지.
- **폐기동 탈출 가능성 / 마차 보급 체계** — ch005~010에서 구체화. ch003에서 강신우가 "탈출"이라는 단어를 내면에서 직접 떠올리는 것은 허용하나, 구체 수단(마차 보급 주기, 경비 동선 등)은 금지.
- **암시장·변두리 도시 지리** — ch011~ 진입.
- **미다스 씨 구상** — 아직 씨앗도 없음.
- **표식 감청 위장 마법진 설계** — ch015. 이 화에서 표식을 관찰하는 묘사조차 금지 (FS-023 재확인 회피).
- **마법진 리팩토링 적극 시도** — Stage 1은 ch021~. 이 화에서는 "언젠가는"식 직관만 유지.

### 6-3. 이 화의 서사 허용 예외
- **폐기동 풍경 도입부 3인칭 객관 묘사**: novel-config §4.1 "새 등장인물 첫 씬은 3인칭 객관 묘사로 시작, 이후 강신우 관점으로 복귀" 규칙 확장 적용. 새 공간 도입 시에도 같은 허용. 풍경 카메라 워크 → 강신우 근접 시점으로 복귀.

| 강신우 ↔ 둥근 안경 학자 | 경계 (관찰자) | -10 | -15 (이별, 노트만 남겨짐. 독자 긴장 소재로 은근히 남김 — 이송 시 "학자는 배웅하지 않았다"식의 한 줄 가능) |

### 7-2. 신규 관계 (ch003 도입)
| 관계 | 단계 | 핵심 포인트 |
|------|------|------------|
| 강신우 ↔ 폐기동 동료 호문클루스들 | **관찰 대상 / 죄책감 씨앗** | 이름 없음, 번호만. 강신우는 그들을 "자기가 될 뻔한 미래"로 봄. 도와줄 수단 없음 → 감정 기록만. FS-022 배치. |
| 강신우 ↔ 회수 요원 | 무관심 / 권태 공유 | 관심 밀도 실험동의 1/5 수준. 강신우의 연기 허용 오차 확대의 근거. |

### 7-3. alive-tracker 갱신 키
- 강신우의 첫 **타인 지향 감정** 씨앗 (ch002까지는 오로지 자기 생존 계산). 이것이 Part 3 정치공작 분노 동력의 원형.

### ch003 제한
- **"마력등의 호흡" 직유 금지** (3연속 방지). 빛 묘사는 필요하면 다른 비유로 — 예: "마력등은 박동 없이 얼어붙어 있었다" / "복도보다 희미한, 잔광에 가까운 빛" 등.

- **"턱을 잡는 정육점 손길"** — 회수 요원이 검진을 하지 않는 설정이므로 자연스럽게 회피됨.
- **"3초 지연" 수치 반복** — 반응 속도 수치는 이미 주입으로 0.25초까지 개선됨. 같은 수치 반복 대신 "폐기동 기본 허용 오차"라는 새 개념으로 전환 권장.

## 9. 감정 흐름

| 화 | 엔딩 감정 |
|----|----------|
| ch001 | 기묘한 안도 (병원비 면제) + 가격표 요구의 결의 (중립~긍정) |
| ch002 | 계산된 결의 + "반가움"의 기습 (상승 곡선) |
| **ch003 목표** | 불안 → 관찰 → **고독** → 결심 (story-framework 명시). 엔딩 훅 강도 3~4. |

- ch001 "기묘한 안도" → ch002 "반가움" → ch003 "고독"으로 **감정 반전 1회** 필요. 연속 상승 2화는 독자 피로. ch003 중반은 고독·죄책감으로 내려가고 엔딩은 "계산으로 변환한 결심"으로 미세 반등.
- **코미디→고통 패턴**: ch001·ch002 모두 돈미새 코미디 톤 유지했음. ch003은 고통 비중을 한 단계 끌어올려도 무방 (폐기동 동료들 관찰). 단 전면 어둠은 금지 — 돈미새 계산기가 끝까지 작동해야 함.

## 10. chapter-creator 주의 플래그

### 10-1. 필수 준수
1. **프로그래머 비유 2회 엄수** (ch002 verifier MINOR 지적 반영). 3회는 상한선이지만 2회로 타이트닝. 가능 후보:
   - 폐기동 = "창고 보관 중인 재고" / "Dead letter queue" / "Orphaned process 풀" 중 1개
   - 24일 수명 카운트다운 = "deprecated API / EOL(End-of-Life) 알림" / "잔여 스토리지 용량" 중 1개
   - **금지 비유 (재사용 불가)**: 스택 트레이스 (ch001), 믹스커피 (ch001), SaaS / Subscription (ch002), "박자 차이" 표식 감지 (ch002), EOF (ch001)
2. **분량 13,500자 이하 지향**. ch001 16,987 → ch002 14,522 → ch003 **13,500** 타이트닝. 드래프트 12,000~14,000자 권장, 최종 컷 후 12,000~13,000자 목표.
3. **수치 박스 1~2회**. 도입부 1회 (폐기동 도착 시 현재 상태) + 엔딩 1회 (30일 적응 후 결심 시점) 권장. story-framework 명시: "수명 11개월, 잔고 0".
   - 수명 11개월 해석: ch003 중~말 시점에서 "호문클루스 원 수명 약 1년 중 2개월 경과 → 남은 10~11개월"을 계산한 강신우의 관점. 엘릭서 재고(24일)와 별개의 생물학적 한계.
4. **엔딩 훅 강도 3~4** (tone-style). "계산으로 덮은 결심" 권장 — 탈출 키워드는 직접 언급 해도 되나 구체 계획은 ch005~.
5. **돈미새 내면 독백 2회 이상** (novel-config §6). 폐기동에서도 비용·가치 계산 지속.
6. **3인칭 근접 시점 유지**. 단 폐기동 풍경 도입 3인칭 객관 허용 (§6.3).

### 10-2. 지양
- **스택 트레이스 / EOF / 믹스커피 / SaaS / Subscription / 박자 차이** 6개 모티프 **재사용 금지**.
- **연금술사 3인 검진 패턴 반복** — ch002에서 "검진 피로의 누적"으로 이미 완결. ch003에서 다시 불러오면 정체.
- **호문클루스 산업 구조 재설명** — ch002에서 SaaS 비유로 완성. 이 화는 **구조 설명이 아니라 구조 안에서의 인물 관계·감정**에 무게.
- **이리스 힌트 선제 배치** — ch004 전담. "왠지 다른 호문클루스와 달리 움직이는 그림자가..." 같은 힌트 금지.

### 10-3. 기회
- **새 공간**: 폐기동은 ch001~002 검진실에서의 첫 공간 전환. 감각적 대비 활용 (석조 벽 단조로움 → 울타리 너머 하늘 / 환기구 → 외부 공기 / 폐쇄 → 반개방).
- **관찰 대상 밀도 증가**: 검진실은 강신우 vs 연금술사 3인 중심. 폐기동은 **동료 호문클루스 다수**가 배경으로 존재 → 강신우의 죄책감·거리감 층이 새로 생김.
- **감정 곡선의 확장**: ch001~002는 자기 생존 독백 중심. ch003에서 처음으로 **타인(동료 호문클루스)에 대한 감정**이 발생 → Part 3 정치공작 분노 동력의 최초 씨앗 (FS-022).
