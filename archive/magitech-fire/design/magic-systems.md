# 마법 체계 — 마도 공학 프로그래머의 영생 프로젝트

> 이 세계의 마법은 **규칙 기반 시스템**이다. 연금술사는 경험·직관으로, 강신우는 논리·최적화로 접근한다.
> 이 인지 격차가 100화 전체의 경쟁 우위 근거.

---

## 1. 마법의 근본 원리

### 1-1. 마력 (Mana)
- **정의**: 대기·생명체·광물에 편재하는 에너지.
- **측정**: 표준 단위 "아르" (1 아르 ≈ 인간 1분 집중 사고량).
- **농도**: 지역별 편차 (아르테미온 도심: 12 아르/㎥, 한랭대 고대 유적: 80 아르/㎥).
- **획득**: 호흡·흡수·마력석 사용.

### 1-2. 마법진 (Runic Array)
- **정의**: 마력을 특정 형태로 흐르게 하는 기하학적 문양.
- **언어**: 고대 룬 문자 + 기하학.
- **구조**: 입력부 → 처리부 → 출력부 (프로그래머 관점에서는 함수 구조).
- **한계**: 복잡해질수록 실패율 급증. 연금술사들은 경험칙으로 대응, 강신우는 **모듈화·디버깅**으로 대응.

### 1-3. 세 갈래 기술
| 분야 | 사용 방식 | 장점 | 단점 |
|------|----------|------|------|
| **연금술 (Alchemy)** | 물질 변환·합성 | 고체 결과물, 재사용 가능 | 원료·시간 필요 |
| **마도 공학 (Magitech)** | 마법진 + 기계 결합 | 반복 생산 가능 | 신흥 분야, 미성숙 |
| **마법 (Classical Magic)** | 주문 즉석 발동 | 즉각적, 전투용 | 재능 필수, 소수만 가능 |

이 소설의 주 무대: **연금술 vs 마도 공학**. 고전 마법은 배경 요소.

## 2. 엘릭서 (Elixir) — 핵심 자원

### 2-1. 정의
- 마력과 생명 에너지를 응축한 액체.
- 인간 혈액 대용으로 호문클루스의 "순환 매질"로 작동.
- 원료: 달빛석·은빛 이끼·심해 진주·혈화초 (worldbuilding.md §4 참조).

### 2-2. 등급
| 등급 | 순도 | 가격 | 용도 |
|------|------|------|------|
| 1급 | 95%+ | 금화 500닢/병 | 귀족용, 고급 호문클루스 유지 |
| 2급 | 70~95% | 금화 150닢/병 | 일반 시장가 (강신우 기본 소비) |
| 3급 | 50~70% | 금화 50닢/병 | 실험용, 저가 노동 호문클루스 |
| 암시장 | 가변 | 정가의 30~70% | 밀수·변조품 (리노 공급선) |

### 2-3. 호문클루스 연비 시스템

강신우의 **핵심 성장 축**. 신체 개조를 통해 엘릭서 1병당 생존 일수가 증가한다.

| 단계 | 개조 내용 | 1병당 연비 | 월 비용 (2급 기준) | 달성 시점 |
|------|-----------|-----------|------------------|----------|
| **Stage 0** | 제조 기본 상태 | 30일 | 금화 150닢/월 | 1화 시작 |
| **Stage 1** | 마법진 재구축 + 불필요 기관 일부 제거 | 90일 | 금화 50닢/월 | Part 1 후반 (ch020~025) |
| **Stage 2** | 그레고르의 고대 설계도 기반 전면 리팩토링 | 1년 | 금화 12닢/월 | Part 2 말 (ch050) |
| **Stage 3** | 엘릭서 열화 지연 마법진 + 고순도 재투자 | 3~5년 | 금화 5닢/월 (1급 사용 기준) | Part 3 후반 (ch070~075) |
| **Stage 4** | 인공 합성 엘릭서 자가 생산 | 무제한 | 0 (재료비만) | Part 4 후반 (ch090) |

### 2-4. 연비 개선 원리 — 프로그래머 비유

| 개조 방식 | IT 비유 | 실제 효과 |
|----------|--------|----------|
| 불필요 기관 제거 (수면·소화기 일부·생식기 일부) | **Dead Code Elimination** | 엘릭서 소비 지점 감소 |
| 기존 마법진 리팩토링 | **Code Refactoring** | 같은 기능, 절반의 마력 |
| 순환 루프 재구성 | **Caching / Memoization** | 엘릭서 폐열·잔류 마력 재활용 |
| 모듈식 마법진 슬롯 | **Plugin Architecture** | 마도구 장착 가능 (아이언맨 준비) |
| 열화 지연 마법진 | **Garbage Collection 개선** | 엘릭서 순도 저하 속도 감소 |

### 2-5. 강신우의 자기 수술 장면 (서사 장치)
- Part 1 후반과 Part 2 말, **강신우 본인이 직접 메스로 자기 몸을 째고 마법진을 새기는 장면**.
- 호문클루스라 통증은 있어도 장기 손상은 엘릭서 수리 가능.
- 달빛조각사의 "밤샘 조각 노가다" 같은 **결의와 장인정신** 묘사 포인트.
- 이리스가 보조 (나중에는 이리스 자신도 같은 수술 받음).

## 3. 호문클루스 구조

### 3-1. 기본 스펙 (Stage 0)
| 항목 | 값 |
|------|-----|
| 수명 | 기본 1~2년, 엘릭서 주입 시 연장 |
| 감각 | 인간과 동일 (통증·감정·식욕) |
| 수면 | 불필요 |
| 근력·반응속도 | **일반인 2~3배** (실험용 기준). 연금술사가 실험 도중 제어하기 위해 기본 스펙을 높게 설정 |
| 운동 능력 | 일반인 대비 상당히 우수 (뛰기·점프·순간 반응 모두) |
| 치유 | 자연 회복 없음 (엘릭서 + 마법진으로만) |
| 지능 | 제조 시 설정값. 실험용은 평균 이하 설정 (이게 강신우의 연기 근거) |
| 감정 | 약한 영혼 에코만 있음 (표정 없는 인형) |

#### 용도별 근력 스펙 비교
- 실험용: 일반인 **2~3배**
- 노동용: 일반인 **3~5배**
- 전투용: 일반인 **5~10배**
- 특수 전투용 (희귀): 일반인 10배 이상

→ 강신우 603호는 실험용이지만, 이것도 **맨몸으로 성인 남성 2~3명을 제압 가능한 수준**. 탈출 씬(ch007~008)에서 처음 시현됨.

### 3-2. 강신우 603호의 특이점
- **영혼 결합체**: 강신우의 인간 영혼 + 603호의 약한 에코가 융합.
- **감각**: 완전한 인간 수준. 통증 회피·쾌감 인식 정상.
- **지능**: 강신우 본인의 지능 + 603호의 제조 스펙 (튼튼한 뇌 구조).
- **연기 가능**: 표정·눈빛·움직임 조절로 "감정 없는 호문클루스"를 완벽하게 위장.

### 3-3. 외형적 특징 (모든 호문클루스 공통)
- 백발 (제조 과정에서 멜라닌 불안정)
- 창백한 피부
- 왼쪽 쇄골 아래 표식 마법진 (제조 번호 + 길드 봉인)
- 몸 전체에 실험 마법진 (실험체의 경우)

### 3-4. 용도별 분류
| 용도 | 특징 |
|------|------|
| 실험용 (Lab) | 저가, 몸에 각종 마법진, 수명 짧음. 603호 원래 용도. |
| 노동용 (Labor) | 근력·체력 강화. 지능 억제. |
| 전투용 (Combat) | 전투 마법진 각인. 민첩·반응속도 강화. |
| 애완용 (Pet) | 외모 최적화. 성격 조절 (온순). |
| 성노예용 (Pleasure) | 외모 극한 미형. 고가. 순도 1급 엘릭서 필수. |

## 4. 마도 공학 (Magitech) — 강신우의 무대

### 4-1. 정의
- 마법진을 기계·도구에 결합해 반복 사용 가능한 "마도구"를 생산하는 공학.
- 연금술이 "1회성 소모품" 중심이라면, 마도 공학은 "재사용 기기" 중심.

### 4-2. 대표 마도구 예시
| 분류 | 예시 |
|------|------|
| 생활 | 자동 세탁 마도진, 온수 지속 마도구, 조명 마도구 |
| 군사 | 마도 탄환 장전기, 방어 결계기 |
| 운송 | 마도 대차 (마차 대체), 개인용 마도 가마 |
| 산업 | 제련용 고온 마도로, 직조 자동화기 |

### 4-3. 강신우의 발명품 (예정)
- **Part 2 초반**: "세탁 자동화 마도진" — 첫 히트작, 중산층 대상 대박.
- **Part 2 후반**: "귀족용 온도 조절 마도구 침대" — 세라핀 부친의 대량 주문 계기.
- **Part 3**: "개인용 방어 마도 외투" — 자신의 신변 안전 + 귀족 시장 대박.
- **Part 3 후반**: "마도 갑주 초기형" — 암살단 격퇴.
- **Part 4**: "합성 엘릭서 증류기" — 최종 발명. FIRE 달성의 하드웨어.

### 4-4. 마법진 최적화 원리 (강신우 경쟁 우위)

#### 연금술사의 전통 방식
- 선대 스승의 마법진을 통째로 사용.
- 개선 = 수십 년 실험 + 경험.
- 중복 구조·불필요 연결선 방치.

#### 강신우의 리팩토링 방식
- 마법진을 **함수 단위로 분해**.
- 중복 제거, 입출력 인터페이스 정리.
- **같은 기능을 절반의 마력으로 구현** → 생산성 2~3배.
- 고객 관점에선 "저 사람 마도구가 효율이 미친 듯이 좋다" 라는 평판.

## 5. 연금술 vs 마도 공학 격돌 구도 (소설 서사 축)

### Part별 긴장 관계
| 부 | 구도 |
|----|------|
| Part 1 | 강신우 vs 연금술사 길드 감시 (숨기) |
| Part 2 | 마도 공학자 길드 입문 → 연금술과의 경쟁 본격화 |
| Part 3 | 정치공작으로 연금술 길드 수익원 공격 |
| Part 4 | 연금술 독점 붕괴 + 마도 공학 시대의 서막 |

### 상징적 대립
- 연금술 = 귀족·전통·독점·경험주의
- 마도 공학 = 중산층·혁신·공개 시장·합리주의

강신우의 승리는 **"프로그래머가 연금술사를 이긴다"**라는 상징.

## 6. 마법진 기호 & 용어 (간략)

실제 마법진 문자는 작중에서 묘사할 때만 등장. 상세 체계는 구축하지 않음.

- **원형 (Circle)**: 입력·출력 경계
- **삼각 (Triangle)**: 방향 지정
- **룬 문자 (Runic)**: 함수명·변수명 역할
- **연결선 (Linkage)**: 데이터 흐름

강신우의 해석 방식: "이거 그냥 그래프 알고리즘 아니냐? 여기 사이클이 있잖아."

## 7. 한계 & 제약

### 7-1. 강신우의 한계
- 타고난 마력 적음 (호문클루스는 제조 스펙 한계).
- 고전 마법사처럼 즉석 주문 발동 불가.
- 전투력은 **마도구·갑주에 의존**. 맨몸 = 평균 이하.

### 7-2. 마법진 설계 한계
- 복잡도 증가 시 실패율 지수 증가.
- 강신우의 모듈화 접근으로 **복잡도를 선형화** 가능.
- 그러나 "고대 설계도"(그레고르 제공)가 없으면 Stage 2 개조 불가.

### 7-3. 엘릭서 합성 한계
- 원료 대체: 현대 화학 지식으로 일부 대체 (강신우).
- 촉매 문제: Part 4 초반의 난관. "혈화초 대체재"가 핵심 병목.
- 해결: Part 4 중반에 그레고르 유산 + 세라 후원으로 돌파.
