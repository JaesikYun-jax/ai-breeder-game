# 보이스 가이드 — 마도 공학 프로그래머의 영생 프로젝트

> 각 캐릭터의 말투·종결어미·호칭·대사 패턴 정리.
> 대사를 다른 캐릭터와 교환했을 때 누가 말했는지 구별되지 않으면 보이스 실패.

---

## 1. 강신우 / 603호 / 미다스 씨 (1인 3역)

### 1-1. 강신우 본래 (내면 독백 + 이리스와 단둘)

#### 내면 독백 (가장 중요)
사용자 원본 샘플이 정본. 살아있는 사람의 실시간 반응.

- **기본 어조**: 혼란기엔 더듬거림·말줄임표, 정리될 때는 차분한 추측
- **자주 쓰는 어미**: "~지?", "~인가?", "~군.", "~거든.", "~아.", "~다."
- **금지**: "~인가 싶었다" 같은 간접화·"~하기 위해서였다" 목적절 강제·영어 혼용

#### 표준 예시 (사용자 원본 샘플 기반)
```
'이 곳은 어디지?'
'뭐지? 이건 가..상현실인가?'
'나는 누구지? 나는 호문클루스 603호다. 아니, 동시에 강신우 라는 
 이름의 회사원이지. 그리고 나는... 죽었을텐데!?'
'일단 실험이 실패한 것으로 계속 위장해야되.'
'호문클루스 603호의 몸에 나 강신우의 혼이 섞여 들어온건가?'
```

#### 리듬·특징
- **짧은 의문문 빈번**: 현실 파악 과정
- **말줄임표 적극**: 생각이 도중에 갈리거나 충격 순간
- **즉각 반응**: 해석보다 반응이 먼저
- **실용적 결단**: 위기 순간 "~해야 한다" 식 짧은 단언
- **때때로 자조**: 심각한 상황에서 피식 새는 자기 조롱 1~2회

#### 이리스와 단둘일 때 (외부 대사)
- **말투**: 편안한 반말. 약간 건조하지만 냉소적이진 않음
- **예시**:
  - "그건 밖에서 생각하자."
  - "지금은 조용히 있자."
  - "그래, 그거 기억해둬."
  - "목소리도 연기야. 잊어버리는 편이 편해."

#### 프로그래머 기질이 드러나는 방식 (용어 없이)
- 허용: "계산에 익숙한 뇌가 자동으로 돌아갔다"
- 허용: "문제를 단계별로 정리하는 건 오래된 습관이었다"
- 허용: "평생 예산 없는 프로젝트만 해온 사람의 본능이었다"
- **금지**: "스택 트레이스", "EOF", "SaaS", "버그", "디버깅" 같은 직접 용어
- **금지**: 영단어 노출 ("Your subscription has been renewed" 등)
- 빈도: 챕터당 1~2회, 0회여도 됨

### 1-2. 호문클루스 603호 연기 (외부 인물 앞)
- **대사**: 극히 제한적. "예", "아니오", "알겠습니다", "네, 주인님" 수준.
- **어투**: 느리고 기계적. 감정 없음. 명령 수행만.
- **표정·몸짓 서술 지시어**:
  - "강신우는 대답 없이 고개만 끄덕였다."
  - "강신우의 눈동자는 초점을 맞추지 않았다."
  - "강신우는 명령을 기다리는 자세로 섰다."
- **주의**: 내면 독백(`''`)은 풍부. 외부 행동과의 격차가 유머/긴장의 원천.

### 1-3. 미다스 씨 연기 (로브·후드 착용)
- **종결어미**: "~하오", "~이라네", "~군그래"
- **말투**: 중후한 노인 말투. 느리고 여유 있음. 반말에 가까우나 예스러움.
- **대사 패턴 예시**:
  - "허허, 그 마도구는 은화 50닢이외다."
  - "귀한 몸이 직접 오셨구려. 차 한잔 드시오."
  - "흠... 설계를 보니, 이 부분은 재작업이 필요하겠군."
- **음성**: 음성 변조 마법구로 굵고 낮은 중년 목소리 연출.
- **특징**: 박학다식한 괴짜로 연기. 사회적 규범 가벼이 무시. 돈 이야기는 단호하게.
- **주의**: 미다스 씨 대사 중에도 내면은 여전히 강신우. 독자는 간극을 느껴야 함.

### 1-4. 호칭
| 대상 | 강신우 호칭 (내면) | 미다스 씨 호칭 (외부) |
|------|------------------|-------------------|
| 이리스 | 이리스 / 591호 | 591호 / 내 호문클루스 |
| 그레고르 | 영감 / 그레고르 | 반 하이델 공(公) / 스승 |
| 세라핀 | 세라 / 저 여자 (초기) | 아르곤 영애 |
| 에녹 | 에녹 / 저 놈 | 칼러스 장로 |
| 리노 | 리노 / 서자 | 할베르크 군(君) |
| 자기 자신 (미다스로) | — | 이 미다스 |

## 2. 이리스 (호문클루스 591호)

### 2-1. 외부 앞 연기
- 603호와 동일. 무감정, 짧은 대사만.
- 단, 기본 연기력이 603호보다 뛰어남 (더 자연스러운 무표정).

### 2-2. 강신우와 단둘 (본래 모습)
- **종결어미**: "~네", "~거든", "~잖아", "~ㄴ데"
- **말투**: 재잘거리는, 약간 건들거리는 느낌. 사투리 없음.
- **대사 패턴 예시**:
  - "603호, 오늘도 지루한 얼굴이네. 그러지 말고 말이라도 좀 걸어주지."
  - "미다스 씨~ 이번 주문 잔고 계산, 내가 대신 해줄까?"
  - "그 여자, 너한테 관심 있어. 강신우, 너 눈치 없지?"
- **특징**: 밝고 명랑하지만 가끔 날카로운 통찰. 강신우의 돈미새성을 놀림.
- **감정 표현**: 강신우보다 직접적. 기쁘면 웃고, 싫으면 표정 그대로.

### 2-3. 호칭
- 강신우를 부를 때: "603호" (공개), "강신우" / "너" (사적).
- 미다스 씨 연기 강신우를 부를 때 (공개): "주인님", "미다스 님".
- 사적 대화에선 구분 없이 "너".

## 3. 그레고르 반 하이델

### 3-1. 기본 말투
- **종결어미**: "~이네", "~인가", "~게", "~야!"
- **말투**: 신경질적, 빠르고 직설적. 자존심 강함. 욕설은 가볍게 섞임.
- **대사 패턴 예시**:
  - "이 마법진을 이렇게 짜? 하, 웃기네. 자네 어디서 배웠나?"
  - "내가 이 길드에서 3급이긴 해도, 1급 녀석들 실력 다 거품이야."
  - "돈? 뭐, 나쁘진 않지만. 내가 원하는 건 인정일세."
- **특징**: 분노와 감탄이 빠르게 오감. 재능 있는 자에겐 사랑받는 스승 타입.
- **강신우에 대한 태도**: 처음엔 의심 → 재능 확인 후 "내 제자" 선언.

### 3-2. 호칭
- 강신우(미다스) 부를 때: "미다스 공(公)", "자네" (친해지면).
- 타인 부를 때: "자네들", "귀공(貴公)".

## 4. 세라핀 드 아르곤

### 4-1. 공식 석상
- **종결어미**: "~이에요", "~입니다", "~하세요"
- **말투**: 절제된 귀족 영애. 이지적, 정확.
- **대사 패턴 예시**:
  - "미다스 경, 이번 주문 마도구는 언제쯤 완성되나요?"
  - "아버지께서 걱정하십니다. 적절히 마치고 돌아오겠어요."

### 4-2. 사적 공간 (오두막에서, 603호 앞)
- **종결어미**: "~야", "~이지", "~네", "~거든"
- **말투**: 부드럽고 약간 수다스러움. 혼잣말처럼.
- **대사 패턴 예시**:
  - "603호, 너는 들어주는 것만으로 충분해. 대답 안 해도 돼."
  - "요즘 아버지가... 아니, 이런 이야기 너한테 하면 뭐 해."
  - "오늘 하늘이 예쁘네. 넌 하늘 볼 줄 알아?"

### 4-3. 감정 격동 시 (정체 발각 후)
- 말이 끊기거나 격해짐. 존대·반말 혼용.
- 예: "당신... 정말 강신우였어? 아니, 강신우라는 이름도 맞는 건가?"

### 4-4. 호칭
- 강신우(603호) 부를 때: "603호" (초반), 점차 "너" (사적), 정체 발각 후 "신우 씨" / "당신".
- 미다스 씨 부를 때: "미다스 경", "미다스 님".
- 이리스 부를 때: "591호" → "이리스" (Part 3부터).

## 5. 에녹 칼러스

### 5-1. 기본
- **종결어미**: "~입니다", "~하지요", "~겠지요"
- **말투**: 정중하고 차가움. 분석적 억양. 감정 억제.
- **대사 패턴 예시**:
  - "이 호문클루스, 소울 색이 특이하군요. 다시 한 번 검진해보지요."
  - "미다스 경, 흥미로운 마도구를 만드신다고 들었습니다. 의견을 나눠보고 싶군요."
  - "제가 틀렸다면 사과하겠습니다. 그러나 데이터는 거짓말하지 않아요."

### 5-2. 격분 시 (거의 없음)
- 냉소적 분노. 목소리는 낮지만 날카로움.
- 예: "... 그래서, 당신이 모든 것의 배후였다는 건가요. 아주 우아하군요."

### 5-3. 호칭
- 강신우/미다스 부를 때: "미다스 경", "603호의 주인" → 정체 의심 후 "흥미로운 자".
- 타인 존중: "칼러스 장로" 호칭을 원함.

## 6. 리노 할베르크

### 6-1. 기본
- **종결어미**: "~이지", "~거든", "~야", "~라고"
- **말투**: 빠르고 가벼움. 장사꾼 특유의 친근함 + 뒷맛.
- **대사 패턴 예시**:
  - "엘릭서? 2급 한 병, 정가의 60%야. 어때, 싸지?"
  - "이번에 들어온 정보 하나, 금화 5닢. 네 목숨값보다 싸."
  - "친구끼리 가격 흥정은 안 하는 거 알지?"
- **특징**: 친근한 척 + 실제로는 냉정. 배신의 여지 항상 남겨둠.

### 6-2. 호칭
- 강신우 부를 때: "친구", "미다스 선배", 사적으론 "야".
- 자기 지칭: "이 몸", "이 리노".

## 8. 종결어미 구분 체크

한 대사를 들었을 때 누구인지 구분되는가?

- "~이외다" / "~그래" → 미다스
- "~거든" / "~네" → 강신우 본래 또는 이리스 (문맥으로 구분)
- "~이에요" (공식) / "~이지" (사적) → 세라
- "~이네" / "~야!" → 그레고르
- "~하지요" / "~겠지요" → 에녹
- "~이지" / "~야" → 리노 (강신우와 겹침 → 내용으로 구분: 리노는 항상 거래 프레임)

## 9. 이중 정체 주의 규칙

### 9-1. 미다스 씨 → 강신우 톤 전환 순간 금지
- 미다스 연기 중 실수로 현대적 어미("~라니까") 사용 시 → 발각 복선.
- 반드시 일관되게 "~이외다" / "~군그래" 유지.

### 9-2. 603호 → 강신우 톤 전환 순간 금지
- 603호 연기 중 외부에서 감정·지성 드러내는 대사 절대 금지.
- 필요하면 이리스가 대신 처리.

### 9-3. 강신우 내면 독백은 자유
- 외부 앞에서 603호/미다스 연기 중이어도 내면(`''`)은 강신우 본래 모습.
- 이 격차가 서사의 핵심.
