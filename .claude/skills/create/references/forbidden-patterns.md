# 금지 패턴 P1~P5 — AI 표류 5종

episode-creator가 본문 저장 후 셀프 grep으로 점검하고, quality-verifier가 VOICE_PATTERN 축에서 재검증한다.
각 패턴은 **장르·작품 무관 generic 패턴**이다. 작품별 특화는 `novel-config.md`의 `voice_docs:`와 `forbidden_words:` 필드에서 오버라이드한다.

---

## P1. 결론 선언 (Pre-Resolution)

### 정의
사건이 진행되기 전에 결과·감정·판단·의미를 미리 진술하는 패턴.

### 왜 나쁜가
웹소설은 다음 줄이 궁금해서 읽는 매체다. 결론을 먼저 알면 페이지터너 효과가 죽는다. 1인칭에서는 화자도 모르는 채 발견해야 독자도 같이 발견한다.

### Before
```
주인공은 자신의 능력이 특별하다는 것을 깨달았다. 그것은 인생을
바꿀 발견이었다. 그는 천천히 화면을 들여다보았다.
```
문제: "특별하다", "인생을 바꿀 발견"이 묘사 앞에 옴 → 김 빠짐.

### After
```
손가락이 멈췄다.

다른 사람들 화면이랑은 달랐다. 글자가 아니라— 함수처럼 생긴 무언가.

…뭐야 이게.
```
결론 없이 발견 과정만 묘사. 마지막 한 줄이 진짜 후크.

### 검출 키워드 (Grep)
다음이 사건 묘사 **앞에** 나오면 P1 의심:
- `깨달았다` / `알게 되었다` / `인지했다`
- `발견했다` / `확신했다` / `분명했다`
- `그것은 ~이었다` / `그것이 의미하는 것은`
- `~인 것을 알았다` / `~였던 것이다`
- `~하게 될 줄은 그때는 몰랐다` (회상형 스포일러)

발견·판단 문장은 **묘사가 끝난 뒤**에만 허용. 짧게.

### 상한
한 화에서 위 키워드가 묘사 **앞**에 등장한 횟수 **0회**.

## P2. 정돈된 문장 (Over-Polished Prose)

### 정의
모든 문장이 비슷한 길이·종결어미·구조로 정돈되어 리듬과 구어체가 사라진 패턴.

### 왜 나쁜가
1인칭이든 3인칭이든 호흡이 있어야 한다. 단어 하나, 짧은 외침, 길게 늘어지는 묘사가 들쭉날쭉해야 사람이 쓴 글로 읽힌다.

### Before
```
나는 깊은 한숨을 내쉬었다. 머리가 복잡했다. 어떻게 이 상황을
헤쳐나가야 할지 알 수 없었다. 결국 나는 행동하기로 결심했다.
이내 자리에서 일어섰다.
```
문제: 5문장 모두 `~다`로 종결. 길이도 비슷. 접속사(`결국`, `이내`) 남발.

### After
```
한숨.

…됐고.

머리 굴려봐야 답 안 나온다. 일단 움직이고, 부서지면 그때 본다.

자리에서 일어섰다.
```
1단어 / 단어+말줄임 / 자조적 긴 문장 / 행동. 리듬 살아있음.

### 검출 휴리스틱
**5문장 슬라이딩 윈도우**에서 다음 중 2개 이상이면 P2 의심:
- 종결어미가 모두 `~다`
- 모든 문장이 12자 이상
- 모든 문장이 주어로 시작
- `결국`, `이내`, `그러자`, `그렇게` 류 접속사 2회 이상

### 상한
- 5문장 윈도우 위반 화당 **1개 이하**
- 단문(1~5자) 단락이 화당 **3회 이상** 등장 권장 (작품 톤에 따라 조정)

## P3. 용어 과잉 설명

### 정의
세계관 용어·시스템·고유명사가 등장할 때마다 친절히 풀어 설명하는 패턴.

### 왜 나쁜가
독자는 장르 클리셰에 익숙하다. 모르는 용어도 맥락에서 추측하는 재미가 있다. 위키식 설명 문단은 즉시 페이지 뒤로 넘기게 만든다.

### Before
```
헌터 협회로 향했다. 헌터 협회란 한국에 게이트가 열린 이후 각성한
인간들을 등록하고 등급을 분류하며 던전 출입을 관리하는 정부 산하
기관이다. 5년 전 설립되어 현재는 전국에 17개 지부를 두고 있다.
```
문제: 위키백과식 설명. 지금 알 필요 없는 정보.

### After
```
협회 1층 로비. 잿빛 정장 직원이 무표정하게 번호표를 내민다.
"F급은 저쪽 창구로." 손가락이 가리키는 방향. 줄. 또 줄.
4년째 똑같다.
```
"F급", "협회"라는 단어로 충분. 구조는 디테일에서 자연스럽게 드러남.

### 예외
세계관 용어 첫 등장 시 **한 문장 이내**의 짧은 단서는 OK:
- 부적합: "스킬 컴파일러란 ~한 능력으로 ~할 수 있는 것이다."
- 적합: "스킬을 코드로 본다. 수정도 된다는 게 문제지."

설명이 **두 문장 이상** 이어지면 무조건 P3.

### 검출 휴리스틱
- 한 문단에 정의문(`~란 ~이다`, `~는 ~을 의미한다`)이 2개 이상
- 위키 어조의 사실 나열 (설립 연도, 본부 위치, 직원 수 등)
- 인물 첫 등장 시 5줄 이상 외양·이력 묘사

### 상한
정의문 한 문단 **2개 이상** 0회. 위키식 사실 나열 문단 0회.

## P4. 단어 무한 반복

### 정의
같은 단어·표현·구문을 한 화 내에 반복적으로 사용하는 패턴.

### 왜 나쁜가
- 텍스트 매체에서 가장 빨리 들통나는 AI 신호
- 독자 댓글에서 즉시 지적됨
- 어휘 풀의 부족이 드러남

### Before — 한 화 안에서
- "심장이 뛰었다" 4회
- "마치 ~처럼" 6회
- "한편" (장면 전환) 3회

### After
각 표현은 **한 화에 1회만**. 같은 의미는 다른 표현으로:
- 신체 반응 → 첫 번째: 박동 / 두 번째: 행동 변화 / 세 번째: 침묵
- 비유 → "마치" 대신 콜론·줄바꿈·직접 비교로

### 검출 키워드 (Grep, 자세한 한도는 word-blacklist.md)
- `마치`, `~듯이`, `~처럼` → 각각 화당 2회 이하
- `심장`, `두근`, `뛰었다` 류 → 합쳐서 화당 3회 이하
- `한편`, `그러나`, `하지만` 문장 시작 → 화당 4회 이하
- `있었다` → 화당 5회 이하 (기존 가드레일)
- `~고 있었다` → 화당 3회 이하 (기존 가드레일)
- `것이었다` → 화당 1회 이하 (기존 가드레일)

자세한 카테고리별 빈도는 `word-blacklist.md` 참조.

### 상한
모든 카테고리 한도 내. 깨달음 클리셰("그래, 이거였다") 화당 **0회**.

## P5. 사람이 안 쓸 것 같은 이상한 문장

### 정의
의미는 통하지만 한국인 화자가 자연스럽게는 쓰지 않는 어색한 어순·번역체·과잉 수식 패턴.

### 왜 나쁜가
- 즉시 "AI 번역체" 신호로 인지됨
- 몰입을 깨는 가장 직접적 요인
- 1인칭에서 발견되면 화자 정체성이 흔들림

### Before — 번역체/어색
```
나의 손은 떨림을 멈추지 않았다.
그것은 마치 그동안 내가 알고 있던 모든 것을 부정하는 듯한 감각이었다.
나로서는 그 사실을 받아들이기에는 시간이 필요했다.
```

### After — 자연체
```
손이 안 떨렸다. 이게 더 이상하지.

…그동안 알았던 게 다 거짓말이라는 거잖아.

받아들이는 데 시간이 좀 걸렸다.
```

### 검출 신호
1. **`~의 ~`의 남발** (영어 of 직역체)
   - 부적합: "나의 손의 떨림" → 적합: "내 손이 떨림"
2. **`것`의 과잉**
   - 부적합: "내가 본 것은 충격적인 것이었다" → 적합: "보고 충격받았다"
3. **수동태 남용**
   - 부적합: "그 사실에 의해 흔들렸다" → 적합: "그 사실 때문에 흔들렸다"
4. **`~는 듯한 감각/느낌/기분`** 류
   - 부적합: "마치 시간이 멈춘 듯한 감각이 들었다"
   - 적합: "시간 멈춤. 진짜로."
5. **번역체 관용구**
   - `그것은 사실이었다` / `나로서는` / `~을 의미했다` / `~에 다름 아니다`

검출되면 입으로 한 번 읽어본다. 어색하면 **확정 P5**.

### 상한
- `나의` (대명사+조사) 화당 **2회 이하** (소유 강조 시만)
- `것이다`, `것이었다` 합쳐서 **한 문단 1회** (이미 기존 가드레일)
- `~듯한 감각/느낌` 화당 **1회 이하**
- 수동태(`~에 의해`) 화당 **2회 이하**

## 종합 자기검토 체크리스트

본문 저장 직후 5개 항목을 그렙 + 휴리스틱으로 점검:

```
[ ] P1: 결론 선언 키워드가 묘사 앞에 0회
[ ] P2: 5문장 윈도우 위반 1개 이하 + 단문 단락 3회 이상
[ ] P3: 정의문 한 문단 2개 이상 0회
[ ] P4: 모든 반복 키워드 한도 이내, 깨달음 클리셰 0회
[ ] P5: 번역체·`것` 남발·수동태 한도 이내
```

5개 모두 통과해야 PASS 가능. 하나라도 실패하면 episode-creator가 해당 부분만 수정 후 재검토.

## 작품별 오버라이드

novel-config.md에 다음 섹션이 있으면 추가 적용한다:

```yaml
voice_patterns:
  P1_extra_keywords:                  # P1에 추가할 검출 키워드
    - "~을 직감했다"
  P4_extra_word_limits:               # P4에 추가할 화당 빈도 한도
    "박자": 1                          # dclass-hero "박자" 어휘 제한
  P5_extra_signals:                   # P5에 추가할 어색 표현
    - "~한 것이 아닐 수 없다"
```

작품별 voice/tone 문서(`voice_docs:`)는 episode-creator가 사전 로드해서 함께 적용한다.
